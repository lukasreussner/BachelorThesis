Abs Limit Start/ Abs Limit Stop

Sets an absolute limit value at the start or stop frequency of the range [dBm].

This parameter is only available if the limit check is set accordingly (see "Limit Check").

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​LIMit:​STARt

[SENSe:​]LIST:​RANGe<range>:​LIMit:​STOP
