INSTrument[:SELect] <ChannelType> | <ChannelName>

This command activates a new measurement channel with the defined channel type, or selects an existing measurement channel with the specified name.

See also INSTrument:​CREate[:​NEW].

For a list of available channel types see INSTrument:​LIST?.

Parameters:
<ChannelType>

Channel type of the new channel.

For a list of available channel types see Table "Available channel types and default channel names in Signal and Spectrum Analyzer mode".

DOCSis

DOCSIS 3.1 option, R&S FSW–K192

<ChannelName>

String containing the name of the channel.

Example: INST DOCS

Activates a measurement channel for the R&S FSW DOCSIS 3.1 application.

INST 'DOCSIS'

Selects the measurement channel named 'DOCSIS' (for example before executing further commands for that channel).
