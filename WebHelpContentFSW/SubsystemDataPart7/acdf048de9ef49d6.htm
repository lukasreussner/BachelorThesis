DISPlay[:WINDow<n>]:TABLe:ITEM:POWer:ALL <State>

This command adds and removes all Power results from the Result Summary.

Suffix: 
<n>
irrelevant
Setting parameters: 
<State>
ON | OFF | 1 | 0
Example: DISP:TABL:ITEM:POW:ALL ON

Displays all power result.

Usage: Setting only
