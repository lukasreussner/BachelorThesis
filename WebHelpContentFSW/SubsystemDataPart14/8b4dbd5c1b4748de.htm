Time Reference (UE mode)

Defines which slot is used as a time reference for synchronization.

"UpPTS"

Uses the Uplink Pilot Time Slot (UpPTS) as a time reference (see also "Frames, Subframes and Slots")

"Slot 1"

Uses slot 1 as a time reference.

Remote command: 

[SENSe:​]CDPower:​TREF
