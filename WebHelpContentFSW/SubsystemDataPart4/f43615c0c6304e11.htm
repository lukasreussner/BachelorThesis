This command starts the measurement and reads out the result of the Amplitude Droop.

When the measurement is started the R&S FSW is automatically set to single sweep mode. Further results of the measurement can then be queried without restart of the measurement via the FETCh:BURSt subsystem.

For details on the Amplitude Droop see Table "Modulation accuracy parameters".

Both the current result and the statistical evaluation of all results over a number of frames (specified by "Statistic Count") are provided.

For details on how the statistical evaluation is performed see Table "Calculated summary results".

Return values: 
<Result>
numeric value

Amplitude droop

Default unit: dB
Example: READ:BURS:ADR:SDEV?
Usage: Query only
