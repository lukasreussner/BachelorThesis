Range

Which part of the detected pulse is evaluated for sidelobe results is configurable, similarly to the result range for common pulse results.

By default ("Result Range" mode), the configured "Result Range" is also used to evaluate sidelobes.

In "Manual" mode you can define the length and alignment of the sidelobe range differently to the result range.

Remote command: 

SENSe:​TRACe:​MEASurement:​DEFine:​TSRange:​RANGe
