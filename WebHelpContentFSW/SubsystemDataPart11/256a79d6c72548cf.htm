CALCulate<n>:DELTamarker<m>:Y?

This command queries the relative position of a delta marker on the y-axis.

If necessary, the command activates the delta marker first.

To get a valid result, you have to perform a complete measurement with synchronization to the end of the measurement before reading out the result. This is only possible for single measurement mode.

See also INITiate<n>:​CONTinuous.

The unit depends on the application of the command.

Suffix: 
<m>

Marker

<n>

Window

Return values: 
<Result>

Result at the position of the delta marker.

The unit is variable and depends on the one you have currently set.

Example: INIT:CONT OFF

Switches to single sweep mode.

INIT;*WAI

Starts a sweep and waits for its end.

CALC:DELT2 ON

Switches on delta marker 2.

CALC:DELT2:Y?

Outputs measurement value of delta marker 2.

Usage: Query only
Manual operation: See "Delta Marker 1 / Marker 2 / Marker 3 / ... Marker 16 / Norm / Delta"
