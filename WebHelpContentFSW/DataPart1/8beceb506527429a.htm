Power Gsm Carrier

Defines the power of the GSM carrier (if available, see "Carrier Adjacent to RF Bandwidth Edge"). Possible values are from 0 dBm to 100 dBm in 1 dB steps.

This setting is only available for "Band Category"2.

This setting is required to calculate the SEM limits according to standard 3GPP 37.141 V12.2.0.

Remote command: 

[SENSe:​]ESPectrum<sb>:​MSR:​GSM:​CARRier
