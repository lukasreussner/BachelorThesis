[SENSe:]ADJust:ALL

This command initiates a measurement to determine and set the ideal settings for the current task automatically (only once for the current measurement).

This includes:

  • Center frequency
  • Reference level
  • Scaling
Example: ADJ:ALL
Usage: Event
Manual operation: See "Adjusting all Determinable Settings Automatically ( Auto All )"
