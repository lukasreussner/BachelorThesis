CONFigure:PSWeep:Z<n>:RESult <Result>

This command selects the result type displayed on the z-axis of the parameter sweep diagram.

Suffix: 
<n>

Window

Parameters:
<Result>

See table below for supported result types.

Example: CONF:PSW:Z:RES EVM

Displays the EVM against two parameters in the Parameter Sweep result display.

Manual operation: See "Selecting the result type displayed in the Parameter Sweep diagram"
