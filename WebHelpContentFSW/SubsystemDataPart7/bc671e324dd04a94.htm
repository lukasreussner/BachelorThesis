Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Performing Amplifier Measurements > Retrieving Numeric Results > Retrieving Results of the Result Summary > Retrieving the Modulation Accuracy > FETCh:MACCuracy:SRERror:MAXimum[:RESult]?
FETCh:MACCuracy:SRERror:MAXimum[:RESult]?
FETCh:MACCuracy:SRERror:MINimum[:RESult]?
FETCh:MACCuracy:SRERror:CURRent[:RESult]?

This command queries the Sample Rate Error as shown in the Result Summary.

Return values: 
<SampleRateError>
<numeric value>

Minimum, maximum or current SampleRateError, depending on the command syntax.

Default unit: Hz
Example: FETC:MACC:SRER:CURR?

would return, e.g.

-0.023
Usage: Query only
Manual operation: See "Results to check modulation accuracy"
