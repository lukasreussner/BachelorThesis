Conversion Loss

Defines how the conversion loss is handled. The following methods are available:

"Average"

Defines the average conversion loss for the entire frequency range in dB.

"Table"

Defines the conversion loss via the table selected from the list. Predefined conversion loss tables are often provided with the external mixer and can be imported to the R&S FSW. Alternatively, you can define your own conversion loss tables. Imported tables are checked for compatibility with the current settings before being assigned.

For details on conversion loss tables, see Conversion Loss Tables.

For details on importing tables, see "Import Table".

Remote command: 

Average for range 1:

[SENSe:​]MIXer<x>:​LOSS[:​LOW]

Table for range 1:

[SENSe:​]MIXer<x>:​LOSS:​TABLe[:​LOW]

Average for range 2:

[SENSe:​]MIXer<x>:​LOSS:​HIGH

Table for range 2:

[SENSe:​]MIXer<x>:​LOSS:​TABLe:​HIGH
