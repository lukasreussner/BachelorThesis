[SENSe:]SWEep:TIME <Time>

This command defines the capture time.

This command is available when [SENSe:]SWEep:TIME:AUTO has been turned off.

Note that when you change the capture time, the capture length is adjusted automatically to the new capture time.

Parameters:
<Time>
<numeric value>
Default unit: s
Example: SWE:TIME:AUTO OFF
SWE:TIME 10MS

Defines a sweep time of 10 ms.

Manual operation: See "Manual definition"
