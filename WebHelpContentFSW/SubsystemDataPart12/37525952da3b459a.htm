Setting the Evaluation Range

In some scenarios, such as in "Evaluating the Rising and Falling Edges", the result range contains symbols that are not supposed to be considered for the EVM or other calculated parameters that are displayed in the Result Summary. Thus, you would not include them in the evaluation range.

To change the evaluated data

  1. Start from the configuration described in "Evaluating the Rising and Falling Edges".

  2. Display the I/Q constellation diagram of the signal in window 1(Source: "Meas&Ref Signal", Result type: "Constellation I/Q", see "Changing the Display Configuration").

    A clear 8PSK constellation is displayed.

    Evaluation lines properly adjusted

  3. In order to understand the effect of an incorrectly set evaluation range, change the evaluation range to include the entire result range.

    1. In the "Overview", select "Evaluation Range".

    2. Enable the "Entire Result Range" option.

    The displayed constellation diagram is no longer clear, it contains additional points. This is due to the fact that the constellation diagram now displays symbol instants that are beyond the burst.

    Evaluation lines not properly adjusted

All measurement windows that consider the evaluation range are marked with two small red lines in the title bar.
