TRACe:IQ:BWIDth?

Queries the currently used analysis bandwidth.

For DOCSIS 3.1 downstream measurements, a fixed bandwidth of 192.0 MHz is used.

For DOCSIS 3.1 upstream measurements, a fixed bandwidth of 96.0 MHz is used.

Example: TRAC:IQ:BWID?
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Usage: Query only
Manual operation: See "Analysis Bandwidth (ABW)"
