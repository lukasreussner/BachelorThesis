FETCh:PNOise<t>:USER<range>:RPM?

This command queries the residual PM for a particular user range.

The trace that is queried depends on CALCulate<n>:​EVALuation:​USER<range>:​TRACe.

Suffix: 
<t>

Trace

<range>
1...3

Selects the user range.

Example: 

FETC:PNO:USER2:RPM?

Queries the residual PM for user range 2.

Usage: Query only
Manual operation: See "Residual Noise"
