CALCulate<n>:MARKer<m>:Z?

This command queries the marker position on the z-axis of three-dimensional result displays.

This command returns the type of value displayed in the selected result display (EVM, Power or Allocation ID).

Suffix: 
<m>

Marker

<n>

Window

Return values: 
<Position>
<numeric value>
Default unit: Depends on result display
Example: 

//Query marker position

CALC:MARK:Z?
Usage: Query only
