Symbols

The detected symbols (i.e. the detected bits) displayed in a table;

The default result type is a hexadecimal symbol table.

Other formats for the symbol table are available, but no other result types (see "Symbol Table").

Remote command: 

LAY:ADD? '1',BEL, SYMB(see LAYout:​ADD[:​WINDow]?)
