CALCulate<n>:LIMit<k>:ESPectrum<sb>:PCLass<PClass>[:EXCLusive] <State>

This command selects the power class used by the measurement if CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​MODE is set to manual.

Note that:

  • You can only use power classes for which limits are defined.
Suffix: 
<n>, <k>

irrelevant

<sb>
1 | 2 | 3

Sub block in a Multi-SEM measurement

<PClass>
1...4

power class

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: CALC:LIM:ESP:PCL1 ON

Activates the first defined power class.
