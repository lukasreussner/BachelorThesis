Restore Original Display

Restores the original display, that is, the originally calculated displays for the entire capture buffer, and closes all zoom windows.

Remote command: 

Single zoom:

DISPlay[:​WINDow<n>]:​ZOOM[:​STATe]

Multiple zoom:

DISPlay[:​WINDow<n>]:​ZOOM:​MULTiple<zoom>[:​STATe] (for each multiple zoom window)
