Auto Recording

If enabled, the SCPI Recorder automatically records the required SCPI commands and parameter values for the settings and functions you use while operating the R&S FSW.

To view the list of currently recorded SCPI commands at any time, select the SCPI Recorder icon in the toolbar.

Recording is stopped when you deactivate "Auto Recording".

To continue recording, reactivate "Auto Recording".

To start a new SCPI command list, select "Clear All" before activating "Auto Recording".

Note:

  • Some parameters cannot be set by a SCPI command.
    In this case, no SCPI command found is entered in the list instead of a command when you record settings automatically.
  • The R&S FSW automatically clears the SCPI command list after booting.
