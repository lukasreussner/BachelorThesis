CONFigure:CHANnel:NFFT <NFFTsubcarriers>

Defines the length of the FFT duration, which corresponds to the number of phsyical subcarriers.

Parameters:
<NFFTsubcarriers>
FFT2K | FFT4K | FFT8K
FFT2K

Upstream only:

2048 subcarriers at = 50 kHz spacing; FFT length = 2048 samples

FFT4K

Downstream: 4096 subcarriers at = 50 kHz spacing; FFT length = 4096 samples

Upstream: 4096 subcarriers at = 25 kHz spacing; FFT length = 4096 samples

FFT8K

Downstream only:

8192 subcarriers at 25 kHz spacing; FFT length = 8192 samples

*RST:FFT4K
Example: CONF:CHAN:NFFT FFT8K
