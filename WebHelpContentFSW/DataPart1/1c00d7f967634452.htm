Used Power Classes:

Defines which power classes are considered for the SEM measurement. Limits can be defined only for used power classes. It is only possible to select either one specific power class or all the defined power classes.

If "All" is selected, the power class that corresponds to the currently measured power in the reference range is used for monitoring. The limits assigned to that power class are applied (see "Abs Limit Start / Stop <n>" and "Rel Limit Start / Stop <n>").

Remote command: 

CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>[:​EXCLusive]

To define all limits in one step:

CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>:​LIMit[:​STATe]
