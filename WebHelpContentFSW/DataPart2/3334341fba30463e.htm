[SENSe:]POWer:ACHannel:FILTer:ALPHa:UALTernate<ch> <Alpha>

This command defines the roll-off factor for the upper alternate channels' weighting filter for asymmetrical MSR signals. To configure the factor for the upper alternate channels, use the [SENSe:​]POWer:​ACHannel:​FILTer:​ALPHa:​UALTernate<ch> command.

Suffix: 
<ch>
1 to 11

Alternate channel number

Parameters:
<Alpha>

Roll-off factor

Range: 0  to  1
*RST:0.22
Manual operation: See "Weighting Filters"
