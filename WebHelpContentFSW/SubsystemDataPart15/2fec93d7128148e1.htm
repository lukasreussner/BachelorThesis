Inactive Channel Threshold

Defines the minimum power that a single channel must have compared to the total signal to be recognized as an active channel.

The default value is -60 dB. With this value, the Code Domain Power Analyzer detects all channels with signals such as the CDMA2000 test models. Decrease the "Inactive Channel Threshold" value if not all channels contained in the signal are detected.

Remote command: 

[SENSe:​]CDPower:​ICTReshold
