Multi Source

For multi source diagrams, the data for one source is provided in one trace, the data for the other source in another trace. By default, the data for the spectrum of the Real/Imag data from the capture buffer or measurement are provided in trace1, whereas the data for the spectrum of the Real/Imag data of the error are provided in trace 2. However, the data sources for the traces can be changed (see CALCulate<n>:​TRACe<t>[:​VALue]).

For details on the results for real/imag traces see "Cartesian Diagrams".
