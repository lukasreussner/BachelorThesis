VSA (R&S FSW-K70) > Configuration > Measurement Filter Settings > Type > Alpha/BT
Alpha/BT

Defines the roll-off factor (Alpha) or the filter bandwidth (BT).

The roll-off factor or filter bandwidth are available for RC, RRC and Gauss filters.

If the measurement mode is automatically selected according to the transmit filter, this setting is identical to the "Alpha/BT" value in the modulation settings (see "Alpha/BT").

Remote command: 

Measurement filter: [SENSe:​]DDEMod:​MFILter:​ALPHa

Transmit filter: [SENSe:​]DDEMod:​TFILter:​ALPHa
