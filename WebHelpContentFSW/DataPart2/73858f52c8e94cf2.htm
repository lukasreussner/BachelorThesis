DIAGnostic:SERVice:INPut[:SELect] <Signal>

This command activates or deactivates the use of an internal calibration signal as input for the R&S FSW.

Parameters:
<Signal>
CALibration

Uses the calibration signal as RF input.

MCALibration

Uses the calibration signal for the microwave range as RF input.

RF

Uses the signal from the RF input.

AIQ

Uses the Analog Baseband calibration signal as input to the optional Analog Baseband interface. This signal is only available if the R&S FSW-B71 option is installed.

*RST:RF
Example: DIAG:SERV:INP CAL

Uses the calibration signal as RF input.
