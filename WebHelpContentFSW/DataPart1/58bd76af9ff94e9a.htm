DISPlay[:WINDow<n>]:TRACe<t>:SMOothing[:STATe] <State>

This command turns trace smoothing for a particular trace on and off.

If enabled, the trace is smoothed by the value specified using DISPlay[:​WINDow<n>]:​TRACe<t>:​SMOothing:​APERture.

For more information see "Trace Smoothing".

Suffix: 
<n>

Window

<t>

Trace

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: 

DISP3:TRAC2:SMO ON

Turns on trace smoothing for trace 2 in window 3

Manual operation: See "Smoothing"
