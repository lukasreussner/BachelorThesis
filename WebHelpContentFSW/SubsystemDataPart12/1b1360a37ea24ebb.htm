TRACe<n>:IQ:BWIDth?

This command queries the bandwidth of the resampling filter ("Usable I/Q Bandwidth").

Suffix: 
<n>

Window

Return values: 
<Bandwidth>

Usable I/Q bandwidth

Default unit: Hz
Example: TRAC1:IQ:BWID?
Usage: Query only
Manual operation: See "Usable I/Q Bandwidth"
