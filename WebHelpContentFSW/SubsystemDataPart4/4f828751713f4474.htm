Carrier Selection/Carrier

Specifies the carrier at which the reference powers for the MCWN measurement are measured (if reference power measurement is enabled, see "Enabling a reference power measurement (Measure)").

In "Auto" mode, the carrier with the maximum power level is selected as a reference.

In "Manual" mode, you must specify the carrier to be used as a reference in the "Carrier" field. All active carriers can be selected (see "Active carriers").

For details see "Reference measurement".

Remote command: 

CONFigure:​SPECtrum:​MODulation:​REFerence:​CARRier[:​AUTO]

CONFigure:​SPECtrum:​MODulation:​REFerence:​CARRier:​NUMBer
