R&S FSW with Activated I/Q Bandwidth Extension Option B5000

The bandwidth extension option R&S FSW-B5000 provides measurement bandwidths up to 5 GHz.

The memory extension option R&S FSW-B106 is not available together with the B5000 option.

MSRA operating mode

If the R&S FSW-B5000 bandwidth extension option is active, MSRA operating mode is not available.

Maximum record length with activated I/Q bandwidth extension option B5000

The maximum record length provided by the R&S FSW depends on the data rate that the oscilloscope can process, which in turn depends on the memory updates installed on it. Assuming the oscilloscope allows for the maximum of 400 MSamples, the maximum record length can be estimated approximately as:

Maximum record length with activated I/Q bandwidth extension option B5000

Sample rate

Maximum record length (approx.)

10 kHz to 10 GHz

((400 MSamples * SampleRate) / 20 GHz) - 100

10 GHz to 20 GHz

((400 MSamples * SampleRate) / 20 GHz) - 1000

Example: 

For the maximum sample rate of 20 GHz and the maximum bandwidth of 5 GHz on the R&S FSW, and the maximum memory depth of 400 MSamples and fixed sample rate of 20 GHz on the oscilloscope, the maximum record length is approximately:

((400 MSamples * 20 GHz) / 20 GHz) - 100 = 399.999900 MSamples

Using an external trigger or power splitter mode

When using an external trigger in common B5000 mode (which uses another channel on the oscilloscope), or in power splitter mode (which uses two input channels on the oscilloscope), the maximum memory size, and thus record length, available for a single input channel may be reduced by half. For details see the oscilloscope's data sheet and documentation.

For details on the power splitter mode see "Power Splitter Mode".
