SENSe:TRACe:MEASurement:DEFine:TSRange:KOTime:AUTO <State>

Defines an excluded area around the center of the time sidelobe range, assuming this is the mainlobe, in which no results are calculated.

This command is only available if the additional option R&S FSW-K6S is installed.

Parameters:
<State>
ON | OFF | 1 | 0
ON | 1

The determined mainlobe 3 dB width is used (see[SENSe:​]PULSe:​TSIDelobe:​MWIDth?)

OFF | 0

You can define the length of the keep-out time using SENSe:​TRACe:​MEASurement:​DEFine:​TSRange:​KOTime:​LENGth.

*RST:1
Manual operation: See "Keep-Out Time"
