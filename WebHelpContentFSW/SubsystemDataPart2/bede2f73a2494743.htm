Ripple

The ripple is calculated as the difference between the maximum and minimum deviation from the pulse top reference, within a user specified interval.

For more information see "Ripple"

Note: The percentage ratio values are calculated in %V if the "Measurement Level" is defined in V (see "Reference Level Unit"), otherwise in %W.
