Importing and Exporting I/Q Data and Results

The I/Q data to be evaluated in the R&S FSW 802.11ad application can not only be measured by the R&S FSW 802.11ad application itself, it can also be imported to the application, provided it has the correct format. Furthermore, the evaluated I/Q data from the R&S FSW 802.11ad application can be exported for further analysis in external applications.

For details on importing and exporting I/Q data see "I/Q Data Import and Export".
