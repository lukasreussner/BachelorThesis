[SENSe:]ADEMod<n>:SPECtrum:BANDwidth[:RESolution] <Bandwidth>
[SENSe:]ADEMod<n>:SPECtrum:BWIDth[:RESolution] <Bandwidth>

Defines the resolution bandwidth for data acquisition.

From the specified RBW and the demodulation span set by [SENSe:​]ADEMod<n>:​SPECtrum:​SPAN[:​MAXimum] or [SENSe:​]BWIDth:​DEMod, the required measurement time is calculated. If the available measurement time is not sufficient for the given bandwidth, the measurement time is set to its maximum and the resolution bandwidth is increased to the resulting bandwidth.

This command is identical to [SENSe:​]BANDwidth[:​RESolution].

Suffix: 
<n>

irrelevant

Parameters:
<Bandwidth>
refer to data sheet
*RST:61.2 kHz
Example: 

ADEM:SPEC:BAND 61.2kHz

Sets the resolution bandwidth to 61.2 kHz.
