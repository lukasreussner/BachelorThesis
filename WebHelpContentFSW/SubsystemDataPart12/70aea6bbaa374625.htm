Signal Description

Access: "Overview" > "Signal Description"

or: [MEAS CONFIG] > "Signal Description"

The signal description provides information on the expected input signal, which optimizes burst and pattern detection and allows for the application to calculate an ideal reference signal. The signal description consists of information on the used modulation and on the signal's structure.
