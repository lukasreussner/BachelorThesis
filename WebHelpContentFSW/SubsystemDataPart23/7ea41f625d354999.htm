Evaluation range for the constellation diagram

The "Evaluation Range" dialog box defines the type of constellation points that are displayed in the "Constellation Diagram".

By default the application displays all constellation points of the data that have been evaluated. However, you can filter the results by several aspects.

  • Modulation
    Filters the results to include only the selected type of modulation.
  • Allocation
    Filters the results to include only a certain type of allocation.
  • Symbol (OFDM)
    Filters the results to include only a certain OFDM symbol.
  • Carrier
    Filters the results to include only a certain subcarrier.

The result display is updated when you make the changes.

Note that the constellation selection is applied to all windows in split screen mode if the windows contain constellation diagrams.

Remote command: 

Modulation: [SENSe:​][LTE:​][CC<cc>:​]MODulation:​SELect

Allocation: [SENSe:​][LTE:​][CC<cc>:​]ALLocation:​SELect
