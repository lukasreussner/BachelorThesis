CONFigure:REFSignal:GOS:NPOSition <Frequency>

This command defines the offset of a notch relative to the center frequency in the internally generated reference signal.

Parameters:
<Frequency>
<numeric value>
Default unit: Hz
Example: CONF:REFS:GOS:NPOS 10000

Defines a notch offset of 10 kHz.

Manual operation: See "Notch Position"
