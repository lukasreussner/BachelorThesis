Rise Base Point Time

The time the amplitude starts rising above 0 %.

The power value of the rise base point and the fall base point is assumed to be equal and is defined by the "Base Power" parameter found in the "Amplitude Parameters" group of the table configuration (see "Base Power").

Remote command: 

[SENSe:​]PULSe:​EMODel:​RBPTime?
