[SENSe:]SWEep:FMODE:STATe <State>

If enabled, the constellation uses a single color for all modulations, which improves performance. However, individual modulation types cannot be selected.

If disabled, different colors are used for different modulation types (see "Constellation"). Which modulation types are displayed (depending on the selected object) is configurable.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:1
Example: SENS:SWE:FMOD:STAT ON
Manual operation: See "Fast Mode (Single Color)"
