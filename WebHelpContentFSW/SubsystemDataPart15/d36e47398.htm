[SENSe:]CDPower:SET <SetNo>

This command selects a specific set for further analysis. The number of sets has to be defined with the [SENSe:​]CDPower:​SET:​COUNt command before using this command.

Parameters:
<SetNo>
Range: 0  to  SET COUNT -1
Increment: 1
*RST:0
Example: CDP:SET:COUN 10

Selects the 11th set for further analysis (counting starts with 0).

Manual operation: See "Set to Analyze"
