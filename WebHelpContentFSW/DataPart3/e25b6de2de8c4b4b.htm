Ping Client

Ping is a utility that verifies the connection between the LXI-compliant instrument and another device. The ping command uses the ICMP echo request and echo reply packets to determine whether the LAN connection is functional. Ping is useful for diagnosing IP network or router failures. The ping utility is not password-protected.

To initiate a ping between the LXI-compliant instrument and a second connected device:

  1. Enable "ICMP Ping" on the "Advanced LAN Configuration" page (enabled after an LCI).

  2. Enter the IP address of the second device without the ping command and without any further parameters into the "Destination Address" field (e.g. 10.113.10.203).

  3. Select "Submit".
