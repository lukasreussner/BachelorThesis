Info. in Hopping Bits

Defines the information available in the hopping bits according to the PDCCH DCI format 0 hopping bit definition.

The information in the hopping bits determines whether type 1 or type 2 hopping is used in the subframe and, in case of type 1, additionally determines the exact hopping function to use.

For more information on PUSCH frequency hopping refer to 3GPP TS36.213.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​PUSCh:​FHOP:​IIHB
