MMEMory:NAME <FileName>

This command has several purposes, depending on the context it is used in.

  • It creates a new and empty file.
  • It defines the file name for screenshots taken with HCOPy[:IMMediate<device>]. Note that you have to route the printer output to a file.
Parameters:
<FileName>

String containing the path and name of the target file.

Example: MMEM:NAME 'C:\R_S\INSTR\USER\PRINT1.BMP'

Selects the file name.
