INPut<ip>:DIQ:RANGe:COUPling <State>

If enabled, the reference level for digital input is adjusted to the full scale level automatically if the full scale level changes.

This command is only available if the optional Digital Baseband Interface is installed.

Suffix: 
<ip>
Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
