OFDM Channel Description (Downstream)

Access: "Overview" > "Signal Description" > "OFDM Channel Description"

or: MEAS CONFIG > "Signal Description" > "OFDM Channel Description": "Downstream"

The general OFDM channel transmission settings are configured in the "Signal Description" dialog box.

The OFDM channel description for upstream channels is described in "OFDM Channel Description (Upstream)".

The OFDM channel information detected in the PLC of the measured signal is displayed in the "PLC Messages (downstream only)" result display.

└ Stream Direction
└ Center Frequency
└ OFDM Spectrum Location
└ NFFT (FFT length)
└ Cyclic Prefix CP
└ Roll-off
└ Time-Interleaving Depth
└ PLC Start Index L
└ PLC Modulation
└ PLC Number of Subcarriers (Np)
└ NCP Modulation
└ Continuous Pilots and Excluded Subcarrier Configuration
