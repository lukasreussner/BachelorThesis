Evaluation Methods for Code Domain Analysis

Access: "Overview" > "Display Config"

The captured I/Q data can be evaluated using various different methods without having to start a new measurement. All evaluation methods available for the selected CDMA2000 measurement are displayed in the evaluation bar in SmartGrid mode.

For details on working with the SmartGrid, see Laying out the Result Display with the SmartGrid.

The selected evaluation not only affects the result display, but also the results of the trace data query (see TRACe<n>[:​DATA]?).

The Code Domain Analyzer provides the following evaluation methods for measurements in the code domain:

└ Bitstream
└ Channel Table
└ Table Configuration
└ Code Domain Power / Code Domain Error Power
└ Composite Constellation
└ Composite EVM
└ Mag Error vs Chip
└ Peak Code Domain Error
└ Phase Error vs Chip
└ Power vs PCG
└ Power vs Symbol
└ Result Summary
└ Symbol Constellation
└ Symbol EVM
└ Symbol Magnitude Error
└ Symbol Phase Error
