Peak List State

Activates/deactivates the marker peak list. If activated, the peak list is displayed and the peaks are indicated in the trace display.

For each listed peak the frequency/time ( "X-value" ) and level ( "Y-Value" ) values are given.

Remote command: 

CALCulate<n>:​MARKer<m>:​FUNCtion:​FPEaks:​STATe
