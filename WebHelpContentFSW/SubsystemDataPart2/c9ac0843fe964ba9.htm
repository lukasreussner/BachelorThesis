Export Range

Defines the range of the I/Q data to store.

"Entire Capture"

The entire capture buffer is exported.

"Result Range"

The result range only (that is, the currently selected pulse; see "Pulse Selection") is exported.

Remote command: 

MMEMory:​STORe<n>:​IQ:​RANGe
