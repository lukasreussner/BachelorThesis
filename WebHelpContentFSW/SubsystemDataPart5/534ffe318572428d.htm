Avionics (VOR/ILS) Measurements application (R&S FSW-15) > Remote Commands to Perform VOR/ILS Measurements > Analyzing VOR/ILS Measurements > Working with Markers > Individual Marker Settings > CALCulate<n>:MARKer<m>:TRACe
CALCulate<n>:MARKer<m>:TRACe <Trace>

This command selects the trace the marker is positioned on.

Note that the corresponding trace must have a trace mode other than "Blank".

If necessary, the command activates the marker first.

Suffix: 
<n>

Window

<m>

Marker

Parameters:
<Trace>
Example: CALC:MARK3:TRAC 2

Assigns marker 3 to trace 2.

Manual operation: See "Assigning the Marker to a Trace"
