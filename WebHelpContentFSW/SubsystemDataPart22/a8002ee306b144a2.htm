CSI RS Weights Phase

For the CSI RS Weights Phase result display, the command returns one value for each subcarrier that has been analyzed.

<Phase>, ...

The unit degrees.

The following parameters are supported.
  • TRACE1
    Returns the phase of the measured weights of the reference signal (RS) carriers over one subframe.
