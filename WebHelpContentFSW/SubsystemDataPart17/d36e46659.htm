CONFigure:CDPower[:BTS]:CTABle:DELete

This command deletes the selected channel table. The channel table to be deleted is selected with the command CONFigure:​CDPower[:​BTS]:​CTABle:​NAME.

Example: CONF:CDP:CTAB:NAME 'NEW_TAB'

Defines the channel table name to be deleted.

CONF:CDP:CTAB:DEL

Deletes the table.

Manual operation: See "Deleting a Table"
