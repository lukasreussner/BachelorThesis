Usable I/Q Bandwidth

Shows the usable I/Q bandwidth which depends on the selected sample rate. For details see "Sample Rate, Symbol Rate and I/Q Bandwidth".

This information is provided for reference only.

Note: In diagrams in the frequency domain (Spectrum transformation, see "Result Type Transformation") the usable I/Q bandwidth is indicated by vertical blue lines.

Remote command: 

TRACe<n>:​IQ:​BWIDth?
