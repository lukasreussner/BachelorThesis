Avionics (VOR/ILS) Measurements application (R&S FSW-15) > Analysis > Markers > Marker Positioning Functions > Select Marker
Select Marker

Selects the subsequent marker (marker 1/2/3/4 or delta marker) to be edited or to be used for a marker function. The currently selected marker number is highlighted.

Remote command: 

Marker selected via suffix <m> in remote commands.
