Hopping BW b_hop

Defines the parameter bhop.

bhop is a UE specific parameter that defines the frequency hopping bandwidth. SRS frequency hopping is active if bhop < BSRS.

For more information refer to 3GPP TS 36.211, chapter 5.5.3.2 "Mapping to Physical Resources" for the Sounding Reference Signal.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SRS:​BHOP
