[SENSe:]REFSig:TIME?

This command queries the length of the reference signal as shown in the "Acquisition" dialog box.

Return values: 
<Duration>
<numeric value>
Default unit: s
Example: REFS:TIME?

would return, e.g.:

0.00125
Usage: Query only
Manual operation: See "Automatic adjustment"
