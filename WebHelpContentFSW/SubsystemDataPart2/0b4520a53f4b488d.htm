Pulse Results

Displays the measured pulse parameters in a table of results. Which parameters are displayed can be configured in the "Result Configuration" (see "Result Configuration"). The currently selected pulse is highlighted blue. The pulses contained in the current capture buffer are highlighted green.

Note: You can apply an additional filter after demodulation to help filter out unwanted signals (see "FM Video Bandwidth").

Limit check

Optionally, the measured results can be checked against defined limits (see "Limit Settings for Table Displays"). The results of the limit check are indicated in the Pulse Results table as follows:

Limit check results in the result tables

Display color

Limit check result

White

No limit check active for this parameter

Green

Limit check passed

Red, asterisk before

Limit check failed; limit exceeds lower limit

Red, asterisk behind

Limit check failed; limit exceeds upper limit

Note: The results of the limit check are for informational purposes only; special events such as stopping the measurement are not available.

Note: Optionally, limit lines can be displayed in the "Parameter Distribution" and "Parameter Trend" diagrams. You can drag these lines to a new position in the window. The new position is maintained, the limit check is repeated, and the results of the limit check in any active table displays are adapted.
