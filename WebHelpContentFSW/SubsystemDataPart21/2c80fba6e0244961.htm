DOCSIS measurements (R&S FSW-K192/-K193) > Remote Commands for DOCSIS 3.1 Measurements > Configuring the DOCSIS 3.1 I/Q Measurement (Modulation Accuracy) > Demodulation > [SENSe:]DEMod:NCP:AUTO
[SENSe:]DEMod:NCP:AUTO <FrameConfig>

This command determines how frames are configured.

Parameters:
<FrameConfig>
SIGNal | USER
SIGNAL

Frames (NCP content) are detected in the signal automatically during demodulation.

USER

Frames must be configured manually (see "Codeword/Frame Configuration").

Example: DEM:NCP:AUTO SIGNAL
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Manual operation: See "Auto Detection: NCP Content (downstream only)"
