*OPT?

Option identification query

Queries the options included in the instrument. For a list of all available options and their description refer to the data sheet.

Return values: 
<Options>

The query returns a list of all installed and activated options, separated by commas, where:

B<number> describes hardware options

K<number> describes software options

For PSA89600 emulation, the option is indicated as "B7J" for the *OPT? query ("B7J, 140" if SYST:PSA:WID is activated).

(See SYSTem:​PSA:​WIDeband.)

Note that K9 (Power Meter) and K14 (Spectrograms) are displayed for compatibility reasons only; in fact they are standard functionality of the R&S FSW base unit.

Usage: Query only
