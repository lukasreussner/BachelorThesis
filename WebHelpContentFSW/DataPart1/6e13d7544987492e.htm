[SENSe:]SWEep:POINts <SweepPoints>

This command defines the number of sweep points to analyze after a sweep.

For EMI measurements, 200001 sweep points are available.

Parameters:
<SweepPoints>
Range: 101  to  100001
*RST:1001
Example: SWE:POIN 251
Example: 

See "Programming Example: Performing a Basic Frequency Sweep".

Manual operation: See "Sweep Points"
