Configuring RF Measurements

RF measurements are performed in the Spectrum application, with some predefined settings as described in "RF Measurement Types and Results".

For details on configuring these RF measurements in a remote environment, see Configuring and Performing Measurements.

The 1xEV-DO RF measurements must be activated in 1xEV-DO applications, see "Activating the Measurement Channel".

The individual measurements are activated using the CONFigure:​CDPower[:​BTS]:​MEASurement command (see "Selecting a Measurement").

In addition to the common RF measurement configuration commands described for the base unit, some special commands are available in 1xEV-DO applications.
