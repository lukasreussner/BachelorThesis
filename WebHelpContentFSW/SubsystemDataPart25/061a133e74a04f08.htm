CONFigure[:LTE]:EUTRa:FREQuency <Frequency>

This command defines the center frequency of an E-UTRA channel.

Prerequisites for this command
  • Select in band deployment of an NB-IoT carrier (CONFigure[:​LTE]:​DEPLoyment).
Parameters:
<Frequency>
<numeric value>
Default unit: Hz
Example: 

//Define E-UTRA channel center frequency

CONF:DEPL INB
CONF:EUTR:FREQ 1GHZ
Manual operation: See "Defining physical settings for NB-IoT inband and guardband deployment"
