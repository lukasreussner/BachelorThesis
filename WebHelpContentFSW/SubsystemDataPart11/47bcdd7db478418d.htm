[SENSe:]STATistic<n>:TYPE <Statistic Type>

Defines which hops/chirps are included in the statistical evaluation.

Suffix: 
<n>
1..n

Window

Parameters:
<Statistic Type>
SELected | ALL
SELected

Only the selected hop/chirp from each sweep (capture) is included in the statistical evaluation.

ALL

All measured hops/chirps from each sweep (capture) are included in the statistical evaluation.

Manual operation: See "Selected Hop/Selected Chirp vs All Hops/All Chirps"
