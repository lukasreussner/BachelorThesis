Number of Carriers

Defines the number of carriers in the signal. The "Span" and "Carrier Spacing" are adapted according to the following formula:

Number of carriers -1 = Span / Carrier spacing

For details see "Dependency of the span on the number of carriers".

Remote command: 

[SENSe:​]CARRier:​COUNt
