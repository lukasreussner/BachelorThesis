Analog Demodulation (R&S FSW-K7) > Configuration > Demodulation > Scaling
Scaling

Access: "Overview" > "Demod Settings" > "Scaling"

Or: "Meas Setup" > "Demod" > "Scaling" tab

The scaling parameters define the range of the demodulated data to be displayed.

Window-specific configuration

The settings in this dialog box are specific to the selected window. To configure the settings for a different Analog Demodulation window, select the window outside the displayed dialog box, or select the window from the "Specifics for" selection list in the dialog box.
