Composite Data Constellation (MS application only)

This result display is only available in the MS application for subtypes 2 or 3.

The Composite Data Constellation shows the channel constellation of the modulated composite data signal at symbol level. The results are displayed for the special composite data channel, regardless of which channel is selected.

Remote command: 

LAY:ADD? '1',RIGH, CDC, see LAYout:​ADD[:​WINDow]?
