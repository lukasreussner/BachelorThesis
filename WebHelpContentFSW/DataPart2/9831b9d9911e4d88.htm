[SENSe:]CORRection:FRESponse:USER:FLISt<fli>:SELect <FilePath>

Loads an additional frequency response (.fres) file to the current configuration.

Suffix: 
<fli>
1..n

Index in frequency response file list

Use [SENSe:​]CORRection:​FRESponse:​BASeband:​USER:​FLISt<fli>:​SIZE? to determine the maximum index number.

Parameters:
<FilePath>
string

Path and file name

The default directory for .fres files is C:\R_S\INSTR\USER\Fresponse.

Example: SENS:CORR:FRES:USER:FLIS2:SEL 'C:\FRes.fres'
Manual operation: See "Add Freq Resp File"
