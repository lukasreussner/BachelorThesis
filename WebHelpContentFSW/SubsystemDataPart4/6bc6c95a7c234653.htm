CALCulate<n>:MARKer<m>:AOFF

This command turns off all markers.

Suffix: 
<n>

Window

<m>

Marker

Example: CALC:MARK:AOFF

Switches off all markers.

Usage: Event
Manual operation: See "All Marker Off"
