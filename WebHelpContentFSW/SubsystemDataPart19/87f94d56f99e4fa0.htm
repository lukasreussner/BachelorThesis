CALCulate<n>:LIMit<li>:UPPer[:DATA]?

This command queries the y-axis values for the specified limit line.

Suffix: 
<n>
1..n
<li>
1..n

Limit line for unused tone error RU group:

// 1 = -35dB LHS

// 2 = RUIdx-3

// 3 = RUIdx-2

// 4 = RUIdx-1

// 5 = RUIdx

// 6 = RUIdx+1

// 7 = RUIdx+2

// 8 = RUIdx+3

// 9 = -35dB RHS

Usage: Query only
Manual operation: See "Unused Tone Error Summary"
