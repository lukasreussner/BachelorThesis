CONFigure:US:NORMalize:CMTSref:SCLock <RefSClock>

Defines the reference value used to normalize the CM sample clock error, if normalization is enabled (see CONFigure:​US:​NORMalize:​CM:​CERRor:​STATe).

Specify the unit (and thus type) of the reference value using CONFigure:​US:​NORMalize:​CMTSref:​SCLock:​UNIT.

Parameters:
<RefSClock>
Default unit: HZ
Example: CONF:US:NORM:CM:CERR:STAT ON
CONF:US:NORM:CMTS:SCL:UNIT PPM
CONF:US:NORM:CMTS:SCL 1.2
Manual operation: See "CMTS Reference Sample Clock (Error) (upstream only)"
