Marker Usage

Markers help you analyze your measurement results by determining particular values in the diagram. Thus you can extract numeric values from a graphical display both in the time and frequency domain. In addition to basic markers, sophisticated marker functions are provided for special results such as noise or demodulation.

Markers in Spectrogram Displays

In the spectrogram result display, you can activate up to 16 markers or delta markers at the same time. Each marker can be assigned to a different frame. Therefore, in addition to the frequency you also define the frame number when activating a new marker. If no frame number is specified, the marker is positioned on the currently selected frame. All markers are visible that are positioned on a visible frame.
