Error Vs Carrier

Three trace types are provided for gain imbalance/quadrature error evaluation:

TRACE1

The minimum gain imbalance/quadrature error value - over the analyzed PPDUs - for each of the Nused subcarriers

TRACE2

The average gain imbalance/quadrature error value - over the analyzed PPDUs - for each of the Nused subcarriers

TRACE3

The maximum gain imbalance/quadrature error value - over the analyzed PPDUs - for each of the Nused subcarriers

Each gain imbalance/quadrature error value is returned as a floating point number, expressed in units of dB.

Supported data formats (see FORMat[:​DATA]): ASCii|UINT
