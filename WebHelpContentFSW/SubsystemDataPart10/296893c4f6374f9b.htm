Spurious Measurements application (R&S FSW-K50) > Configuration
Configuration

Access: MODE > "Spurious"

Spurious measurements require a special application on the R&S FSW.

For details see Applications, Measurement Channels, and Operating Modes.

When you switch a measurement channel to the R&S FSW Spurious measurements application the first time, a set of parameters is passed on from the currently active application. After initial setup, the parameters for the measurement channel are stored upon exiting and restored upon re-entering the channel. Thus, you can switch between applications quickly and easily.

When you activate a measurement channel in the R&S FSW Spurious measurements application, the "Spurious" menu is displayed and provides access to the most important configuration functions.
