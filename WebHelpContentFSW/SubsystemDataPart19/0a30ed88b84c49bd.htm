Nsts to use

Defines the PPDUs taking part in the analysis depending on their Nsts.

Note: The terms in brackets in the following description indicate how the setting is referred to in the "Signal Field" result display ("NSTS" column, see "Signal Field").

"Auto, same type as first PPDU:""(A1st)"

All PPDUs using the Nsts identical to the first recognized PPDU are analyzed.

"Auto, individually for each PPDU""(AI)"

All PPDUs are analyzed

"Meas only the specified Nsts""(M ...)"

Only PPDUs with the Nsts specified for the "Nsts" setting are analyzed

"Demod all with specified Nsts""(D ...)"

The "Nsts" setting is used for all PPDUs.

Remote command: 

[SENSe:​]DEMod:​FORMat:​NSTSindex:​MODE
