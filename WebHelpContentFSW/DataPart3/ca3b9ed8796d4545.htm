Bias Current

Single-diode mixers generally require a DC voltage which is applied via the LO line. This DC voltage is to be tuned to the minimum conversion loss versus frequency. Such a DC voltage can be set via the "BIAS" function using the D/A converter of the R&S FSW. The value to be entered is not the voltage but the short-circuit current. The current is defined in the "Bias Settings" or set to the value of the conversion loss table (see "Bias Value" and "Bias".

Bias circuit of the R&S FSW

The voltage U0 at the output of the operational amplifier can be set in the range –2.0 to +2.0 V. An open-circuit voltage Ubias of –0.5 to +0.5 V is obtained accordingly at the output of the voltage divider. A short-circuit current of Ishort = U0 / 200 Ω = 10 mA to + 10 mA is obtained for a short circuit at the output of the voltage divider. In order to use biasing it is not important to know the exact current flowing through the diode since the conversion loss must be set to a minimum with the frequency. Therefore, it makes no difference whether the setting is performed by an open-circuit voltage or by a short-circuit current. A DC return path is ensured via the 66 Ω resistor, which is an advantage in some mixers.
