Trace Mode

Defines the update mode for subsequent traces.

For details, see Analyzing Several Traces - Trace Mode.

The available trace modes depend on the selected result display. Not all evaluations support all trace modes.

For the Magnitude Overview Absolute result display, only the trace modes "Clear/ Write" and "View" are available. For the Magnitude Absolute result display, the trace modes "Average", "MinHold", "MaxHold" are applied to the individual result ranges and thus possibly do not provide useful results.

"Clear/ Write"

Overwrite mode (default): the trace is overwritten by each measurement.

"Max Hold"

The maximum value is determined over several measurements and displayed. The R&S FSW saves each trace point in the trace memory only if the new value is greater than the previous one.

"Min Hold"

The minimum value is determined from several measurements and displayed. The R&S FSW saves each trace point in the trace memory only if the new value is lower than the previous one.

"Average"

The average is formed over several measurements.

The "Statistic Count" determines the number of averaging procedures.

"View"

The current contents of the trace memory are frozen and displayed.

"Blank"

Removes the selected trace from the display.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​MODE
