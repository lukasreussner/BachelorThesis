[SENSe:]ADEMod<n>:MTIMe <Time>

This command defines the measurement time for analog demodulation.

Suffix: 
<n>

irrelevant

Parameters:
<Time>
*RST:62.5us
Example: 

ADEM:MTIM 62.5us

Sets the measurement time to 62.5 μs.
