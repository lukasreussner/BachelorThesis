NB-IoT Downlink (R&S FSW-K106) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Parameter Estimation > Estimating Parameters > [SENSe:][LTE:]DL:DEMod:BESTimation
[SENSe:][LTE:]DL:DEMod:BESTimation <State>

This command turns boosting estimation for downlink signals on and off.

Note that boosting estimation is always active.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:ON
Example: 

//Turn on boosting estimation

DL:DEM:BEST ON
Manual operation: See "Boosting Estimation"
