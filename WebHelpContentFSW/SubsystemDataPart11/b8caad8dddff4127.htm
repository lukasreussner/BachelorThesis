CALCulate<n>:AR:TIME:PERCent <TimePercent>

For CALCulate<n>:​AR:​TIME:​PERCent:​STATeTRUE,the length of the time gate, that is, the duration (or height) of the analysis region, is defined as a percentage of the full measurement time. The time gate start is the start of the capture buffer plus an offset defined by CALCulate<n>:​AR:​TIME:​STARt.

Suffix: 
<n>
1..n

irrelevant

Parameters:
<TimePercent>

percentage of the full measurement time

Manual operation: See "Linked analysis time span"
