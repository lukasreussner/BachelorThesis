Demodulation Bandwidth

The R&S FSW Avionics (VOR/ILS) measurements application captures I/Q data using digital filters with quasi-rectangular amplitude responses. The demodulation bandwidth defines the width of the filter's flat passband.

For more information, see "Demodulation Bandwidth".

Depending on the selected DBW mode, the value is either determined automatically or can be defined manually.

"Auto mode"

(Default) The DBW is determined automatically by the R&S FSW Avionics (VOR/ILS) measurements application.

For ILS measurements: 12.5 kHz

For VOR measurements: 25 kHz

"Manual mode"

The user-defined DBW is used. For a list of available demodulation bandwidths, see Table "Available demodulation bandwidths and measurement times for ILS measurements" and Table "Available demodulation bandwidths and measurement times for VOR measurements".

Remote command: 

[SENSe:​]ADEMod:​BWIDth:​DEModulation

[SENSe:​]ADEMod:​BWIDth:​DEModulation:​AUTO
