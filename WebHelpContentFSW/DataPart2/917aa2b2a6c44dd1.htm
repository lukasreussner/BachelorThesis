RBW

Defines the resolution bandwidth. The available resolution bandwidths are specified in the data sheet. Numeric input is always rounded to the nearest possible bandwidth.

If "Auto" is selected, the resolution bandwidth is coupled to the selected span (for span > 0). If the span is changed, the resolution bandwidth is automatically adjusted.

If the resolution bandwidth is defined manually, a green bullet is displayed next to the "RBW" display in the channel bar.

For more information see "Separating Signals by Selecting an Appropriate Resolution Bandwidth".

For measurements on I/Q data in the frequency domain, the maximum RBW is 1 MHz.

For EMI measurements using the quasipeak detector, the 1 MHz RBW filter is not available (see "Resolution Bandwidth and Filter Types").

Remote command: 

[SENSe:​]BANDwidth[:​RESolution]

[SENSe:​]BANDwidth[:​RESolution]:​AUTO
