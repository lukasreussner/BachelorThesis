*IDN Format

Defines the response format to the remote command *IDN? (see *IDN?). This function is intended for re-use of existing control programs together with the R&S FSW.

"Leg"

Legacy format, as in the R&S FSP/FSU/FSQ family.

"New"

R&S FSW format.

Remote command: 

SYSTem:​FORMat:​IDENt
