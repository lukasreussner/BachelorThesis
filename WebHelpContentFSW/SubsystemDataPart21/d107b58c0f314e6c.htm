Automatic Grid Scaling

Activates or deactivates automatic scaling of the y-axis for the specified trace display. If enabled, the R&S FSW DOCSIS 3.1 application automatically scales the y-axis to best fit the measurement results.

If disabled, the y-axis is scaled according to the specified "Minimum / Maximum" and "Number of Divisions".

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO
