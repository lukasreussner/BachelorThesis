CONFigure:WCDPower[:BTS]:CTABle:DELete

This command deletes the selected channel table. The channel table to be deleted is selected with the command CONFigure:​WCDPower[:​BTS]:​CTABle:​NAME.

Example: CONF:WCDP:CTAB:NAME 'NEW_TAB'

Defines the channel table name to be deleted.

CONF:WCDP:CTAB:DEL

Deletes the table.

Mode: BTS application only
Manual operation: See "Deleting a Table"
