n(1)_DMRS

Defines the part of the demodulation reference signal index that is broadcast. It is valid for the whole cell.

The index applies when multiple shifts within a cell are used. It is used for the calculation of the DMRS sequence.

The n_DMRS parameter can be found in 3GPP TS36.211 V8.5.0, 5.5.2.1.1 Reference signal sequence.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​DRS:​NDMRs
