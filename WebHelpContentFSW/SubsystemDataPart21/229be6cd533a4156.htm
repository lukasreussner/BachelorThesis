[SENSe:]FRAMe:COUNt <Value>

If the statistic count is enabled (see [SENSe:​]FRAMe:​COUNt:​STATe), the specified number of frames is taken into consideration for the statistical evaluation (maximally the number of frames detected in the current capture buffer).

If disabled, all detected frames in the current capture buffer are considered.

Parameters:
<Value>
*RST:1
Example: SENS:FRAM:COUN:STAT ON
SENS:FRAM:COUN 10
Manual operation: See "Frame Statistic Count / Number of Frames to Analyze"
