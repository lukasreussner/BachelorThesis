Message: Pattern Symbols Incorrect

This message can only appear if the pattern waveform has already been found. It indicates that the measured signal contains a pattern that is similar (but not identical) to the pattern specified in the signal description dialog. Technically speaking, the correlation of the signal with the pattern waveform gives a value above the specified I/Q correlation threshold specified in the pattern search dialog. However, not all symbols match the pattern specified in the signal description dialog.

Solution:

Make sure that the correct pattern is specified in the "Signal Description" dialog.
