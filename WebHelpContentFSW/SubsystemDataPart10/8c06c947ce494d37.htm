Trace Settings

Access: TRACE

The trace settings determine how the measured data is analyzed and displayed in the window. In the result displays for the R&S FSW Spurious measurements application, only one (clear/write) trace is available and it cannot be configured except for the number of trace points. However, the result trace can be exported to a file (see "Trace / Table Export Configuration").
