CALCulate<n>:CHRDetection:TABLe:FREQuency:ALL[:STATe] <Visability>[, <Scaling>]

If enabled, all frequency parameters are included in the result tables (see "Frequency parameters").

Note that only the enabled columns are returned for the CALCulate<n>:​CHRDetection:​TABLe:​RESults? query.
