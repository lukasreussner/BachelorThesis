CALCulate<n>:MARKer<m>:FUNCtion:FPEaks[:IMMediate] <Peaks>

This command initiates a peak search.

Suffix: 
<n>

Window

<m>

Marker

Parameters:
<Peaks>

This parameter defines the number of peaks to find during the search.

Note that the actual number of peaks found during the search also depends on the peak excursion you have set with CALCulate<n>:​MARKer<m>:​PEXCursion.

Range: 1  to  200
Example: CALC:MARK:PEXC 5

Defines a peak excursion of 5 dB, i.e. peaks must be at least 5 dB apart to be detected as a peak.

CALC:MARK:FUNC:FPE 10

Initiates a search for 10 peaks on the current trace.
