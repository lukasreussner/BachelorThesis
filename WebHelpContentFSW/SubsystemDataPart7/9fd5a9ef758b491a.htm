Turning the parameter sweep on and off

Before you can use the Parameter Sweep functionality, you have to turn it on separately.

When you turn it on, the application starts the Parameter Sweep in single sweep mode (RUN SGL and RUN CONT both start the Parameter Sweep in that case). When the Parameter Sweep is on, other measurements are not possible, and vice versa.

Turning on the Parameter Sweep also expands the channel bar by several labels that carry information about the progress of the Parameter Sweep.

Remote command: 

CONFigure:​PSWeep[:​STATe]
