FETCh:UTESummary:ALL?

Returns the results of the Unused Tone Error Summary. The result is a comma-separated list of values for up to 37 measurement points in the channel. Which subcarriers are measured depends on the size and position of the RU being transmitted.

This result is required by the IEEE 802.11ax standard for HE trigger-based PPDUs with a maximum channel bandwidth of 80 MHz.

Return values: 
<RU group>
-35 dB LHS | RUIdx-3 | RUIdx-2 | RUIdx-1 | RUIdx | RUIdx+1 | RUIdx+2 | RUIdx+3 | -35 dB RHS

Set of n subcarriers, where n is the number of subcarriers in the resource unit to be checked.

For details see "Unused Tone Error"

Default unit: -
<RU size>
1 | 2 | 4 | 9 | 18 | 37 | 72

Actual size of the RU, indicated as a factor of 26

Default unit: -
<Min>
numeric value

Minimum EVM

Default unit: dB
<Mean>

Mean EVM

Default unit: dB
<Max>

Maximum EVM

Default unit: dB
<Limit>

Limit for error vector magnitude as specified by the IEEE 802.11ax standard

Default unit: dB
<Delta>

Limit - Max = distance of maximum value to limit for an individual RU group

Default unit: dB
<Max RU26 Idx>
integer

Index of the subcarrier with the maximum value, based on an RU size of 26

Default unit: -
<Limit check>
PASS | FAIL | NONE

Result of the limit check for the individual subcarrier

NONE

Limit line not evaluated

Example: 
FETC:UTES:ALL?
//Result:
//-35 dB LHS,-1,nan,nan,nan,nan,nan,-1,NONE,
//RUIdx-3,-1,nan,nan,nan,nan,nan,-1,NONE,
//RUIdx-2,-1,nan,nan,nan,nan,nan,-1,NONE,
//RUIdx-1,-1,nan,nan,nan,nan,nan,-1,NONE,
//RUIdx,4,-59.64,-58.80,-58.11,-30.00,-28.11,2,PASS,
//RUIdx+1,4,-68.39,-67.93,-67.28,-32.00,-35.28,7,PASS,
//RUIdx+2,4,-66.06,-65.14,-63.41,-35.00,-28.41,12,PASS,
//RUIdx+3,4,-68.06,-67.31,-65.97,-35.00,-30.97,13,PASS,
 //-35 dB RHS,4,-73.80,-71.81,-69.82,-35.00,-34.82,17,PASS
Usage: Query only
Manual operation: See "Unused Tone Error Summary"
