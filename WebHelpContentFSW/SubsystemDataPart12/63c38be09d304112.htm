CALCulate<n>:TRACe<t>:ADJust[:VALue] <Reference>

This command defines the reference point for the display.

Suffix: 
<n>

Window

<t>

Trace

Setting parameters: 
<Reference>
TRIGger | BURSt | PATTern
TRIGger

The reference point is defined by the start of the capture buffer.

BURSt

The reference point is defined by the start/center/end of the burst.

PATTern

The instrument selects the reference point and the alignment.

*RST:TRIGger
