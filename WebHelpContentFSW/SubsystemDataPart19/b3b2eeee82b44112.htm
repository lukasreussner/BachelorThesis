CONFigure:BURSt:PVT[:IMMediate]

This remote control command configures the measurement type to be Power vs Time.

Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the Magnitude Capture buffer in window 1 at the top of the screen and the selected result type in window 2 below that. Any other active windows are closed.

Use the LAYout commands to change the display (see "Working with Windows in the Display").
