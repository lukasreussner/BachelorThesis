CALCulate<n>:LIMit<k>:CONTrol:OFFSet <Offset>

This command defines an offset for a complete limit line.

Compared to shifting the limit line, an offset does not actually change the limit line definition points.

Suffix: 
<n>

irrelevant

<k>

Limit line

Parameters:
<Offset>

Numeric value.

The unit depends on the scale of the x-axis.

*RST:0
Manual operation: See "X-Offset"
