Gain

Displays the gain in magnitude from the reference trace to the measurement trace in dB or percent. This diagram characterizes the transmission behavior of the DUT.

Note: This result display requires a calibration measurement (see "Calibrate (Execute Reference Calibration)").

Remote command: 

LAY:ADD? '1',RIGH,GAIN

(see LAYout:​ADD[:​WINDow]?)
