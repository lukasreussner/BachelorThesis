Defining Display Lines

  1. Display lines are configured in the "Lines Config" dialog box. To display this dialog box, press the LINES key and then "Lines Config".

  2. Select the "Display Lines" tab.

  3. To define a vertical line, select "Vertical Line 1" or 2 and enter the x-value at which the line is to be displayed.

    To define a horizontal line, select "Horizontal Line 1" or 2 and enter the y-value at which the line is to be displayed.
