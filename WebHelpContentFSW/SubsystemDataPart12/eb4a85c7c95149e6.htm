VSA Window Configuration

For each window you can select a different evaluation method (result type), based on the data source selected in the "Display Configuration". Further window settings are available for some result types.

Manual configuration of VSA windows is described in "Result Window Configuration".

Useful commands for configuring the window described elsewhere:
  • LAYout:​ADD[:​WINDow]?
