Channel Table

Two different commands are available to retrieve the channel table results:

  • TRAC:DATA? TRACEx commands return detailed trace information for each channel
  • TRAC:DATA? CTABle provides the maximum values of the timing/phase offset between each assigned channel and the pilot channel
