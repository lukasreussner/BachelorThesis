Network and Remote Control Settings

Access: SETUP > "Network + Remote"

Network settings in secure user mode

Be sure to store all network settings before"SecureUser Mode" is enabled; see "Protecting Data Using the Secure User Mode".

If the currently stored network settings are not suitable, you must correct them each time you switch on the R&S FSW in secure user mode, as the settings are not stored permanently in this case.

The remote commands required to define these settings are described in "Configuring the Network and Remote Control".

Step-by-step instructions are provided in "How to Set Up a Network and Remote Control".
