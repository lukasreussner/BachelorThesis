PHICH N_g

Defines the variable Ng.

Ng in combination with the number of resource blocks defines the number of PHICH groups in a downlink subframe. The standard specifies several values for Ng that you can select from the dropdown menu.

If you need a customized configuration, you can set the number of PHICH groups in a subframe by selecting the "Custom" menu item and define the number of PHICH groups directly with "PHICH Number of Groups".

Remote command: 

CONFigure[:​LTE]:​DL[:​CC<cc>]:​PHICh:​NGParameter
