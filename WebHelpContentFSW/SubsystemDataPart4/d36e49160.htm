GSM application (R&S FSW-K10) > Remote Commands to Perform GSM Measurements > Configuring and Performing GSM I/Q Measurements > Frontend Configuration > Frequency > CONFigure[:MS]:ARFCn
CONFigure[:MS]:ARFCn <Value>

This command specifies the Absolute Radio Frequency Channel Number (ARFCN) to be measured. Setting the ARFCN updates the frequency.

Parameters for setting and query: 
<Value>
numeric value
Range: 0 to 1023 (some values may not be allowed depending on the selected frequency band)
Default unit: NONE
Example: CONF:ARFC 5
Manual operation: See "ARFCN"
