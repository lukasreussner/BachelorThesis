CONFigure:CDPower:CTABle:ORDer <CODE | MIDamble>

This command selects sorting of the channel table in code order or midamble order.

Parameters:
<CODE | MIDamble>
CODE

Channels are sorted in code order.

MIDamble

Channels are sorted in midamble order.

*RST:CODE
Example: CONF:CDP:CTAB:ORD

Sorts the channels in code order.

Manual operation: See "Channel Table Sort Order"
