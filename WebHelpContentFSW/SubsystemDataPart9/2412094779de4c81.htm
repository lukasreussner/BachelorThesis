CALCulate<n>:SNOise:DECades[:STATe] <State>

This command turns the spot noise calculation on every 10x offset frequency on and off.

Suffix: 
<n>

Window

Parameters:
<State>
ON | OFF | 1 | 0
*RST:1
Example: CALC:SNO:DEC ON

Turns the spot noise calculation for each decade start on.

Manual operation: See "On All Decade Edges"
