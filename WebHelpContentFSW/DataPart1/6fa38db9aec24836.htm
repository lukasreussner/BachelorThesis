Reference: Transducer Factor File Format

Transducer factor data can be exported to a file in ASCII (CSV) format for further evaluation in other applications. Transducer factors stored in the specified ASCII (CSV) format can also be imported to the R&S FSW for other measurements (see "Import"). This reference describes in detail the format of the export/import files for transducer factors. Note that the bold data is mandatory, all other data is optional.

Different language versions of evaluation programs may require a different handling of the decimal point. Thus, you can define the decimal separator to be used (see "Decimal Separator").
