Retrieving Results

The following commands are required to retrieve the calculated VSA parameters.

All results that are not based on the capture buffer data are calculated for a single result range only (see "Result Range"). To retrieve the results for several result ranges, use the [SENSe:​]DDEMod:​SEARch:​MBURst:​CALC command to move from one result range to the next.
