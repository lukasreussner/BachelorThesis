INPut<ip>:CONNector <ConnType>

Determines whether the RF input data is taken from the RF input connector or the optional Analog Baseband I connector. This command is only available if the Analog Baseband interface (R&S FSW-B71) is installed and active for input. It is not available for the R&S FSW67 or R&S FSW85.

For Multicarrier Wideband Noise (MCWN) measurements, only input from the RF input connector is allowed.

For more information see Processing Data from the Analog Baseband Interface.

Suffix: 
<ip>

irrelevant

Parameters:
<ConnType>
RF | AIQI
RF

RF input connector

AIQI

Analog Baseband I connector

*RST:RF
Example: INP:CONN:AIQI

Selects input from the analog baseband I connector.
