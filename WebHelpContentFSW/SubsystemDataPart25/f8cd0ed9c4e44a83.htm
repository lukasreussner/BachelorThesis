Remote Control

The following remote control commands are required to configure and perform LTE NB-IoT measurements in a remote environment. The R&S FSW must already be set up for remote operation in a network as described in the base unit manual.

Universal functionality
Note that basic tasks that are also performed in the base unit in the same way are not described here. In particular, this includes:
  • Data management:
    –General Data Storage and Loading Commands
    –Storing and Loading Instrument Settings
    –Storing or Printing Screenshots
  • Basic instrument configuration: Configuring the R&S FSW
  • Common status registers: Using the Status Register
