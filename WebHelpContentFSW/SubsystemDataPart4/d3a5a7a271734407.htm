CONFigure[:MS]:BSEarch <State>

This command toggles between active burst search and inactive burst search.

Note

This command is retained for compatibility with R&S FS-K5 only. Use CONFigure:MS:SYNC:MODE BURSt or CONFigure:MS:SYNC:MODE ALL instead (see CONFigure[:​MS]:​SYNC:​MODE).

Parameters for setting and query: 
<State>
1 | 0 | ON | OFF
ON | 1

Burst search on

OFF | 0

Burst search off

*RST:1
