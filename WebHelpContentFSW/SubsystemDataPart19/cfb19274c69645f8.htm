Power Interval Search

If enabled, the R&S FSW WLAN application initially performs a coarse burst search on the input signal in which increases in the power vs time trace are detected. Further time-consuming processing is then only performed where bursts are assumed. This improves the measurement speed for signals with low duty cycle rates.

However, for signals in which the PPDU power levels differ significantly, this option should be disabled as otherwise some PPDUs may not be detected.

Remote command: 

[SENSe:​]DEMod:​TXARea
