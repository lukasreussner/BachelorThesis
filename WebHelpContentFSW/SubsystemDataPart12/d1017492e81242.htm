[SENSe:]DDEMod:SBANd <SidebandPos>

This command selects the sideband for the demodulation.

Note that this command is maintained for compatibility reasons only. Use the SENS:SWAP:IQ command for new remote control programs (see [SENSe:​]SWAPiq).

Setting parameters: 
<SidebandPos>
NORMal | INVerse
NORMal

Normal (non-inverted) position

INVerse

Inverted position

*RST:NORMal
