Modulation

The constellation diagram is restricted to the specified modulation type(s) of the selected "Object".

A specific modulation cannot be selected if "Object" is set to "All" and "Fast Mode (Single Color)" is "ON" (default).

Optionally, the points for each modulation can be displayed by a different color, see "Fast Mode (Single Color)" and "Constellation".

Remote command: 

[SENSe:​]MODulation:​SELect
