Zoomed Displays

You can zoom into the diagram to visualize the measurement results in greater detail. Using the touchscreen or a mouse pointer you can easily define the area to be enlarged.

Graphical zoom and the number of sweep points

Note that (graphical) zooming is merely a visual tool, it does not change any measurement settings, such as the number of sweep points!

You should increase the number of sweep points before zooming, as otherwise the resolution of the trace in the zoomed region is poor (see "How Much Data is Measured: Sweep Points and Sweep Count").
