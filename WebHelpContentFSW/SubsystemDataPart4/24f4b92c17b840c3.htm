Limit Check Results

The following commands are required to query the results of a limit check.

Currently, limit check results can only be queried for the following result displays:
  • "PvT Full Burst"
  • "Modulation Spectrum Graph"
  • "Transient Spectrum Graph"
  • "Spectrum Graph"
Useful commands for retrieving limit check results described elsewhere:
  • READ:​BURSt:​SPOWer:​SLOT<Slot>:​LIMit:​FAIL?
  • FETCh:​SPECtrum:​MODulation:​LIMit:​FAIL?
Remote commands exclusive to retrieving limit check results:
List of Commands
CALCulate<n>:LIMit<k>:CONTrol:DATA?
CALCulate<n>:LIMit<k>:FAIL?
CALCulate<n>:LIMit<k>:LOWer[:DATA]?
CALCulate<n>:LIMit<k>:UPPer[:DATA]?
