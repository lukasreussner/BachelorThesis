OUTPut:DIQ[:STATe] <State>

This command turns continuous output of I/Q data to the optional Digital Baseband Interface on and off.

Using the digital input and digital output simultaneously is not possible.

If digital baseband output is active, the sample rate is restricted to 100 MHz (200 MHz if enhanced mode is possible; max. 160 MHz bandwidth).

See also "Digital I/Q enhanced mode".

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: OUTP:DIQ ON
