LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring Transmit On/Off Power Measurements > CONFigure[:LTE]:OOPower:NFRames
CONFigure[:LTE]:OOPower:NFRames <Frames>

This command defines the number of frames that are analyzed for On/Off Power measurements.

Parameters:
<Frames>
<numeric value>
Example: CONF:OOP:NFR 10

Defines 10 frames to be analyzed.

Manual operation: See "Number of Frames"
