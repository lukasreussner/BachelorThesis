Marker Search Settings

Access: "Overview" > "Analysis" > "Marker" tab > "Search Config" tab

Markers are commonly used to determine peak values, i.e. maximum or minimum values, in the measured signal. Configuration settings allow you to influence the peak search results.

Depending on the type of result display, different settings are available.

└ Search Mode for Next Peak in X-Direction
└ Search Mode for Next Peak in Y-Direction
└ Marker Search Type
└ Marker Search Area
└ Peak Excursion
