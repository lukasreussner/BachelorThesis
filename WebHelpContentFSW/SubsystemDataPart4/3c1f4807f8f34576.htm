GSM application (R&S FSW-K10) > Remote Commands to Perform GSM Measurements > Selecting the Measurement > CONFigure:MEASurement
CONFigure:MEASurement <MeasType>

This command selects the measurement to be performed in the GSM application.

Note: if you switch between the IQ measurement and MCWN measurement, the enable, positive and negative transition settings in the status registers are set to their default values. Thus, you must reconfigure the transitions after switching measurements, if necessary. (See "Querying the Status Registers").

Parameters:
<MeasType>
IQ

Default I/Q measurement to determine the modulation accuracy, modulation /transient spectrum, trigger parameters etc.

MCWNoise

Sweep measurement to determine noise and intermodulation in multicarrier setups.

*RST:IQ
Example: 

CONF:MEAS MCWN

Example: 

See "Programming Example: Measuring the Wideband Noise for Multiple Carriers".
