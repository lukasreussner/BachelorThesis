Configuring and Performing Sweeps

The "Sweep" settings define how often data is captured from the input signal by the R&S FSW GSM application.

Useful commands for configuring sweeps described elsewhere:
  • [SENSe:​]SWEep:​TIME
  • INITiate<n>:​REFResh
