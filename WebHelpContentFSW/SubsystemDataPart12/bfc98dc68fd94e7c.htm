Modulation Type

Defines the modulation type used for the pattern symbols. The following types are available:

  • PSK
  • QAM
  • User Modulation
    Uses the selected user-defined modulation loaded from a file.
    The currently loaded file is indicated in the channel bar.

For more information on these modulation types, see "Symbol Mapping".

Remote command: 

[SENSe:​]DDEMod:​PATTern:​FORMat

 
└ Load User Modulation
