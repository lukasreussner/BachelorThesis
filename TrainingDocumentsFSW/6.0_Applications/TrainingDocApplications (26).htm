Spectrum

In the Spectrum application the provided functions correspond to those of a conventional spectrum analyzer. The analyzer measures the frequency spectrum of the RF input signal over the selected frequency range with the selected resolution and sweep time, or, for a fixed frequency, displays the waveform of the video signal. This application is used in the initial configuration.

For details see "Measurements and Results".
