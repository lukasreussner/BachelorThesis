Abort Self Alignment

As long as the self-alignment data is being collected, the procedure can be canceled using the "Abort Self Alignment" button.

Note: If you start a self-alignment remotely, then select the "Local" softkey while the alignment is still running, the instrument only returns to the manual operation state after the alignment is completed. In this case, you cannot abort a self-alignment manually.

For details, see "Basics on Alignment".
