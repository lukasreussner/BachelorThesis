Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Retrieving Results > Retrieving Parameter Results > Retrieving Power / Amplitude Parameters > [SENSe:]PULSe:POWer:PMIN?
[SENSe:]PULSe:POWer:PMIN? <QueryRange>

Returns the Peak-to-Min Power Ratio for the specified pulse(s).

Query parameters: 
<QueryRange>
SELected | CURRent | ALL
SELected

Currently selected pulse

CURRent

Detected pulses in the current capture buffer

ALL

All detected pulses in the entire measurement.

Return values: 
<Result>
<char_data>
Usage: Query only
Manual operation: See "Peak-to-Min Power Ratio"
