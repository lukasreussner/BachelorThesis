150 kHz Highpass

Enables or disables the use of an additional 150 kHz highpass filter to protect the R&S FSW LISN from excessive input.

The filter is available for the ENV 216 network only.

For more information see "Controlling V-Networks (LISN)".

Remote command: 

INPut<ip>:​LISN:​FILTer:​HPASs[:​STATe]
