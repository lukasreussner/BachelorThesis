[SENSe:]DDEMod:KDATa:PRBS:FEEDback[:STATe] <State>

Determines how the feedback bit is calculated.

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

The xor connected bit from the specified feedback positions is fed into the last shift register.

ON | 1

The xor connected bit from the specified feedback positions is inverted before it is fed into the last shift register.

*RST:0
Example: DDEM:KDAT:PRBS:FEED:STAT ON
Manual operation: See "Negate Feedback Path"
