CALCulate<n>:TRACe<t>:ADJust:ALIGnment[:DEFault] <Alignment>

This command defines where the reference point is to appear in the result range.

Suffix: 
<n>

Window

<t>

Trace

Setting parameters: 
<Alignment>
LEFT | CENTer | RIGHt
LEFT

The reference point is at the start of the result range.

CENTer

The reference point is in the middle of the result range.

RIGHt

The reference point is displayed at the end of the result range.

*RST:LEFT
