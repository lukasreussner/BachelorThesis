Configuring Result Display Characteristics

Access: "Overview" > "Result Config" > "Display Settings"

The application allows you to define the information displayed in various graphical result displays.

Scope of the scaling

The functionality of the "Display Settings" is only available when you have selected one of the results displays that support this feature from the "Specifics for:" dropdown menu at the bottom of the dialog box.

(In this case, the functionality to adjust the Time Domain result display.)
