General Instrument Setup > Frequency Response Correction (R&S FSW-K544) > User-defined Frequency Response Correction Settings > Load Settings
Load Settings

Loads a stored saveset for a user-defined frequency response correction scenario. Existing settings in the dialog box are overwritten.

Only .dfl files can be loaded. The default storage directory for correction files is C:\R_S\INSTR\USER\FResponse.

Remote command: 

[SENSe:​]CORRection:​FRESponse:​USER:​LOAD
