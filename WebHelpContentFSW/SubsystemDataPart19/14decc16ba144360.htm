CALCulate<n>:LIMit<li>:ACPower:ACHannel:RESult?
CALCulate<n>:LIMit<li>:ACPower:ALTernate<ch>:RESult?

This command queries the state of the limit check for the adjacent or alternate channels in an ACLR measurement.

To get a valid result, you have to perform a complete measurement with synchronization to the end of the measurement before reading out the result. This is only possible for single measurement mode.

See also INITiate<n>:​CONTinuous.

Suffix: 
<n>, <li>

irrelevant

<ch>
1 to 11

Alternate channel number

Return values: 
<LowerChan>, <UpperChan>
text value

The command returns two results. The first is the result for the lower, the second for the upper adjacent or alternate channel.

PASSED

Limit check has passed.

FAIL

Limit check has failed.

Example: INIT:IMM;*WAI;
CALC:LIM:ACP:ACH:RES?
PASSED,PASSED

See Example: Configuring and Performing an ACLR Measurement.

Usage: Query only
