CALCulate<n>:PSPectrum:MAXFrequency <MaxFrequncy>

Defines the maximum frequency span for which the Spectrum is calculated. Internally, the span is limited by the number of possible interpolation samples (100 000).

Suffix: 
<n>
1..n

Window

Parameters:
<MaxFrequncy>
Range: >0  to  1/10 of sample rate
Default unit: HZ
Manual operation: See "Maximum Frequency"
