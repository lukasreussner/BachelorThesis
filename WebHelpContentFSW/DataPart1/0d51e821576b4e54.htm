[SENSe:]BANDwidth:VIDeo <Bandwidth>

This command defines the video bandwidth.

The command decouples the video bandwidth from the resolution bandwidths.

Parameters:
<Bandwidth>
refer to data sheet
*RST:AUTO is set to ON
Example: BAND:VID 10 kHz
