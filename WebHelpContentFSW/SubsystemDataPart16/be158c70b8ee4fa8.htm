Code Display Order

Defines the sorting of the channels for the Code Domain Power and Code Domain Error result displays.

For further details on the code order, refer to "Code Display and Sort Order".

"Hadamard"

By default, the codes are sorted in Hadamard order, i.e. in ascending order.

The power of each code is displayed; there is no visible distinction between channels. If a channel covers several codes, the display shows the individual power of each code.

"Bit-Reverse"

Bundles the channels with concentrated codes, i.e. all codes of a channel are next to one another. Thus you can see the total power of a concentrated channel.

Remote command: 

[SENSe:​]CDPower:​ORDer
