X-Axis Reference Position

Defines the position of the "X-Axis Reference Value" on the x-axis. The position is defined as a percentage value, where 0 % refers to the beginning (left side), 100 % refers to the end (right side) of the diagram. The x-axis is adapted so that the reference value is displayed at the reference position.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​X[:​SCALe]:​RPOSition
