[SENSe:]DDEMod:SEARch:MBURst:STARt:SAMPles?

This command queries the start sample of the current result range within the capture buffer.

Tip: to query the start symbol, use [SENSe:​]DDEMod:​SEARch:​MBURst:​STARt[:​SYMBols]?.

Example: 
INIT:CONT OFF 
//switch to single sweep mode 
INIT;*WAI 
//perform single sweep 
SENS:DDEM:SEAR:MBUR:CALC 1 
//switch to first result range 
SENS:DDEM:SEAR:MBUR:START:SAMP? 
//query start of current (=first) result range
// in the capture buffer 
SENS:DDEM:SEAR:MBUR:CALC 2 
//switch to second result range 
SENS:DDEM:SEAR:MBUR:START:SAMP? 
//query start of current (=second) result range
// in the capture buffer
Usage: Query only
