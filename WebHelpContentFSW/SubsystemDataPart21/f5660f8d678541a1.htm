CONFigure:DS:CHANnel:CPES<i>:SUBCarrier:TYPE <SCType>

Defines the type of configuration for the specified entry in the "Continuous Pilots and Excluded Subcarrier Assignment" table.

Suffix: 
<i>
1..n

index in the "Continuous Pilots and Excluded Subcarrier Assignment" table

Parameters:
<SCType>
CPIL | ESUB
PLC

Physical link channel

(Query only, always available as first entry.)

CPIL

Continuous pilot

ESUB

Excluded subcarrier

Example: CONF:DS:CHAN:CPES2:SUBC:TYPE CPIL
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Manual operation: See "Type"
