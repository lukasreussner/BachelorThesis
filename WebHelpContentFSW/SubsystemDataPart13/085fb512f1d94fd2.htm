(CPICH) Slot

Selects the (CPICH) slot for evaluation. This affects the following evaluations (see also "Evaluation Methods for Code Domain Analysis"):

  • Code Domain Power
  • Peak Code Domain Error
  • Result Summary
  • Composite Constellation
  • Code Domain Error Power
  • Channel Table
  • Power vs Symbol
  • Symbol Const
  • Symbol EVM
  • Bitstream
