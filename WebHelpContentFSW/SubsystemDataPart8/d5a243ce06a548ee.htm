Meas Mode (Auto Manual)

Selects the measurement mode for the hot and cold power measurements.

For more information about the measurement modes see "Measurement Modes".

In manual measurement mode, the application opens a dialog box when you start a measurement. For more information about its contents see "Performing a manual measurement".

Remote command: 

[SENSe:​]CONFigure:​CONTrol

[SENSe:​]CONFigure:​MEASurement
