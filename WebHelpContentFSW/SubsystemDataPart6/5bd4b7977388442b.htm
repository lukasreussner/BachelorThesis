Range per Division

Defines the value range to be displayed per division. Since the display consists of 10 divisions by default, the displayed range is:

Range = 10 * <Range per Division>

Note: If fewer divisions are displayed (e.g. because the window is reduced in height), the range per division is increased in order to display the same result range in the smaller window. In this case, the range per division does not correspond to the actual display.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​PDIVision
