Transient Analysis (R&S FSW-K60) > Remote Commands to Perform Transient Analysis > Retrieving Results > Retrieving Information on Detected Hops > [SENSe:]HOP:STATe[:INDex]?
[SENSe:]HOP:STATe[:INDex]? <QueryRange>

Returns the hop state indexes from the Results table for the specified hop(s).

Query parameters: 
<QueryRange>
SELected | CURRent | ALL
SELected

Selected hop

CURRent

Detected hops in the current capture buffer

ALL

All hops detected in the entire measurement

Return values: 
<Result>
integer
Usage: Query only
Manual operation: See "State Index"
