Configuring Oscilloscope Baseband Input

The following commands define settings for analog baseband input from an oscilloscope.

The commands for analog baseband input from an oscilloscope are similar, but not identical to those used for analog baseband using the optional Analog Baseband interface on the R&S FSW.
