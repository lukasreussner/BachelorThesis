CALCulate<n>:DELTamarker<m>:X:RELative?

This command queries the relative position of a delta marker on the x-axis.

If necessary, the command activates the delta marker first.

Suffix: 
<n>

Window

<m>

Marker

Return values: 
<Position>

Position of the delta marker in relation to the reference marker.

Example: CALC:DELT3:X:REL?

Outputs the frequency of delta marker 3 relative to marker 1 or relative to the reference position.

Usage: Query only
Manual operation: See "Marker 1 / Marker 2 / Marker 3 / Marker 4"
See "Delta Marker 1 / Marker 2 / Marker 3 / ... Marker 16 / Norm / Delta"
