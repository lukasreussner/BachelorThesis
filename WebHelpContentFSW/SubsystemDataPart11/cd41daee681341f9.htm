CALCulate<n>:DISTribution:HOP:TIMing <XAxis>, <YAxis>

Configures the Parameter Distribution result display for hop timing parameters.

Suffix: 
<n>
1..n

Window

Setting parameters: 
<XAxis>
BEGin | DWELl | SWITching

Hop parameter to be displayed on the x-axis. For a description of the available parameters see "Hop Parameters""Chirp Parameters".

BEGin

Hop begin time

DWELl

Hop dwell time

SWITching

Hop switching time

<YAxis>
COUNt | OCCurrence

Parameter to be displayed on the y-axis.

COUNt

Number of hops in which the parameter value occurred.

OCCurance

Percentage of all measured hops in which the parameter value occurred.

*RST:COUNt
Example: CALC:DIST:HOP:TIM SWIT,COUN
Usage: Setting only
