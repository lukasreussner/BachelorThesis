CALCulate<n>:FSK:DEViation:COMPensation <RefDevComp>

This command defines whether the deviation error is compensated for when calculating the frequency error for FSK modulation.

Note that this command is maintained for compatibility reasons only. For newer remote programs, use [SENSe:​]DDEMod:​NORMalize:​FDERror.

Setting parameters: 
<RefDevComp>
ON | OFF
ON | 1

Scales the reference signal to the actual deviation of the measurement signal.

OFF | 0

Uses the entered nominal deviation for the reference signal.

*RST:ON
