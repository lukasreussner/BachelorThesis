Burst Type

Assigns a burst type to the selected slot.

The following burst types are supported:
  • Normal (NB)
  • Higher Symbol Rate (HB)
  • Access (AB)

The graphical slot structure is adapted according to the selected burst type.

Note: The "Slot" settings are dependant on each other, and only specific combinations of these parameters are available in this dialog box (see "Dependency of Slot Parameters").

Remote command: 

CONFigure[:​MS]:​CHANnel:​SLOT<Number>:​TYPE
