Drop-Out Time

Defines the time the input signal must stay below the trigger level before triggering again.

For more information on the drop-out time, see Trigger Drop-Out Time.

When using the optional 5 GHz bandwidth extension (R&S FSW-B2000/B5000) with an IF power trigger, the drop-out time defines the width of the robust width trigger. By default it is set to 1 μs. For external triggers, no drop-out time is available when using the B2000/B5000 option.

(For details, see Triggering)

Remote command: 

TRIGger[:​SEQuence]:​DTIMe
