Limit Offset to Detection Threshold

Defines a limit line as an offset to the detection threshold for each range. The limit line is indicated by a red line in the "Spurious Detection Spectrum". Values that exceed this limit are indicated red in the "Spurious Detection Table". If a violation occurs, the global limit check over all ranges is indicated as failed.

For all spurs to be indicated as violations, set this threshold to 0.

Note: The limit line functionality used in the R&S FSW base unit is not supported in the R&S FSW Spurious measurements application.

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​LOFFset
