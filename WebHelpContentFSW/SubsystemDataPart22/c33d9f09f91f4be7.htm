Conflict

In case of a conflict, the application shows the type of conflict and the ID of the allocations that are affected. Possible conflicts are:

  • bandwidth error (">BW")
    A bandwidth error occurs when the number of resource blocks in the subframe exceeds the bandwidth you have set.
  • RB overlap errors
    An RB overlap error occurs if one or more allocations overlap. In that case, check if the length and offset values of the allocations are correct.
