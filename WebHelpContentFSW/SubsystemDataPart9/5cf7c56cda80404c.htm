MMEMory:LOAD<n>:TRACe <Trace>, <FileName>

This command imports trace data from the specified window to an ASCII file.

For details on the file format see Reference: ASCII File Export Format.

Suffix: 
<n>

Window

Parameters:
<Trace>

Number of the trace to be stored

(This parameter is ignored for FORMat:​DIMPort:​TRACes ALL).

<FileName>

String containing the path and name of the import file.
