LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Read Numeric Results > Result for Selection > FETCh[:CC<cc>]:SUMMary:OSTP:MAXimum?
FETCh[:CC<cc>]:SUMMary:OSTP:MAXimum?
FETCh[:CC<cc>]:SUMMary:OSTP:MINimum?
FETCh[:CC<cc>]:SUMMary:OSTP[:AVERage]?

This command queries the OSTP.

Suffix: 
<cc>

Component Carrier

Return values: 
<OSTP>
<numeric value>

Minimum, maximum or average OSTP, depending on the last command syntax element.

Default unit: dBm
Example: 

//Query average OSTP

FETC:SUMM:OSTP?
Usage: Query only
Manual operation: See "Result Summary"
