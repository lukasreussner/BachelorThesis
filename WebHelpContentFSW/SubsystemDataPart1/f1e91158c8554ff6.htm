Input Configuration

Defines whether the input is provided as a differential signal via all four input channel connectors on the oscilloscope, or as a plain I/Q signal via two simple-ended lines.

"Single-ended"

I, Q data only

"Differential"

I, Q and inverse I,Q data

(Not available for R&S FSW85)

Remote command: 

INPut<ip>:​IQ:​OSC:​BALanced[:​STATe]
