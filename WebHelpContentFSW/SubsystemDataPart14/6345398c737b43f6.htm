Configuring Input via the Optional Analog Baseband Interface

The following commands are required to control the optional Analog Baseband Interface in a remote environment. They are only available if this option is installed.

For more information on the Analog Baseband Interface see Processing Data from the Analog Baseband Interface.

Useful commands for Analog Baseband data described elsewhere:
  • INP:SEL AIQ (see INPut<ip>:​SELect)
  • [SENSe:​]FREQuency:​CENTer
For calibration:
  • DIAGnostic:SERVice:INPut[:SELect]
  • DIAGnostic:SERVice:INPut:AIQ[:TYPE]
  • [SENSe:​]PROBe<p>:​SETup:​CMOFfset
