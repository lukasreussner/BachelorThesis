WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Retrieving Results > Numeric Modulation Accuracy, Flatness and Tolerance Results > Error Parameter Results > FETCh:SCDetailed:POWer:RU?
FETCh:SCDetailed:POWer:RU? <SCDetailedObject>

Returns the power in all PPDUs for each RU. The result is a comma-separated list of power values, one per RU.

Tip: to obtain the results for an individual subcarrier, use FETCh:​SCDetailed:​POWer[:​SC]?.

Query parameters: 
<SCDetailedObject>
HELTf | DPILot
HELTf

Includes long training field (HE-LTF) subcarriers only

DPILot

Includes data and pilot subcarriers only

Default unit: dBm
Example: FETC:SCD:POW:RU?
Usage: Query only
