I/Q Mode

Defines the format of the input signal.

For more information, see "Processing Oscilloscope Baseband Input".

"I/Q"

Both components of the complex input signal (in-phase component, quadrature component) are filtered and resampled to the sample rate of the application. The input signal is down-converted with the center frequency (Low IF I).

"I Only / Low IF I"

The input signal at the channel providing I data is filtered and resampled to the sample rate of the application.

The input signal is down-converted with the center frequency (Low IF I).

Remote command: 

INPut<ip>:​IQ:​OSC:​TYPE
