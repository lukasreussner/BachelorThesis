Problem: identification code not as expected

That is OK if DUT and R&S FSW Avionics (VOR/ILS) measurements application are not synchronized. Sending "MUC" can give you "C MU" or "UC MU" or "UC MU", etc.

Possible Solution: Increase the measurement time to get the complete code word, .e.g "UC MUC MU".
