Pulse Measurements (R&S FSW-K6/6S) > Analysis > Markers > Marker Positioning Functions > Search Next Minimum
Search Next Minimum

Sets the selected marker/delta marker to the next (higher) minimum of the selected trace. If no marker is active, marker 1 is activated.

Remote command: 

CALCulate<n>:​MARKer<m>:​MINimum:​NEXT

CALCulate<n>:​MARKer<m>:​MINimum:​LEFT

CALCulate<n>:​MARKer<m>:​MINimum:​RIGHt

CALCulate<n>:​DELTamarker<m>:​MINimum:​NEXT

CALCulate<n>:​DELTamarker<m>:​MINimum:​LEFT

CALCulate<n>:​DELTamarker<m>:​MINimum:​RIGHt
