[SENSe:]MIXer<x>:HARMonic:HIGH:STATe <State>

This command specifies whether a second (high) harmonic is to be used to cover the band's frequency range.

Suffix: 
<x>

irrelevant

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: MIX:HARM:HIGH:STAT ON
Example: 

See "Programming Example: Working with an External Mixer" and "Configuring a conversion loss table for a user-defined band".
