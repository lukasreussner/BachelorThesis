Linked Markers

If enabled, the markers in all Transient Analysis diagrams - regardless of the x-axis unit - are linked, i.e. when you move a marker in one window, the markers in all other windows are moved to the same position in time. Linking is also possible across spectrogram and spectrum displays.

Remote command: 

CALCulate<n>:​MARKer<m>:​LINK
