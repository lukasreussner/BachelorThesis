Message: Data Sequences have Incorrect # Symbols

The length of the data sequences taken from a known data file does not correspond with the specified result range length (see "Result Length"). Specify the correct number of symbols as the result length.

Tip: the result length of the data sequences in the known data file is indicated in the "Signal Description" dialog box, in the "Known Data" tab (see "Known Data").
