TRACe:IQ:WBANd[:STATe] <State>

This command determines whether the wideband provided by bandwidth extension options is used or not (if installed).

Parameters:
<State>
ON | OFF | 1 | 0
ON | 1

If enabled, installed bandwidth extension options can be used. They are activated for bandwidths > 80 MHz, if the bandwidth is not restricted by the TRACe:​IQ:​WBANd:​MBWidth command. Otherwise, the currently available maximum bandwidth is allowed.

(See "Sample Rate and Maximum Usable I/Q Bandwidth for RF Input").

This parameter corresponds to the "Auto" setting in manual operation (with TRACe:​IQ:​WBANd:​MBWidth 512 MHz).

OFF | 0

The bandwidth extension options R&S FSW-B160/-B320/-B512/-B1200/-B2001 are deactivated; the maximum analysis bandwidth is restricted to 80 MHz. This parameter corresponds to the "80 MHz" setting in manual operation.

*RST:1
Manual operation: See "Maximum Bandwidth"
