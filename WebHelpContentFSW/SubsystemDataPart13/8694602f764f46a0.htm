Copying a Table

Copies an existing channel table definition. The details of the selected channel are displayed in the "Channel Table" dialog box. See "Channel Details".

Remote command: 

BTS measurements:

CONFigure:​WCDPower[:​BTS]:​CTABle:​COPY

UE measurements:

CONFigure:​WCDPower:​MS:​CTABle:​COPY
