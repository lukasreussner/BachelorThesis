Transient Analysis (R&S FSW-K60) > Remote Commands to Perform Transient Analysis > Retrieving Results > Retrieving Information on Detected Hops > [SENSe:]HOP:FREQuency:RMSFm?
[SENSe:]HOP:FREQuency:RMSFm? <QueryRange>

Returns the RMS Frequency Deviation from the Results table for the specified hop(s).

Query parameters: 
<QueryRange>
SELected | CURRent | ALL
SELected

Selected hop

CURRent

Detected hops in the current capture buffer

ALL

All hops detected in the entire measurement

Return values: 
<Result>
Usage: Query only
Manual operation: See "Frequency Deviation (RMS)"
