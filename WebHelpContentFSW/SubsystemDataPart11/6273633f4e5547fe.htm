Returns the statistical value for the hop switching time from the statistics table for the specified hop(s).

Query parameters: 
<QueryRange>
CURRent | ALL
CURRent

Detected hops in the current capture buffer

ALL

All hops detected in the entire measurement

Return values: 
<Result>
Usage: Query only
