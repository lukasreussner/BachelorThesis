[SENSe:][LTE:]DL:DEMod:PRData <Reference>

This command selects the type of reference data to calculate the EVM for the PDSCH.

Parameters:
<Reference>
AUTO

Automatic identification of reference data.

ALL0

Reference data is 0, according to the test model definition.

Example: 

//Select reference data for PDSCH demodulation

DL:DEM:PRD ALL0
Manual operation: See "PDSCH Reference Data"
