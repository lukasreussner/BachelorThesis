WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Demodulation > [SENSe:]BANDwidth:CHANnel:AUTO
[SENSe:]BANDwidth:CHANnel:AUTO <State>

Defines whether the channel bandwidth to be analyzed is determined automatically.

This command is only available for standards IEEE 802.11a, ac, n.

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

Switches the function off

ON | 1

Switches the function on

*RST:1
