Magnitude

Displays the spectrum of the input signal. In contrast to the Spectrum application, the frequency values are determined using an Discrete frequency transformation (DFT) from the recorded I/Q data set.

Remote command: 

LAY:ADD? '1',RIGH,MAGN

(see LAYout:​ADD[:​WINDow]?)
