Traces

The trace settings determine how the measured data is analyzed and displayed on the screen. In TD-SCDMA applications, only one trace per window can be configured for Code Domain Analysis.

List of Commands
DISPlay[:WINDow<n>]:TRACe<t>:MODE
DISPlay[:WINDow<n>]:TRACe<t>[:STATe]
