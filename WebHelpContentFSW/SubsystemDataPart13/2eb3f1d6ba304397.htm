Saving the Table

Saves the changes to the table and closes the "Carrier Table Settings" dialog box.

The new or edited table is stored in the default directory for carrier tables: C:\R_S\INSTR\USER\chan_tab\carrier_table\.

Remote command: 

[SENSe:​]TAERror:​SAVE
