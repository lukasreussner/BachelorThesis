[SENSe:]CDPower:MIMO <State>

Activates or deactivates single antenna MIMO measurement mode.

Channels that have modulation type MIMO-QPSK or MIMO-16QAM are only recognized as active channels if this setting is ON.

For details see "MIMO".

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: SENS:CDP:MIMO ON
Mode: BTS application only
Manual operation: See "MIMO"
