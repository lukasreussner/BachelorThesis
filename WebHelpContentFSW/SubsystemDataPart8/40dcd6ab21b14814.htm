[SENSe:]FREQuency:STARt <Frequency>

This command defines the start frequency.

If you change the start frequency, the application creates a new frequency list.

Parameters:
<Frequency>
*RST:RST value
Example: FREQ:STAR 900MHZ

Defines a start frequency of 900 MHz.
