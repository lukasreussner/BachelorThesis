[SENSe:]DDEMod:TFILter[:STATe] <TXFilterState>

Use this command to switch the transmit filter off. To switch a transmit filter on, use the [SENSe:​]DDEMod:​TFILter:​NAME command.

Setting parameters: 
<TXFilterState>
ON | OFF
OFF | 0

Switches the transmit filter off.

ON | 1

Switches the transmit filter specified by [SENSe:​]DDEMod:​TFILter:​NAME on. However, this command is not necessary, as the [SENSe:​]DDEMod:​TFILter:​NAME command automatically switches the filter on.

*RST:ON
Manual operation: See "Transmit Filter Type"
