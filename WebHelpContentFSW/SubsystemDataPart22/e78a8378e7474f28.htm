LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Signal Capture > Data Capture > [SENSe:][LTE:]FRAMe:COUNt:STATe
[SENSe:][LTE:]FRAMe:COUNt:STATe <State>

This command turns manual selection of the number of frames you want to analyze on and off.

Parameters:
<State>
ON

You can set the number of frames to analyze.

OFF

The R&S FSW analyzes the frames captured in a single sweep.

*RST:ON
Example: 

//Turn on manual selection of number of frames

FRAM:COUN:STAT ON
Manual operation: See "Overall Frame Count"
