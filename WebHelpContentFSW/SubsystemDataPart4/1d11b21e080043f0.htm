CONFigure[:MS]:MCARrier:CARRier<c>[:STATe]?

This command queries the activity of the selected carrier.

Note: to activate a carrier, define its absolute frequency using CONFigure[:​MS]:​MCARrier:​CARRier<c>:​FREQuency.

Suffix: 
<c>
1..16

Active carrier

Return values: 
<State>
ON | OFF | 1 | 0
*RST:0
Example: CONF:MCAR:CARR3?
Example: 

See "Programming Example: Measuring the Wideband Noise for Multiple Carriers".

Usage: Query only
Manual operation: See "Active carriers"
