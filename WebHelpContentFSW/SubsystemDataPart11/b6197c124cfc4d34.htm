FM Settling Tolerance

Defines the allowed deviation from the detected FM signal model state where the hop or chirp is considered as "settled". One global FM settling tolerance is defined (in Hertz) for all hops or chirps.

Remote command: 

CALCulate<n>:​HOPDetection:​FMTolerance

CALCulate<n>:​CHRDetection:​FMTolerance
