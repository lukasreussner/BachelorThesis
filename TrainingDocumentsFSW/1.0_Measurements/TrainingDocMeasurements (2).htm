Measuring a Basic Signal

We will start out by measuring a simple sinus wave, using the internal calibration signal as the input.

To display the internal 64 MHz calibration signal

  1. Press the PRESET key to start out in a defined instrument configuration.

  2. Press the SETUP key on the front panel.

  3. Tap the "Service" softkey.

  4. Tap the "Calibration Signal" tab.

  5. Tap the "Calibration Frequency RF" option. Leave the frequency at the default 64 MHz, with a narrowband spectrum.

    The calibration signal is now sent to the RF input of the R&S FSW. By default, a continuous frequency sweep is performed, so that the spectrum of the calibration signal is now displayed in the standard level versus frequency diagram.

    Calibration signal as RF input
Instrument warmup time

Note that the instrument requires an initial warmup time after switching it on. A message in the status bar ("Instrument warming up...") indicates that the operating temperature has not yet been reached. Wait until this message is no longer displayed before you start a measurement.

To optimize the display

To optimize the display for the calibration signal, we will adjust the main measurement settings.

  1. Set the center frequency to the calibration frequency:

    1. Tap the "Overview" softkey to display the configuration "Overview".

    2. Tap the "Frequency" button.

    3. In the "Center" field, enter 64 on the number pad on the front panel.

    4. Press the "MHz" key next to the number pad.

  2. Reduce the span to 20 MHz:

    1. In the "Span" field of the "Frequency" dialog box, enter 20 MHz.

    2. Close the "Frequency" dialog box.

  3. Set the reference level to -25 dBm:

    1. In the configuration "Overview", tap the "Amplitude" button.

    2. In the "Value" field of the "Amplitude" dialog box, enter -25 dBm.

    The display of the calibration signal is now improved. The maximum at the center frequency (=calibration frequency) of 64 MHz becomes visible.
