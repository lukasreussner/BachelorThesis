How to Operate the Instrument Without a Network

To operate the instrument without a network connection either temporarily or permanently, no special measures are necessary. Windows  automatically detects the interruption of the network connection and does not set up the connection when the instrument is switched on.

If you are not prompted to enter the user name and password, proceed as described in "How to Activate or Deactivate the Automatic Login Mechanism".
