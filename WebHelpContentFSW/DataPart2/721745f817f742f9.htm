[SENSe:]MIXer<x>:BIAS[:LOW] <BiasSetting>

This command defines the bias current for the low (first) range.

(See also "Bias Current").

This command is only available if the external mixer is active (see [SENSe:​]MIXer<x>[:​STATe]).

Parameters:
<BiasSetting>
