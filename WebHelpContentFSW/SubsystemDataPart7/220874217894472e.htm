Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Sweep Configuration > [SENSe:]SWEep:IQAVg:COUNt
[SENSe:]SWEep:IQAVg:COUNt <Count>

This command defines the average count for I/Q averaging.

Parameters:
<Count>
<numeric value> (integer only)
Range: 1  to  10000
*RST:1
Example: 

//Average over 10 data captures

SWE:IQAV:COUN 10
Manual operation: See "Averaging the I/Q data"
