Remote Commands > Managing Settings and Results > Selecting the Items to Store > MMEMory:SELect:CHANnel[:ITEM]:DEFault
MMEMory:SELect:CHANnel[:ITEM]:DEFault
MMEMory:SELect[:ITEM]:DEFault

This command selects the current settings as the only item to store to and load from a configuration file.

Depending on the used command, either the items from the entire instrument (MMEMory:SELect[:ITEM]...), or only those from the currently selected channel (MMEM:SELect:CHANnel[:ITEM]...) are stored or loaded.

Usage: Event
Manual operation: See "Items:"
