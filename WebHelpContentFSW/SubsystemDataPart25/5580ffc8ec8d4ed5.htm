NB-IoT Downlink (R&S FSW-K106) > Remote Control > Analysis > Result Settings > UNIT:EVM
UNIT:EVM <Unit>

This command selects the EVM unit.

Parameters:
<Unit>
DB

EVM results returned in dB

PCT

EVM results returned in %

*RST:PCT
Example: 

//Display EVM results in %

UNIT:EVM PCT
Manual operation: See "EVM Unit"
