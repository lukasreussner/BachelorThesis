Configuring Outputs

Access: "Overview" > "Input / Output" > "Output"

The "Output" tab of the "Input / Output" dialog box contains settings to configure the various signal outputs available on the R&S FSW.

The functionality is the same as in the Spectrum application. For more information about the output functions, refer to the R&S FSW User Manual.
