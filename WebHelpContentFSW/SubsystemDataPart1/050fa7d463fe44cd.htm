I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Configuring I/Q Analyzer Measurements > Configuring the Data Input and Output > Configuring the 2 GHz / 5 GHz Bandwidth Extension (R&S FSW-B2000/B5000) > SYSTem:COMMunicate:RDEVice:OSCilloscope:TCPip
SYSTem:COMMunicate:RDEVice:OSCilloscope:TCPip <Address>

Defines the TCPIP address or computer name of the oscilloscope connected to the R&S FSW via LAN.

Note: The IP address is maintained after a [PRESET], and is transferred between applications.

Parameters:
<Address>

computer name or IP address

Example: SYST:COMM:RDEV:OSC:TCP '192.0.2.0'
Example: SYST:COMM:RDEV:OSC:TCP 'FSW43-12345'
Manual operation: See "TCPIP Address / Computer Name"
