[SENSe:]CDPower:UCPich:PATTern <Pattern>

This command defines which pattern is used for signal analysis for the user-defined CPICH (see [SENSe:​]CDPower:​UCPich[:​STATe]).

This command only applies to antenna 1.

Note that this command is maintained for compatibility reasons only. Use [SENSe:​]CDPower:​UCPich:​ANT<antenna>[:​STATe] for new remote control programs.

Parameters:
<Pattern>
1

fixed usage of "Pattern 1" according to standard

2

fixed usage of "Pattern 2" according to standard

*RST:2
Example: 

SENS:CDP:UCP:PATT 1

Mode: BTS application only
