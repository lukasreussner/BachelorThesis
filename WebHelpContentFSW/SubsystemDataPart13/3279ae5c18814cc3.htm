Configuration Overview

Access: [Meas Config] > "Overview"

For Time Alignment Error measurements, the "Overview" provides quick access to the following configuration dialog boxes (listed in the recommended order of processing):

  1. "Select Measurement"

    See "Measurements and Result Display"

  2. "Scrambling Code"

    See "BTS Scrambling Code"

  3. "Input/ Frontend"

    See "Data Input and Output Settings"

  4. (Optionally:) "Trigger"

    See "Trigger Settings"

  5. "Signal Capture"

    See "Signal Capture (Data Acquisition)"

  6. "Synchronization"

    See "Synchronization (BTS Measurements Only)"

  7. "Carrier Table"

    See "Carrier Table Configuration"

  8. "Display Configuration"

    See "Evaluation Methods for Code Domain Analysis" and "Evaluation Methods"

All settings required for Time Alignment Error measurements are identical to those described for Code Domain Analysis (see "Code Domain Analysis").

For TAE measurement on multiple base stations, however, the carrier table must be defined.
