Configuration Overview

Access: all menus

Throughout the measurement channel configuration, an overview of the most important currently defined settings is provided in the "Overview".

In addition to the main measurement settings, the "Overview" provides quick access to the main settings dialog boxes. The individual configuration steps are displayed in the order of the data flow. Thus, you can easily configure an entire measurement channel from input over processing to output and analysis by stepping through the dialog boxes as indicated in the "Overview".

In particular, the "Overview" provides quick access to the following configuration dialog boxes (listed in the recommended order of processing):

  1. Input Settings

    See "RF Input Settings"

  2. Amplitude Settings

    See "Amplitude and Y-Axis Scaling"

  3. Frequency Settings

    See "Frequency Settings"

  4. Trigger Settings

    See "Trigger Settings"

  5. Multi-Carrier Configuration

    See "Multi-Carrier Configuration"

  6. Output Settings

    See "Output Settings"

  7. Analysis

    See "Analysis"

  8. Display Configuration

    See "Display Configuration"

The main configuration settings and dialog boxes are also available via the "MCGD Meas" menu which is displayed when you press the [MEAS CONFIG] key.

For a description see
  • "Carrier Spacing"
  • "Group Delay Mode"
  • "Measurement Time"
  • "Calibrate (Execute Reference Calibration)"
  • "Measurement Configuration"
  • "Display Configuration"

In this documentation, only the most convenient method of accessing the dialog boxes is indicated - usually via the "Overview".

To configure settings
► 

Select any button in the "Overview" to open the corresponding dialog box.

Select a setting in the channel bar (at the top of the measurement channel tab) to change a specific setting.

For step-by-step instructions on configuring Multi-Carrier Group Delay measurements, see "How to Perform Measurements in the Multi-Carrier Group Delay Application".
