CONFigure:DPD:SEQuence <State>

This command selects the order in which the AM/AM and AM/PM distortion are applied.

Available when both have been turned on.

Prerequisites for this command
  • Turn on polynomial DPD (CONFigure:DDPD[:STATe]).
  • Turn on both AM/AM and AM/PM calculation (CONFigure:DPD:AMAM[:STATe] / CONFigure:DPD:AMPM[:STATe]).
Parameters:
<Order>
AMFirst

Calculates the AM/AM distortion first, then the AM/PM distortion.

PMFirst

Calculates the AM/PM distortion first, then the AM/AM distortion.

Example: 

//Calculates the AM/AM curve first

CONF:DPD:SEQ AMF
Manual operation: See "Selecting the order of model calculation"
