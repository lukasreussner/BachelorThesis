Number of Adjacent Channels (Adj Count)

Defines the number of adjacent channels above and below the Tx channel block in an MSR signal. You must define the carrier channel to which the relative adjacent-channel power values refer (see "Reference Channel").

For details on MSR signals, see "Measurement on Multi-Standard Radio (MSR) Signals".
