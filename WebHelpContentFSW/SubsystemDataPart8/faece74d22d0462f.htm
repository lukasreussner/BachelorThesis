Noise Source

Selects the type of noise source you are using for the measurement.

The noise source type affects the data type that the table contains. For a "Noise Diode", the table contains the ENR values of the noise source you are using. For a "Resistor", the table contains the noise temperatures of the resistors (Thot and Tcold).

For more information on the noise source types, see "Noise Source".

Remote command: 

[SENSe:​]CORRection:​ENR[:​MEASurement]:​TYPE
