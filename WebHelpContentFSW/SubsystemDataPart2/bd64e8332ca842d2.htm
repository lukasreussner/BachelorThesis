Reference Marker

Defines a marker as the reference marker which is used to determine relative analysis results (delta marker values).

If the reference marker is deactivated, a different reference marker is automatically selected; the delta marker remains active.

Remote command: 

CALCulate<n>:​DELTamarker<m>:​MREF
