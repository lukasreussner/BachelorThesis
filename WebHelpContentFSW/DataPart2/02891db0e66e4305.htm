Print

Starts to print or store the selected screen contents to a file (see "Print Content Settings").

Whether the output is sent to the printer or stored in a file or the clipboard depends on the selected printing device and the printing device settings (see "Printer Settings").

If the output is stored to a file, a file selection dialog box is opened to select the filename and location. The default path is C:\R_S\INSTR\USER.

Remote command: 

HCOPy[:​IMMediate<device>]

HCOPy[:​IMMediate<device>]:​NEXT
