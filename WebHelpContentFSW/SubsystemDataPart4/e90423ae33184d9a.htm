Modulation Accuracy Measurement Configuration

GSM measurements require a special application on the R&S FSW, which you activate using the MODE key.

For details see Applications, Measurement Channels, and Operating Modes.

When you switch a measurement channel to the GSM application the first time, a set of parameters is passed on from the currently active application. After initial setup, the parameters for the measurement channel are stored upon exiting and restored upon re-entering the channel. Thus, you can switch between applications quickly and easily.

When you activate a measurement channel in the GSM application, a GSM modulation accuracy measurement for the input signal is started automatically with the default configuration. The "GSM" menu is displayed and provides access to the most important configuration functions.

For a description see
  • "Slot to Measure"
  • "Slot Scope"
  • "Measurement Settings"
  • "Data Acquisition"
  • "Result Configuration"
  • "Display Configuration"
  • "Configuration Overview"

The MARKER FUNCT and LINES menus are currently not used.

Importing and Exporting I/Q Data

The I/Q data to be evaluated in the GSM application (Modulation Accuracy measurement only) can not only be captured by the GSM application itself, it can also be imported to the application, provided it has the correct format. Furthermore, the evaluated I/Q data from the GSM application can be exported for further analysis in external applications.

The import and export functions are available in the "Save/Recall" menu which is displayed when you select the "Save" or "Open" icon in the toolbar.

For details on importing and exporting I/Q data see Import/Export Functions.
