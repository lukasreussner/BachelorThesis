Import

Opens a file selection dialog box and loads the limit line from the selected file in .CSV format.

Note that a valid import file must contain a minimum of required information for the R&S FSW.

For details on the file format see "Reference: Limit Line File Format".

Remote command: 

MMEMory:​LOAD<n>:​LIMit
