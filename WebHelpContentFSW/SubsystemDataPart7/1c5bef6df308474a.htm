Parameter Sweep

The Parameter Sweep result display is a result display that shows a result of the DUT (for example the EVM) against two (custom) measurement parameters. The results of this measurement are displayed in graphical and numerical form.

The Parameter Sweep is a good way to find, for example, the location of the ideal delay time of the RF signal and the envelope signal in case you are measuring an amplifier that supports envelope tracking or to determine the characteristics and behavior of an amplifier over different frequencies and levels.

For more information about supported parameters and how to set them up see "Selecting the data to be evaluated during the Parameter Sweep".
