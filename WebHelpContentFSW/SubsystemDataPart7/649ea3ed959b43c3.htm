INPut<ip>:SELect <Source>

This command selects the signal source for measurements, i.e. it defines which connector is used to input data to the R&S FSW.

If no additional input options are installed, only RF input is supported.

If no additional input options are installed, only RF input or file input is supported.

This command only selects the RF input or file input. To select the analog baseband input, you have to use INPut:SELect:BBANalog[:STATe].

Suffix: 
<ip>
1 | 2

irrelevant

Parameters:
<Source>
RF

Radio Frequency ("RF INPUT" connector)

FIQ

I/Q data file (selected by INPut<ip>:​FILE:​PATH)

For details see "Basics on Input from I/Q Data Files".

*RST:RF
