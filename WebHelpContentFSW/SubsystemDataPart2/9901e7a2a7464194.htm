Defines the valid value range for the limit check for the selected parameter if limit check is active (CALCulate<n>:​TABLe:​<ParameterGroup>:​<Parameter>:​LIMit:​STATeON).

Commands for the parameter group <TSIDelobe> are only available if the additional option R&S FSW-K6S is installed.

For details on the individual parameters see "Pulse Parameters".

Suffix: 
<n>
1..n

Window

Parameters:
<LowerLimit>

Lower limit of the valid value range.

Default unit: S
<UpperLimit>

Upper limit of the valid value range.

Default unit: S
