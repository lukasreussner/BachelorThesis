[SENSe:]DDEMod:SEARch:SYNC:CATalog <Patterns>

This command reads the names of all patterns stored on the hard disk. The file names are returned as a comma-separated list of strings, one for each file name (without the file extension).

Setting parameters: 
<Patterns>
CURRent | ALL
CURRent

Only patterns that belong to the current standard

ALL

All patterns

*RST:ALL
Example: :DDEM:SEAR:SYNC:CAT? CURR

Result:

'GSM_AB0','GSM_AB1','GSM_AB2','GSM_TSC1'
