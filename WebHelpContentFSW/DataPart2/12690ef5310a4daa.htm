Remote Commands > Configuring the R&S FSW > Configuring the Reference Frequency > [SENSe:]ROSCillator:O640
[SENSe:]ROSCillator:O640 <State>

If enabled, a 640 MHz reference signal is provided to the REF OUTPUT 640 MHZ connector.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: ROSC:O640 ON
Manual operation: See "Reference Frequency Output"
