[SENSe:][LTE:][CC<cc>:]MODulation:SELect <Modulation>

This command filters the displayed results in the constellation diagram by a particular type of modulation.

Suffix: 
<cc>

Component Carrier

Parameters:
<Modulation>
ALL

Shows the results for all modulation types.

<numeric_value>

Shows the results for a single modulation type.

Modulation types are mapped to numeric values. For the code assignment, see "Return Value Codes".

*RST:ALL
Example: 

//Display results for all elements with a QPSK modulation

MOD:SEL 2
Manual operation: See "Evaluation range for the constellation diagram"
