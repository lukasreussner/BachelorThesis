CONFigure:CDPower:CTABle:NAME <ChannelTable>

This command selects an existing channel table or creates a new one. Use this command to edit the channel table. To use a channel table for a measurement, use the CONFigure:​CDPower:​CTABle:​SELect command.

Parameters:
<ChannelTable>

<string> = name of the channel table

*RST:RECENT
Example: CONF:CDP:CTAB:NAME 'NEW_TAB'

Selects channel table for editing. If a channel table with this name does not exist, a new channel table by that name is created.

Manual operation: See "Name"
