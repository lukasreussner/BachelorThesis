Applying a global tolerance value

Applies a global "Tolerance" value to all hop states in the table at once. By default, a tolerance of 1/2 the "Step Size" is applied when a series of states is generated.

To edit the tolerance value for an individual hop state, select the value directly in the "Hop States" table and enter the new value.

Remote command: 

CALCulate<n>:​HOPDetection:​STATes:​TABLe:​TOLerance
