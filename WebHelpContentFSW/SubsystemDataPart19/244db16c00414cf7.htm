Number of Divisions

Defines the number of divisions to be used for the x-axis or y-axis. By default, the x-axis or y-axis is divided into 10 divisions.

This command is currently only supported for Amplitude Modulation measurements.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​DIVisions
