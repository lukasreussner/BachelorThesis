I/O Logging

Activates or deactivates the SCPI error log function. All remote control commands received by the R&S FSW are recorded in the following log file:

C:\R_S\INSTR\ScpiLogging\ScpiLog.txt

Logging the commands may be extremely useful for debug purposes, e.g. in order to find misspelled keywords in control programs.

Remote command: 

SYSTem:​CLOGging
