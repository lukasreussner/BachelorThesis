Trace Settings

Access: [Trace] > "Trace Config"

You can configure the settings for up to 6 individual traces for the following result displays, which are based on the defined result range (see "Result Range"):
  • "Pulse Frequency"
  • "Pulse Magnitude"
  • "Pulse Phase"
  • "Pulse Phase (Wrapped)"

Trace data can also be exported to an ASCII file for further analysis. For details see "Trace / Data Export Configuration".
