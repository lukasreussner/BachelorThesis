Evaluation Methods for WLAN IQ Measurements

The captured I/Q data from the WLAN signal can be evaluated using various different methods without having to start a new measurement or sweep. Which results are displayed depends on the selected evaluation.

Result display windows

All evaluations available for the selected WLAN measurement are displayed in SmartGrid mode.

To activate SmartGrid mode, do one of the following:
  • Select the "SmartGrid" icon from the toolbar.
  • Select the "Display Config" button in the configuration "Overview" (see "Display Configuration").
  • Press the [MEAS CONFIG] hardkey and then select the "Display Config" softkey.

To close the SmartGrid mode and restore the previous softkey menu select the "Close" icon in the right-hand corner of the toolbar, or press any key.

For details on working with the SmartGrid, see Laying out the Result Display with the SmartGrid.

MIMO measurements

When you capture more than one data stream (MIMO measurement setup, see "Signal Processing for MIMO Measurements (IEEE 802.11ac, n)"), each result display contains several tabs. The results for each data stream are displayed in a separate tab. In addition, an overview tab is provided in which all data streams are displayed at once, in individual subwindows.

The selected evaluation method not only affects the result display in a window, but also the results of the trace data query in remote control (see TRACe[:​DATA]).
