VSA (R&S FSW-K70) > Configuration > Signal Description > Known Data > Load Data File
Load Data File

If "Known Data" is activated, this function displays a file selection dialog box to select the XML file that contains the known data. Once a file has been selected, any additional information provided by the file is displayed at the bottom of the dialog box.

Tip: An auxiliary tool to create Known Data files from data that is already available in the VSA application is provided on the instrument free of charge. See "To create a Known Data file using the recording tool for sequences".

Remote command: 

[SENSe:​]DDEMod:​KDATa[:​NAME]
