VISA Libraries

VISA is a standardized software interface library providing input and output functions to communicate with instruments. The I/O channel (LAN or TCP/IP, USB, ...) is selected at initialization time by one of the following:

  • The channel–specific address string ("VISA resource string") indicated in Table "Remote control interfaces and protocols"
  • An appropriately defined VISA alias (short name).
A VISA installation is a prerequisite for remote control using the following interfaces:
  • "GPIB Interface (IEC 625/IEEE 418 Bus Interface)"
  • "LAN Interface"
  • "USB Interface"

For more information about VISA, refer to the user documentation.
