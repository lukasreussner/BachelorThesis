Link Trend M1 to Selected Pulse

If enabled, marker M1 in Parameter Trend displays is linked to the pulse selection. Thus, if you move the marker M1 to a different pulse, the "Pulse Selection" is set to the same pulse, and vice versa.

Note that this function is only available if "Linked Markers Across Windows" is also enabled.

Remote command: 

CALCulate<n>:​MARKer<m>:​LINK:​TRENd
