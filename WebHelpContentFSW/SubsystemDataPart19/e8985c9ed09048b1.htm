Selecting the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance)

Any of the following commands can be used to return to the WLAN IQ measurement. Each of these results is automatically determined when the WLAN IQ measurement is performed.

The selected measurement must be started explicitly (see "Starting a Measurement")!

Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the Magnitude Capture buffer in window 1 at the top of the screen and the selected result type in window 2 below that. Any other active windows are closed.

Use the LAYout commands to change the display (see "Configuring the Result Display").
