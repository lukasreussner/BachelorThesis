CONFigure:CDPower[:BTS]:PVTime:LIST:RESult?

Queries the list evaluation results. The results are a comma-separated list containing the following values for each list range:

Return values: 
<RangeNo>

consecutive number of list range

<StartTime>

Start time of the individual list range

<StopTime>

Stop time of the individual list range

<AverageDBM>

Average power level in list range in dBm.

<AverageDB>

Average power level in list range in dB.

<MaxDBM>

Maximum power level in list range in dBm.

<MaxDB>

Maximum power level in list range in dB.

<MinDBM>

Minimum power level in list range in dBm.

<MinDB>

Minimum power level in list range in dB.

<LimitCheck>

Result of limit check for the list range.

0

Passed

1

Failed

<Reserved1>

0; currently not used

<Reserved2>

0; currently not used

Usage: Query only
Manual operation: See "Power vs Time (BTS application only)"
