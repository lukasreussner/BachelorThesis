Copy to User Limit Line

Creates a new user limit line from the data of a phase noise limit line.

The file is stored in the default folder for user limit lines. You can load and edit the limit line via the "Select Limit Line" dialog box. For more information see "Select Limit Line".

Remote command: 

CALCulate:​PNLimit:​COPY<k>
