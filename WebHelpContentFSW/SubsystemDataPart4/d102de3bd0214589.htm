SCPIR

This parameter is only available for AQPSK modulation.

It specifies the Subchannel Power Imbalance Ratio (SCPIR). The value of SCPIR affects the shape of the AQPSK constellation (see "AQPSK Modulation"). For an SCPIR of 0 dB the constellation is square (as in "normal" QPSK), while for other values of SCPIR the constellation becomes rectangular.

Remote command: 

CONFigure[:​MS]:​CHANnel:​SLOT<s>:​SCPir
