CALibration:MCGD:STATe?

This command queries the calibration status of the Multi Carrier Group Delay application.

Return values: 
<State>
ON | 1

Calibration has been performed, reference data is available.

OFF | 0

Calibration has not yet been performed or is currently running, reference data is not yet available.

Usage: Query only
