General Instrument Setup > Transducers > Transducer Settings > Transducer Factors > Import
Import

Opens a file selection dialog box and loads the transducer factor from the selected file in .CSV format.

Note that a valid import file must contain a minimum of required information for the R&S FSW. For details on the file format see "Reference: Transducer Factor File Format".

Remote command: 

MMEMory:​LOAD<n>:​TFACtor
