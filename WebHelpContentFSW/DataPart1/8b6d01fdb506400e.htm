[SENSe:]ESPectrum<sb>:RRANge?

This command queries the reference range.

Suffix: 
<sb>
1 | 2 | 3

Sub block in a Multi-SEM measurement

Return values: 
<RefRange>

Number of the current reference range.

Range: 1  to  30
