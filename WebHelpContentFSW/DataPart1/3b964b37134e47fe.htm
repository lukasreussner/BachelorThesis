General Instrument Setup > Transducers > Transducer Settings > Transducer Factors > Name
Name

Defines the transducer line name. All names must be compatible with the Windows  conventions for file names. The transducer data is stored under this name (with a .TDF extension) in the C:\Program Files (x86)\Rohde-Schwarz\FSW\<version>\trd directory.

Remote command: 

[SENSe:​]CORRection:​TRANsducer:​SELect
