Avionics (VOR/ILS) Measurements application (R&S FSW-15) > Remote Commands to Perform VOR/ILS Measurements > Analyzing VOR/ILS Measurements > Working with Markers > General Marker Settings > DISPlay[:WINDow<n>]:MTABle
DISPlay[:WINDow<n>]:MTABle <DisplayMode>

This command turns the marker table on and off.

Suffix: 
<n>

irrelevant

Parameters:
<DisplayMode>
ON | 1

Turns on the marker table.

OFF | 0

Turns off the marker table.

AUTO

Turns on the marker table if 3 or more markers are active.

*RST:AUTO
Example: DISP:MTAB ON

Activates the marker table.

Manual operation: See "Marker Table Display"
