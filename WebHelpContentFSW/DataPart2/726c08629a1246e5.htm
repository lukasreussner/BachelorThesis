INSTrument:CREate[:NEW] <ChannelType>, <ChannelName>

This command adds an additional channel.

The number of channels you can configure at the same time depends on available memory.

See also
  • INSTrument[:​SELect]
  • INSTrument:​CREate:​DUPLicate
Parameters:
<ChannelType>

Channel type of the new channel.

For a list of available channel types see INSTrument:​LIST?.

<ChannelName>

String containing the name of the channel. The channel name is displayed as the tab label for the channel.

Note: If the specified name for a new channel already exists, the default name, extended by a sequential number, is used for the new channel (see INSTrument:​LIST?).

Example: INST:CRE IQ, 'IQAnalyzer2'

Adds an additional I/Q Analyzer channel named "IQAnalyzer2".
