Global Sweep Mode

Selects the analysis mode for all half decades. The sweep mode defines the way the application processes the data.

For more information see "Sweep Modes".

"Normal"

Uses spectrum analyzer data for the data analysis.

"I/Q / FFT"

Uses I/Q data for the data analysis.

Remote command: 

[SENSe:​]LIST:​BWIDth[:​RESolution]:​TYPE
