DOCSIS measurements (R&S FSW-K192/-K193) > Remote Commands for DOCSIS 3.1 Measurements > Configuring the DOCSIS 3.1 I/Q Measurement (Modulation Accuracy) > Signal Description > OFDM Downstream Channel Description > CONFigure:DS:CHANnel:SPECtrum:FREQuency
CONFigure:DS:CHANnel:SPECtrum:FREQuency <Frequency>

Specifies the center frequency in Hz of the subcarrier 0 of the OFDM channel (fsc0), which defines the beginning of the OFDM spectrum.

The default value is derived from the current center frequency. If the spectrum location is changed, the center frequency is adapted accordingly (see [SENSe:​]FREQuency:​CENTer).

Parameters:
<Frequency>
Default unit: Hz
Example: CONF:DS:CHAN:SPEC:FREQ 1285000
Manual operation: See "OFDM Spectrum Location"
