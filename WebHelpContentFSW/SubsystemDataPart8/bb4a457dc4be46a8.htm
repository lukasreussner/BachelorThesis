SYSTem:COMMunicate:GPIB:RDEVice:GENerator:ADDRess <Number>

Changes the IEC/IEEE-bus address of the external generator.

This command is only valid if External Generator Control (R&S FSW-B10) is installed.

Parameters:
<Number>
Range: 0  to  30
*RST:28
Example: SYST:COMM:GPIB:RDEV:GEN:ADDR 15
Example: 

See Programming Example for External Generator Control.

Manual operation: See "GPIB Address / TCPIP Address / Computer Name"
