Oversampling

Access: "Overview" > "Display Config" > "More"

Defines the sample basis for statistical evaluation. This setting is only available for the result type transformation "Statistics".

"Symbols only"

Statistics are calculated for symbol instants only.

"Infinite"

Statistics are calculated for all trace points (symbol instants and intermediate times).

"Auto"

Oversampling is automatically set to "Symbols only" (Statistics are calculated for symbol instants only).

Remote command: 

CALCulate<n>:​STATistics:​MODE
