Spectral Overview

Displays a spectrum diagram of the "Spectral Overview" (see "Measurement Process"), meant to obtain an overview of the input signal and the required measurement settings. A continuous trace is shown for the entire measurement span.

The specified detection threshold for each range/span is indicated by a blue line in the diagram (only if it is within the displayed power region, see "Detection Threshold").

Remote command: 

LAY:ADD? '1',RIGH,SOVerview, see LAYout:​ADD[:​WINDow]?

Storing results:

MMEMory:​STORe<n>:​TRACe
