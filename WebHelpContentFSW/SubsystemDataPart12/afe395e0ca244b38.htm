CALCulate<n>:TRACe<t>[:VALue] <TrRefType>

This commands selects the signal to be used as the data source for a trace.

Suffix: 
<t>
1..6

Trace

Setting parameters: 
<TrRefType>
MEAS | REF | ERRor | TCAP
MEAS

Measurement signal

REF

Reference signal

ERR

Error

TCAP

Capture buffer

*RST:Depends on the current measurement.
Usage: SCPI confirmed
