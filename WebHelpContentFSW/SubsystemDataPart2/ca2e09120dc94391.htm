Selected Pulse vs All Pulses

Defines which pulses are included in the statistical evaluation.

For details see "Trace Statistics".

"Selected pulse"

Only the selected pulse from each measurement is included in the statistical evaluation.

"All Pulses"

All measured pulses from each measurement are included in the statistical evaluation.

Remote command: 

[SENSe:​]STATistic<n>:​TYPE
