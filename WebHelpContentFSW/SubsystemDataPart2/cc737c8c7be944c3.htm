TRACe<n>:IQ:SCAPture:BOUNdary?

This remote control command returns an array of sample indices for the start of each captured data segment. The length of the array depends on the number of trigger events specified by [SENSe:​]SWEep:​SCAPture:​EVENts.

Suffix: 
<n>
1..n

Window

Return values: 
<Data>
Example: 

See TRACe<n>:​IQ:​SCAPture:​TSTamp:​SSTart?.

Usage: Query only
Manual operation: See "Magnitude Capture"
