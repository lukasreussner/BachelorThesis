CONFigure:US:CHANnel:PCONfig<pri>:MINislot:COUNt <MinislotCount>

Defines the number of minislots for which the specified configuration applies.

Suffix: 
<pri>
1..n

index in the "Profile Configuration (Upstream)" table

Parameters:
<MinislotCount>

Number of minislots

Range: 1 to 237
*RST:10
Example: CONF:US:CHAN:PCON2:MIN:COUN 4
Manual operation: See "Number of Minislots"
