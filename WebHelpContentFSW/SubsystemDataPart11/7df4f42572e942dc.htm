Record Length

The "Measurement Time" and record length are interdependent and define the amount of data to be captured.

For details see "Data Acquisition".

The maximum record length in the R&S FSW Transient Analysis application is limited only by the available memory ("memory limit reached" message is shown in status bar). Note, however, that increasing the record length (and thus reducing the available memory space) may restrict the number of measurement channels that can be activated simultaneously on the R&S FSW.

Remote command: 

[SENSe:​]RLENgth
