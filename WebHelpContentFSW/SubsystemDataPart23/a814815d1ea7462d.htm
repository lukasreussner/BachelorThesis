Group Hopping

Turns group hopping for the demodulation reference signal on and off.

The group hopping pattern is based on 17 hopping patterns and 30 sequence shift patterns. It is generated by a pseudo-random sequence generator.

If on, PUSCH and PUCCH use the same group hopping pattern.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​DRS:​GRPHopping
