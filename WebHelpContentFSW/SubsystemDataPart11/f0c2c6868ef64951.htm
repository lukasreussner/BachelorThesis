Starting the Transient Analysis Application

The Transient Analysis application adds a new application to the R&S FSW.

To activate the Transient Analysis application

  1. Press the [MODE] key on the front panel of the R&S FSW.

    A dialog box opens that contains all operating modes and applications currently available on your R&S FSW.

  2. Select the "Transient Analysis" item.

    The R&S FSW opens a new measurement channel for the Transient Analysis application.

For details see Applications, Measurement Channels, and Operating Modes.

The measurement is started immediately with the default settings. It can be configured in the Transient "Overview" dialog box, which is displayed when you select the "Overview" softkey from any menu (see "Configuration Overview").
