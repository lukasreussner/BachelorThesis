Export

Opens a file selection dialog box and stores the currently displayed limit line to the defined file in .CSV format.

For details on the file format see "Reference: Limit Line File Format".

The limit line can be imported again later by the R&S FSW for use in other measurements.

Remote command: 

MMEMory:​STORe<n>:​LIMit
