INPut<ip>:GAIN:STATe <State>

This command turns the preamplifier on and off.

If activated, the input signal is amplified by 20 dB.

If option R&S FSW-B22 is installed, the preamplifier is only active below 7 GHz.

If option R&S FSW-B24 is installed, the preamplifier is active for all frequencies.

Suffix: 
<ip>
1 | 2

irrelevant

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

Switches the function off

ON | 1

Switches the function on

*RST:0
Example: INP:GAIN:STAT ON

Switches on 20 dB preamplification.

Manual operation: See "Preamplifier"
