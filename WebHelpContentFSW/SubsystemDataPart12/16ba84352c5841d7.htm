Measurement Filter Settings

You can configure the measurement filter to be used.

Manual configuration of the measurement filter is described in "Measurement Filter Settings".

For more information on measurement filters, refer to "Measurement Filters".

Useful commands for defining measurement filters described elsewhere:
  • [SENSe:​]DDEMod:​FILTer:​ALPHa
  • [SENSe:​]DDEMod:​FILTer[:​STATe]
