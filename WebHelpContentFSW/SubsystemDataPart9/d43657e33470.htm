FETCh:PNOise:SPURs:RANDom?

This command queries the Random Jitter result.

Return values: 
<Jitter>
Default unit: s
Example: FETC:PNO:SPUR:RAND?

would return, e.g.

3.59e-09
Usage: Query only
Manual operation: See "Spur List"
