WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Signal Capturing > MIMO Capture Settings > CONFigure:WLAN:DUTConfig
CONFigure:WLAN:DUTConfig <NoOfAnt>

This remote control command specifies the number of antennas used for MIMO measurement.

Parameters:
<NoOfAnt>
TX1 | TX2 | TX3 | TX4 | TX5 | TX6 | TX7 | TX8

TX1: one antenna,

TX2: two antennas etc.

*RST:TX1
Example: CONF:WLAN:DUTC TX1
Manual operation: See "DUT MIMO Configuration"
