[SENSe:]POWer:SEM:UL:REQuirement <Requirement>

This command selects the requirements for a spectrum emission mask.

Parameters:
<Requirement>
GEN | NS3 | NS4 | NS67
GEN

General spectrum emission mask.

NS3 | NS4 | NS67

Spectrum emission masks with additional requirements.

Example: POW:SEM:UL:REQ NS3

Selects a spectrum emission mask with requirement for network signalled value NS3.

Manual operation: See "SEM Requirement"
