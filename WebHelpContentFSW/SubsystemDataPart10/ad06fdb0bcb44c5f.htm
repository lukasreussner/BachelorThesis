Configuring the Y-Axis Scaling

The scaling for the vertical axis is highly configurable, using either absolute or relative values. These commands are described here.

List of Commands
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:AUTO
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:MAXimum
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:MINimum
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:PDIVision
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:RPOSition
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:RVALue
