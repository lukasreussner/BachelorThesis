OUTPut:IF:IFFRequency <Frequency>

This command defines the frequency for the IF output of the R&S FSW. The IF frequency of the signal is converted accordingly.

This command is available in the time domain and if the IF/VIDEO/DEMOD output is configured for IF.

If the [IF WIDE OUTPUT] connector is used (TRACe:IQ:WBANd ON, see TRACe:​IQ:​WBANd[:​STATe]), this command is available as a query only. It returns the used IF output frequency which is defined automatically by the application according to the center frequency.

For more information see "IF and Video Signal Output".

Parameters:
<Frequency>
*RST:50.0 MHz
Manual operation: See "Data Output"
