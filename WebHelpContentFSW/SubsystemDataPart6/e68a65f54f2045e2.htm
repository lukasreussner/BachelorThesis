[SENSe:]ADEMod:MCPHase:SPACing <CarrierSpacing>

Sets/queries the carrier spacing in Hz between the multiple carriers.

Note that this command is maintained for compatibility reasons only. Use [SENSe:​]CARRier:​SPACing for new remote control programs.

Parameters:
<CarrierSpacing>
Default unit: HZ
