Results to check modulation accuracy

Raw EVM

Error vector magnitude between synchronized reference and measured signal.

FETCh:​MACCuracy:​REVM:​CURRent[:​RESult]?

Raw Model EVM

Error vector magnitude between synchronized measured and model signal.

FETCh:​MACCuracy:​RMEV:​CURRent[:​RESult]?

Frequency Error

Difference of the RF frequency of the reference signal compared to the measured signal.

If the offset is very high, it is likely that the reference frequency sources are not coupled correctly, e.g. if the analyzer is configured for external reference frequency, but the cable is not connected.

FETCh:​MACCuracy:​FERRor:​CURRent[:​RESult]?

Sample Rate Error

Sample rate difference between reference and measured signal.

FETCh:​MACCuracy:​SRERror:​CURRent[:​RESult]?

Magnitude Error

Difference in magnitude between the reference signal and the measured signal.

FETCh:​MACCuracy:​MERRor:​CURRent[:​RESult]?

Phase Error

Phase difference between reference and measured signal.

FETCh:​MACCuracy:​PERRor:​CURRent[:​RESult]?

Quadrature Error

Phase deviation of the 90° phase difference between the real (I) and imaginary (Q) part of the signal.

Within an ideal transmitter, the I and Q signal parts are mixed with an angle of 90° by the I/Q output mixer. Due to hardware imperfections, the signal delay of I and Q may be different and thus lead to an angle non-equal to 90°.

FETCh:​MACCuracy:​QERRor:​CURRent[:​RESult]?

Gain Imbalance

Gain difference between the real (I) and imaginary (Q) part of the signal.

This effect is typically generated by two separate amplifiers with a different gain in the I and Q path of the analog baseband signal generation.

FETCh:​MACCuracy:​GIMBalance:​CURRent[:​RESult]?

I/Q Imbalance

Combination of Quadrature error and Gain imbalance.

The I/Q imbalance parameter is a representation of the combination of Quadrature error and gain imbalance.

FETCh:​MACCuracy:​IQIMbalance:​CURRent[:​RESult]?

I/Q Offset

Shift of the measured signal compared to the ideal I/Q constellation in the I/Q plane.
