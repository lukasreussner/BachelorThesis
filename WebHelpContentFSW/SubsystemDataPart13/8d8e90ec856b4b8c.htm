Frequency Error vs Slot

For each value to be displayed, the difference between the frequency error of the corresponding slot to the frequency error of the first (zero) slot is calculated (based on CPICH slots). This helps eliminate a static frequency offset of the whole signal to achieve a better display of the actual time-dependant frequency diagram.

Remote command: 

LAY:ADD? '1',RIGH, FESLot, see LAYout:​ADD[:​WINDow]?

TRACe<n>[:DATA]? ATRACE
