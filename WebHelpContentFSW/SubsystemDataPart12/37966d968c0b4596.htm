Adding patterns to a standard

Adds the selected patterns in the list of available patterns ("All Patterns") to the list of assigned patterns ("Standard Patterns").

For details see "To add a predefined pattern to a standard".

Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​PATTern:​ADD
