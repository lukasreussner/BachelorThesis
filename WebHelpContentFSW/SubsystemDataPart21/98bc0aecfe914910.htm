CONFigure:DS:CHANnel:FCONfig<cwi>:SYMBol:COUNt <SCStart>

Defines the number of symbols to which the selected codeword is assigned.

Suffix: 
<cwi>
1..n

codeword index in the "Codeword / Frame Configuration" table

Parameters:
<SCStart>
integer
Range: 1  to  4
Example: CONF:DS:CHAN:FCON2:SYMB:COUN 3
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Manual operation: See "Number of Symbols"
