TD-SCDMA Measurements (R&S FSW-K76/-K77) > Remote Commands for TD-SCDMA Measurements > Activating the TD-SCDMA Applications > INSTrument[:SELect]
INSTrument[:SELect] <Mode>

This command activates a new measurement channel with the defined channel type, or selects an existing measurement channel with the specified name.

Also see

  • INSTrument:​CREate[:​NEW]
  • Programming Example: Performing a Sequence of Measurements
Parameters:
<Mode>
BTDS

TD-SCDMA BTS mode (R&S FSW-K76 option)

MTDS

TD-SCDMA UE mode (R&S FSW-K77 option)
