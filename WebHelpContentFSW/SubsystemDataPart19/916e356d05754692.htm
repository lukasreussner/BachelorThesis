Reference Status

LEDs indicate the connection state of the device providing the individual reference frequency for each channel (for "Reference Frequency Coupling").

Meaning of LED colors

Color

State

gray

external reference off or IP address not available/valid

red

external reference on and IP address valid, but no external reference detected

green

external reference on and IP address accessible
