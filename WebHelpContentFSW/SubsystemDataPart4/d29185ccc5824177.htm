CONFigure:SPECtrum:MODulation:REFerence:PLEVel <Level>

This command defines the reference power level for MCWN measurements (if no reference measurement is performed, see CONFigure:​SPECtrum:​MODulation:​REFerence:​MEASure).

Parameters:
<Level>

power level in dBm

*RST:0.00
Example: CONF:SPEC:MOD:REF:MEAS OFF
CONF:SPEC:MOD:REF:PLEV 35
Manual operation: See "Power Level"
