CALCulate<n>:FSK:DEViation:REFerence:RELative <FSKRefDev>

This command defines the deviation to the reference frequency for FSK modulation as a multiple of the symbol rate.

Setting parameters: 
<FSKRefDev>
numeric value
Range: 0.1  to  60
*RST:1
Default unit: None
Manual operation: See "FSK Ref Deviation (FSK only)"
