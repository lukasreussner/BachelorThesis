CALCulate<n>:SGRam:FRAMe:COUNt?
CALCulate<n>:SPECtrogram:FRAMe:COUNt?

This command queries the number of frames that are contained in the selected result display (depends on the evaluation basis).

Suffix: 
<n>
1..n

Window

Return values: 
<Frames>

The maximum number of frames depends on the history depth.

Range: 1  to  history depth
Increment: 1
Example: INIT:CONT OFF

Selects single sweep mode.

LAY:REPL 2,SGR

Replaces the result display in window 2 by a spectrogram.

DISP:WIND2:EVAL REG

Defines the analysis region as the evaluation basis for the spectrogram in window 2.

CALC:SGR:FRAM:COUN?

Queries the number of frames in the spectrogram based on the analysis region.

Usage: Query only
