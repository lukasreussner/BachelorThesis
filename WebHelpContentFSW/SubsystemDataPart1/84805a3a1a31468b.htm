Importing and Exporting I/Q Data and Results

Alternatively to capturing I/Q data by the I/Q Analyzer itself, stored I/Q data from previous measurements or other applications can be imported to the I/Q Analyzer. Furthermore, I/Q data processed in the I/Q Analyzer can be stored to a file for further evaluation in other applications.

I/Q data can only be exported in applications that process I/Q data, such as the I/Q Analyzer or optional applications.

For details on importing and exporting I/Q data see "I/Q Data Import and Export".
