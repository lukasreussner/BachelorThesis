[SENSe:]CDPower:SET <SetNo>

This command selects a specific set for further analysis. The number of sets to capture has to be defined with the [SENSe:​]CDPower:​SET command before using this command.

Parameters:
<SetNo>
Range: 0  to  SET COUNT -1
Increment: 1
*RST:0
Example: CDP:SET:COUN 10

Selects the 10th set for further analysis.

Manual operation: See "Set to Analyze"
