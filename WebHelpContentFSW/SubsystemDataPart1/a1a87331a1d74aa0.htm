[SENSe:]MIXer<x>:FREQuency:STOP <Freq>

This command sets or queries the frequency at which the external mixer band stops.

Suffix: 
<x>

irrelevant

Parameters:
<Freq>
numeric value
Example: MIX:FREQ:STOP?

Queries the stop frequency of the band.

Example: 

See "Programming Example: Working with an External Mixer" and "Configuring a conversion loss table for a user-defined band".

Manual operation: See "RF Start / RF Stop"
