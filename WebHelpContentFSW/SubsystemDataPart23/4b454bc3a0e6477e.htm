CONFigure[:LTE]:UL[:CC<cc>]:SUBFrame<sf>:ALLoc:PRECoding:CBINdex <CBIndex>

This command selects the codebook index for a PUSCH allocation.

Suffix: 
<cc>

Component Carrier

<sf>

Subframe

Parameters:
<CBIndex>
Range: 0  to  5
*RST:0
Example: CONF:UL:SUBF:ALL:PREC:CBIN 1

Selects codebook index 1 for the PUSCH allocation.

Manual operation: See "Enhanced PUSCH Configuration"
