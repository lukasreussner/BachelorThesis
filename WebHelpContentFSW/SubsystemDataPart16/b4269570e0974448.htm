Automatic Settings
MSRA operating mode

In MSRA operating mode, the following automatic commands are not available, as they require a new data acquisition. However, 1xEV-DO applications cannot perform data acquisition in MSRA operating mode.

Useful commands for adjusting settings automatically described elsewhere:
  • DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO ONCE
