CALCulate<n>:SGRam:HDEPth <Depth>

This command defines the number of frames to be stored in the R&S FSW memory.

Suffix: 
<n>
1..n

irrelevant

Parameters:
<Depth>
Example: CALC:SGR:SPEC 1500

Sets the history depth to 1500.

Manual operation: See "History Depth"
