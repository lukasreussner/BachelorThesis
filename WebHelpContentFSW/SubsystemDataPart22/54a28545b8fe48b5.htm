LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Input / Frontend > Configuring the Input > INPut:DIQ:RANGe[:UPPer]:UNIT
INPut:DIQ:RANGe[:UPPer]:UNIT <Unit>

Defines the unit of the full scale level. The availability of units depends on the measurement application you are using.

This command is only available if the optional Digital Baseband Interface is installed.

Parameters:
<Level>
VOLT | DBM | DBPW | WATT | DBMV | DBUV | DBUA | AMPere
*RST:Volt
