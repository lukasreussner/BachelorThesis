Defining the Result Range

The result range determines which part of the capture buffer, burst or pattern is displayed.

Manual configuration of the result range is described in "Result Range Configuration".

Useful commands for result ranges described elsewhere:
  • [SENSe:​]DDEMod:​SEARch:​MBURst:​STARt[:​SYMBols]?
  • DISPlay[:​WINDow<n>]:​TRACe<t>:​X[:​SCALe]:​STOP?
