Channel Power Configuration

Access: "Overview" > "Select Measurement" > "Channel Power ACLR" > "CP / ACLR Config"

Both Channel Power (CP) and Adjacent-Channel Power (ACLR) measurements are available.

If the "Multi-Standard Radio" standard is selected (see "Standard"), the "ACLR Setup" dialog box is replaced by the "MSR ACLR Setup" dialog box. See "MSR ACLR Configuration" for a description of these settings.

The remote commands required to perform these tasks are described in "Measuring the Channel Power and ACLR".
