Marker Table Display

Defines how the marker information is displayed.

For more details, see Marker Results.

"On"

Displays the marker information in a table in a separate area beneath the diagram.

"Off"

No separate marker table is displayed.

The marker information is displayed within the diagram area.

"Auto"

(Default) If more than two markers are active, the marker table is displayed automatically.

The marker information for up to two markers is displayed in the diagram area.

Remote command: 

DISPlay[:​WINDow<n>]:​MTABle
