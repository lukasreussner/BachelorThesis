CONFigure:MODeling:AMPM:ORDer <Order>

This command defines the order (or degree) of the AM/PM model polynomials that are calculated by the application.

Parameters:
<Order>

String containing the polynomials to be calculated.

You can either select a range of polynomials (e.g. "1-7"), a selection of polynomials (e.g. "1,3,5") or a combination of both (e.g. "1,3-5").

Range: 0  to  18
*RST:"1-7"
Example: CONF:MOD:AMPM:ORD "1,3-5"

Calculates the polynomials to the 1st, 3rd, 4th and 5th degree.

Manual operation: See "Selecting the degree of the polynomial"
