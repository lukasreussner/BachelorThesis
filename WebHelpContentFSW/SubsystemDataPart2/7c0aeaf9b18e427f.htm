Configuring Limit Checks

For each parameter in the result tables you can activate a limit check and define the valid value ranges. For details see "Pulse Results".

Useful commands for configuring limit checks described elsewhere:
  • CALCulate<n>:​DISTribution:​LLINes[:​STATe]
  • CALCulate<n>:​TRENd:​LLINes[:​STATe]

For commands required to retrieve the results of the limit check for individual parameters see "Retrieving Limit Results".
