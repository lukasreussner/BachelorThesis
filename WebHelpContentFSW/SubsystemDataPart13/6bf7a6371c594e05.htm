Code Power Display

For Code Domain Power evaluation:

Defines whether the absolute power or the power relative to the chosen reference is displayed.

Note: In UE measurements, relative powers are always in relation to the total signal power.

"TOT"

Relative to the total signal power

"CPICH"

Relative to the power of the CPICH

[SENSe:​]CDPower:​PDISplay
