Frame Count

Determines how many frames are plotted during a single sweep (as opposed to a continuous sweep). The maximum number of possible frames depends on the history depth (see "History Depth").

For more details see Time Frames.

Remote command: 

CALCulate<n>:​SPECtrogram:​FRAMe:​COUNt
