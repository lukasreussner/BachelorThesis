Returns the statistical value for the fall time over the specified pulses.

Query parameters: 
<QueryRange>
CURRent | ALL
CURRent

Detected pulses in the current capture buffer

ALL

All detected pulses in the entire measurement.

Return values: 
<Result>
<char_data>
Usage: Query only
