CONFigure:REFSignal:SEGMent <Segment>

This command selects the segment of the reference signal that should be used in the measurement when the reference signal is based on a multi segment waveform file.

Parameters:
<Segment>
<numeric value>: (integer only)
Range: Depends on the number of segments in the waveform file.
*RST:0
Example: CONF:REFS:SEGM 3

Selects the 3rd segment in the waveform file.

Manual operation: See "Using multi-segment waveform files"
