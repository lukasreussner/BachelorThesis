Optimizing and Troubleshooting the Measurement

If the results do not meet your expectations, try the following methods to optimize the measurement:

Synchronization fails:
  • Check the center frequency.
  • Perform an automatic reference level adjustment.
  • In BTS mode:
    When using an external trigger, check whether an external trigger signal is being sent to the R&S FSW and check the "PN offset".
  • In MS mode, check the "Long Code Mask" and "Long Code Offset".
  • Make sure "Invert Q" is off.
