[SENSe:]DDEMod:PATTern:FRAMe:EDIT:TEXT <Description>

Defines the description for the frame structure in a previously loaded file.

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Note that the file must be loaded for editing before the description can be defined using this command (see [SENSe:​]DDEMod:​PATTern:​FRAMe:​EDIT).

Parameters for setting and query: 
<Description>
string
Example: SENS:DDEM:PATT:FRAM:MODE USER
SENS:DDEM:PATT:FRAM:EDIT 'D:\MyFrameStructure.xml'
SENSe:DDEMod:PATTern:FRAMe:EDIT:TEXT 'description for the frame structure'
Manual operation: See "Description"
