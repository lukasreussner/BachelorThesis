CALCulate<n>:LIMit<k>:ESPectrum<sb>:PCLass:COUNt <NoPowerClasses>

This command sets the number of power classes to be defined.

This command must be executed before any new power class values can be defined using CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>:​MAXimum and CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>:​MINimum.

Suffix: 
<n>, <k>

irrelevant

<sb>
1 | 2 | 3

Sub block in a Multi-SEM measurement

Parameters:
<NoPowerClasses>
1 to 4
*RST:1
Example: CALC:LIM:ESP:PCL:COUN 2

Two power classes can be defined.
