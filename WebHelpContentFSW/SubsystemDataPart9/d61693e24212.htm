Marker 1 ... Marker x

Selects and turns the corresponding marker on and off.

Turning on a marker also opens an input field to define the horizontal position of the marker.

In the "Marker Configuration" dialog box, you can also define the horizontal position of each marker ("x-value").

By default, the first marker you turn on is a normal marker, all others are delta markers.

Remote command: 

CALCulate<n>:​MARKer<m>[:​STATe]

CALCulate<n>:​DELTamarker<m>[:​STATe]
