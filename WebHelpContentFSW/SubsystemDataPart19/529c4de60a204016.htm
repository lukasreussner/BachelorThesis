Frequency Sweep Measurements

When you activate a measurement channel in WLAN mode, an IQ measurement of the input signal is started automatically (see "WLAN I/Q Measurement (Modulation Accuracy, Flatness and Tolerance)"). However, some parameters specified in the WLAN 802.11 standard require a better signal-to-noise level or a smaller bandwidth filter than the default measurement on I/Q data provides and must be determined in separate measurements based on RF data (see "Frequency Sweep Measurements"). In these measurements, demodulation is not performed.

Selecting the measurement type

WLAN measurements require a special operating mode on the R&S FSW, which you activate using the [MODE] key.

For details see Applications, Measurement Channels, and Operating Modes.

► 

To select a frequency sweep measurement type, do one of the following:

  • Select the "Overview" softkey. In the "Overview", select the "Select Measurement" button. Select the required measurement.
  • Press the [MEAS] key. In the "Select Measurement" dialog box, select the required measurement.

The R&S FSW WLAN application uses the functionality of the R&S FSW base system (Spectrum application) to perform the WLAN frequency sweep measurements. Some parameters are set automatically according to the WLAN 802.11 standard the first time a measurement is selected (since the last [PRESET] operation). These parameters can be changed, but are not reset automatically the next time you re-enter the measurement. Refer to the description of each measurement type for details.

The main measurement configuration menus for the WLAN frequency sweep measurements are identical to the Spectrum application.

For details refer to Common Measurement Settings.
