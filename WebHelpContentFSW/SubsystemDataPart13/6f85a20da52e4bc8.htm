EVM vs Chip

When the trace data for this evaluation is queried, a list of vector error values of all chips at the selected slot is returned (=2560 values). The values are calculated as the square root of the square difference between the received signal and the reference signal for each chip, normalized to the square root of the average power at the selected slot.
