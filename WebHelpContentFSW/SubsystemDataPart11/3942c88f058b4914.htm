[SENSe:]RLENgth <SampleCount>

This command defines the record length (in samples) for the current measurement. Note that the record length and the measurement time are interdependent (see [SENSe:​]MTIMe).

Parameters:
<SampleCount>

The maximum record length is limited only by the available memory.

*RST:140000
