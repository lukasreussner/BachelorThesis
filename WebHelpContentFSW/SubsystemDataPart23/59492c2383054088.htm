LTE Uplink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Signal Capture > Data Capture > [SENSe:][LTE:]FRAMe:COUNt:AUTO
[SENSe:][LTE:]FRAMe:COUNt:AUTO <State>

This command turns automatic selection of the number of frames to analyze on and off.

Parameters:
<State>
ON

Selects the analyzed number of frames according to the LTE standard.

OFF

Turns on manual selection of the number of frames.

Example: 

//Turn on automatic selection of analyzed frames

FRAM:COUN:AUTO ON
Manual operation: See "Auto According to Standard"
