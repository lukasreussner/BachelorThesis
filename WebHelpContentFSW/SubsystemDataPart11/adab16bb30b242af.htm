Display Configuration

Access: "Overview" > "Display Config"

The captured signal can be displayed using various evaluation methods. All evaluation methods available for the Transient Analysis application are displayed in the evaluation bar in SmartGrid mode.

Up to six evaluation methods can be displayed simultaneously in separate windows. The evaluation methods available for Transient Analysis are described in "Evaluation Methods for Transient Analysis".

For details on working with the SmartGrid see Laying out the Result Display with the SmartGrid.
