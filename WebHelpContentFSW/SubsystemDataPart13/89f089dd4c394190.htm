Automatic Settings
MSRA operating mode

In MSRA operating mode, the following commands are not available, as they require a new data acquisition. However, 3GPP FDD applications cannot perform data acquisition in MSRA operating mode.

Useful commands for adjusting settings automatically described elsewhere:
  • DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO ONCE
  • [SENSe:​]CDPower:​LCODe:​SEARch[:​IMMediate]?
