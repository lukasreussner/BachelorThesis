TRACe<n>[:DATA]? <Trace>

This command queries current trace data and measurement results for the specified window. For each of the measurement points (number is determined by record length, see [SENSe:​]RLENgth?) the power level, phase or group delay value is returned.

Suffix: 
<n>
1..n
Query parameters: 
<Trace>
TRACe1 | TRACe2 | TRACe3 | TRACe4

Defines the trace whose data is returned.

Example: 

TRAC3:DATA? TRACE1

Returns the measured values for trace 1 in window 3.

Usage: Query only
