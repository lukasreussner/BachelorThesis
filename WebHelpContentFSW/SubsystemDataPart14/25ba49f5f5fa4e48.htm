CALCulate<n>:MARKer<m>:MINimum[:PEAK]

This command moves a marker to the minimum level.

If the marker is not yet active, the command first activates the marker.

Suffix: 
<n>

Window

<m>

Marker

Usage: Event
Manual operation: See "Search Minimum"
