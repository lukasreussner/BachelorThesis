Connecting a PC via the GPIB Interface

You can connect a PC to the R&S FSW via the GPIB interface to send remote commands to control and operate the instrument. You can configure the GPIB address and the ID response string. The GPIB language is set as SCPI by default but can be changed to emulate other instruments.

A GPIB interface is integrated on the rear panel of the instrument.
