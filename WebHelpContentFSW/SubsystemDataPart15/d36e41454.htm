[SENSe:]CDPower:LCODe:MASK <Mask>

Defines the long code mask of the mobile in hexadecimal form.

Note: For the default mask value of 0 the long code offset (see ) is not taken into consideration.

Parameters:
<Mask>
Range: #H0  to  #H4FFFFFFFFFF
*RST:#H0
Example: INST:SEL MC2K

'Activate cdma2000 MS; by default, "CDP relative" is displayed in screen A and "Result Summary" in screen B.

INIT:CONT OFF

'Select single sweep

TRIG:SOUR:EXT

'Select external trigger source

CDP:LCOD:MASK '#HF'

'Define long code mask

INIT;*WAI

'Start measurement with synchronization

Manual operation: See "Long Code Mask"
