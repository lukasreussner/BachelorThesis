CALCulate<n>:FLINe<k> <Frequency>

This command defines the position of a frequency line.

Suffix: 
<n>

Window

<k>

Limit line

Parameters:
<Frequency>

Note that you can not set a frequency line to a position that is outside the current span.

Range: 0 Hz  to  Fmax
*RST:(STATe to OFF)
Example: CALC:FLIN2 120MHz

Sets frequency line 2 to a frequency of 120 MHz.

Manual operation: See "Vertical Line 1/ Vertical Line 2"
