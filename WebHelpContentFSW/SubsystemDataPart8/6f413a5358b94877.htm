Noise Figure Measurements (R&S FSW-K30) > Configuration
Configuration

Noise figure measurements require a special application on the R&S FSW, which you activate using the [MODE] key.

For details, see Applications, Measurement Channels, and Operating Modes.

The [Noise Source Control] connector on the R&S FSW is also a prerequisite for the R&S FSW Noise measurements application. Without this connector, no measurement can be performed.

When you activate a measurement channel in the R&S FSW Noise measurements application, a measurement for the input signal is defined with the default configuration. The "Noise Figure" menu is displayed and provides access to the most important configuration functions.

For a description, see
  • "Configuring the Noise Source"
  • "Configuring Additional Loss"
  • "Configuring the Analyzer"
  • "Using the Uncertainty Calculator"
Unavailable hardkeys

Note that the [AMPT], [AUTO SET], [BW], [TRIG] and [MKR FUNC] keys have no contents and no function in the R&S FSW Noise measurements application.
