VSA (R&S FSW-K70) > Remote Commands for VSA > Configuring VSA > Signal Description > Modulation > [SENSe:]DDEMod:USER:NAME
[SENSe:]DDEMod:USER:NAME <Name>

Selects the file that contains the user-defined modulation to be loaded.

Setting parameters: 
<Name>
string

Path and file name of the *.vam file

Example: SENS:DDEM:FORM UQAM

Define the use of a user-defined modulation

SENS:DDEM:USER:NAME 'D:\MyModulation.vam'

Select the file name to be loaded

Manual operation: See "Load User Modulation"
