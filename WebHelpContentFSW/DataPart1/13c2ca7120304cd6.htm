[SENSe:]POWer:ACHannel:SBLock<sb>:BANDwidth[:CHANnel<ch>] <Bandwidth>

This command defines the bandwidth of the specified MSR Tx channel.

This command is for MSR signals only (see CALCulate<n>:​MARKer<m>:​FUNCtion:​POWer:​PRESet).

For details on MSR signals see "Measurement on Multi-Standard Radio (MSR) Signals".

Suffix: 
<sb>
1 | 2 | 3 | 4 | 5

sub block number

<ch>
1 to 18

Tx channel number

Parameters:
<Bandwidth>

Bandwidth in Hz

Example: 

See "Example: Configuring and Performing an MSR ACLR Measurement".

Manual operation: See "Tx Channel Bandwidth"
