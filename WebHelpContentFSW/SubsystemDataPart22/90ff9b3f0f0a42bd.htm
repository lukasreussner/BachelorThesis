Area ID

Defines the ID for an MBFSN area.

Radio cells that transmit the same content to multiple users will form a so called MBSFN area. Multiple cells can belong to such an area, and every cell can be part of up to eight MBSFN areas. There could be up to 256 different MBSFN areas defined, each one with an own identity.

The area ID (NIDMBFSN) is defined in 3GPP 36.211.

Remote command: 

CONFigure[:​LTE]:​DL[:​CC<cc>]:​MBSFn:​AI:​ID
