Gap Channel Bandwidths

The gap channel bandwidth is normally predefined by the transmission technology standard. The correct bandwidth is set automatically for the selected technology. The bandwidth for each channel is indicated by a colored bar in the display (if the gap is not too narrow, see "Channel display for MSR signals").

Remote command: 

[SENSe:​]POWer:​ACHannel:​BANDwidth:​GAP<gap>
