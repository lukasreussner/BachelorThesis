Peak Excursion

Defines the minimum level value by which the signal must rise or fall after a detected spur so that a new spur is detected.

Note: If noise peaks are detected as potential spurs, and in particular, if the noise level is displayed above the detection threshold after spurious detection, increase the peak excursion. Noise peaks are removed by the spot search, but the additional process slows down the measurement.

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​PEXCursion
