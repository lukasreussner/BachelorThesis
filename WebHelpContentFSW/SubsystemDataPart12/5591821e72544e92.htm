Constellation Frequency

The instantaneous frequency of the source signal (without inter-symbol interference) as an X/Y plot; only the symbol decision instants are drawn and not connected.

Available for source types:
  • Meas & Ref Signal
Result display "Constellation Frequency"
Remote commands:

LAY:ADD? '1',BEL,MEAS

to define the required source type (see LAYout:​ADD[:​WINDow]?)

CALC:FORM CONF

to define the result type (see CALCulate<n>:​FORMat)

TRAC:DATA? TRACE1

to query the trace results (see TRACe<n>[:DATA]? TRACE<n> and "Polar Diagrams")
