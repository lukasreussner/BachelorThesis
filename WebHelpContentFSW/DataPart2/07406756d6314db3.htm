DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe] <Range>

This command defines the display range of the y-axis (for all traces).

Note that the command works only for a logarithmic scaling. You can select the scaling with DISPlay[:​WINDow<n>]:​TRACe<t>:​Y:​SPACing.

Suffix: 
<n>

Window

<t>

irrelevant

Parameters:
<Range>
Range: 1 dB  to  200 dB
*RST:100 dB
Example: DISP:TRAC:Y 110dB
Example: 

See "Programming Example: Performing a Basic Frequency Sweep".

Manual operation: See "Range"
