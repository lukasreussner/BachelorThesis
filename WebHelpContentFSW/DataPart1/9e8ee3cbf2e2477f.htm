MMEMory:STORe<n>:LIMit <FileName>,<LimitLineName>

This command exports limit line data to an ASCII (CSV) file.

For details on the file format see "Reference: Limit Line File Format".

Suffix: 
<n>

irrelevant

Parameters:
<FileName>

String containing the path and name of the target file.

<LimitLineName>

Name of the limit line to be exported.

Example: MMEM:STOR:LIM 'C:\TEST', 'UpperLimitLine'

Stores the limit line named "UpperLimitLine" in the file TEST.CSV.

Manual operation: See "Export"
