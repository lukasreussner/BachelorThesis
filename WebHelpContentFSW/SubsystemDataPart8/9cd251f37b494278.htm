Result Table

Shows the measurement results in numerical form in a table.

The contents of the table depend on the "Display Settings". By default it shows the noise figure, gain and y-factor results. Each row represents one measurement point. Each column represents one result type. The first column shows the measurement frequency.

If you display the uncertainty result, it is displayed in the "Noise" column next to the Noise Figure result. Note that the uncertainty is displayed only after you have turned on the uncertainty calculation and also include the result in the display.

The result table shows either the RF or the IF, depending on your selection.

For more information, see "Configuring Numerical Results".
