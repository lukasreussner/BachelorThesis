Configuring Data Acquisition

The following remote commands are required to configure which data is to be acquired and then demodulated in a remote environment.

MSRA/MSRT operating mode

In MSRA/MSRT operating mode, only the MSRA/MSRT Master channel actually captures data from the input signal. The data acquisition commands for the Analog Demodulation application in MSRA/MSRT mode define the analysis interval.

For details on the MSRA operating mode see Applications and Operating Modes. For details on the MSRT operating mode see Applications and Operating Modes.
