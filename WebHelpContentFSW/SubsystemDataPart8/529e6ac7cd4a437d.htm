[SENSe:]CONFigure:MODE:DUT <DUTType>

This command selects the type of DUT you are testing.

Note that you have to use [SENSe:]CONFigure:MODE:SYSTem:LO to select if the LO or IF are fixed.

Parameters:
<DUTType>
AMPLifier

Measurements on fixed frequency DUTs.

DOWNconv

Measurements on down-converting DUTs.

UPConv

Measurements on up-converting DUTs.

*RST:AMPLifier
Example: CONF:MODE:DUT DOWN

Selects the measurement mode for a down-converting DUT.

Manual operation: See "Mode"
