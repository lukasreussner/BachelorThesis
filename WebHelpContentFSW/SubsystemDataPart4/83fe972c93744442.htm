CONFigure[:MS]:POWer:PCARrier <Power>

Defines the maximum output power per carrier, which determines the limit lines for the modulation spectrum (I/Q measurements) and MCWN measurement.

This value is ignored if CONFigure[:​MS]:​POWer:​PCARrier:​AUTO is ON.

Parameters:
<Power>

maximum output power in dBm

*RST:0 dBm
Example: CONF:POW:PCAR:AUTO OFF
CONF:POW:PCAR 4 dBm
Manual operation: See "Maximum Output Power per Carrier (multicarrier measurements only)"
