CONFigure:WCDPower[:BTS]:CTABle:NAME <Name>

This command creates a new channel table file or selects an existing channel table in order to copy or delete it.

Parameters:
<Name>
<file name>
*RST:RECENT
Example: 

CONF:WCDP:CTAB:NAME 'NEW_TAB'

Mode: BTS application only
Manual operation: See "Name"
