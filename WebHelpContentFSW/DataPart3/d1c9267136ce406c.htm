Y-Axis

Describes the vertical axis on which the data points of the limit line are defined. Includes the following settings:

  • Level unit
  • Scaling mode: absolute or relative (dB/%) values
    Relative limit values refer to the reference level.
  • Limit type: upper or lower limit; values must stay above the lower limit and below the upper limit to pass the limit check
Remote command: 

CALCulate<n>:​LIMit<k>:​UNIT

CALCulate<n>:​LIMit<k>:​LOWer:​SPACing

CALCulate<n>:​LIMit<k>:​UPPer:​SPACing
