INPut:DIQ:SRATe:AUTO <State>

If enabled, the sample rate of the digital I/Q input signal is set automatically by the connected device.

This command is only available if the optional Digital Baseband Interface is installed.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
