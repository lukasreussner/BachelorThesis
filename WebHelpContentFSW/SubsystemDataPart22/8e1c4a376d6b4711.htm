Spatial Multiplexing

Turns on precoding for spatial multiplexing according to 3GPP TS 36.211.

If you are using spatial multiplexing, you can also define the number of layers for any allocation and the codebook index.

The number of layers of an allocation in combination with the number of code words determines the layer mapping. The available number of layers depends on the number of transmission antennas. Thus, the maximum number of layers you can select is eight.

The codebook index determines the precoding matrix. The available number of indices depends on the number of transmission antennas in use. The range is from 0 to 15. The application automatically selects the codebook index if you turn on the "Cyclic Delay Diversity" (CDD).
