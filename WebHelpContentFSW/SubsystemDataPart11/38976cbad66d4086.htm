INITiate<n>:REFResh

This function is only available if the Sequencer is deactivated (SYST:SEQ:OFF) and only for slave applications in MSRA/MSRT mode, not the MSRA/MSRT Master.

The data in the capture buffer is re-evaluated by the currently active slave application only. The results for any other slave applications remain unchanged.

Suffix: 
<n>

irrelevant

Example: SYST:SEQ:OFF

Deactivates the scheduler

INIT:CONT OFF

Switches to single sweep mode.

INIT;*WAI

Starts a new data measurement and waits for the end of the sweep.

INST:SEL 'IQ ANALYZER'

Selects the IQ Analyzer channel.

INIT:REFR

Refreshes the display for the I/Q Analyzer channel.

Usage: Event
Manual operation: See "Refresh (MSRA/MSRT only)"
