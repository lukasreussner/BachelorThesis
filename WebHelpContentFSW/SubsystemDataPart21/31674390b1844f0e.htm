Band Configuration Table

If you set "Band Auto Config" to "User-defined", you can configure how bands are detected manually.

You can define the offsets and bandwidths for the bands (in relation to the detected miniblock sets) as a number of subcarriers or a frequency range.

Remote command: 

CONFigure:​US:​CHANnel:​SBPower:​CONFigure:​OFFSet:​CHANnel<ch>:​SCARrier

CONFigure:​US:​CHANnel:​SBPower:​CONFigure:​OFFSet:​CHANnel<ch>[:​FREQuency]
