Pulse Measurements

The Pulse slave application requires an instrument equipped with the Pulse Measurements option, R&S FSW-K6. This slave application provides measurement functions for pulsed signals.

For details refer to Welcome to the Pulse Measurements Application.

Remote command: 

INST:SEL PULSE, see INSTrument[:​SELect]
