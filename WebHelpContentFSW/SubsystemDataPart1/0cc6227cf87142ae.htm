Querying the Status Registers

The R&S FSW-I/Q Analyzer uses the standard status registers of the R&S FSW.

The following status registers of the R&S FSW status reporting system are used by the Digital Baseband Interface (R&S FSW-B17). The commands required to query the status registers specific to the Digital Baseband Interface (R&S FSW-B17) are described with the registers.

For details and a description of the common R&S FSW status registers see Status Reporting System.

*RST does not influence the status registers.
