Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Pulse Detection > [SENSe:]DETect:LIMit
[SENSe:]DETect:LIMit <State>

If enabled, the number of pulses to be detected is restricted. When the maximum number is exceeded, measurement is stopped for the current capture buffer. This limitation can be used to speed up the measurement if only a small number of pulses is of interest.

The maximum number of pulses to be detected is defined using the [SENSe:​]DETect:​LIMit:​COUNt command.

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

Switches the function off

ON | 1

Switches the function on

*RST:0
Manual operation: See "Detection Limit"
