[SENSe:]POWer:SEM:PIOV <Power>

This command defines the power of the NB-IoT carrier on which the calculation of the SEM limits is based.

Prerequisites for this command
  • Select manual SEM limit calculation mode ([SENSe:]POWer:SEM:PIOM).
Parameters:
<Power>
<numeric value>
*RST:0
Example: 

//Define NB-IoT power manually

POW:SEM:PIOM MAN
POW:SEM:PIOV -43
Manual operation: See "Power NB-IoT Carrier"
