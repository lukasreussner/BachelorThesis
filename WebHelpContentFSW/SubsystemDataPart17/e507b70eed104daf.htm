1xEV-DO Measurements (R&S FSW-K84/-K85) > Remote Commands for 1xEV-DO Measurements > Configuring Code Domain Analysis > Configuring Triggered Measurements > Configuring the Triggering Conditions > TRIGger[:SEQuence]:TIME:RINTerval
TRIGger[:SEQuence]:TIME:RINTerval <Interval>

This command defines the repetition interval for the time trigger.

Parameters:
<Interval>
2.0 ms to 5000
Range: 2 ms  to  5000 s
*RST:1.0 s
Example: TRIG:SOUR TIME

Selects the time trigger input for triggering.

TRIG:TIME:RINT 50

The measurement starts every 50 s.
