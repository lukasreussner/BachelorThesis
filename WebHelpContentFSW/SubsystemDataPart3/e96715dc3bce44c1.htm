[SENSe:]ADEMod<n>:SPECtrum:SPAN:ZOOM <Span>

This command sets the span (around the center frequency) for RF spectrum result display.

The span is limited to the demodulation bandwidth (see [SENSe:​]BWIDth:​DEMod).

Suffix: 
<n>

irrelevant

Parameters:
<Span>
*RST:5 MHz
Example: 

ADEM:SPEC:SPAN:ZOOM 200 kHz

Sets the rF span to 200 kHz

Manual operation: See "Span"
