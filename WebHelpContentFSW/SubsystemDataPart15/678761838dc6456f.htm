Marker Table

Displays a table with the current marker values for the active markers.

This table is displayed automatically if configured accordingly (see "Marker Table Display").

Tip: To navigate within long marker tables, simply scroll through the entries with your finger on the touchscreen.

Remote command: 

LAY:ADD? '1',RIGH, MTAB, see LAYout:​ADD[:​WINDow]?

Results:

CALCulate<n>:​MARKer<m>:​X

CALCulate<n>:​MARKer<m>:​Y?
