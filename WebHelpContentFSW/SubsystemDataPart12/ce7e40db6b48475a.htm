Using the Transmit Filter as a Measurement Filter (Auto)

If the "Auto" option is enabled, the measurement filter is defined automatically depending on the transmit filter specified in the "Modulation" settings (see "Transmit Filter Type").

Note: If a user-defined transmit filter is selected and the measurement filter is defined automatically, a Low-ISI measurement filter according to the selected user filter is calculated and used.

Remote command: 

[SENSe:​]DDEMod:​MFILter:​AUTO
