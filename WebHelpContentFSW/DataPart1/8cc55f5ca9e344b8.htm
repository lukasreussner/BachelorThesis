TTL Handshake

If available for the specified generator type, this option activates TTL synchronization via handshake for GPIB connections.

Using the TTL interface allows for considerably higher measurement rates than pure GPIB control, because the frequency stepping of the R&S FSW is directly coupled with the frequency stepping of the generator.

For more information on TTL synchronization, see "TTL synchronization".

For an overview of which generators support TTL synchronization see "Overview of Supported Generators".

Remote command: 

SYSTem:​COMMunicate:​RDEVice:​GENerator:​LINK
