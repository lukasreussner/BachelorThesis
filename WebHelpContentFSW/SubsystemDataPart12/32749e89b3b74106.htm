Offset

The offset of the pattern is defined in relation to the start of the useful part of the burst (see "Useful length"). If the position of the pattern within the burst is known, it is recommended that you define the offset. That accelerates the pattern search and enhance the accuracy of the burst search.

Remote command: 

[SENSe:​]DDEMod:​STANdard:​SYNC:​OFFSet:​STATe

[SENSe:​]DDEMod:​STANdard:​SYNC:​OFFSet[:​VALue]
