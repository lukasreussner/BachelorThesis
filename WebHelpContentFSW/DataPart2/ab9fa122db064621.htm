General Instrument Setup > Frequency Response Correction (R&S FSW-K544) > User-defined Frequency Response Correction Settings > Additional Frequency Response active
Additional Frequency Response active

Activates or deactivates the use of additional frequency response (.fres) files. The correction data is these files is applied after any correction settings in active touchstone files. Only active files are included in filter calculation.

Remote command: 

[SENSe:​]CORRection:​FRESponse:​USER:​FSTate
