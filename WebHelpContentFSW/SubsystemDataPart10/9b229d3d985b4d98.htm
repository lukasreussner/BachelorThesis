[SENSe:]LIST:RANGe<range>:THReshold:STOP <Stop>

Defines an absolute threshold that the power level must exceed for a peak to be detected as a true spur.

The stop value must be higher than the start value.

Suffix: 
<range>
1..n
Setting parameters: 
<Stop>
Range: -200 dBm to 0 dBm
*RST:0 dBm
Default unit: DBM
Manual operation: See "Spur Detection Threshold Start/ Spur Detection Threshold Stop"
