Noise Figure Measurements (R&S FSW-K30) > Configuration > Selecting DUT Characteristics > Mode
Mode

Selects the measurement mode.

The required measurement mode depends on the type of DUT you are testing. For more information, see "DUT Types".

Note: For upconverting measurements, if a preamplifier is used, make sure the upconverted frequencies for the measurement stay below the maximum frequency the preamplifier supports.

Remote command: 

DUT type: [SENSe:​]CONFigure:​MODE:​DUT

LO type: [SENSe:​]CONFigure:​MODE:​SYSTem:​LO
