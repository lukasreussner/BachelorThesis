Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Applying a System Model > CONFigure:MODeling:LRANge
CONFigure:MODeling:LRANge <Level>

This command defines the modeling level range.

Parameters:
<Level>
<numeric value>
Default unit: dB
Example: CONF:MOD:LRAN 30

Defines a modeling level range of 30 dB.

Manual operation: See "Defining the modeling range"
