Trace Data Output Formats

All formats are supported for trace data output: display units (command O1), display units in two byte binary data (command O2 or TDF B and MDS W), display units in one byte binary data (command O4 or TDF B and MDS B) and physical values (commands O3 or TDF P). With format "display units" the level data is converted into value range and resolution of the 8566/8568 models. On transition to REMOTE state the number of trace points are reconfigured in order to be conform to the selected instrument model (1001 for 8566A/B and 8568 A/B, 601 for 8560E to 8565E, 401 for 8594E).
