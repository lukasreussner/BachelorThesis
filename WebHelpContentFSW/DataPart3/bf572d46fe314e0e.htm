[SENSe:]NPRatio:CHANnel:INTegration:BWIDth <arg0>

Defines the bandwidth to be used for total power calculation as an absolute value. This value is only considered for [SENSe:​]NPRatio:​CHANnel:​INTegration:​AUTOOFF.

The entire specified bandwidth is used, including any defined notches.

Parameters:
<arg0>
*RST:72 MHz
Default unit: HZ
Example: NPR:CHAN:INT:AUTO OFF

Selects manual definition of the integration bandwidth.

NPR:CHAN:INT:BWID 100000000

Sets the integration bandwidth to 100 MHz.

Manual operation: See "Integration Bandwidth"
