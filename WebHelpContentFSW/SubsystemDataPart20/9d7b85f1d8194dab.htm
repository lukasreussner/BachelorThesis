Evaluation Methods for IEEE 802.11ad Modulation Accuracy Measurements

Access: "Overview" > "Display Config"

Or: [MEAS] > "Display Config"

The R&S FSW 802.11ad application provides various different methods to evaluate the captured signal without having to start a new measurement or sweep. Which results are displayed depends on the selected measurement and evaluation.

The selected evaluation method not only affects the result display in a window, but also the results of the trace data query in remote control (see TRACe<n>[:​DATA]?).

All evaluations available for the selected IEEE 802.11ad measurement are displayed in SmartGrid mode.

For details on working with the SmartGrid see Laying out the Result Display with the SmartGrid.

The IEEE 802.11ad measurements provide the following evaluation methods:

└ Bitstream
└ Channel Frequency Response
└ Constellation
└ EVM vs Symbol
└ Freq. Error vs Symbol
└ Header information
└ Magnitude Capture
└ Phase Error vs Symbol
└ Phase Tracking vs Symbol
└ Power Spectrum
└ PvT Full PPDU
└ PvT Rising Edge
└ PvT Falling Edge
└ Result Summary
