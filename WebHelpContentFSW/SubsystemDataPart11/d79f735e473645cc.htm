Automatic Range Scaling

Defines whether the result range length is determined automatically according to the width of the selected hop/chirp (see "Select Hop / Select Chirp").

"OFF"

Switches automatic range scaling off

"ON"

Switches automatic range scaling on

Remote command: 

CALCulate<n>:​RESult:​RANGe:​AUTO
