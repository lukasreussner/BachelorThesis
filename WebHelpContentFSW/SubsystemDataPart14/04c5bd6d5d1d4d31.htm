[SENSe:]CDPower:SULC <SyncUL>

Defines the code used for synchronization on the UpPTS (see "Time Reference (UE mode)").

This command is available for UE mode (K77) only.

Parameters:
<SyncUL>
integer

For details on available values depending on the scrambling code see Table "Possible SYNC-UL codes depending on scrambling code".

Range: 0  to  255
*RST:0
Example: CDP:SULC 28

Sets the code 28.

Manual operation: See "SYNC-UL Code (UE only)"
