Gated Trigger

Activates and deactivates gating for statistics functions for the ACP and the CCDF measurements. If activated, the trigger source is changed to "External Trigger 1". The gate ranges are defined using the "Edit Gate Ranges" function.

For details on using a gated trigger see "APD and CCDF Basics - Gated Triggering".

Remote command: 

[SENSe:​]SWEep:​EGATe:​TRACe<t>[:​STATe<range>]
