Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Applying a System Model > CONFigure:MODeling:NPOints
CONFigure:MODeling:NPOints <Points>

This command defines the number of modeling points.

Parameters:
<Points>
<numeric value>: (integer only)
*RST:50
Default unit: ---
Example: CONF:MOD:NPO 50

Calculates the model based on 50 points.

Manual operation: See "Defining the modeling range"
