Y-Offset

Shifts a limit line that has relative values for the y-axis (levels or linear units such as volt) vertically.

This setting does not have any effect on limit lines that are defined by absolute values for the y-axis.

Remote command: 

CALCulate<n>:​LIMit<k>:​LOWer:​OFFSet

CALCulate<n>:​LIMit<k>:​UPPer:​OFFSet
