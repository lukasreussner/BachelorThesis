Wideband

This setting defines which option is returned when the *OPT? query is executed, depending on the state of the wideband option.

It is only available for PSA89600 emulation.

"Off"

No wideband is used.

The option is indicated as "B7J".

"40 MHz"

The 40 MHz wideband is used.

The option is indicated as "B7J, 140".

"80 MHz"

The 80 MHz wideband is used.

The option is indicated as "B7J, 122".

Remote command: 

SYSTem:​PSA:​WIDeband
