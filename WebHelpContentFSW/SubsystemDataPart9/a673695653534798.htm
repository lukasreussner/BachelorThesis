FETCh:PNOise<t>:RMS?

This command queries the residual RMS jitter for the first trace.

Suffix: 
<t>

Trace

Example: 

FETC:PNO2:RMS?

Queries the RMS jitter for the first trace in the diagram.

Usage: Query only
Manual operation: See "Residual Noise"
