Save As

Saves the current SCPI command list to the C:\R_S\INSTR/ScpiRecordings directory in ASCII format with the file extension .inp.

Tip: You can execute the command list in an .inp file without further editing using the IECWIN tool provided with the R&S FSW, see "The IECWIN Tool".
