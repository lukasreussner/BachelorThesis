Limit Offset to Detection Threshold

Defines a limit line as an offset to the detection threshold for all spans. Values that exceed this limit are indicated red in the "Spurious Detection Table". If a violation occurs, the global limit check over all spans is indicated as failed (see also "Spurious Detection Spectrum").

Note: The limit line functionality used in the R&S FSW base unit is not supported in the R&S FSW Spurious measurements application.

Remote command: 

[SENSe:​]DIRected:​LOFFset
