RC

The Radio Configuration (RC) can be customized for two channel types. For the PDCH, you can set the configuration to either 10 (QPSK), 10 (8PSK) or 10 (16QAM). For CHAN channels, you can set the radio configuration to 1-2 or 3-5.

For details on radio configurations, see "Radio Configuration".
