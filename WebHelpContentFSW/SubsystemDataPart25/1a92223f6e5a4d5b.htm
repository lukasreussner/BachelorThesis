Common Suffixes

In the LTE NB-IoT measurement application, the following common suffixes are used in remote commands:

Common suffixes used in remote commands in the LTE NB-IoT measurement application

Suffix

Value range

Description

<m>

1..4

Marker

<n>

1..16

Window (in the currently selected channel)

<t>

1..6

Trace

<ant>

1..2

Selects an antenna for MIMO measurements.

<cc>

1..5

Selects a component carrier.

Irrelevant for the NB-IoT application.

<k>

---

Selects a limit line.

Irrelevant for the NB-IoT application.

<np>

0...20

Selects a NPUSCH (NB-IoT uplink only)
