Importing and Exporting I/Q Data and Results

The I/Q data to be evaluated in the GSM application can not only be measured by the GSM application itself, it can also be imported to the application, provided it has the correct format. Furthermore, the evaluated I/Q data from the GSM application can be exported for further analysis in external applications.

For details on importing and exporting I/Q data see "I/Q Data Import and Export".

List of Commands
MMEMory:LOAD:IQ:STATe
MMEMory:STORe<n>:IQ:COMMent
MMEMory:STORe:IQ:FORMat
MMEMory:STORe<n>:IQ:STATe
