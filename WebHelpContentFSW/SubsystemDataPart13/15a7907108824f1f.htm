Input Connector

Determines whether the RF input data is taken from the [RF input] connector (default) or the optional [Baseband Input I] connector. This setting is only available if the optional Analog Baseband Interface is installed and active for input. It is not available for the R&S FSW67.

For more information, see Processing Data from the Analog Baseband Interface.

Remote command: 

INPut<ip>:​CONNector
