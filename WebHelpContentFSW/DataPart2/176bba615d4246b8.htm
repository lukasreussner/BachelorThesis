SYSTem:DFPRint?

Creates an *.xml file with information on installed hardware, software, image and FPGA versions. The *.xml file is stored under C:\Program Files (x86)\Rohde-Schwarz\FSW\<version>\devicedata\xml\ on the instrument.

Usage: Query only
Manual operation: See "Save Device Footprint"
