Adjust Settings

If harmonic measurement was performed in the frequency domain, a new peak search is started in the frequency range that was set before starting the harmonic measurement. The center frequency is set to this frequency and the reference level is adjusted accordingly.

If harmonic measurement was performed in the time domain, this function adjusts the reference level only.

Remote command: 

CALCulate<n>:​MARKer<m>:​FUNCtion:​HARMonics:​PRESet
