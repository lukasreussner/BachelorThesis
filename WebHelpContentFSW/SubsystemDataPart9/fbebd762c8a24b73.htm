Trace Math Function

Defines which trace is subtracted from trace 1. The result is displayed in trace 1.

"T1-T3->T1"

Subtracts trace 3 from trace 1

T2-T3->T2

Subtracts trace 3 from trace 2

To switch off the trace math, use the "Trace Math Off" button.

Remote command: 

CALCulate<n>:​MATH[:​EXPRession][:​DEFine]

CALCulate<n>:​MATH:​STATe
