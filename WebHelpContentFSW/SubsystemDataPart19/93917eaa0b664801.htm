WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Demodulation > CONFigure:WLAN:SMAPping:TX<ch>
CONFigure:WLAN:SMAPping:TX<ch> <STSI>...

This remote control command specifies the mapping for all streams (real & imaginary data pairs) and timeshift for a specified antenna.

Suffix: 
<ch>
1..n
Parameters:
<STSI>

Imag part of the complex element of the STS-Stream

Example: CONF:WLAN:SMAP:TX 1.0,1.0,2.0,2.0,3.0,3.0,4.0,4.0,1e-9
Manual operation: See "User Defined Spatial Mapping"
