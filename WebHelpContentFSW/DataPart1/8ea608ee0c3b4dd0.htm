CALCulate<n>:LIMit<k>:UPPer:THReshold <Limit>

This command defines an absolute limit for limit lines with a relative scale.

The R&S FSW uses the threshold for the limit check, if the limit line violates the threshold.

Suffix: 
<n>

irrelevant

<k>

Limit line

Parameters:
<Limit>

Numeric value.

The unit depends on CALCulate<n>:​LIMit<k>:​UNIT.

*RST:-200
Default unit: dBm
Manual operation: See "Threshold"
