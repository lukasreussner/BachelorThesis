Base Station Maximum Output Power

Defines the maximum output power of the base station. Possible values are from 0 dBm to 100 dBm in 1 dB steps.

This setting is only available for base stations with a medium range "Base Station Class".

This value is required to calculate the SEM limits according to standard 3GPP 37.141 V12.2.0.

Remote command: 

[SENSe:​]ESPectrum<sb>:​MSR:​MPOWer
