Couple Windows

If enabled, markers in all diagrams with the same x-axis (time or symbols) have coupled x-values (except for capture buffer display). That means, if you move the marker in one diagram, it is moved in all coupled diagrams.

Remote command: 

CALCulate<n>:​MARKer<m>:​LINK
