CALCulate<n>:MARKer<m>:FUNCtion:TDPIch

This command sets the marker to channel 16.128.

Suffix: 
<n>

Window

<m>

Marker

Example: CALC:MARK:FUNC:TDP

Activates marker and positions it at TDPICH 16.128.

CALC:MARK:Y?

Queries value of the relative Code Domain Power of the transmit diversity pilot channel.

Mode: BTS application only
Manual operation: See "Marker To TDPICH"
