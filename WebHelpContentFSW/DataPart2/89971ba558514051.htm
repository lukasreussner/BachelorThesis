Common Measurement Settings > Bandwidth, Filter and Sweep Configuration > Bandwidth, Filter and Sweep Settings > Default Coupling
Default Coupling

Sets all coupled functions to the default state ("Auto"). In addition, the ratio "RBW/VBW" is set to "Sine[1/1]" and the ratio "Span/RBW" to 100.

For more information see "Coupling VBW and RBW".

Remote command: 

[SENSe:​]BANDwidth[:​RESolution]:​AUTO

[SENSe:​]BANDwidth:​VIDeo:​AUTO

[SENSe:​]SWEep:​TIME:​AUTO
