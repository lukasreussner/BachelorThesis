Accessing the Start Menu

The Windows "Start" menu provides access to the Windows  functionality and installed programs.

To open the "Start" menu:

► 

In Windows 7:

Select the "Windows" icon on the R&S FSW toolbar, or press the "Windows" key or the CTRL + ESC key combination on the (external) keyboard.

All necessary system settings can be defined in the "Start > Control Panel" menu (for required settings refer to the Windows  documentation and to the hardware description).
