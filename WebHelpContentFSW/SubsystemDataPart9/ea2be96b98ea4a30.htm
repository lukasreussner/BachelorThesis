Using Markers

Markers are used to mark points on traces, to read out measurement results and to select a display section quickly. The application provides 4 markers.

By default, the application positions the marker on the lowest level of the trace.You can change a marker position in several ways.

  • Enter a particular offset frequency in the input field that opens when you activate a marker.
  • Move the marker around with the rotary knob or the cursor keys.
  • Drag the marker around using the touchscreen.
