Sequencer State

Activates or deactivates the Sequencer. If activated, sequential operation according to the selected Sequencer mode is started immediately.

For details on sequential operation see The Sequencer Concept.

Remote command: 

SYSTem:​SEQuencer

INITiate<n>:​SEQuencer:​IMMediate

INITiate<n>:​SEQuencer:​ABORt
