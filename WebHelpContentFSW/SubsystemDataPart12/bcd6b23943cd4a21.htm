Spectrum (Measurement + Error)

This display combines two diagrams in one. The first trace displays the spectrum of the real/image data from the measured signal. The second trace displays the spectrum of the real/image data of the error. Optionally, the data source of the traces can be switched. Which source is currently displayed for which trace is indicated in the window title bar.

(See also Figure "Window title bar information in VSA application").

Remote commands:

LAY:ADD? '1',BEL,MCOM

to define the required source type (see LAYout:​ADD[:​WINDow]?)

CALC:FEED 'XTIM:DDEM:MEAS:ERR'

to define the result type (see CALCulate<n>:​FEED)

CALC:TRAC MEAS; CALC:TRAC2:ERR

to define trace1 to be based on the measurement data and trace 2 on the error (default, see CALCulate<n>:​TRACe<t>[:​VALue])

TRAC:DATA? TRACE1

to query the trace results for measurement data (see TRACe<n>[:DATA]? TRACE<n> and "Symbols")

TRAC:DATA? TRACE2

to query the trace results for error data
