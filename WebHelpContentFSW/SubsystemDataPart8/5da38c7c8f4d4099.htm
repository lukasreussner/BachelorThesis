[SENSe:]MIXer<x>:FREQuency:STARt? <Freq>

This command queries the frequency at which the external mixer band starts.

Suffix: 
<x>

irrelevant

Parameters:
<Freq>
numeric value
Example: MIX:FREQ:STAR?

Queries the start frequency of the band.

Example: 

See Programming Example: Working with an External Mixer and Programming Example: Working with an External Mixer.

Usage: Query only
