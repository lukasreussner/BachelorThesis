CONFigure:BURSt:PREamble:SELect <ErrType>

This remote control command specifies whether frequency or phase results are displayed when the measurement type is set to Error Vs Preamble (CONFigure:​BURSt:​PREamble[:​IMMediate]).

Parameters:
<ErrType>
FREQuency | PHASe
FREQuency

Displays frequency error results for the preamble of the measured PPDUs only

PHASe

Displays phase error results for the preamble of the measured PPDUs only

Example: 

CONF:BURS:PRE:SEL PHAS

Manual operation: See "Freq. Error vs Preamble"
See "Phase Error vs Preamble"
