[SENSe:]ADEMod<n>:PRESet:STORe <Standard>

This command saves the current Analog Demodulation measurement configuration.

Standard definitions are stored in an xml file. The default directory for Analog Demodulation standards is C:\R_S\INSTR\USER\predefined\AdemodPredefined.

Suffix: 
<n>

irrelevant

Parameters:
<Standard>

String containing the file name.

You can save the file in a subdirectory of the directory mentioned above. In that case, you have to include the relative path to the file.

Manual operation: See "Save Standard"
