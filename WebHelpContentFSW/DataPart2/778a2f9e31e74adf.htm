CALCulate<n>:LIMit<k>:ACPower:GAP<gap>:ABSolute:STATe <State>

This command turns the absolute limit check for the specified gap (CACLR) channel on and off.

You have to activate the general ACLR limit check before using this command with CALCulate<n>:​LIMit<k>:​ACPower[:​STATe].

Suffix: 
<n>, <k>

irrelevant

<gap>
1 | 2

Gap (CACLR) channel number

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Manual operation: See "Limit Checking"
