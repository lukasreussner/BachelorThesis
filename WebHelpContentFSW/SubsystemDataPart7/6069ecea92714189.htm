CONFigure:DDPD:COUNt:CURRent?

This command queries the process of the direct DPD sequence (number of current iteration).

  • Turn on direct DPD (CONFigure:DDPD[:STATe]).
  • Start a DPD sequence (CONFigure:DDPD:STARt).
Return values: 
<Iterations>
Example: 

//Define number of iterations

CONF:DDPD:COUN 10

//Query process of measurement

CONF:DDPD:COUN:CURR?

would return, e.g.

7 (out of 10)
Usage: Query only
