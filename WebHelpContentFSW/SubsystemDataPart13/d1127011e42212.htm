[SENSe:]CDPower:LCODe:SEARch:LIST?

This command returns the automatic search sequence (see [SENSe:​]CDPower:​LCODe:​SEARch[:​IMMediate]?) as a comma-separated list of results for each detected scrambling code.

Return values: 
<Code1>

Scrambling code in decimal format.

Range: 16 * n, with n = 0...511
<Code2>

Scrambling code in hexadecimal format.

Range: 0x0000h – 0x1FF0h, where the last digit is always 0
<CPICHPower>

Highest power value for the corresponding scrambling code.

Example: 

SENS:CDP:LCOD:SEAR:LIST?

Result:

16,0×10,-18.04,32,0×20,-22.87,48,0×30,-27.62,64,0×40,-29.46

(Explanation in table below)

Usage: Query only
Mode: BTS application only
Manual operation: See "Scrambling Codes"
