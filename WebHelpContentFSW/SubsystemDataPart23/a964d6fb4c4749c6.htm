N_PUCCH

Defines the resource index for PUCCH format 1/1a/1b respectively 2/2a/2b.

You can select the PUCCH format manually or allow the application to determine the PUCCH format automatically based on the measurement.

It is also possible to define NPUCCH on a subframe level by selecting the "Per Subframe" menu item. For more information see "Configuring Subframes".

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​PUCCh:​NPAR
