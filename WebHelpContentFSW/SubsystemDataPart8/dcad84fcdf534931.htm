Marker Config

Opens the "Marker Configuration" dialog box.

The "Marker Configuration" dialog box contains all marker functions necessary to set up the four markers supported by the application.

  • Selected
    Highlights the currently selected marker.
  • State
    Turns a marker on and off.
  • X-value
    Defines the marker position on the horizontal axis.
  • Type
    Selects the marker type. For more information see "Marker Type".
  • Trace
    Selects the trace the marker is positioned on.
