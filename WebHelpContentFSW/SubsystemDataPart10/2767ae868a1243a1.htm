Trace / Table Export Configuration

Access: "Trace" > "Trace/Table Export"

The R&S FSW provides various evaluation methods for the results of the performed measurements. However, you may want to evaluate the data with other, external applications. In this case, you can export the measurement data to an ASCII file.

The standard data management functions (e.g. saving or loading instrument settings) that are available for all R&S FSW applications are not described here.

See Data Management for a description of the standard functions.
