Channel Table Configuration

You can configure which parameters are displayed in the Channel Table by selecting the table header. A "Table Configuration" dialog box is displayed in which you can select the columns to be displayed.

By default, only active channels are displayed. To display all channels, including the inactive ones, enable the "Show Inactive Channels" option.
