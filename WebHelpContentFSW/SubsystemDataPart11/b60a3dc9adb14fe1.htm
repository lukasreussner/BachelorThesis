Result Range Reference Point

Defines the reference point for positioning the result range. The "Offset" is given with respect to this value.

"Rise"

The result range is defined in reference to the rising edge.

"Center"

The result range is defined in reference to the center of the hop/chirp top.

"Fall"

The result range is defined in reference to the falling edge.

Remote command: 

CALCulate<n>:​RESult:​REFerence
