Direction

Determines which parameter controls the other.

"<-"

Parameter 2 controls parameter 1. If parameter 2 is changed, parameter 1 is adapted. If parameter 1 is changed, parameter 2 remains unchanged.

Remote command: 

INST:COUP:USER:REL RTOL

 

"->"

Parameter 1 controls parameter 2. If parameter 1 is changed, parameter 2 is adapted. If parameter 2 is changed, parameter 1 remains unchanged.

Remote command: 

INST:COUP:USER:REL LTOR

 

"<->"

Both parameters are equal. If one parameter is changed, the other parameter is adapted and vice versa.

Remote command: 

INST:COUP:USER:REL BID
