CONFigure:REFSignal:CGW:READ

This command transfers a reference signal designed on a signal generator into the R&S FSW-K18.

Example: CONF:REFS:CGW:READ

Imports the reference signal data from the generator.

Usage: Event
Manual operation: See "Designing a reference signal on a signal generator"
