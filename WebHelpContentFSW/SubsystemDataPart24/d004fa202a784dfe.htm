MSRA Mode > Configuration > Data Acquisition and Bandwidth Settings > Sweep Settings > Continue Single Sweep
Continue Single Sweep

After triggering, repeats data acquisition without deleting the trace of the last measurement. If the Sequencer is off, only the evaluation for the currently displayed measurement channel is updated.

While the measurement is running, the "Continue Single Sweep" softkey and the [RUN SINGLE] key are highlighted. The running measurement can be aborted by selecting the highlighted softkey or key again.

Remote command: 

INITiate<n>:​CONMeas
