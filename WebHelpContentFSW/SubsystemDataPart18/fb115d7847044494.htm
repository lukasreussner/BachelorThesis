I/Q Correlation Threshold

The I/Q correlation threshold decides whether a match is accepted or not during a pattern search (see also "I/Q Pattern Search"). If the parameter is set to 100%, only I/Q patterns that match totally with the input signal are found. This is only the case for infinite SNR.

If the threshold "Auto" option is enabled, the default value of 90% is used. As long as the pattern is found, there is no need to change this parameter. However, if the pattern is very short (approximately < 10 symbols) or if the signal is highly distorted, tuning this parameter helps the pattern search to succeed. To define a threshold manually, disable the "Auto" option.

Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​IQCThreshold

[SENSe:​]DDEMod:​SEARch:​PATTern:​CONFigure:​AUTO
