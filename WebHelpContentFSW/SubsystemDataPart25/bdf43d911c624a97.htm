CONFigure[:LTE]:DL[:CC<cc>]:PLC:CID <CellId>

This command defines the cell ID.

Suffix: 
<cc>

Component Carrier

Parameters:
<CellId>
AUTO

Automatically defines the cell ID.

<numeric value> (integer only)

Number of the cell ID.

Range: 0  to  503
Example: 

//Select cell ID

CONF:DL:PLC:CID 15
Manual operation: See "Configuring the Physical Layer Cell Identity"
