[SENSe:]DDEMod:SEARch:SYNC:NSTate <NState>

This command selects the degree of modulation (number of permitted states). The pattern must have been selected before using [SENSe:​]DDEMod:​SEARch:​SYNC:​NAME.

The number of permitted states depends on the modulation mode.

Setting parameters: 
<NState>
numeric value
