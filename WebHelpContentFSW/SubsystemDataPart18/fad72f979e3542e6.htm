VSA (R&S FSW-K70) > Configuration > Input, Output and Frontend Settings > Frontend Settings > Optimization
Optimization

Selects the priority for signal processing after the RF attenuation has been applied.

This function is only available under the following conditions:
  • Bandwidth extension R&S FSW-B160/-B320 Extension Board 1, Revision 2 or higher, R&S FSW-B512, or the real-time option R&S FSW-B160R is installed (these options provide a separate wideband processing path in the R&S FSW)
  • An I/Q bandwidth higher than 80 MHz is used (only in this case the wideband path is used)
  • The optional Digital Baseband Interface is not active

"Low distortion"

(Default:) Optimized for low distortion by avoiding intermodulation

"Low noise"

Optimized for high sensitivity and low noise levels

If this setting is selected, "Low noise" is indicated in the channel information bar.

Remote command: 

INPut<ip>:​ATTenuation:​AUTO:​MODE
