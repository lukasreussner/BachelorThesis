CONFigure:POWer:RESult:P3DB:REFerence <RefPower>

This command defines the input power corresponding to the gain reference required to calculate the Compression Points.

The command is available when you turn off CONFigure:POWer:RESult:P3DB[:STATe].

Parameters:
<RefPower>
<numeric value>
Default unit: dBm
Example: CONF:POW:RES:P3DB OFF
CONF:POW:RES:P3DB:REF 3

Reference point is the gain measured at an input power of 3 dBm.

Manual operation: See "Configuring compression point calculation"
