MMEMory:STORe:DPD <FileName>

This command generates and stores a waveform containing the DPD in a file you have specified.

Prerequisites for this command
  • DPD method "Generate Predistorted Waveform File" has to be selected (CONFigure:DPD:METHod = WFILe)
  • The DPD calculation has been initiated with CONFigure:DPD:FILE:GENerate.
Setting parameters: 
<FileName>

String containing the file name.

Example: CONF:DPD:METH WFIL
CONF:DPD:FILE:GEN
MMEM:STOR:DPD 'DPD_WV'

Calculates the DPD within the Amplifier application, transfers the result to the signal generator and saves it in a file.

Usage: Setting only
