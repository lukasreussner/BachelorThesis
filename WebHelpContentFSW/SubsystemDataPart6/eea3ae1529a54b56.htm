[SENSe:]ADEMod:RLENgth?

This command returns the record length set up for current measurement settings.

Note that this command is maintained for compatibility reasons only. Use the [SENSe:​]RLENgth? command for new remote control programs.

Return values: 
<SampleCount>
Usage: Query only
