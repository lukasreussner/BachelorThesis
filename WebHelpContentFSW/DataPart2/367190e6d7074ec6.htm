CALCulate<n>:SGRam:FRAMe:COUNt <Frames>
CALCulate<n>:SPECtrogram:FRAMe:COUNt <Frames>

This command defines the number of frames to be recorded in a single sweep.

This value applies to all spectrograms in the channel.

Suffix: 
<n>

irrelevant

Parameters:
<Frames>

The maximum number of frames depends on the history depth.

Range: 1  to  history depth
Increment: 1
*RST:1
Example: INIT:CONT OFF

Selects single sweep mode.

CALC:SGR:FRAM:COUN 200

Sets the number of frames to 200.

Manual operation: See "Frame Count"
