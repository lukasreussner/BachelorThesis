Trigger Offset

Defines the time offset between the trigger event and the start of the sweep .

Offset > 0:

Start of the sweep is delayed

Offset < 0:

Sweep starts earlier (pretrigger)

Remote command: 

TRIGger[:​SEQuence]:​HOLDoff[:​TIME]
