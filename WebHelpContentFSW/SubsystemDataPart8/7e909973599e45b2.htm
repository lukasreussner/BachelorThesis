[SENSe:]CORRection:ENR[:MEASurement]:TABLe:TEMPerature:LIST?

This command queries all temperature tables available in the application.

Return values: 
<Tables>

String containing the names of the tables as a comma separated list.

Example: CORR:ENR:TABL:TEMP:LIST

would return, e.g.

'Table1,Table2,Table3'
Usage: Query only
