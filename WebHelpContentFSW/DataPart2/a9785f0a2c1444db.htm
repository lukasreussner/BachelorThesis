CALCulate<n>:LIMit<k>:ESPectrum<sb>:PCLass<PClass>:MINimum <Level>

This command defines the lower limit of a particular power class.

Note:
  • The first power class always has a lower limit of -200 dBm.
  • The lower limit of a power class must always be the same as the upper limit of the previous power class.
  • The power class must already exist (see CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass:​COUNt).
Suffix: 
<n>, <k>

irrelevant

<sb>
1 | 2 | 3

Sub block in a Multi-SEM measurement

<PClass>
1...4

power class

Parameters:
<Level>
Range: -200 dBm  to  199.9 dBm
Example: CALC:LIM:ESP:PCL2:MIN -40 dBm

Sets the minimum power value of the second power class to -40 dBm.

Manual operation: See "PMin/ PMax"
