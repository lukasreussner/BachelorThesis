Export Data to ASCII File for All Windows

Exports the data from all currently displayed traces and tables in the R&S FSW Spurious measurements application for export to an ASCII file.

The results are output in the same order as they are displayed on the screen: window by window, trace by trace, and table row by table row.

Remote command: 

MMEMory:​STORe:​SPUR:​MEAS
