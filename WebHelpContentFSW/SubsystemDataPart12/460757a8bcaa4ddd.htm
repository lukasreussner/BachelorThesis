Transmit Filter Type

Defines the type of transmit filter

An overview of available transmit filters is provided in "Transmit Filters".

For more information on transmit filters, see "Modulation and Demodulation Filters".

Remote command: 

[SENSe:​]DDEMod:​TFILter:​NAME

To define the name of the transmit filter to be used.

[SENSe:​]DDEMod:​TFILter[:​STATe]

To switch off the transmit filter.

[SENSe:​]DDEMod:​TFILter:​USER

To select a user-defined filter.
