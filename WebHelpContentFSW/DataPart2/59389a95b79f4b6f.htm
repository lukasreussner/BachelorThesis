Specifics for Window

Selects the windows of the selected channel for which the parameters are coupled. This setting is only available for Analog Demodulation channels. Individual windows can only be coupled for frequency markers.

Remote command: 

INSTrument:​COUPle:​USER<uc>:​WINDow:​LIST?
