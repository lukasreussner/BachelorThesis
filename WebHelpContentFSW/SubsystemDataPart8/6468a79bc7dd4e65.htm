Trace Config

Opens a dialog box made up out of three tabs to configure traces.

The first tab contains functionality to configure a trace. For more information see "Traces".

The second tab contains functionality to export trace data. For more information see "Trace Export".

The third tab contains functionality to copy traces. For more information see "Copy Trace".
