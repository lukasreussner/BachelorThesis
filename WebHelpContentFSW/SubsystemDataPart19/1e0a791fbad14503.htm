WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Demodulation > CONFigure:WLAN:RUConfig:SEGMent<seg>:CHANnel<ch>:RULocation<rui>:USER<mu>:NSTS
CONFigure:WLAN:RUConfig:SEGMent<seg>:CHANnel<ch>:RULocation<rui>:USER<mu>:NSTS <NoStreams>

For MIMO measurements only:

Number of space-time streams (NSTS) for each user

Suffix: 
<seg>
1..n

80-MHz-Segment to be configured

Segment 1: first 242 subcarriers of the channel

Segment 2: last 242 subcarriers of the channel (for 160 MHz channels and MU-PPDU (DL) only)

<ch>
1..n

20-MHz-subchannel that RU allocation applies to

<rui>
1..n

Index of the entry in the RU allocation table

<mu>
1..n

User index for RU in MIMO mode

Parameters:
<NoStreams>

Number of space-time streams (NSTS) for each user

Range: 1 to 8
Example: CONFigure:WLAN:RUConfig:SEGMent1:CHANnel1:RULocation1:USER1:NSTS?
//Result: 1
Manual operation: See "Nsts per User"
