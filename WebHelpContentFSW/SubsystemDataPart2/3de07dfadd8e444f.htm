Filter type

Defines the filter to be used for demodulation.

"Flat"

Standard flat demodulation filter

"Gauss"

Filter with optimized settling behavior (default)

Note: For Gaussian filters whose -3dB bandwidth is large compared to the maximum I/Q bandwidth, the ideal Gaussian filter shape would exceed the maximum I/Q bandwidth at its outer edges. Thus, the actual filter only follows the ideal Gaussian filter shape in the inner range of the set I/Q bandwidth. At a certain frequency offset it must deviate from the ideal Gauss filter and drop off faster.
For details see "Effects of Large Gauss Filters".

Remote command: 

[SENSe:​]BWIDth:​DEMod:​TYPE
