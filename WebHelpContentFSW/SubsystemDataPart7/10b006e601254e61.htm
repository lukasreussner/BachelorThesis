[SENSe:]SWEep:IQAVg:COUNt:CURRent?

This command queries the current measurement out of a sequence of measurements that averages I/Q data.

  • Turn on I/Q averaging ([SENSe:]SWEep:IQAVg[:STATe]).
  • Define an average count > 1 ([SENSe:]SWEep:IQAVg:COUNt).
Return values: 
<Measurement>
Example: 

//Define number of measurements

SWE:IQAV:COUN 10

//Query process of measurement

SWE:IQAV:COUN:CURR?

would return, e.g.

7 (out of 10)
Usage: Query only
