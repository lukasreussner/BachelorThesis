Optimized Settings (Adjust Settings)

All instrument settings for the selected channel setup (channel bandwidth, channel spacing) can be optimized automatically.

The adjustment is carried out only once. If necessary, the instrument settings can be changed later.

The following settings are optimized by "Adjust Settings":

  • "Frequency Span"
  • "Resolution Bandwidth (RBW)"
  • "Video Bandwidth (VBW)"
  • "Detector"
  • " Trace Averaging"

Note: The reference level is not affected by this function. To adjust the reference level automatically, use the "Setting the Reference Level Automatically (Auto Level)" function in the AUTO SET menu.

Remote command: 

[SENSe:​]POWer:​ACHannel:​PRESet
