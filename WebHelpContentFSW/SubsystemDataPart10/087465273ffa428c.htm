Display Configuration

Access: "Overview" > "Display Config"

The captured signal can be displayed using various evaluation methods. All evaluation methods available for the R&S FSW Spurious measurements application are displayed in the evaluation bar in SmartGrid mode.

Up to sixteen evaluation methods can be displayed simultaneously in separate windows. The evaluation methods available for Spurious are described in "Evaluation Methods".

For details on working with the SmartGrid see Laying out the Result Display with the SmartGrid.
