DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:RLEVel:OFFSet <Offset>

This command defines a reference level offset (for all traces in all windows).

Suffix: 
<n>, <t>

irrelevant

Parameters:
<Offset>
Range: -200 dB  to  200 dB
*RST:0dB
Example: DISP:TRAC:Y:RLEV:OFFS -10dB
