DISPlay:CMAP<item>:DEFault<colors>

This command resets the color scheme for the display.

Suffix: 
<item>

Selects the item for which the color scheme is to be defined.

For more information see "CMAP Suffix Assignment".

<colors>
1...4

1

Current colors with a white background and a black grid.

2

Optimized colors.

3

Customized colors.

4

Current screen colors (setting for hardcopies).

Example: DISP:CMAP:DEF2

Selects default setting 2 for setting the colors.

Usage: Event
Manual operation: See "Screen Colors"
