Maximum number of trace points

If the number of samples within the result range (see "Result Range") is larger than this value, the trace data is reduced to the defined maximum number of trace points using the selected detector.

Restricting this value can improve performance during statistical evaluation of large result range lengths.

Remote command: 

[SENSe:​]SWEep:​POINts
