Evaluation Range

Access: "Overview" > "Analysis" > "Evaluation Range" tab

The evaluation range defines which channel (Code Number), PCG or set is analyzed in the result display.

For CDMA2000 MS measurements, the branch to be analyzed can also be defined.

└ Code Number
└ Power Control Group
└ Set to Analyze
└ Branch (MS application only)
