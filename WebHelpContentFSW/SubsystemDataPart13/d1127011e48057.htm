CONFigure:WCDPower:MS:CTABle:COMMent <Comment>

This command defines a comment for the selected channel table:

Prior to this command, the name of the channel table has to be defined with command CONFigure:​WCDPower:​MS:​CTABle:​NAME. The values of the table are defined with command CONFigure:​WCDPower:​MS:​CTABle:​DATA.

Parameters:
<Comment>
Example: CONF:WCDP:MS:CTAB:NAME 'NEW_TAB'

Defines the channel table name.

CONF:WCDP:MS:CTAB:COMM 'Comment for table 1'

Defines a comment for the table.

Mode: UE application only
Manual operation: See "Comment"
