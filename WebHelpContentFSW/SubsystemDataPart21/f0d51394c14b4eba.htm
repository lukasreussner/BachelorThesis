Configuring Frequency Sweep Measurements on DOCSIS 3.1 Signals

The R&S FSW DOCSIS 3.1 application uses the functionality of the R&S FSW base system (Spectrum application, see Configuring and Performing Measurements) to perform the DOCSIS 3.1 frequency sweep measurements. The R&S FSW DOCSIS 3.1 application automatically sets the parameters to predefined settings as described in "Configuring Frequency Sweep Measurements on DOCSIS 3.1 Signals".

The DOCSIS 3.1 RF measurements must be activated for a measurement channel in the R&S FSW DOCSIS 3.1 application, see "Activating DOCSIS 3.1 Measurements".

For details on configuring these RF measurements in a remote environment, see Configuring and Performing Measurements.
