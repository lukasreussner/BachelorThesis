[SENSe:][LTE:][CC<cc>:]PREamble:SELect <Subframe>

This command selects a particular preamble for measurements that analyze individual preambles.

The command is available in PRACH analysis mode.

Suffix: 
<cc>

Component Carrier

Parameters:
<Subframe>
ALL

Analyzes all preambles.

<numeric value>

Analyzes a single preamble.

*RST:ALL
Example: 

//Analyze all preambles

PRE:SEL ALL
Manual operation: See "Preamble Selection"
