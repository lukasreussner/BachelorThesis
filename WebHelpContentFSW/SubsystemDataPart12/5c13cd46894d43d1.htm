Symbols

For the symbol table result diagrams, the command returns one value for each number in the table. The number of returned values depends on the modulation scheme you have selected.

If no parameter is specified, the command always returns the values in the decimal format. If the parameter STRace is used, the format depends on the currently defined symbol format (see CALCulate<n>:​FORMat).
