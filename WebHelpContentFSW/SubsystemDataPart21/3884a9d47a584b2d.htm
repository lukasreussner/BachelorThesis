DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:MINimum <Min>

Defines the minimum value to be displayed on the y-axis of the specified evaluation diagram.

For automatic scaling with a fixed range (see DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO:​FIXed:​RANGe), the minimum defines the fixed lower limit.

Suffix: 
<n>

Window

<t>

Trace

Parameters:
<Min>
Example: DISP:WIND2:TRAC:Y:SCAL:MIN -20
Manual operation: See "Minimum / Maximum"
