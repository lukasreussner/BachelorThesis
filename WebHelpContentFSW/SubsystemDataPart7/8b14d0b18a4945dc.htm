CONFigure:PSWeep:Y:SETTing <Setting>

This command selects the parameter type for the second parameter controlled by the Parameter Sweep.

Parameters:
<Setting>
BIAS

Controls the envelope bias.

DELay

Controls the delay between envelope and RF signal.

FREQuency

Controls the frequency.

POWer

Controls the output level.

Example: CONF:PSW:Y:STAT ON
CONF:PSW:Y:SETT FREQ
CONF:PSW:Y:STAR 10MHZ
CONF:PSW:Y:STOP 100MHZ
CONF:PSW:Y:STEP 1MHZ

Configure the second parameter with start, stop and stepsize values.
