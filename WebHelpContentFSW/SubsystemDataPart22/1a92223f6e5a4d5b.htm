Common Suffixes

In the LTE measurement application, the following common suffixes are used in remote commands:

Common suffixes used in remote commands in the LTE measurement application

Suffix

Value range

Description

<m>

1..4

Marker

<n>

1..16

Window (in the currently selected channel)

<t>

1..6

Trace

<a>

0..110

Selects a subframe allocation.

<in>

1..4

Selects an instrument for MIMO measurements.

<ant>

1..4

Selects an antenna for MIMO measurements.

<cc>

1..5

Selects a component carrier. The actual number of supported component carriers depends on the selected measurement

<cluster>

1..2

Selects a cluster (uplink only).

<cw>

1..n

Selects a codeword.

<k>

---

Selects a limit line.

Irrelevant for the LTE application.

<sf>

DL: 0..49

UL: 0..9

Selects a subframe.
