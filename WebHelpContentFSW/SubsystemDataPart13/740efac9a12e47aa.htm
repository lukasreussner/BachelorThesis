Constellation Parameter B

For Bitstream evaluation:

Defines the constellation parameter B. According to 3GPP specification, the mapping of 16QAM symbols to an assigned bitstream depends on the constellation parameter B. This parameter can be adjusted to decide which bit mapping should be used for bitstream evaluation.

Remote command: 

[SENSe:​]CDPower:​CPB
