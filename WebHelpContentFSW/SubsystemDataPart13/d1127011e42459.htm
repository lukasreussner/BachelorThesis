[SENSe:]CDPower:LCODe[:VALue] <ScramblingCode>

This command defines the scrambling code in hexadecimal format.

Parameters:
<ScramblingCode>
Range: #H0  to  #H1fff
*RST:#H0
Example: 

SENS:CDP:LCOD #H2

Defines the scrambling code in hexadecimal format.

Manual operation: See "Format Hex/Dec"
