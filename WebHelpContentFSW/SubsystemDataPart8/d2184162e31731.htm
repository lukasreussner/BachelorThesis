[SENSe:]CORRection:ENR[:MEASurement]:SPOT <ENR>

This command defines the constant ENR for all measurement points during the actual measurement.

Parameters:
<ENR>
Range: -999.99  to  999.99
*RST:15
Default unit: dB
Example: CORR:ENR:MODE SPOT
CORR:ENR:SPOT 30

Selects constant ENR value mode and defines an ENR of 30 dB for all measurement points.

Manual operation: See "Measurement"
