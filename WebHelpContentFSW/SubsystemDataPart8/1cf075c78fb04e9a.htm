(Measurement) Points

Defines the number of measurement points for single frequency measurements.

For single frequency measurements, the number of measurement points corresponds to the number of measurements (index values) performed on a single frequency.

For more information, see "Single frequency measurements".

Remote command: 

[SENSe:​]SWEep:​POINts
