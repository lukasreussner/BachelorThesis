SYSTem:HPADditional <State>

Allows the use of HP commands in addition to SCPI commands for R&S FSP/FSQ/FSU emulation (see SYSTem:​LANGuage).

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

Switches the function off

ON | 1

Switches the function on

*RST:0
Manual operation: See "Language"
See "HP Additional"
