CONFigure[:LTE]:DL[:CC<cc>]:SUBFrame<sf>:ALLoc<a>[:CW<cw>]:MODulation <Modulation>

This command selects the modulation of an allocation in a downlink subframe.

Suffix: 
<a>

Allocation

<cc>

Component Carrier

<cw>

Codeword

<sf>

Subframe

Parameters:
<Modulation>
QPSK

QPSK modulation

QAM16

16QAM modulation

QAM64

64QAM modulation

QAM256

256QAM modulation

Q1K

1024QAM modulation

*RST:QPSK
Example: CONF:DL:SUBF2:ALL5:CW2:MOD QAM64

Selects a 64QAM modulation for the second codeword of allocation 5 in subframe 2.

Manual operation: See "Modulation"
