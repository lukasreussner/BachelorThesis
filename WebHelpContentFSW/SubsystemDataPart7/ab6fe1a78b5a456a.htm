Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Applying a System Model > CONFigure:MODeling:SEQuence
CONFigure:MODeling:SEQuence <State>

This command selects the sequence in which the models are calculated.

Parameters:
<State>
AMFirst

Calculates the AM/AM model before calculating the AM/PM model.

PMFirst

Calculates the AM/PM model before calculating the AM/AM model.

*RST:AMFirst
Example: CONF:MOD:SEQ AMF

Calculates the AM/AM model first.

Manual operation: See "Turning system modeling on and off"
