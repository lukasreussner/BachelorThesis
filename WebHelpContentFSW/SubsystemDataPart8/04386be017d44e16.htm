Import / Export

Opens a dialog box to select an ENR table to import or export.

An import copies the ENR table into the default ENR table directory. An export copies the table to a location outside the default ENR table directory, e.g. a memory stick. The file extension has to be *.freq.

For details on the required file format see "Reference: ENR Table File Format".
