Fast SEM

Activates "Fast SEM" mode for all ranges in the sweep list. For details, see "Fast SEM Measurements".

Note: If you deactivate "Fast SEM" mode while "Symmetrical Setup" mode is on, "Symmetrical Setup" mode is automatically also deactivated.
If you activate "Fast SEM" mode while "Symmetrical Setup" mode is on, not all range settings can be configured symmetrically automatically.

Remote command: 

[SENSe:​]ESPectrum<sb>:​HighSPeed
