[SENSe:]CORRection:FRESponse:USER:FLISt<fli>:REMove

Removes the selected frequency response (.fres) file from the current configuration.

Suffix: 
<fli>
1..n

Index in frequency response file list

Use [SENSe:​]CORRection:​FRESponse:​BASeband:​USER:​FLISt<fli>:​SIZE? to determine the maximum index number.

Example: SENS:CORR:FRES:USER:FLIS2:REM
Usage: Event
Manual operation: See "Remove Frequency Response File"
