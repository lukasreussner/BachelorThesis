[SENSe:]CDPower:PRESet

This command resets the 3GPP FDD channel to its predefined settings. Any RF measurement is aborted and the measurement type is reset to Code Domain Analysis.

Note that this command is retained for compatibility reasons only. For new R&S FSW programs use SYSTem:​PRESet:​CHANnel[:​EXEC].
