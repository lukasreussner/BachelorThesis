Window Function

In the I/Q analyzer you can select one of several FFT window types.

For details on the effects of FFT windowing functions see "Basics on FFT".

The following window types are available:
  • Blackman-Harris
  • Flattop
  • Gauss
  • Rectangular
  • 5-Term
Remote command: 

[SENSe:​]IQ:​FFT:​WINDow:​TYPE
