Number of Notches

Defines the number of notches for which results are determined. A maximum of 25 notches can be defined. Note that even if bandwidths for further notches are defined, only the number specified here are actually calculated and displayed.

See "NPR Results" for more information about how notches are treated in the measurement.

Remote command: 

[SENSe:​]NPRatio:​NOTCh<notch>:​COUNt
