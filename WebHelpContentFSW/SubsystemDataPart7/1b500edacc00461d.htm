Selecting the modeling scale

The input power range is split into several equally spaced subranges (= modeling points) for the calculation of the amplifier model.

With the "Modeling Scale", you can select whether the split is done on a logarithmic or linear basis.

Remote command: 

CONFigure:​MODeling:​SCALe
