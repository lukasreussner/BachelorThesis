CALCulate<n>:DISTribution:X?

Queries the x-axis values of the specified Parameter Distribution display.

Suffix: 
<n>
1..n

Window

Return values: 
<XAxis>
<char_data>

The number of values is defined by CALCulate<n>:​DISTribution:​NBINs. The used unit depends on the selected parameter.

Example: CALC:DIST:X?
Usage: Query only
Manual operation: See "Parameter Distribution"
See "X-Axis"
