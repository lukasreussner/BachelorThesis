[SENSe:]HOP:STATe:STAFrequency:AVERage? <QueryRange>
[SENSe:]HOP:STATe:STAFrequency:MAXimum? <QueryRange>
[SENSe:]HOP:STATe:STAFrequency:MINimum? <QueryRange>
[SENSe:]HOP:STATe:STAFrequency:SDEViation? <QueryRange>

Returns the statistical value for the nominal hop state frequency from the statistics table for the specified hop(s).

Query parameters: 
<QueryRange>
CURRent | ALL
CURRent

Detected hops in the current capture buffer

ALL

All hops detected in the entire measurement

Return values: 
<Result>
Usage: Query only
