Common Measurement Settings > Data Input and Output > Power Sensors > Power Sensor Settings > Use Ref Level Offset
Use Ref Level Offset

If activated, takes the reference level offset defined for the analyzer into account for the measured power (see "Shifting the Display (Offset)").

If deactivated, takes no offset into account.

Remote command: 

[SENSe:​]PMETer<p>:​ROFFset[:​STATe]
