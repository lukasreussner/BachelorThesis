FETCh:PLCMessages:NCP:ASSignment?

Queries the subcarrier to codeword assignment defined by the Next Codeword Pointer (NCP) from the PLC messages

Return values: 
<Result>
Example: FETC:PLCM:NCP:ASS?
//Result:
0:1:1108,2988:1:4095,1145,1218,1291
Usage: Query only
