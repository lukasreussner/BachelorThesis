CALCulate<n>:MARKer<m>:FUNCtion:MDEPth:SEARchsignal ONCE

This command initiates a search for the signals required for the AM depth measurement.

Note that the command does not perform a new measurement, but looks for the signals on the current trace.

Suffix: 
<n>

Window

<m>

Marker

Parameters:
ONCE
Example: CALC:MARK:FUNC:MDEP:SEAR ONCE

Executes the search of an AM modulated signal at the currently available trace.

Example: 

See "Example: Measuring the AM Modulation Depth".

Usage: Event
Manual operation: See "Search Signals"
