[SENSe:][LTE:][CC<cc>:]SLOT:SELect <Slot>

This command filters the displayed results in the constellation diagram by a particular slot.

Suffix: 
<cc>

Component Carrier

Parameters:
<Slot>
S0

Slot 0

S1

Slot 1

ALL

Both slots

*RST:ALL
Example: 

//Display results for all slots

SLOT:SEL ALL
Manual operation: See "Slot Selection"
