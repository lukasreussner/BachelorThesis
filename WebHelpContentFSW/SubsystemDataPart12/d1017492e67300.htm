[SENSe:]DDEMod:EQUalizer:LOAD <Name>

This command selects a user-defined equalizer. The equalizer mode is automatically switched to USER (see [SENSe:​]DDEMod:​EQUalizer:​MODE).

Setting parameters: 
<Name>
string

Path and file name (without extension)

Example: DDEM:EQU:LOAD 'D:\MyEqualizer'

Selects equalizer named MyEqualizer in directory D.

Manual operation: See "Store/Load Current Equalizer"
