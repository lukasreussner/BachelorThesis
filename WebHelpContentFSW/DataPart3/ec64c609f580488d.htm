LAN

The LAN interface can be used to connect the R&S FSW to a local network for remote control, printouts or data transfer. The assignment of the RJ-45 connector supports twisted-pair category 5 UTP/STP cables in a star configuration (UTP stands for unshielded twisted pair, and STP for shielded twisted pair).

For details, see "Network and Remote Operation".
