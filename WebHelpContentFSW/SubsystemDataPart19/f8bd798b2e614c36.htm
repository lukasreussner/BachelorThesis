WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Demodulation > [SENSe:]DEMod:FORMat:NSTSindex
[SENSe:]DEMod:FORMat:NSTSindex <Index>

Defines the PPDUs taking part in the analysis depending on their Nsts.

This command is only available for the IEEE 802.11 ac standard.

This command is available for DEM:FORM:NSTS:MODE MEAS or DEM:FORM:NSTS:MODE DEM (see [SENSe:​]DEMod:​FORMat:​NSTSindex:​MODE).

Parameters:
<Index>
Example: SENS:DEM:FORM:NSTS:MODE MEAS
SENS:DEM:FORM:NSTS 1
Manual operation: See "Nsts"
