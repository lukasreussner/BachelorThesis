Loading Data
MMEM:CDIR?
//Returns the path of the current directory.
//e.g.
C:\R_S\Instr\user\
MMEM:CDIR 'C:\R_S\INSTR\USER\Results'
//Changes the current directory.
MMEM:CAT? 'C:\R_S\INSTR\USER\Results\*.xml'
//or
MMEM:CAT? '*.xml'
//Returns a list of all xml files in the directory 'C:\R_S\INSTR\USER\Results'.
MMEM:CAT:LONG? '*.xml'
//Returns additional information about the xml files in the directory 'C:\R_S\INSTR\USER\Results'.
