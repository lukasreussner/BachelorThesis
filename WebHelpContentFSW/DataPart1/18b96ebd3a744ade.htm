Frequency and Span Information in Diagram Footer

The information in the diagram footer (beneath the diagram) depends on the current application, measurement, and result display.

For a default measurement in the Spectrum mode, the Diagram result display contains the following information, for example:

Label

Information

CF

Center frequency

Span

Frequency span (frequency domain display)

ms/

Time per division (time domain display)

Pts

Number of sweep points or (rounded) number of currently displayed points in zoom mode
