Ref Level

Defines the reference level for the signal levels in the currently active unit (dBm, dBµV, etc.).

For the APD function this value corresponds to the right diagram border. For the CCDF function there is no direct representation of this value on the diagram as the x-axis is scaled relatively to the measured mean power.

For details on the results of the measurement see "APD and CCDF Results".

Remote command: 

CALCulate<n>:​STATistics:​SCALe:​X:​RLEVel
