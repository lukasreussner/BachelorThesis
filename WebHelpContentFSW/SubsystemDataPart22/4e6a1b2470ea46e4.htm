Non-MBSFN Region Length

Selects the length of the MBSFN control data region at the start of the MBSFN subframe.

If you select a region length of '1', the first symbol in an MBFSN subframe carries data of the control channel. All other symbols of an MBSFN region may be used by the PMCH.

If you select a region length of '2', the first two symbols in an MBFSN subframe carry data of the control channel.

Remote command: 

CONFigure[:​LTE]:​DL[:​CC<cc>]:​MBSFn:​AI:​NMRL
