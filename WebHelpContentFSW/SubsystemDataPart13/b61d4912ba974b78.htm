CALCulate<n>:MARKer<m>:FUNCtion:TAERror:RESult? <ResultType>

This command queries the result of a time alignment measurement (see "Time Alignment Error Measurements").

Suffix: 
<n>

Window

<m>

Marker

Query parameters: 
<ResultType>
TAERror

Returns the time offset between the two antenna signals in chips.

Example: 

CALC:MARK:FUNC:TAER:RES? TAER

Usage: Query only
Mode: BTS application only
Manual operation: See "Result List"
