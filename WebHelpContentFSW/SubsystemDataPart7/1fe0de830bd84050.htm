DISPlay[:WINDow<n>]:TRACe<t>:X[:SCALe]:MINimum <Value>

This command defines the value at the bottom of the y-axis.

Suffix: 
<n>

Window

<t>

irrelevant

Parameters:
<Value>
<numeric value>
Default unit: Depends on the result display.
Example: DISP:TRAC:X:AUTO OFF
DISP:TRAC:X:MIN -10DBM
DISP:TRAC:X:MAX -110DBM

The x-axis covers a level range of 100 dB.

Manual operation: See "Scaling the x-axis manually"
