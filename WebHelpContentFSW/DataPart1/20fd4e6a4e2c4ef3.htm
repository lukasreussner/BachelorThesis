Sweep Repeat

Controls a repeated sweep of the E1 and MKPK HI HP model commands (for details on the commands refer to "Reference: GPIB Commands of Emulated HP Models"). If the repeated sweep is OFF, the marker is set without sweeping before.

Note: In single sweep mode, switch off this setting before you set the marker via the E1 and MKPK HI commands in order to avoid sweeping again.

This setting is only available if a HP language is selected (see "Language").

Remote command: 

SYSTem:​RSWeep
