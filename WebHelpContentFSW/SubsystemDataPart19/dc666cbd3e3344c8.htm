Selecting a Common RF Measurement for WLAN Signals

The following commands are required to select a common RF measurement for WLAN signals in a remote environment.

For details on available measurements see "Frequency Sweep Measurements".

The selected measurement must be started explicitly (see "Starting a Measurement")!

Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the Magnitude Capture buffer in window 1 at the top of the screen and the selected result type in window 2 below that.

Use the LAYout commands to change the display (see "Configuring the Result Display").
