Measurement Time

Defines the net, settled measurement length; internally, the R&S FSW Avionics (VOR/ILS) measurements application captures data slightly longer to allow for all filters to settle.

"Auto"

(Default:) The required time (1 s) is determined by the R&S FSW Avionics (VOR/ILS) measurements application. The currently used measurement time is indicated for reference only

"Manual"

The measurement time is defined manually; enter the measurement time in seconds

For a list of available measurement times depending on the "Demodulation Bandwidth", see Table "Available demodulation bandwidths and measurement times for ILS measurements" and Table "Available demodulation bandwidths and measurement times for VOR measurements".

Remote command: 

[SENSe:​]SWEep:​TIME

[SENSe:​]SWEep:​TIME:​AUTO
