Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Configuring the Results > Selecting the Pulse > SENSe:TRACe:MEASurement:DEFine:PULSe:SELected
SENSe:TRACe:MEASurement:DEFine:PULSe:SELected <PulseNumber>

Selects a particular pulse for which the traces, parameters and results are displayed, or queries the number of the selected pulse. The number of the current or all detected pulses can be queried using [SENSe:​]PULSe:​NUMBer? or [SENSe:​]PULSe:​ID?.

Note that this command causes an error if no measurement results are available.

Parameters:
<PulseNumber>
Range: 0  to  number of detected pulses
*RST:0
