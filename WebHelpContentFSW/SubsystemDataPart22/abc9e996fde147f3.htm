Marker

Commands to position markers and query marker positions are the same as in the base unit.

For three-dimensional resulr diagrams, the application supports a couple of additional commands.

List of Commands
CALCulate<n>:MARKer<m>:Z?
CALCulate<n>:MARKer<m>:Z:ALL?
