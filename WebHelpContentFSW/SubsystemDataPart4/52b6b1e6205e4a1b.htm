EVM, Phase Error, Magnitude Error Trace Results

The error vector magnitude (EVM), as well as the phase and magnitude errors are calculated and displayed for each symbol. Thus, the TRAC:DATA? query returns one value per symbol. The number of symbols depends on the burst type, modulation and number of carriers used for transmission, as well as the oversampling factor used internally by the R&S FSW GSM application. The following table provides an overview of the possible number of symbols.
