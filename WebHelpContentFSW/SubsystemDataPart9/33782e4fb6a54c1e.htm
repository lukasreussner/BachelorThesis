CALCulate<n>:LIMit<li>:NAME <Name>

This command selects a limit line that already exists or defines a name for a new limit line.

Suffix: 
<n>

Window

<li>

Limit line

Parameters:
<Name>

String containing the limit line name.

*RST:REM1 to REM8 for lines 1 to 8
