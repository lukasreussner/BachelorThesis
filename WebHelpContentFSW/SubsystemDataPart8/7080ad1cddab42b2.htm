Continuous Gate

Activates or deactivates continuous gating.

This setting is only available if "Gated Trigger" is "On".

If no external trigger is active yet when continuous gating is activated, external trigger 1 is automatically activated as the trigger source.

For details see Gated Measurements.

Remote command: 

[SENSe:​]SWEep:​EGATe:​CONTinuous[:​STATe]
