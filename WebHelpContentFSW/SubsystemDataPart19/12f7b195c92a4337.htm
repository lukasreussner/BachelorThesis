Analysis

The following commands define general result analysis settings concerning the traces and markers in standard WLAN measurements. Currently, only one (Clear/Write) trace and one marker are available for standard WLAN measurements.

Analysis for RF measurements

General result analysis settings concerning the trace, markers, lines etc. for RF measurements are identical to the analysis functions in the Spectrum application except for some special marker functions and spectrograms, which are not available in the R&S FSW WLAN application.
