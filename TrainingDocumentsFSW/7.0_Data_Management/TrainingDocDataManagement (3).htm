Continuous Value Update

If activated, the power sensor data is updated continuously during a sweep with a long sweep time, and even after a single sweep has completed.

This function cannot be activated for individual sensors.

If the power sensor is being used as a trigger (see "Using the power sensor as an external trigger"), continuous update is not possible; this setting is ignored.

Remote command: 

[SENSe:​]PMETer<p>:​UPDate[:​STATe]
