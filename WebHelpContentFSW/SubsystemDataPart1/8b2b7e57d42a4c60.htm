I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Configuring I/Q Analyzer Measurements > Configuring the Data Input and Output > Configuring Input via the Optional Analog Baseband Interface > CALibration:AIQ:DCOFfset:Q
CALibration:AIQ:DCOFfset:Q <Offset>

This command defines a DC offset of the Q input from the Analog Baseband interface (R&S FSW-B71).

Parameters:
<Offset>
numeric value

DC offset

*RST:0
Default unit: V
Example: CAL:AIQ:DCOF:Q 0.001
