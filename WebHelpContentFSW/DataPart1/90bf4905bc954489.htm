Configuring search limits for OBW measurement

The occupied bandwidth of the signal can also be determined within defined search limits instead of for the entire signal.

The OBW measurement uses the same search limits as defined for marker search (see "Search Limits"). However, only the left and right limits are considered.

For step-by-step instructions, see "How to determine the OBW for a multicarrier signal using search limits".

The remote commands required to perform these tasks are described in "Measuring the Occupied Bandwidth".
