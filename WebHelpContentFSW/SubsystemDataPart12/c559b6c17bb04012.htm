Symbols

The pattern definition is a symbol table consisting of one or more symbols. The number of symbols is indicated as the "Size" to the left of the symbol table.

A scrollbar beneath the input area allows you to scroll through the table for long patterns. The numbers beneath the scrollbar indicate the sequential number of the following symbols, from left to right:

  • the first symbol
  • the currently selected symbol
  • the last symbol
Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​DATA
