General Configuration

The following remote control command control general configuration of the application.

The remote control commands to select the result displays for I/Q measurements are described in "Configuring the Screen Layout".

List of Commands
CONFigure[:LTE]:MEASurement
MMEMory:LOAD:IQ:STATe
SYSTem:PRESet:CHANnel[:EXEC]
