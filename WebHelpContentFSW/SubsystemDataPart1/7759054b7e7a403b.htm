INPut<ip>:IQ:OSC:IMPedance:PTYPe <PadType>

Defines the type of matching pad used for impedance conversion for analog baseband input.

For RF input, use the INPut<ip>:​IMPedance:​PTYPe command.

For analog baseband input without an oscilloscope, use the INPut<ip>:​IQ:​IMPedance:​PTYPe command.

Suffix: 
<ip>
1 | 2

irrelevant

INP:IQ:OSC:IMP:PTYP MLP
