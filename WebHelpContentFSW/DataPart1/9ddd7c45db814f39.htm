DISPlay[:WINDow<n>]:STATe <State>

This command changes the display state of the selected measurement window.

Note that this command is maintained for compatibility reasons only. Use the LAYout commands for new remote control programs

(See "Working with Windows in the Display").

Suffix: 
<n>

Window

Parameters:
<State>
ON | OFF | 1 | 0
OFF | 0

The window is closed.

*RST:0
