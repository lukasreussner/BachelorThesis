Transm. Comb. k_TC

Defines the transmission comb kTC.

The transmission comb. is a UE specific parameter. For more information refer to 3GPP TS 36.211, chapter 5.5.3.2 "Mapping to Physical Resources" for the Sounding Reference Signal.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SRS:​TRComb
