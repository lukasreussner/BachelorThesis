Detector

Defines the detector used to combine overlapping FFT frames for the spectrogram result display.

"Sum"

Calculates the sum of all values in one sample point

"Average"

Calculates the linear average of all values in one sample point

"RMS"

Calculates the RMS of all values in one sample point

"Maximum"

Determines the largest of all values in one sample point

"Minimum"

Determines the minimum of all values in one sample point

"Sample"

Selects the last measured value for each sample point

Remote command: 

[SENSe:​][WINDow<n>:​]SGRam|SPECtrogram:​DETector:​FUNCtion
