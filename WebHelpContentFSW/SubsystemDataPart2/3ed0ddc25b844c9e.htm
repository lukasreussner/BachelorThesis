SENSe:TRACe:MEASurement:DEFine:RRANge:AUTO <State>

If enabled, the result range length is determined automatically according to the width of the selected pulse (see SENSe:​TRACe:​MEASurement:​DEFine:​PULSe:​SELected).

Parameters:
<State>
OFF

Switch the function off

ON

Switch the function on

ONCE

Execute the function once and then switch it off
