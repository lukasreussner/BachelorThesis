How to Work With a Power Sensor

The following step-by-step instructions demonstrate how to set up a power sensor. For details on individual functions and settings see "Power Sensor Settings".

The remote commands required to perform these tasks are described in "Working with Power Sensors".

Power sensors can also be used to trigger a measurement at a specified power level, e.g. from a signal generator.

This is described in "How to Configure a Power Sensor as an External (PSE) Trigger".
