OUTPut:TRIGger<port>:LEVel <Level>

This command defines the level of the (TTL compatible) signal generated at the trigger output.

This command works only if you have selected a user defined output with OUTPut:​TRIGger<port>:​OTYPe.

Suffix: 
<port>

Selects the trigger port to which the output is sent.

2 = trigger port 2 (front)

3 = trigger port 3 (rear)

Parameters:
<Level>
HIGH

5 V

LOW

0 V

*RST:LOW
Example: 

See Programming Example: Performing a Basic Frequency Sweep.

Example: OUTP:TRIG2:LEV HIGH
Manual operation: See "Level"
