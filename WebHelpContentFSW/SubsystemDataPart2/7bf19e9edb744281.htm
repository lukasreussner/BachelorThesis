CALCulate<n>:TRACe<t>[:VALue] <EvalType>

Defines which signal component (I/Q) is evaluated in which trace for the "Pulse I and Q" result display. This setting is not available for any other result displays. By default, the I component is displayed by trace 1, while the Q component is displayed by trace 4.

Suffix: 
<n>

Window

<t>

Trace

Parameters:
<EvalType>
ITIMe | QTIMe
ITIMe

The I component is evaluated by the selected trace.

QTIMe

The Q component is evaluated by the selected trace.

Example: CALC2:TRAC2 QTIM

Trace 2 in window 2 evaluates the Q component of the signal.

Manual operation: See "Evaluation"
