Phase Noise (R&S FSW-K40) > Analysis > Using Limit Lines > Selecting Standard Limit Lines > Select Limit Line > View Filter
View Filter

Turns filter for the list of limit lines on and off.

By default, the list includes all limit lines that are stored on the R&S FSW.

"Show Compatible"

Filters the list of limit lines by compatibility.

If on, the list includes only those limit lines that are compatible to the current measurement setup.

"Show Lines For PNoise"

Filters the list of limit lines by compatibility to phase noise measurements.

If on, the list includes only those limit lines that are compatible to phase noise measurements.
