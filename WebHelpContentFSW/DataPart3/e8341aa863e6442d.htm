IF Power/BB Power

Depending on the input source type, this softkey defines triggering of the measurement using:

  • The baseband power (for baseband input via the optional Digital Baseband Interface or the optional Analog Baseband interface)
  • The third intermediate frequency (RF input)
    The R&S FSW is triggered as soon as the trigger threshold is exceeded around the selected frequency (= start frequency in the frequency sweep). The trigger threshold depends on the defined trigger level, as well as on the RF attenuation and preamplification. A reference level offset, if defined, is also considered. The trigger bandwidth at the intermediate frequency depends on the RBW and sweep type. For details on available trigger levels and trigger bandwidths see the data sheet.
  • The "robust width" of a pulse on the oscilloscope's input channel, when using the optional 2 GHz / 5 GHz bandwidth extension (R&S FSW-B2000/B5000)

Note: Be aware that in auto sweep type mode, due to a possible change in sweep types, the trigger bandwidth may vary considerably for the same RBW setting.
