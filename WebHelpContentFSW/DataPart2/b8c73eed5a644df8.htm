Standard Files

Access: "Overview" > "Select Measurement" > "Spectrum Emission Mask" > "Standard Files"

You can save the current measurement settings as a user-defined standard (XML file), or load stored measurement settings. Furthermore, you can delete an existing settings file.

For details, see "How to Manage SEM Settings Files".

Standard files for sub blocks (Multi-SEM measurements)

If more than one sub blocks are defined, the "Standard Files" tab and softkey are not available. To load a standard file for an individual sub block, use the "Standard / MSR Settings" setting in the "Sub Blocks" tab.
