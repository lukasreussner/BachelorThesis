Configuring Spur Search Spans

In the "Directed Search Settings" you define the frequencies at which spurs are to be expected: the search is directed to those frequencies. Furthermore, you define the criteria by which a spur is detected. For each frequency, the following parameters must be defined.

└ Number
└ Frequency
└ Search Span
└ Detection Threshold
└ Minimum Spur SNR
└ Detection Mode
└ Conflict
