Loop Bandwidth

Defines the speed of internal synchronization with the reference frequency. The setting requires a compromise between performance and increasing phase noise.

For a variable external reference frequency with a narrow tuning range (+/- 0.5 ppm), the loop bandwidth is fixed to 0.1 Hz and cannot be changed.
