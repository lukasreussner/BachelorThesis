CONFigure[:MS]:AUTO:TRIGger ONCE

This command is used to perform a single measurement that determines the trigger offset automatically.

This command is not available in MSRA mode.

Note that in Signal and Spectrum Analyzer mode, if the Sequencer is active, this command cannot be aborted via the ABORt command! This can lead to a hang up situation when no trigger signal is available or the trigger level is not set correctly. Use a device clear to abort the operation correctly.

Example: CONF:AUTO:TRIG ONCE
Usage: Setting only
Manual operation: See "Automatic Trigger Offset"
