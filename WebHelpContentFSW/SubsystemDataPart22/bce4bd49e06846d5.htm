Beamforming Selection

Filters the displayed results to include only certain antenna port(s).

The availability of antenna ports depends on the number of transmission antennas and the number of beamforming layers you are testing.

In addition, you can select the antenna port for each type of reference signal separately (UE reference signal, cell reference signal and CSI reference signal).

Remote command: 

Cell RS: CONFigure[:​LTE]:​DL[:​CC<cc>]:​BF:​AP:​CELL

UE RS: CONFigure[:​LTE]:​DL[:​CC<cc>]:​BF:​AP[:​UERS]

CSI RS: CONFigure[:​LTE]:​DL[:​CC<cc>]:​BF:​AP:​CSI
