Scale Mode

For Gain display only: Defines the scaling method for the y-axis.

For more information see "Scaling".

"Logarithmic"

Logarithmic scaling (only available for logarithmic units - dB...)

"Linear"

Linear scaling in percentages from 0 to 100

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​MODE
