CDP Average

The Code Domain Analysis is averaged over all slots in the set. For channel types Data and Preamble this calculation assumes that preambles of different lengths do not occur in the slots. If active, "ALL" is displayed in the "Slot" field in the channel bar.

This function is required by the 1xEV-DO standard.

Remote command: 

[SENSe:​]CDPower:​AVERage
