INPut<ip>:SELect <Source>

This command selects the signal source for measurements, i.e. it defines which connector is used to input data to the R&S FSW.

Suffix: 
<ip>

irrelevant

Parameters:
<Source>
RF

Radio Frequency ("RF INPUT" connector)

*RST:RF
