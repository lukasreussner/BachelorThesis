Configuration According to Standards

Access: "Overview" > "Setup Standard"

Various predefined settings files for common standards are provided for use with the Analog Demodulation application. In addition, you can create your own settings files for user-specific measurements.

For details on which settings are defined and an overview of predefined standards see "Predefined Standards and Settings".
