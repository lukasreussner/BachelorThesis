Measurement Time

The measurement time and "Record Length" are interdependent and define the amount of data to be captured.

For details see "Data Acquisition".

The maximum measurement time in the R&S FSW Transient Analysis application is limited only by the available memory ("memory limit reached" message is shown in status bar). Note, however, that increasing the measurement time (and thus reducing the available memory space) may restrict the number of measurement channels that can be activated simultaneously on the R&S FSW.

Remote command: 

[SENSe:​]MTIMe
