Modulation Mapping

The available mapping types depend on the "Modulation Type" and "Modulation Order".

Note: When using PRBS generators, select the SMx mapping, which corresponds to the mapping used by R&S SMx signal generators.

For more information on the modulation mapping, refer to "Symbol Mapping"

Remote command: 

[SENSe:​]DDEMod:​MAPPing[:​VALue]
