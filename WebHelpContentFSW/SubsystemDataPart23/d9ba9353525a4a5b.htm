CONFigure[:LTE]:UL[:CC<cc>]:PUCCh:NPAR <Format>

This command defines the N_PUCCH parameter in the PUCCH structure settings.

Suffix: 
<cc>

Component Carrier

Parameters:
<Format>
<numeric value>
<numeric value>
AUTO

Determines the N_PUCCH based on the measurement.

SUBF

Selects the definition of N_PUCCH on subframe level.

*RST:0
Example: CONF:UL:PUCC:NPAR 2

Sets N_PUCCH to 2.

Manual operation: See "N_PUCCH"
