VSA (R&S FSW-K70) > Remote Commands for VSA > Configuring VSA > Signal Description > Modulation > [SENSe:]DDEMod:MSK:FORMat
[SENSe:]DDEMod:MSK:FORMat <MSKformat>

This command defines the specific demodulation order for MSK.

Setting parameters: 
<MSKformat>
TYPE1 | TYPE2 | NORMal | DIFFerential
TYPE1 | NORMal

Demodulation order MSK is used.

TYPE2 | DIFFerential

Demodulation order DMSK is used.

*RST:TYPE1
Manual operation: See "Modulation Order"
