Modulation

Access: "Overview" > "Signal Description" > "Modulation"

or: [MEAS CONFIG] > "Signal Description" > "Modulation"

The "Modulation" settings contain modulation and transmit filter settings. A live preview of the Constellation I/Q trace using the currently defined settings is displayed at the bottom of the dialog box to visualize the changes to the settings. The preview area is not editable directly.

The modulation settings vary depending on the selected modulation type; in particular, FSK modulation provides some additional settings.
