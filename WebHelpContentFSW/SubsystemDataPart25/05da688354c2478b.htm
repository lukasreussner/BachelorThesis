EVM vs Symbol x Carrier

The EVM vs Symbol x Carrier shows the EVM for each carrier in each symbol.

The horizontal axis represents the symbols. The vertical axis represents the carriers. Different colors in the diagram area represent the EVM. The color map for the power levels is provided above the diagram area.

Remote command: 

Selecting the result display: LAY:ADD ? '1',LEFT,EVSC

Querying results: TRACe:DATA?
