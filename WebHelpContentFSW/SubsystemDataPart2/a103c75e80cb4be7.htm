Retrieving Marker Results

The following commands are required to retrieve marker results.

Useful commands for retrieving marker results described elsewhere:
  • CALCulate<n>:​DELTamarker<m>:​X
  • CALCulate<n>:​MARKer<m>:​X
Remote commands exclusive to retrieving marker results:
List of Commands

CALCulate<n>:DELTamarker<m>:Y?
