Link Markers Across Windows

Turns marker coupling across result windows on and off.

When you link markers, moving a marker in one result display moves the marker to the same sample in another window. This is useful to compare results in result displays that have different information on their x- and y-axis (for example the AM/AM and AM/PM results).

Remote command: 

CALCulate<n>:​MARKer<m>:​LINK
