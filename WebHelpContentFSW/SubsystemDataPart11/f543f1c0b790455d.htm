CALCulate<n>:CHRDetection:PDEViation:OFFSet:BEGin <Time>

Defines the beginning of the measurement range for phase deviation results as an offset in seconds from the chirp start. This command is only available if the reference is EDGE (see CALCulate<n>:​CHRDetection:​FDEViation:​REFerence).

For details on the measurement range parameters see "Measurement Range".

Suffix: 
<n>
1..n

irrelevant

Parameters:
<Time>
Default unit: S
Example: CALC:CHRD:PDEV:RANG EDGE
CALC:CHRD:PDEV:OFFS:BEG 3e-6 S
Manual operation: See "Offset Begin / Offset End"
