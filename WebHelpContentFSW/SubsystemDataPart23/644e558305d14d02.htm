MMEMory:LOAD[:CC<cc>]:TMOD:DL <TestModel>

This command loads an EUTRA test model (E-TM).

The test models are in accordance with 3GPP TS 36.141.

Suffix: 
<cc>

Component Carrier

Setting parameters: 
<TestModel>
'E-TM1_1__20MHz'

EUTRA Test Model 1.1 (E-TM1.1)

'E-TM1_2__20MHz'

EUTRA Test Model 1.2 (E-TM1.2)

'E-TM2__20MHz'

EUTRA Test Model 2 (E-TM2)

'E-TM2A__20MHz'

EUTRA Test Model 2a (E-TM2)

'E-TM3_1__20MHz'

EUTRA Test Model 3.1 (E-TM3.1)

'E-TM3_1A__20MHz'

EUTRA Test Model 3.1a (E-TM3.1)

'E-TM3_2__20MHz'

EUTRA Test Model 3.2 (E-TM3.2)

'E-TM3_3__20MHz'

EUTRA Test Model 3.3 (E-TM3.3)

To select a test model for a different bandwidth, replace "20MHz" with either "1_4MHz", "3MHz", "5MHz", "10MHz" or "15MHz".

Example: MMEM:LOAD:TMOD:DL 'E-TM2__10MHz'

Selects test model 2 for a 10 MHz bandwidth.

Usage: Setting only
