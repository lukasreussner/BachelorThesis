Starting the Application

The amplifier measurement application adds a new type of measurement to the R&S FSW.

To activate the amplifier application

  1. Press the MODE key on the front panel of the R&S FSW.

    A dialog box opens that contains all operating modes and applications currently available on your R&S FSW.

  2. Select the "Amplifier" item.

    The R&S FSW opens a new measurement channel for the amplifier application.

    All settings specific to amplifier measurements are in their default state.

For details, see Applications, Measurement Channels, and Operating Modes.
