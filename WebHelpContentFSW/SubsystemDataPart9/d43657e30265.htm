Phase Noise (R&S FSW-K40) > Remote Control Commands for Phase Noise Measurements > Using Limit Lines > Using Phase Noise Limit Lines > CALCulate:PNLimit:TRACe
CALCulate:PNLimit:TRACe <Trace>

This command selects the trace to assign a phase noise limit line to.

Parameters:
<Trace>
Range: 1  to  6
*RST:-
Example: CALC:PNL:TRAC 1

Assigns the phase noise limit line to trace 1.

Manual operation: See "Selected Traces"
