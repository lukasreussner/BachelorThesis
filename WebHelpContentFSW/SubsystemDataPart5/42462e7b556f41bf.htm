CALCulate<n>:AVIonics:SHD:RESult?

Queries the result of the harmonic distortion measurement performed at the position of the marker H1 (defined by CALCulate<n>:​AVIonics:​SHD:​FREQuency).

For details see "Avionics Parameters".

Suffix: 
<n>
1..n

irrelevant

Return values: 
<Value>

Distortion measured at the frequency specified by CALCulate<n>:​AVIonics:​SHD:​FREQuency in relation to the modulation measured at the fundamental frequency (CALCulate<n>:​AVIonics:​THD:​FREQuency:​FUNDament) in percent

Range: 0  to  100
Default unit: percent
Example: CALC:AVI:THD:FREQ:FUND '90'
CALC:AVI:SHD:FREQ 100MHZ
CALC:AVI:SHD:RES?
Usage: Query only
Manual operation: See "DIST"
See "Fixed markers (H1, F1, F2)"
