[SENSe:]MIXer<x>:LOSS:TABLe:HIGH <FileName>

This command defines the file name of the conversion loss table to be used for the high (second) range.

Suffix: 
<x>

irrelevant

Parameters:
<FileName>

String containing the path and name of the file.

Example: MIX:LOSS:TABL:HIGH 'MyCVLTable'
Manual operation: See "Conversion Loss"
