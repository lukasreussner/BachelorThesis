Configuring Data Acquisition

The following commands are required to configure the actual data acquisition.

Configuring data acquisition is only possible for the MSRA Master channel. In MSRA slave application channels, these commands define the analysis interval (see "Commands Specific to MSRA Slave Applications"). Be sure to select the correct measurement channel before executing these commands.
