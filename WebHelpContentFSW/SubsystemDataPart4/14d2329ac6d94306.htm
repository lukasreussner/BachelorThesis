Trigger Source

Defines the trigger source. If a trigger source other than "Free Run" is set, "TRG" is displayed in the channel bar and the trigger source is indicated.

Note: The GSM measurements can be performed in "Free Run" (untriggered) mode; however, an external trigger or a power trigger can speed up measurements.
For more information see "Trigger settings".

Note: Trigger source for MSRA Master. Any trigger source other than "Free Run" defined for the MSRA Master is ignored when determining the frame start in the R&S FSW GSM application (see "Trigger settings"). For this purpose, the trigger is considered to be in "Free Run" mode.

Remote command: 

TRIGger[:​SEQuence]:​SOURce

 
└ Free Run
└ External Trigger 1/2/3
└ IF Power
└ RF Power
