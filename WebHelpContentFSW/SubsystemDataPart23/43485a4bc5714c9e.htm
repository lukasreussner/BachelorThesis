[SENSe:][LTE:]FRAMe:SSUBframe <State>

This command turns the analysis of a single subframe only on and off.

Parameters:
<State>
ON | OFF
*RST:OFF
Example: FRAM:SSUB ON

Evaluates a single subframe only.

Manual operation: See "Single Subframe Mode"
