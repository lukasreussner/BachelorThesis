Reference Frequency

Selects the source of the generator reference frequency.

The internal reference is that of the signal generator itself. When you select an external reference, you can use another frequency reference, for example that of the R&S FSW.

Remote command: 

CONFigure:​GENerator:​EXTernal:​ROSCillator

CONFigure:​GENerator:​EXTernal:​ROSCillator:​LEDState?
