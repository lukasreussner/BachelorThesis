Evaluation Methods

Access: "Overview" > "Display Config"

Or: MEAS

The data that was measured by the R&S FSW Spurious measurements application can be evaluated using various different methods. All evaluation methods available for the Spurious measurements are displayed in the selection bar in SmartGrid mode.

For details on working with the SmartGrid, see Laying out the Result Display with the SmartGrid.

By default, the following result displays are provided for Spurious measurements:
  • "Spectral Overview"
  • "Spurious Detection Spectrum"
  • "Spurious Detection Table"
