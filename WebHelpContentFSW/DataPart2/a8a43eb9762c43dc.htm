Common Mode Offset

Sets the common mode offset. The setting is only available if a differential probe is connected to the R&S FSW.

If the probe is disconnected, the common mode offset of the probe is reset to 0.0 V.

For details, see See Also.

Remote command: 

[SENSe:​]PROBe<p>:​SETup:​CMOFfset
