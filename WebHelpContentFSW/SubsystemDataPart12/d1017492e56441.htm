VSA (R&S FSW-K70) > Remote Commands for VSA > Configuring VSA > Signal Description > Modulation > [SENSe:]DDEMod:QAM:FORMat
[SENSe:]DDEMod:QAM:FORMat <QAMformat>

This command defines the specific demodulation order for QAM.

Setting parameters: 
<QAMformat>
NORMal | DIFFerential | NPI4 | MNPI4
NORMal

Demodulation order QAM is used.

DIFFerential

Demodulation order DQAM is used.

NPI4

Demodulation order π/4-16QAM is used.

MNPI4

Demodulation order -π/4-32QAM is used.

*RST:NORMal
Manual operation: See "Modulation Order"
