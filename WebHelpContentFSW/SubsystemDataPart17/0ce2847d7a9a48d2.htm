Operation Mode

The operation mode is used for the channel search.

"Access"

The signal can contain only PICH (always available) and DATA channels.

"Traffic"

The signal can contain all channels (PICH/RRI/DATA/ACK and DRC). PICH and RRI are always available.

Remote command: 

[SENSe:​]CDPower:​OPERation
