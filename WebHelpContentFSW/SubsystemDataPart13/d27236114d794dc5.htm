LAYout:MOVE[:WINDow] <WindowName>, <WindowName>, <Direction>
Setting parameters: 
<WindowName>

String containing the name of an existing window that is to be moved.

By default, the name of a window is the same as its index. To determine the name and index of all active windows in the active channel, use the LAYout:​CATalog[:​WINDow]? query.

<WindowName>

String containing the name of an existing window the selected window is placed next to or replaces.

By default, the name of a window is the same as its index. To determine the name and index of all active windows in the active channel, use the LAYout:​CATalog[:​WINDow]? query.

<Direction>
LEFT | RIGHt | ABOVe | BELow | REPLace

Destination the selected window is moved to, relative to the reference window.

Example: LAY:MOVE '4','1',LEFT

Moves the window named '4' to the left of window 1.

Example: LAY:MOVE '1','3',REPL

Replaces the window named '3' by window 1. Window 3 is deleted.

Usage: Setting only
