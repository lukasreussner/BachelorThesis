[SENSe:]CDPower:LEVel:ADJust

This command adjusts the reference level to the measured channel power. This ensures that the settings of the RF attenuation and the reference level are optimally adjusted to the signal level without overloading the R&S FSW or limiting the dynamic range by an S/N ratio that is too small.

Note that this command is retained for compatibility reasons only. For new R&S FSW programs use [SENSe:​]ADJust:​LEVel.
