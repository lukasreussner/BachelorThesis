SYSTem:COMMunicate:PRINter:SELect<device> <PrinterName>

This command selects the printer that processes jobs sent by the R&S FSW.

Use HCOPy:​DESTination<device> to select another output destination.

Suffix: 
<device>
1 | 2

Printing device.

Parameters:
<PrinterName>

String containing the printer name.

Use

•SYSTem:​COMMunicate:​PRINter:​ENUMerate:​FIRSt? and

•SYSTem:​COMMunicate:​PRINter:​ENUMerate[:​NEXT]?

to query all available printers.

*RST:NONE
Manual operation: See "Printer Name"
