SEM Measurements

Access: "Overview" > "Select Measurement"

Or: [MEAS] > "Select Measurement"

When you activate a measurement channel in IEEE 802.11ad mode, an IQ measurement of the input signal is started automatically (see "IEEE 802.11ad Modulation Accuracy Measurement"). However, some parameters specified in the IEEE 802.11ad standard require a better signal-to-noise level or a smaller bandwidth filter than the default measurement on I/Q data provides and must be determined in separate measurements based on RF data (see "SEM Measurements"). In these measurements, demodulation is not performed.

The R&S FSW 802.11ad application uses the functionality of the R&S FSW base system (Spectrum application) to perform the IEEE 802.11ad SEM measurements. Some parameters are set automatically according to the IEEE 802.11ad standard the first time a measurement is selected (since the last [PRESET] operation). These parameters can be changed, but are not reset automatically the next time you re-enter the measurement. Refer to the description of each measurement type for details.

The main measurement configuration menus for the IEEE 802.11ad SEM measurements are identical to the Spectrum application.

For details refer to Common Measurement Settings.
