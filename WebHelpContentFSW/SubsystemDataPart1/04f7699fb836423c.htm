[SENSe:]CORRection:METHod <Type>

This command selects the type of measurement to be performed with the external generator.

This command is only valid if External Generator Control (R&S FSW-B10) is installed and active.

Parameters:
<Type>
REFLection

Selects reflection measurements.

TRANsmission

Selects transmission measurements.

*RST:TRANsmission
Example: CORR:METH TRAN

Sets the type of measurement to "transmission".
