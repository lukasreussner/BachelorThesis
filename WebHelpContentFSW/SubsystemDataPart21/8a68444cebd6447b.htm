FETCh:PLCMessages:PROFile<n>:ASSignment?

Queries the profile to subcarrier assignment from the PLC messages. the result is a comma-separated list of assignments with the following syntax:

<Modulation>,<SubCarriers>

Suffix: 
<n>
1..n

currently irrelevant; only profile A is queried

Return values: 
<Result>
Example: FETC:PLCM:PROF:ASS?
//Result:
QAM1024,1108:1:1207,1145,1218,1291,
ZEROBIT,1208:1:1608,
QAM1024,1609:1:2987
Usage: Query only
