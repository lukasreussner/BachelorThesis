Multi-Carrier Group Delay (R&S FSW-K17) > Remote Commands to Perform Multi-Carrier Group Delay Measurements > Measuring, Loading and Storing Calibration Data > MMEMory:STORe:MCGD:RCALibration
MMEMory:STORe:MCGD:RCALibration <File>

This command stores the current calibration data to the selected file.

Setting parameters: 
<File>

path and file name of the .csv file that will contain the calibration data

Usage: Setting only
Manual operation: See "Storing Calibration Data"
