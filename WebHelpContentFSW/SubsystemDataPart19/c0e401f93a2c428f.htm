Reference Frequency Coupling

For simultaneous MIMO setups, you can set the reference frequency source for all slave devices to the same setting as the master device.

Tip: the LED symbol in the "Ref Status" column indicates the state of the Rx antenna:

Color

State

gray

reference frequency off or IP address not available/valid

red

reference frequency on and IP address valid, but no reference frequency detected

green

reference frequency on and IP address accessible

"Slaves Reference same as Master setting"

Both the master and all slaves use the same reference, according to the setting at the master.

"Slaves: External; Master: Internal"

The slave devices are set to use the external reference from the master. The master device uses its internal reference.

Configure the master to send its reference frequency to all slave devices via one of its [REF OUTPUT] connectors.

(See Reference Frequency Output).

"Off"

Both the master and slave devices use their own internal references; the frequencies are not coupled.

Remote command: 

CONFigure:​WLAN:​ANTMatrix:​SOURce:​ROSCillator:​SOURce
