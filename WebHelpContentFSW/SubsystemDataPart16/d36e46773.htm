[SENSe:]CDPower:ICTReshold <ThresholdLevel>

This command defines the minimum power which a single channel must have compared to the total signal in order to be regarded as an active channel. Channels below the specified threshold are regarded as "inactive".

Parameters:
<ThresholdLevel>
Range: -100  to  10
*RST:-40 dB
Default unit: dB
Example: 

CDP:ICTR -10

Sets the minimum power threshold to -10 dB.

Manual operation: See "Inactive Channel Threshold"
