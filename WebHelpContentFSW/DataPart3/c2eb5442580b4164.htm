[SENSe:]MIXer<x>:PORTs <PortType>

This command selects the mixer type.

Suffix: 
<x>

irrelevant

Parameters:
<PortType>
2 | 3
2

Two-port mixer.

3

Three-port mixer.

*RST:2
Example: MIX:PORT 3
Example: 

See "Programming Example: Working with an External Mixer".

Manual operation: See "Mixer Type"
