CALCulate<n>:LIMit<k>:ESPectrum<sb>:PCLass<PClass>:LIMit[:STATe] <State>

This command selects the limit check mode for each power class.

Suffix: 
<n>, <k>

irrelevant

<sb>
1 | 2 | 3

Sub block in a Multi-SEM measurement

<PClass>
1...4

power class

Parameters:
<State>
ABSolute

Evaluates only limit lines with absolute power values

RELative

Evaluates only limit lines with relative power values

AND

Evaluates limit lines with relative and absolute power values. A negative result is returned if both limits fail.

OR

Evaluates limit lines with relative and absolute power values. A negative result is returned if at least one limit failed.

*RST:REL
Example: CALC:LIM:ESP:PCL:LIM ABS
