General Instrument Setup > Frequency Response Correction (R&S FSW-K544) > User-defined Frequency Response Correction Settings > Save Settings
Save Settings

Stores a saveset for a user-defined frequency response correction scenario. As for all instrument settings, a .dfl file is created for the correction data. Note that only the settings defined in this dialog box are stored, not the contents of the files themselves.

The default storage directory is C:\R_S\INSTR\USER\FResponse.

Remote command: 

[SENSe:​]CORRection:​FRESponse:​USER:​STORe
