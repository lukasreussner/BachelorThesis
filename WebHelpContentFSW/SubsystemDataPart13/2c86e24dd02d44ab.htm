Table Configuration

You can configure which parameters are displayed in the Channel Table by double-clicking the table header.

A "Table Configuration" dialog box is displayed in which you can select the columns to be displayed.

By default, only active channels are displayed. In order to display all channels, including the inactive ones, enable the "Show Inactive Channels" option.

The following parameters of the detected channels are determined by the CDP measurement and can be displayed in the Channel Table evaluation. (For details see "Code Domain Parameters".)
