Transient Analysis (R&S FSW-K60) > Configuration > Hop / Chirp Selection > Select Hop / Select Chirp
Select Hop / Select Chirp

Defines the individual hop or chirp from the current capture buffer for which results are calculated and displayed.

These settings are only available if at least one of the additional options R&S FSW-K60C/-K60H are installed.

Remote command: 

CALCulate<n>:​CHRDetection:​SELected

CALCulate<n>:​HOPDetection:​SELected
