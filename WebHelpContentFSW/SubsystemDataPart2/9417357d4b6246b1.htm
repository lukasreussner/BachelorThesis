Reference for Pulse-Pulse Measurements

Reference pulse on which relative pulse results are based (e.g. for traces normalized to reference pulse, see "Normalizing Traces").

"Fixed"

A fixed pulse number

Relative results for the specified pulse number itself are not valid and are indicated as "...".

"Selected"

The currently selected pulse (see "Pulse Selection")

Relative results for the selected pulse itself are not valid and are indicated as "...".

If you change the value for the reference pulse here, the "Pulse Selection" value is adapted accordingly, and vice versa.

"Before Pulse"

The nth pulse before the currently evaluated pulse, where n is the specified number

No values are available for the first n pulses, as no valid reference pulse is available. These results are indicated as "...".

For example, a value of 2 will use row 1 as the reference row for Pulse-Pulse results for pulse number 3. In this case, pulse numbers 1 and 2 will not have a valid reference row and the Pulse-Pulse results will be invalid for these rows.

"After Pulse"

The nth pulse after the currently evaluated pulse, where n is the specified number

No values are available for the last n pulses, as no valid reference pulse is available. These results are indicated as "...".

For example, a value of 2 will use row 5 as the reference row for Pulse-Pulse results for pulse number 3. In this case, the last two pulse rows will not have a valid reference row and the Pulse-Pulse results will be invalid for these rows.
