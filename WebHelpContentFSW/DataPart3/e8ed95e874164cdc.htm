External Trigger 1/2/3

Data acquisition starts when the TTL signal fed into the specified input connector meets or exceeds the specified trigger level.

(See "Trigger Level (Repetition Interval)").

Note: The "External Trigger 1" softkey automatically selects the trigger signal from the TRIGGER 1 INPUT connector on the front panel.

For details, see "TRIGGER INPUT / OUTPUT" and "TRIGGER 3 INPUT/ OUTPUT".

"External Trigger 1"

Trigger signal from the TRIGGER 1 INPUT connector.

"External Trigger 2"

Trigger signal from the TRIGGER 2 INPUT / OUTPUT connector.

Note: Connector must be configured for "Input" in the "Output" configuration

(See "Trigger Input/Output Settings").

"External Trigger 3"

Trigger signal from the TRIGGER 3 INPUT/ OUTPUT connector on the rear panel.

Note: Connector must be configured for "Input" in the "Output" configuration.

(See "Trigger Input/Output Settings").

Remote command: 

TRIG:SOUR EXT, TRIG:SOUR EXT2

TRIG:SOUR EXT3

See TRIGger[:​SEQuence]:​SOURce

SWE:EGAT:SOUR EXT for gated triggering, see [SENSe:​]SWEep:​EGATe:​SOURce
