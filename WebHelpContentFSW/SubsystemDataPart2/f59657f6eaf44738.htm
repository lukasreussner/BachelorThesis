Activating a limit check for a parameter

To activate a limit check for the selected parameter, set "Limit On/Off" to "ON".

Note: If a limit is defined for a parameter that is displayed in a "Parameter Trend" diagram, the "Auto Scale Once" function is not available for the axis this parameter is displayed on (see also "Automatic Grid Scaling").

Remote command: 

CALCulate<n>:​TABLe:​<ParameterGroup>:​<Parameter>:​LIMit:​STATe
