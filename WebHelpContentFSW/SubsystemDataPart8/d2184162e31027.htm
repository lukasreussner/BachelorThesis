[SENSe:]CONFigure:MODE:SYSTem:IF:FREQuency <Frequency>

This command defines the frequency for DUTs with a fixed IF.

Parameters:
<Frequency>
Range: 0 Hz  to  100 GHz
*RST:10 MHz, if frequency converting mode has been selected
Example: CONF:MODE:SYST:IF:FREQ 1GHZ

Defines a fixed IF of 1 GHz.

Manual operation: See "IF Fixed"
