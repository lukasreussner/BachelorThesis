CONFigure:EQUalizer:TRAin

This command initiates a training sequence for the equalizer filter.

Note that you have to synchronize the measurement before you can initiate a training sequence.

Prerequisites for this command
  • Define a filter length (CONFigure:EQUalizer:FILTer:LENGth).
Usage: Event
Manual operation: See "Using the equalizer"
