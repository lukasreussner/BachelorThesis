SENSe:TRACe:MEASurement:DEFine:FREQuency:OFFSet <Offset>

Defines a known frequency offset to be corrected in the pulse acquisition data.

Use the SENSe:​TRACe:​MEASurement:​DEFine:​FREQuency:​OFFSet:​AUTO to define the frequency offset automatically.

Parameters:
<Offset>
*RST:0
Default unit: HZ
Manual operation: See "Frequency Offset Value"
