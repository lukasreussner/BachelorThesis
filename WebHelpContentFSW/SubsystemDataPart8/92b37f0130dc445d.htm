Auto Level Range

Defines the maximum expected gain of the DUT.

The application uses the auto level range to determine the reference level automatically if the 2nd stage correction is on.

Make sure the range does not exceed the actual gain of the DUT by more than 10 dB.

Remote command: 

SYSTem:​CONFigure:​DUT:​GAIN
