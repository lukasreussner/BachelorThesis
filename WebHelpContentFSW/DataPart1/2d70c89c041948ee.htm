*IDN?

Identification

Returns the instrument identification.

Return values: 
<ID>
"Rohde&Schwarz,<device type>,<part number>/<serial number>,<firmware version>"
Example: 

Rohde&Schwarz,FSW-26,1312.8000K26/100005,1.30

Usage: Query only
Manual operation: See "*IDN Format"
