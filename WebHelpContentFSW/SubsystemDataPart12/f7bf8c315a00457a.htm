STATus:QUESTionable:MODulation<n>:PHASe Register

This register comprises information about limit violations in Phase Error evaluation. It can be queried with commands STATus:QUEStionable:MODulation<n>:PHASe:CONDition and STATus:QUEStionable:MODulation<n>:PHASe[:EVENt].

Bit No

Meaning

0

Error in current RMS value

1

Error in mean RMS value

2

Error in peak RMS value

3-4

These bits are not used

5

Error in current peak value

6

Error in mean peak value

7

Error in peak peak value

8-15

These bits are not used
