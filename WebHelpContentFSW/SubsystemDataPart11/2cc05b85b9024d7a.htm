Automatic vs. Manual Hop/Chirp State Detection

By default, the R&S FSW Transient Analysis application automatically detects the existing hop/chirp states in a pre-measurement. For an initial overview of the signal at hand this detection is usually sufficient. For more accurate results, particularly if the input signal is known in advance, the nominal frequency or chirp values can be defined manually.
