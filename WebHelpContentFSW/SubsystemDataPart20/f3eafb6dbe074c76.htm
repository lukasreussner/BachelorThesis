802.11ad measurements application (R&S FSW-K95) > Remote Commands for IEEE 802.11ad Measurements > Configuring the IEEE 802.11ad Modulation Accuracy Measurement > Configuring the Data Input and Output > Using External Mixers > Basic Settings > [SENSe:]MIXer<x>:LOPower
[SENSe:]MIXer<x>:LOPower <Level>

This command specifies the LO level of the external mixer's LO port.

Suffix: 
<x>

irrelevant

Parameters:
<Level>
numeric value
Range: 13.0 dBm  to  17.0 dBm
Increment: 0.1 dB
*RST:15.5 dBm
Example: MIX:LOP 16.0dBm
Example: 

See "Programming Example: Working with an External Mixer".
