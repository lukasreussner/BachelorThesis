Restoring the User Settings to Default Colors

In addition to the predefined color settings, a user-defined setting can be configured. By default, the same settings as defined in "Default Colors 1" are used. They can then be modified according to user-specific requirements (see "Modifying User-Defined Color Assignments").

The "Set to Default" function restores the original default settings for the user-defined color set. You can select which of the three default settings are restored.

Remote command: 

DISPlay:​CMAP<item>:​PDEFined
