INPut<ip>:SELect <Source>

This command selects the signal source for measurements, i.e. it defines which connector is used to input data to the R&S FSW.

If no additional input options are installed, only RF input is supported.

Suffix: 
<ip>
1 | 2

irrelevant

Parameters:
<Source>
RF

Radio Frequency ("RF INPUT" connector)

DIQ

Digital IQ data (only available with optional Digital Baseband Interface

For details on I/Q input see "Digital Input".

AIQ

Analog Baseband signal (only available with optional Analog Baseband Interface R&S FSW-B71)

For details on Analog Baseband input see "Processing Data from the Analog Baseband Interface".

OBB

Oscilloscope Baseband Input

For details on Oscilloscope Baseband Input see "Processing Oscilloscope Baseband Input".

*RST:RF
