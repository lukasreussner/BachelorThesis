CALCulate<n>:LIMit<li>:STATe <State>

This command turns the limit check for a specific limit line on and off.

To query the limit check result, use CALCulate<n>:​LIMit<li>:​FAIL?.

Note that a new command exists to activate the limit check and define the trace to be checked in one step (see CALCulate<n>:​LIMit<li>:​TRACe<t>:​CHECk).

Suffix: 
<n>

irrelevant

<li>

Limit line

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: CALC:LIM:STAT ON

Switches on the limit check for limit line 1.

Manual operation: See "Disable All Lines"
