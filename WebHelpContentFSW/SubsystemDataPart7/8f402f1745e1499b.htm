Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Applying Digital Predistortion > CONFigure:DPD:FNAMe
CONFigure:DPD:FNAMe <FileName>

This command defines a name for the DPD correction table.

Prerequisites for this command
  • Turn on polynomial DPD (CONFigure:DDPD[:STATe]).
Parameters:
<FileName>

String containing the DPD table file name.

Example: 

//Defines the DPD table name

CONF:DPD:FNAM 'DPDTable'
Manual operation: See "Selecting the DPD shaping method"
