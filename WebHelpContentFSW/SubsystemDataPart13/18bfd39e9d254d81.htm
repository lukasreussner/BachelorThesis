Result List

For one base station only:

Indicates the time delay (in chips) of the signal at antenna 2 relative to the signal at antenna 1.

For multiple base stations and carriers:

Indicates the frequency offset for each carrier and the time delay (in chips) for each antenna, relative to the specified reference carrier. Furthermore, the synchronization state for each antenna is indicated. The overall status indicated above the table is "SYNC OK" only if the signals for all of the antennas for all of the base stations defined in the table are "SYNC OK".
