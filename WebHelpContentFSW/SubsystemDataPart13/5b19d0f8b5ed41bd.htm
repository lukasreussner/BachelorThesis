[SENSe:]CDPower:QINVert <State>

This command inverts the Q-branch of the signal.

Note that this command is maintained for compatibility reasons only. Use the [SENSe:​]SWAPiq command for new remote control programs.

Parameters:
ON | OFF | 1 | 0
*RST:0
