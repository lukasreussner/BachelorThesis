Search Limits ( Left / Right )

If activated, limit lines are defined and displayed for the search. Only results within the limited search range are considered.

For details on limit lines for searches, see "Peak search limits".

Remote command: 

CALCulate<n>:MARKer<m>:X:SLIMits[:STATe]

CALCulate<n>:MARKer<m>:X:SLIMits:LEFT

CALCulate<n>:MARKer<m>:X:SLIMits:RIGHt
