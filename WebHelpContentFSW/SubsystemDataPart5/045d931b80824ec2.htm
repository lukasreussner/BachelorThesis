Avionics (VOR/ILS) Measurements application (R&S FSW-15) > Remote Commands to Perform VOR/ILS Measurements > Configuring and Performing Sweeps > [SENSe:]SWEep:COUNt
[SENSe:]SWEep:COUNt <SweepCount>

This command defines the number of measurements that the application uses to average traces.

Currently the R&S FSW Avionics (VOR/ILS) measurements application does not support averaging measurements. Thus, this command has no effect (the <SweepCount> parameter is always considered to be 1).

Parameters:
<SweepCount>
*RST:1
