Analysis

Access: "Overview" > "Analysis"

General result analysis settings concerning the trace, markers, lines etc. are identical to the analysis functions in the base unit except for the special marker functions.

For a description of the lines functionality see Display and Limit Lines.

Window-specific configuration

The settings in this dialog box are specific to the selected window. To configure the settings for a different R&S FSW Analog Demodulation application window, select the window outside the displayed dialog box, or select the window from the "Specifics for" selection list in the dialog box.

The remote commands required to perform these tasks are described in "Remote Commands for Analog Demodulation Measurements".
