CONFigure:DS:CHANnel:PLC:CARRiers?

Queries the number of subcarriers used by the PLC. The number of subcarriers depends on the FFT length setting (see CONFigure:​CHANnel:​NFFT).

Example: CONF:DS:CHAN:PLC:CARR?
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Usage: Query only
Manual operation: See "PLC Number of Subcarriers (Np)"
