CONFigure:DS:CHANnel:NCP:PCONfig<pri>:SUBCarrier:MODulation <ModType>

Defines the modulation used by the specified row in the NCP profile. Note that the modulation for the row 0 is not editable, and for all other rows, the modulation is always zero bit.

Suffix: 
<pri>
1..n
Parameters:
<ModType>
ZERobit
Manual operation: See "Modulation"
