Slot Selection

Selects a single slot whose results the application displays. Alternatively, you can display the results over all slots.

You can select a single slot for the following measurements.
  • Result Summary
  • EVM vs Carrier / EVM vs Symbol / EVM vs Symbol X Carrier
  • Inband Emission
  • Spectrum Flatness / Spectrum Flatness Difference
  • Group Delay
  • Power vs Symbol X Carrier
  • Constellation Diagram

In PRACH analysis mode, you cannot select a particular slot.

Remote command: 

[SENSe:​][LTE:​][CC<cc>:​]SLOT:​SELect
