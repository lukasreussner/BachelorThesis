Automatic Grid Scaling

Activates or deactivates automatic scaling of the x-axis or y-axis for the specified trace display. If enabled, the R&S FSW WLAN application automatically scales the x-axis or y-axis to best fit the measurement results.

If disabled, the x-axis or y-axis is scaled according to the specified "Minimum / Maximum" and "Number of Divisions".

This command is currently only supported for Amplitude Modulation measurements.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO
