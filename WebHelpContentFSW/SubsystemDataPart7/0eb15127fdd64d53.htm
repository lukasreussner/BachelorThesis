CONFigure:REFSignal:SINFo:SLENgth?

This command queries the sample length of the currently used reference signal.

Return values: 
<Samples>
<numeric value>: (integer only)
Default unit: Samples
Example: CONF:REFS:SINF:SLEN?

would return, e.g.

40000
Usage: Query only
Manual operation: See "Reference signal information"
