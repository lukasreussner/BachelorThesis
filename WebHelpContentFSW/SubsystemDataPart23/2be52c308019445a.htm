PRACH Configuration

Sets the PRACH configuration index as defined in the 3GPP TS 36.211, i.e. defines the subframes in which random access preamble transmission is allowed.

The preamble format is automatically derived from the PRACH Configuration.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​PRACh:​CONF
