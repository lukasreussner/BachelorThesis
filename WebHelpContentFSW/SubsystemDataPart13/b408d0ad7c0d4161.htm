CALCulate<n>:MARKer<m>:FUNCtion:CPICh

This command sets the marker to channel 0.

This command is only available in Code Domain Power and Code Domain Error Power evaluations.

Suffix: 
<n>

Window

<m>

Marker

Example: 

CALC:MARK:FUNC:CPIC

Manual operation: See "Marker To CPICH"
