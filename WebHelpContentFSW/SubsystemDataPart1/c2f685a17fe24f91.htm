Changing the Automatic Measurement Time ( Meastime Manual )

This function allows you to change the measurement duration for automatic setting adjustments. Enter the value in seconds.

For details, see "Adjusting Settings Automatically".

This function is only available for the MSRA/MSRT Master, not for the applications.

Remote command: 

[SENSe:​]ADJust:​CONFigure[:​LEVel]:​DURation:​MODE

[SENSe:​]ADJust:​CONFigure[:​LEVel]:​DURation
