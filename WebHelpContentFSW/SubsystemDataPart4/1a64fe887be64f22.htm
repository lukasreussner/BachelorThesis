Configuration

The default GSM I/Q measurement captures the I/Q data from the GSM signal and determines various characteristic signal parameters such as the modulation accuracy, transient spectrum, trigger to sync, etc. in just one measurement (see "GSM I/Q Measurement Results").

For multicarrier wideband noise (MCWN) measurements, a different configuration is required (see "Multicarrier Wideband Noise (MCWN) Measurements").

The settings required to configure each of these measurements are described here.

Selecting the measurement type
► 

To select a different measurement type, do one of the following:

  • Select the "Overview" softkey. In the "Overview", select the "Select Measurement" button. Select the required measurement.
  • Press the MEAS key. In the "Select Measurement" dialog box, select the required measurement.
