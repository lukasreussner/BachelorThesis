SEM Signal Description

The signal description for SEM measurements contains settings to describe general physical characteristics of the signal you are measuring.

Access: "Overview" > "Signal Description"

For more information on the "LTE Mode", "Test Model", "Channel Bandwidth" and "Cyclic Prefix" see "Selecting the LTE mode", "Using Test Scenarios", "Channel Bandwidth / Number of Resource Blocks" and "Cyclic Prefix".

All other settings available for the SEM measurement are the same as in the spectrum application. For more information, refer to SEM Configuration.
