Delta Sequence Shift

Defines the delta sequence shift ΔSS.

The standard defines a sequence shift pattern fss for the PUCCH. The corresponding sequence shift pattern for the PUSCH is a function of fssPUCCH and the delta sequence shift.

For more information refer to 3GPP TS 36.211, chapter 5.5.1.3 "Group Hopping".

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​DRS:​DSSHift
