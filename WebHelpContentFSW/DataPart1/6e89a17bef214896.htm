Remote Commands > Configuring the R&S FSW > Working with Transducers > [SENSe:]CORRection:TRANsducer:SELect
[SENSe:]CORRection:TRANsducer:SELect <Name>

This command selects a transducer factor.

Parameters:
<Name>

String containing the name of the transducer factor.

If the name does not exist yet, the R&S FSW creates a transducer factor by that name.

Example: CORR:TRAN:SEL 'FACTOR1'
