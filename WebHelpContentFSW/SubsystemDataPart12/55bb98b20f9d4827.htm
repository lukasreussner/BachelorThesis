Generator Polynomial

Determines the coefficients of the polynomial and thus the feedback positions for the PRBS algorithm.

This function requires the R&S FSW-K70P option.

If the PRBS data is generated in accordance with the ITU-T standard, it is sufficient to specify the "PRBS Type". By default, the R&S FSW VSA application automatically determines the corresponding generator polynomial.

If "Auto" is deactivated, specify the coefficients a0, ..., aN of the polynomial manually.

Note that the coefficients must be defined in descending order (e.g. 9;4;0 not 0;4;9)

For details, see "Known Data from PRBS Generators".

Remote command: 

[SENSe:​]DDEMod:​KDATa:​PRBS:​POLYnomial:​AUTO

[SENSe:​]DDEMod:​KDATa:​PRBS:​POLYnomial[:​ORDer]
