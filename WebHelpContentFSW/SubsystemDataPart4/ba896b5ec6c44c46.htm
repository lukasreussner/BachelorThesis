Improving EVM Accuracy

If the EVM results show unexpected power levels, check the following issues.

Extending the data basis

Sporadic distortions in the EVM can be eliminated by evaluating several measurements and determining the average over all traces. Increase the "Statistic Count" in the "Sweep" settings to obtain sufficiently stable results.

Excluding results from adjacent channels

For signals from base stations capable of using multiple carriers, configure the DUT as such in the signal description. In this case, an additional multicarrier (PvT) filter suppresses power from adjacent channels. This filter is also taken into account during the generation of the ideal (reference) signal, otherwise there would be an increase in EVM because the measured signal has a smaller bandwidth compared to the reference signal. Define which PvT filter to use, depending on whether the channel to be measured has a reduced or equal power compared to its adjacent channels (see "Power vs Time Filter").

For single carrier measurements, make sure the correct "Device Type" setting is selected so the correct PvT filter is used for the power measurement.
