CALCulate<n>:DELTamarker<m>:FUNCtion:BPOWer:RESult?

This command queries the results of the band power measurement.

Suffix: 
<n>

Window

<m>

Marker

Return values: 
<Power>

Signal power over the delta marker bandwidth.

Usage: Query only
