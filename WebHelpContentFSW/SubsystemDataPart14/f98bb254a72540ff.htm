TD-SCDMA Measurements (R&S FSW-K76/-K77) > Remote Commands for TD-SCDMA Measurements > Selecting a Measurement > CONFigure:CDPower:MEASurement
CONFigure:CDPower:MEASurement <Measurement>

This command selects the measurement type for the TD-SCDMA BTS application.

For details on these measurements see "Frequency and Time Domain Measurements".

Parameters:
<Measurement>
ACLR

Adjacent Channel Power

CCDF

Complementary Cumulative Distribution Function

CDPower

Code Domain Power

ESPectrum

Spectrum Emission Mask

OBWidth

Occupied Bandwidth

POWer

Channel Power

PVTime

Power vs Time

*RST:CDPower
Example: CONF:CDP:MEAS POW

Selects Signal Channel Power measurement.
