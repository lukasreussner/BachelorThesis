External Analog

Data acquisition starts when the signal fed into the [EXT TRIGGER INPUT] connector on the oscilloscope meets or exceeds the specified trigger level.

This trigger source is only available if the optional 2 GHz bandwidth extension (R&S FSW-B2000) is active in power splitter mode (see "Power Splitter Mode"), or if the baseband oscilloscope option is active.

For details see "Basics on I/Q Data Acquisition and Processing".

Remote command: 

TRIG:SOUR EXT, see TRIGger[:​SEQuence]:​SOURce
