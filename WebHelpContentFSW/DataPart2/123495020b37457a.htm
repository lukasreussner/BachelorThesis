[SENSe:]CORRection:FRESponse:USER:SCOPe <Frames>

Determines whether the frequency response correction settings are applied to all active measurement channels, or only the currently selected channel.

Parameters:
<Frames>
CHANnel | ALL
CURRent

The frequency response correction settings are applied to the currently selected channel only.

To select a channel, use INSTrument[:​SELect].

For a list of available channels, use INSTrument:​LIST?.

ALL

The frequency response correction settings are applied to all active measurement channels.

Example: INST:SEL 'MyIQSpectrum'

SENS:CORR:FRES:USER:SCOP CURR

The filter is applied only to the channel named 'MyIQSpectrum'.

Manual operation: See "Use Settings For"
