Frequency Settings

The frequency span to be measured can be defined using a start and stop frequency, or a center frequency and span; alternatively, it can be set to a specific characteristic value automatically.

Useful commands for frequency settings described elsewhere:
  • CONFigure[:​MS]:​ARFCn
  • [SENSe:​]FREQuency:​CENTer
  • [SENSe:​]FREQuency:​CENTer:​STEP
  • [SENSe:​]FREQuency:​OFFSet
Remote commands exclusive to frequency settings in MCWN measurements
