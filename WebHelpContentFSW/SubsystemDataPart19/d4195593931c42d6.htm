[SENSe:]DEMod:FORMat:BANalyze <Format>

Specifies which PSDUs are to be analyzed depending on their modulation. Only PSDUs using the selected modulation are considered in result analysis.

Note: to analyze all PPDUs that are identical to the first detected PPDU (corresponds to "Auto, same type as first PPDU"), use the command: SENS:DEMO:FORM:BANA:BTYP:AUTO:TYPE FBUR.

To analyze all PPDUs regardless of their format and modulation (corresponds to "Auto, individually for each PPDU") , use the command: SENS:DEMO:FORM:BANA:BTYP:AUTO:TYPE ALL.

See [SENSe:​]DEMod:​FORMat:​BANalyze:​BTYPe:​AUTO:​TYPE.
