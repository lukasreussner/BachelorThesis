FFT Window

In the Transient Analysis application you can select one of several FFT window types.

The following window types are available:
  • Blackman-Harris
  • Flattop
  • Gauss
  • Rectangular
  • Hanning
  • Hamming
  • Chebyshev
Remote command: 

[SENSe:​]SWEep:​FFT:​WINDow:​TYPE
