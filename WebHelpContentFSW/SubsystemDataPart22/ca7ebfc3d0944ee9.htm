LTE Downlink (R&S FSW-K10x) > Remote Control > Analysis > Result Settings > UNIT:BSTR
UNIT:BSTR <Unit>

This command selects the way the bit stream is displayed.

Parameters:
<Unit>
SYMbols

Displays the bit stream using symbols

BITs

Displays the bit stream using bits

*RST:SYMbols
Example: 

//Display bit stream as bits

UNIT:BSTR BIT
Manual operation: See "Bit Stream Format"
