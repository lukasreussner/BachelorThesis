LAYout:WINDow<n>:TYPE?

Queries the window type of the window specified by the index <n>. For a list of possible window types see LAYout:​ADD[:​WINDow]?.

Suffix: 
<n>

Window

Example: 

LAY:WIND2:TYPE?

Response:

MACC

Modulation accuracy

Usage: Query only
