Phase Noise (R&S FSW-K40) > Configuration > Controlling the Measurement > Verify Level
Verify Level

Turns level verification on and off.

If level verification is on, the R&S FSW initiates the phase noise measurement only if the level of the DUT is within a certain level tolerance range. The tolerance range is a level range relative to the nominal level.

For more information see "Level Determination".

Remote command: 

Verify level:

[SENSe:​]POWer:​RLEVel:​VERify[:​STATe]

Level tolerance:

[SENSe:​]POWer:​RLEVel:​VERify:​TOLerance
