DISPlay[:WINDow<n>]:TRACe<t>:NORMalize:PHASe <Phase>

Normalizes pulse phase traces to a specific phase value. For details see "Normalization of pulse phase traces".

This command is valid only for Phase Time and Phase Time Wrapped result displays.

Suffix: 
<n>
1..n

Window

<t>
1..n

irrelevant

Parameters:
<Phase>
floating point value

Phase offset in degrees or radians

*RST:0
Default unit: DEG
Example: DISP:WIND2:TRAC:NORM:PHAS 45
Manual operation: See "Phase Normalization"
