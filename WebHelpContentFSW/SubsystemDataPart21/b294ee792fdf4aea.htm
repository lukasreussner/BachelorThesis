Constellation

This measurement represents the complex constellation points for the currently selected frame as I and Q data. Each I and Q point is returned in floating point format.

Data is returned as a repeating array of interleaved I and Q data in groups of selected carriers per OFDM-Symbol, until all the I and Q data for the analyzed OFDM-Symbols is exhausted.

The following selections are possible:

  • All symbols ([SENS:]SYMB:SELect ALL, see[SENSe:​]SYMBol:​SELect)+ all subcarriers ([SENS:]SUBC:SEL ALL[SENSe:​]SUBCarrier:​SELect
    Number of subcarriers (NFFT) pairs of I and Q data per OFDM-Symbol:
    OFDM-Symbol 0: (I0,0, Q0,0), (I0,1,Q0,1),…,( I0,Nfft-1, Q0,Nfft-1)
    OFDM-Symbol 1: (I1,1, Q1,1), (I1,2,Q1,2), …,( I1,Nfft-1, Q1,Nfft-1)
    …
    OFDM-Symbol 127:
    (I127,0, Q127,0), (I127,1,Q127,1),…,( I127,Nfft-1, Q127,Nfft-1)
  • One symbol only ([SENS:]SYMB:SEL <x>, see[SENSe:​]SYMBol:​SELect) + all subcarriers ([SENS:]SUBC:SEL ALL[SENSe:​]SUBCarrier:​SELect
    Number of subcarriers (NFFT) pairs of I and Q data for selected OFDM-Symbol x:
    (Ix,0, Qx,0), (Ix,1,Qx,1), …,( Ix,Nfft-1, Qx,Nfft-1)
  • All symbols ([SENS:]SYMB:SELect ALL, see[SENSe:​]SYMBol:​SELect) + one subcarrier ([SENS:]SUBC:SEL <y>[SENSe:​]SUBCarrier:​SELect
    One pair of I and Q data (for subcarrier y) per OFDM-Symbol:
    OFDM-Symbol 0: (I0,y, Q0,y)
    OFDM-Symbol 1: (I1,y, Q1,y)
    …
    OFDM-Symbol 127:
    (I127,y, Q127,y)
  • One symbol only ([SENS:]SYMB:SEL <x>, see[SENSe:​]SYMBol:​SELect) + one subcarrier ([SENS:]SUBC:SEL <y>[SENSe:​]SUBCarrier:​SELect
    One pair of I and Q data for subcarrier y and selected OFDM-Symbol x:
    (Ix,y, Qx,y)
