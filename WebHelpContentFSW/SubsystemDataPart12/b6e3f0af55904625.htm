Maximum Bandwidth

Defines the maximum bandwidth to be used by the R&S FSW for I/Q data acquisition.

This setting is only available if a bandwidth extension option greater than 160 MHz is installed on the R&S FSW. Otherwise the maximum bandwidth is determined automatically.

Note: This setting is not available for the optional 2 GHz / 5 GHz bandwidth extension (R&S FSW-B2000/B5000).

For details on the maximum bandwidth see "Sample Rate and Maximum Usable I/Q Bandwidth for RF Input".

"Auto"

(Default) All installed bandwidth extension options are activated. The currently available maximum bandwidth is allowed.

(See "Sample Rate and Maximum Usable I/Q Bandwidth for RF Input").

Note that using bandwidth extension options greater than 160 MHz may cause more spurious effects.

(See "Restricting the maximum bandwidth manually")

"80 MHz"

Restricts the analysis bandwidth to a maximum of 80 MHz.

The bandwidth extension options greater than 160 MHz are deactivated.

"160 MHz"

Restricts the analysis bandwidth to a maximum of 160 MHz. The bandwidth extension option for 320 MHz is deactivated.

(Not available or required if bandwidth extension option for 512 MHz is installed.)

"512 MHz"

Restricts the analysis bandwidth to a maximum of 512 MHz. The bandwidth extension option for 1200 MHz is deactivated.

"1200 MHz"

Restricts the analysis bandwidth to a maximum of 1200 MHz. The bandwidth extension option for 2000 MHz (B2001) is deactivated.

Remote command: 

TRACe:​IQ:​WBANd[:​STATe]

TRACe:​IQ:​WBANd:​MBWidth
