TRACe<n>[:DATA]:X? <Result>

This command queries the horizontal trace data for each measurement point (x-axis values).

In combination with TRACe<n>[:DATA]?, you can thus query the coordinates of each measurement point.

Suffix: 
<n>

Window

Query parameters: 
<TraceNumber>
TRACe1 | TRACe2 | TRACe3 | TRACe4 | TRACe5 | TRACe6
Return values: 
<TraceData>

The type of value depends on the information displayed on the x-axis of the result display whose contents you query.

Example: 

//Query trace data of trace 1 in window 2

TRAC2? TRACE1
TRAC2:X? TRACE1
Usage: Query only
