[SENSe:]DDEMod:PATTern[:STATe] <State>

Determines whether the pattern uses a different modulation type than the data symbols.

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Setting parameters: 
<State>
ON | OFF | 0 | 1
OFF | 0

The pattern uses the same modulation as the data symbols, defined by [SENSe:​]DDEMod:​MAPPing[:​VALue].

ON | 1

The pattern uses a different modulation, configured by [SENSe:​]DDEMod:​PATTern:​MAPPing[:​VALue].

*RST:0
Example: SENS:DDEM:PATT:STAT ON

The pattern uses a different modulation than the data symbols.

Manual operation: See "Modulation Settings for Pattern: Same as for Data Symbols"
