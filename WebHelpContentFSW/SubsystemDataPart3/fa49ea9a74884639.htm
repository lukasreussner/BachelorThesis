SOURce:EXTernal:ROSCillator[:SOURce] <Source>

This command controls selection of the reference oscillator for the external generator.

This command is only valid if External Generator Control (R&S FSW-B10) is installed.

If the external reference oscillator is selected, the reference signal must be connected to the rear panel of the instrument.

Parameters:
<Source>
INTernal

Uses the internal reference.

EXTernal

Uses the external reference; if none is available, an error flag is displayed in the status bar.

*RST:INT
Example: 

//Select an external reference oscillator

SOUR:EXT:ROSC EXT
