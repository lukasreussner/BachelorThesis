CALCulate<n>:STATistics:PRESet

This command resets the scale of the diagram (x- and y-axis).

  • Reference level (x-axis)
    0.0 dBm
  • Display range (x-axis) for APD measurements
    100 dB
  • Display range (x-axis) for CCDF measurements
    20 dB
  • Upper limit of the y-axis
    1.0
  • Lower limit of the y-axis
    1E-6
Suffix: 
<n>

Window

Example: CALC:STAT:PRES

Resets the scaling for statistical functions

Usage: Event
Manual operation: See "Default Settings"
