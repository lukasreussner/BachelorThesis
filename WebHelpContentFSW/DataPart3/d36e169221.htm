SOURce:EXTernal:ROSCillator:EXTernal:FREQuency <Frequency>

This command defines the frequency of the external reference oscillator.

If the external reference oscillator is selected, the reference signal must be connected to the rear panel of the instrument.

Parameters:
<Frequency>
Range: 1 MHz  to  20 MHz
Example: ROSC:EXT:FREQ 13MHZ

Sets the frequency to 13 MHz.

SOUR:EXT:ROSC:EXT:FREQ 13MHZ
Manual operation: See "Reference Frequency Input"
