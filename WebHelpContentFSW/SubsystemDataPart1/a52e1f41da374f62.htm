Data Acquisition

Access: "Overview" > "Bandwidth" > "Data Acquisition" tab

The data acquisition settings define which parts of the input signal are captured for further evaluation in the applications.

Data acquisition settings with advanced FFT parameters
MSRA/MSRT operating mode

In MSRA/MSRT operating mode, only the MSRA/MSRT Master channel actually captures data from the input signal. The data acquisition settings for the I/Q Analyzer application in MSRA/MSRT mode define the analysis interval.

For details on the MSRA operating mode see Applications and Operating Modes. For details on the MSRT operating mode see Applications and Operating Modes.

The remote commands required to perform these tasks are described in "Configuring Data Acquisition".
