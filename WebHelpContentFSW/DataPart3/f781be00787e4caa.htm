Sweep Time

With the RMS detector, a longer "Sweep Time" increases the stability of the measurement results. For recommendations on setting this parameter, see " Sweep Time ".

The "Sweep Time" can be set via the softkey in the "Ch Power" menu and is identical to the general setting in the "Sweep" configuration dialog box.

Remote command: 

[SENSe:​]SWEep:​TIME
