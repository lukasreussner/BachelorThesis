CALCulate<n>:HOPDetection:TABLe:STATe:STAFrequency <State>[, <Scaling>]

If enabled, the hop state frequency parameter is included in the result tables (see "State parameters").

Note that only the enabled columns are returned for the CALCulate<n>:​HOPDetection:​TABLe:​RESults? query.

Suffix: 
<n>
1..n

irrelevant

Parameters:
<State>
*RST:ON
<Scaling>
GHZ | MHZ | KHZ | HZ

Defines the scaling for the frequency parameters

Manual operation: See "State Frequency (Nominal)"
