CONFigure:WCDPower[:BTS]:CTABle:SELect <FileName>

This command selects a predefined channel table file for comparison during channel detection. Before using this command, the "RECENT" channel table must be switched on first with the command CONFigure:​WCDPower[:​BTS]:​CTABle[:​STATe].

Parameters:
<FileName>
*RST:RECENT
Example: CONF:WCDP:CTAB ON

Switches the channel table on.

CONF:WCDP:CTAB:SEL 'CTAB_1'

Selects the predefined channel table 'CTAB_1'.

Mode: BTS application only
Manual operation: See "Selecting a Table"
