Avionics (VOR/ILS) Measurements application (R&S FSW-15) > Remote Commands to Perform VOR/ILS Measurements > Configuring VOR/ILS Measurements > Data Acquisition > [SENSe:]ADEMod:BANDwidth:DEModulation
[SENSe:]ADEMod:BANDwidth:DEModulation <DBW>
[SENSe:]ADEMod:BWIDth:DEModulation <DBW>

This command defines the demodulation bandwidth manually.

For ADEM:BWID:DEM:AUTO OFF, the query returns the currently used DBW.

Parameters:
<DBW>
<numeric value>

For a list of available DBW values see Table "Available demodulation bandwidths and measurement times for ILS measurements" and Table "Available demodulation bandwidths and measurement times for VOR measurements"

*RST:ON
Example: ADEM:BWID:DEM:AUTO OFF
ADEM:BWID:DEM 50KHZ
Manual operation: See "Demodulation Bandwidth"
