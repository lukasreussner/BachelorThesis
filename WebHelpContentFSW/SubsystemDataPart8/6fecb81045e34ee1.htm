Span

Defines the measurement span.

If you change the span, the application changes the start frequency, the stop frequency and the stepsize according to the center frequency and the measurement points.

Note: If a Preamplifier is used, make sure the defined (and possibly upconverted) frequencies for the measurement stay below the maximum frequency the preamplifier supports.

Remote command: 

[SENSe:​]FREQuency:​SPAN
