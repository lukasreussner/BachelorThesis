[SENSe:]CORRection:FRESponse:USER:STATe <State>

Activates or deactivates the general usage of user-defined frequency response correction settings.

Only if activated, the filter is calculated and applied to the results.

For details, see "Frequency Response Correction (R&S FSW-K544)".

Parameters:
<State>
ON | OFF | 0 | 1
OFF | 0

Switches the function off

ON | 1

Switches the function on

*RST:0
Example: SENS:CORR:FRES:USER:STAT
Manual operation: See "State"
