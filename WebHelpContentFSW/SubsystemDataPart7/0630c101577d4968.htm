Notch Width

Defines the width of a notch that you can add to the reference signal.

Within the notch, all carriers of the reference signal have zero amplitude. You can use the noise notch to, for example, determine the noise power ratio (NPR) before and after the DPD.

Remote command: 

CONFigure:​REFSignal:​GOS:​NWIDth
