[SENSe:]FILTer<n>:HPASs:FREQuency:MANual <Frequency>

This command selects the cutoff frequency of the high pass filter for the specified evaluation.

For details on the high pass filters refer to "High Pass".

Suffix: 
<n>

Window

Parameters:
<Frequency>
numeric value
Range: 0  to  3 MHz
*RST:15kHz
Example: 

FILT:HPAS:FREQ:MAN 3MHz

The AF results are restricted to frequencies lower than 3 MHz.

Manual operation: See "High Pass"
