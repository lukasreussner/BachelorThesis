PLC Number of Subcarriers (Np)

Indicates the number of subcarriers used by the PLC (for reference only). The number of subcarriers depends on the "NFFT (FFT length)" setting.

Remote command: 

CONFigure:​DS:​CHANnel:​PLC:​CARRiers?
