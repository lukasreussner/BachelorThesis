[SENSe:]DDEMod:RLENgth[:VALue] <RecordLength>

This command defines the capture length for further processing, e.g. for burst search. The record length is defined in time (S, default) or symbols (SYM).

Note that the maximum record length depends on the sample rate for signal capture (see [SENSe:​]DDEMod:​PRATe). For the default value =4, the maximum is 64000 symbols. For larger sample rates, the maximum record length (in symbols) can be calculated as:

RecordlengthMAX = 256000/ <points per symbol>

Setting parameters: 
<RecordLength>
numeric value
*RST:8000
Default unit: Sym
