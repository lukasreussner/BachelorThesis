According to Frame Structure File

The result length is determined by the total number of symbols in the frame structure configuration.

This setting is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Remote command: 

[SENSe:​]DDEMod:​TIME:​AUTO
