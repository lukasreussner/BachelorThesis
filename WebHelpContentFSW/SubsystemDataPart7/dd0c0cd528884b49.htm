Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Performing Amplifier Measurements > Retrieving Numeric Results > Retrieving Results of the Result Summary > Retrieving Power Results > FETCh:AMAM:CWIDth:CURRent[:RESult]?
FETCh:AMAM:CWIDth:CURRent[:RESult]?

This command queries the AM/AM Curve Width as shown in the Result Summary.

Return values: 
<CurveWidth>
<numeric value>

Current AM/AM Curve Width.

Default unit: dB
Example: FETC:AMAM:CWID:CURR?

would return, e.g.

0.69
Usage: Query only
Manual operation: See "Results to check power characteristics"
