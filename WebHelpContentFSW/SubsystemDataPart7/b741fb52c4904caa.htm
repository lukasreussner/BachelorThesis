Applying Digital Predistortion

Access: "Overview" > "Measurement" > "DPD"

Digital predistortion (DPD) is a method to improve the linearity of an RF power amplifier. Basically, DPD is a set of correction values that are added to the input signal to compensate the non-linearities that occur in the amplifier. The output signal measured by the R&S FSW then shows the corrected amplifier characteristics.

You can compensate non-linearities with the functionality of the amplifier application. The application provides two compensation methods: polynomial DPD and direct DPD.

Note that you can only use one of the two DPD types at any time. When you turn on the polynomial DPD, the R&S FSW automatically turns off the direct DPD and vice versa.

Remote command:

CONFigure:​DDPD[:​STATe]
