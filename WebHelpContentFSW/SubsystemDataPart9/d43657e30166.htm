CALCulate:PNLimit:COPY<k>

Creates a new user limit line from the data of a phase noise limit line.

Suffix: 
<k>
1...8

Number of the limit line the phase noise limit line is copied to. An existing limit line in that slot is overwritten.

Example: CALC:PNL:COPY2

Copies the phase noise limit line to limit line 3.

Usage: Event
Manual operation: See "Copy to User Limit Line"
