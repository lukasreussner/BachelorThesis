CONFigure:BURSt:SPECtrum:MASK[:IMMediate]

This remote control command configures the result display in window 2 to be Spectrum Mask. Results are only displayed after a measurement is executed, e.g. using the INITiate<n>[:​IMMediate] command

Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the Magnitude Capture buffer in window 1 at the top of the screen and the selected result type in window 2 below that. Any other active windows are closed.

Use the LAYout commands to change the display (see "Configuring the Result Display").

Usage: Event
Manual operation: See "Spectrum Emission Mask"
