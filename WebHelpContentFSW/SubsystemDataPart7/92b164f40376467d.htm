Selecting and Configuring the Input Source

Access: "Overview" > "Input / Frontend" > "Input Source"

The R&S FSW-K18 supports the RF input, the optional analog baseband input and optional input from external mixers.

Simultaneous use of the RF input and the Analog Baseband input

Compared to other applications available for the R&S FSW, the R&S FSW-K18 allows you to use both the RF input and the Analog Baseband input simultaneously.

This allows for various specific measurements which require a simultaneous capture of the RF signal, of the supply voltage and of the current drawn by an amplifier. Such a test setup is, for example, required to calculate the instantaneous PAE (Power Added Efficiency), which in turn is of interest for measurements on amplifiers that make use of envelope tracking.

For more information about input from external mixers, refer to Optional External Mixers.
