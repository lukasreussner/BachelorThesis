Carrier Settings

Access: "Overview" > "Signal Description" > "Carriers"

The "Carrier" settings define whether the expected signal contains a single or multiple carriers. Multiple carriers can only be defined if a multicarrier "Device Type" is selected (see "Device Under Test Settings".

Up to 16 carriers can be configured for a single MCWN measurement.

The carriers can also be configured automatically, see "Adjusting the Center Frequency Automatically (Auto Freq)".
