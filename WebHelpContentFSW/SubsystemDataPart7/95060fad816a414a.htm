Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Designing a Reference Signal > CONFigure:REFSignal:GOS:NWIDth
CONFigure:REFSignal:GOS:NWIDth <Frequency>

This command defines the notch width of an internally generated reference signal.

Parameters:
<Frequency>
<numeric value>
Default unit: Hz
Example: CONF:REFS:GOS:NWID 150000

Defines a notch width of 150 kHz.

Manual operation: See "Notch Width"
