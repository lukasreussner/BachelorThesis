Spur Removal / Spur Threshold

Turns spur removal for all traces on and off and defines the threshold for spur removal.

For more information see "Spurs and Spur Removal".

Note that you can also remove spurs for individual traces in the "Trace Config" dialog box.

Remote command: 

Turn spur suppression on and off:

[SENSe:​]SPURs:​SUPPression

Set the threshold:

[SENSe:​]SPURs:​THReshold
