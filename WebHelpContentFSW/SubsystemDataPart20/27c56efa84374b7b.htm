Trace Configuration

Access: [Trace] > "Trace Config"

For the Power vs Time and Channel Frequency Response result displays, a maximum of three traces are available, for all other result displays in the R&S FSW 802.11ad application, only one trace is available. The trace modes cannot be changed.

Trace data can also be exported to an ASCII file for further analysis. For details see "Trace / Data Export Configuration".
