Input Source Settings

Access: "Overview" ≥ "Input/Frontend" > "Input Source Config"

Or: [INPUT/OUTPUT] > "Input Source Config"

The input source determines which data the R&S FSW will analyze.

Further input sources
The R&S FSW 802.11ad application application can also process input from the following optional sources:
  • I/Q Input files
    See "Settings for Input from I/Q Data Files"
  • External mixer
    See Optional External Mixers.
  • 2 GHz bandwidth extension (R&S FSW-B2000)
    See Settings for 2 GHz / 5 GHz Bandwidth Extension (R&S FSW-B2000/B5000).
