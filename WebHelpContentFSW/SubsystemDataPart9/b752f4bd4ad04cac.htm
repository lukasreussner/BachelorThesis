CALCulate<n>:LIMit<li>:LOWer:MODE <Mode>

This command selects the vertical limit line scaling.

Suffix: 
<n>

Window

<li>

Limit line

Parameters:
<Mode>
ABSolute

Limit line is defined by absolute physical values.

The unit is variable.

RELative

Limit line is defined by relative values related to the reference level (dB).

*RST:ABSolute
Manual operation: See "X-Axis"
