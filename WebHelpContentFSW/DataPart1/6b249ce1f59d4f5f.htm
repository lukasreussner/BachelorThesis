CALCulate<n>:MARKer<m>:X:SLIMits:ZOOM[:STATe] <State>

This command adjusts the marker search range to the zoom area for all markers in all windows.

Suffix: 
<n>, <m>

irrelevant

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: CALC:MARK:X:SLIM:ZOOM ON

Switches the search limit function on.

CALC:MARK:X:SLIM:RIGH 20MHz

Sets the right limit of the search range to 20 MHz.

Manual operation: See "Use Zoom Limits"
