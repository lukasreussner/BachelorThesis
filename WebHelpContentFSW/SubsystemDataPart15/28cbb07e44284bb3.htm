OUTPut:TRIGger<port>:PULSe:LENGth <Length>

This command defines the length of the pulse generated at the trigger output.

Suffix: 
<port>

Selects the trigger port to which the output is sent.

2 = trigger port 2 (front)

3 = trigger port 3 (rear)

Parameters:
<Length>

Pulse length in seconds.

Example: 

See Programming Example: Performing a Basic Frequency Sweep.

Example: OUTP:TRIG2:PULS:LENG 0.02
Manual operation: See "Pulse Length"
