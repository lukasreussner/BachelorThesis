Frequency Offset

Shifts the displayed frequency range along the x-axis by the defined offset.

This parameter has no effect on the instrument's hardware, or on the captured data or on data processing. It is simply a manipulation of the final results in which absolute frequency values are displayed. Thus, the x-axis of a spectrum display is shifted by a constant offset if it shows absolute frequencies, but not if it shows frequencies relative to the signal's center frequency.

A frequency offset can be used to correct the display of a signal that is slightly distorted by the measurement setup, for example.

The allowed values range from -100 GHz to 100 GHz. The default setting is 0 Hz.

Note: In MSRA mode, this function is only available for the MSRA Master.

Remote command: 

[SENSe:​]FREQuency:​OFFSet
