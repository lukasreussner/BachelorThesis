Pulse Modulation

Defines the expected pulse modulation:

"Arbitrary"

Modulation not considered (no phase error/frequency error results available)

"CW"

Continuous wave modulation, i.e. only the carrier power is modulated (On/Off)

For CW modulation, additional parameters are available to define the frequency offset.

"Linear FM"

Linear frequency modulation (FM) (The frequency changes linearly over time within each pulse)

For linear pulse modulation, additional parameters are available to define the chirp rate.

"Reference IQ"

A reference pulse is configured (see "Reference Signal Description").

Remote command: 

SENSe:​TRACe:​MEASurement:​DEFine:​PULSe:​MODulation
