Display Configuration

The captured signal can be displayed using various evaluation methods. All evaluation methods available for the selected measurement are displayed in the evaluation bar in SmartGrid mode when you do one of the following:

  • Select the "SmartGrid" icon from the toolbar.
  • Select the "Display Config" button in the "Overview".
  • Press the MEAS key.
  • Select the "Display Config" softkey in any GSM menu.

Up to 16 evaluation methods can be displayed simultaneously in separate windows. The GSM evaluation methods are described in "GSM I/Q Measurement Results" and "Multicarrier Evaluation Methods".

For details on working with the SmartGrid see Laying out the Result Display with the SmartGrid.
