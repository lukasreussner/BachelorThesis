Configuring the Result Display

The following commands are required to configure the screen display in DOCSIS 3.1 I/Q measurements in a remote environment. The corresponding tasks for manual operation are described in "Display Configuration".

The suffix <n> in the following remote commands represents the window (1..16) in the currently selected measurement channel.
