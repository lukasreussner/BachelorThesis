Filter Type

Selects the filter type if "Multicarrier Filter" is activated.

Two filter types are available for selection: a low-pass filter and an RRC filter.

By default, the low-pass filter is active. The low-pass filter affects the quality of the measured signal compared to a measurement without a filter.

The RRC filter comes with an integrated Hamming window. If selected, two more settings become available for configuration: the "Roll-Off Factor" and the "Cut Off Frequency".

For details, see "Multicarrier Mode".

Remote command: 

CONFigure:​CDPower[:​BTS]:​MCARrier:​FILTer:​TYPE
