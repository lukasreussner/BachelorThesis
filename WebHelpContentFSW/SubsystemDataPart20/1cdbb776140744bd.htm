EVM vs Symbol

This result display shows all EVM values per symbol over the number of analyzed PPDUs as defined by the "Evaluation Range" settings (see "PPDU to Analyze / Index of Specific PPDU"). The Tracking/Channel Estimation according to the user settings is applied (see "Tracking").

Vertical lines indicate the start of the next PPDU. The numbers at the bottom of the lines indicate the corresponding symbol positions.

Remote command: 

LAY:ADD? '1',RIGH, EVSY, see LAYout:​ADD[:​WINDow]?

Querying results:

TRACe<n>[:​DATA]?, see "EVM vs Symbol"
