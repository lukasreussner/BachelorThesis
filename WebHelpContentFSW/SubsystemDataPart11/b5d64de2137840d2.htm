RBW

Defines the resolution bandwidth. Numeric input is always rounded to the nearest possible bandwidth.

The resolution bandwidth is coupled to the selected span (see "ABW / RBW").

For more information see "Resolution bandwidth".

Remote command: 

[SENSe:​]BANDwidth[:​WINDow<n>]:​RESolution
