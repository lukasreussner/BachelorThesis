Range

Defines the displayed y-axis range in dB.

The default value is 100 dB.

For Analog Demodulation measurements, time domain scaling is defined in Hz (default: 500 kHz).

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]
