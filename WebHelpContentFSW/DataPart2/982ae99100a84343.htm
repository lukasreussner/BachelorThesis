Margin

Defines a margin for the limit line. The default setting is 0 dB (i.e. no margin).

For details on margins see "Limits and Margins".

Remote command: 

CALCulate<n>:​LIMit<k>:​LOWer:​MARGin

CALCulate<n>:​LIMit<k>:​UPPer:​MARGin
