[SENSe:][LTE:]DL:TRACking:PHASe <Type>

This command selects the phase tracking type.

Parameters:
<Type>
OFF

Deactivate phase tracking

PIL

Pilot only

PILP

Pilot and payload

*RST:OFF
Example: 

//Select phase tracking type

DL:TRAC:PHAS PILPAY
Manual operation: See "Phase"
