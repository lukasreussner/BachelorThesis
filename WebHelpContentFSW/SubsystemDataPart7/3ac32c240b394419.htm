Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Performing Amplifier Measurements > Retrieving Numeric Results > Retrieving Results of the Result Summary > Retrieving the Modulation Accuracy > FETCh:MACCuracy:FERRor:MAXimum[:RESult]?
FETCh:MACCuracy:FERRor:MAXimum[:RESult]?
FETCh:MACCuracy:FERRor:MINimum[:RESult]?
FETCh:MACCuracy:FERRor:CURRent[:RESult]?

This command queries the Frequency Error as shown in the Result Summary.

Return values: 
<FrequencyError>
<numeric value>

Minimum, maximum or current Frequency Error, depending on the command syntax.

Default unit: Hz
Example: FETC:MACC:FERR:MAX?

would return, e.g.

1.2879
Usage: Query only
Manual operation: See "Results to check modulation accuracy"
