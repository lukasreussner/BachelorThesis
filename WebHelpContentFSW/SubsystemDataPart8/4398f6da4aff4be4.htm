Center

Defines the center of the measurement frequency range.

If you change the center frequency, the application changes the start and stop frequency according to the span you have set.

Note: If a Preamplifier is used, make sure the defined (and possibly upconverted) frequencies for the measurement stay below the maximum frequency the preamplifier supports.

Remote command: 

Frequency list measurement:

[SENSe:​]FREQuency:​CENTer

Single frequency measurement:

[SENSe:​]FREQuency:​SINGle
