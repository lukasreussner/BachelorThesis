Symbol Constellation

When the trace data for this evaluation is queried, the real and the imaginary branches of each symbol are transferred:

<Re0>, <Im0>, <Re1>, <Im1>, ...., <Ren>, <Imn>

The number of values depends on the number of symbols and therefore the spreading factor. With transmit diversity activated, the number of values is reduced to half.

For details see "Number of bits per symbol".
