HE PPDU Config

Defines the PPDU configuration, which also contains the assignment of the stations (users) to the resource units (RUs), that is: the channels and subcarriers. For single user (HE SU-PPDU) and trigger-based PPDUs (UL), the configuration is detected automatically by the R&S FSW WLAN application. For multi-user downlink PPDUs (HE MU_PPDU), you must define the configuration manually.

For trigger-based PPDUs, you must also define the length of the HE-LTF field in the PPDUs in the R&S FSW WLAN application.

For details on PPDU configuration see also "Signal Processing for High-Efficiency Wireless Measurements (IEEE 802.11ax)".
