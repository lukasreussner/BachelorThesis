DISPlay[:WINDow<n>]:TRACe<t>:X[:SCALe]:STARt?

This command queries the first value of the x-axis in the specified window in symbols or time, depending on the unit setting for the x-axis.

Note: using the CALCulate<n>:​TRACe<t>:​ADJust:​ALIGnment:​OFFSet command, the burst is shifted in the diagram; the x-axis thus no longer begins on the left at 0 symbols but at a selectable value.

Suffix: 
<n>

Window

<t>

Trace

Example: 

See "Measurement Example 2: GSM EDGE Burst Measurement Based on a Digital Standard".

Usage: Query only
