STBC Field

Defines the PPDUs taking part in the analysis according to the Space-Time Block Coding (STBC) field content.

Note: The terms in brackets in the following description indicate how the setting is referred to in the Signal Field result display ("Format" column, see "Signal Field").

"Auto, same type as first PPDU""(A1st)"

All PPDUs using a STBC field content identical to the first recognized PPDU are analyzed.

"Auto, individually for each PPDU""(AI)"

All PPDUs are analyzed.

"Meas only if STBC field = 1 (+1 Stream)""(M1)"[(IEEE 802.11N)]

Only PPDUs with the specified STBC field content are analyzed.

"Meas only if STBC field = 2 (+2 Stream)""(M2)"[(IEEE 802.11N)]

Only PPDUs with the specified STBC field content are analyzed.

"Demod all as STBC field = 1""(D1)"[(IEEE 802.11N)]

All PPDUs are analyzed assuming the specified STBC field content.

"Demod all as STBC field = 2""(D2)"[(IEEE 802.11N)]

All PPDUs are analyzed assuming the specified STBC field content.

"Meas only if STBC = 1 (Nsts = 2Nss)""(M1)"[(IEEE 802.11ac)]

Only PPDUs with the specified STBC field content are analyzed.

"Demod all as STBC = 1 (Nsts = 2Nss)""(D1)"[(IEEE 802.11ac)]

All PPDUs are analyzed assuming the specified STBC field content.

Remote command: 

CONFigure:​WLAN:​STBC:​AUTO:​TYPE
