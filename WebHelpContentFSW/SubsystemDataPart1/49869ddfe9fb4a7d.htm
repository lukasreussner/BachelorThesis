I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Configuring I/Q Analyzer Measurements > Configuring the Data Input and Output > External Generator Control > Interface Configuration > SYSTem:COMMunicate:TCPip:RDEVice:GENerator:ADDRess
SYSTem:COMMunicate:TCPip:RDEVice:GENerator:ADDRess <Address>

Configures the TCP/IP address for the external generator.

This command is only valid if External Generator Control (R&S FSW-B10) is installed.

Parameters:
<Address>
TCP/IP address between 0.0.0.0 and 0.255.255.255
*RST:0.0.0.0
Example: SYST:COMM:TCP:RDEV:GEN:ADDR 130.094.122.195
Manual operation: See "GPIB Address / TCPIP Address / Computer Name"
