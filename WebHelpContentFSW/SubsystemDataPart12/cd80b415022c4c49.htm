[SENSe:]SWEep:COUNt[:VALue] <SweepCount>

This command sets the statistics count.

For more information see "Statistic Count".

Setting parameters: 
<SweepCount>
numeric value
0

activates "Auto" mode

numeric value > 0

Activates "Manual" mode and sets the statistics count to the corresponding number.

Range: 0  to  5000000
*RST:0
Default unit: None
Usage: SCPI confirmed
Manual operation: See "Statistic Count"
