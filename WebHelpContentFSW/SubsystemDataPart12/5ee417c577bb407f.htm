Enabling Pattern Searches

Enables or disables pattern searches. If "Auto" is selected, pattern search is enabled only if the signal structure defines a pattern in the signal (in the "Signal Structure" tab of the "Modulation & Signal Description" dialog box, see "Pattern Settings").

Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​STATe

[SENSe:​]DDEMod:​SEARch:​SYNC:​AUTO
