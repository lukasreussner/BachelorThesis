Power Class

The following power classes are supported:

(For MCWN measurements no power class is used: "NONE".)

  • 1, …, 8 (BTS)
  • 1, …,5 (MS: GMSK)
  • E1, E2, E3 (MS: all except GMSK)
  • M1, M2, M3 (Micro BTS)
  • P1 (Pico BTS)

The default power class is 2.

Remote command: 

CONFigure[:​MS]:​POWer:​CLASs
