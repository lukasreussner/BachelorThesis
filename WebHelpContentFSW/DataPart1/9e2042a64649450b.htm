MMEMory:SELect[:ITEM]:SPECtrogram <State>
MMEMory:SELect[:ITEM]:SGRam <State>

This command includes or excludes spectrogram data when storing or loading a configuration file.

Depending on the used command, either the items from the entire instrument (MMEMory:SELect[:ITEM]...), or only those from the currently selected channel (MMEM:SELect:CHANnel[:ITEM]...) are stored or loaded.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: MMEM:SEL:SGR ON

Adds the spectrogram data to the list of data subsets.

Manual operation: See "Items:"
