I/Q Analyzer > Configuration > Data Input and Output Settings > Input Source Settings > External Mixer Settings > Mixer Settings > RF Start / RF Stop
RF Start / RF Stop

Displays the start and stop frequency of the selected band (read-only).

The frequency range for the user-defined band is defined via the harmonics configuration (see "Range 1 / Range 2").

For details, see Frequency Ranges.

For details on available frequency ranges, see Table " Frequency ranges for pre-defined bands".

Remote command: 

[SENSe:​]MIXer<x>:​FREQuency:​STARt?

[SENSe:​]MIXer<x>:​FREQuency:​STOP
