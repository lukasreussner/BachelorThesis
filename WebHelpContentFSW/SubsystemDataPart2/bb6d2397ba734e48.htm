Pulse Measurements (R&S FSW-K6/6S) > Analysis > Markers
Markers

Access: "Overview" > "Result Configuration" > "Markers"

Or: [MKR]

Markers help you analyze your measurement results by determining particular values in the diagram. Thus you can extract numeric values from a graphical display.

Window-specific configuration

The settings in this dialog box are specific to the selected window. To configure the settings for a different Pulse window, select the window outside the displayed dialog box, or select the window from the "Specifics for" selection list in the dialog box.
