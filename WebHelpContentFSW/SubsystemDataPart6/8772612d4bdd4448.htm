Magnitude Reference

Displays the spectrum of the reference signal. In contrast to the Spectrum application, the frequency values are determined using DFT from the recorded I/Q data set.

Remote command: 

LAY:ADD? '1',RIGH,RMAG

(see LAYout:​ADD[:​WINDow]?)
