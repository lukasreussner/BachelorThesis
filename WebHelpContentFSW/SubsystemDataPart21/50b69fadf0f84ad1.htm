Trigger Level Mode

By default, the optimum trigger level for power triggers is automatically measured and determined at the start of each sweep (for Modulation Accuracy measurements).

In order to define the trigger level manually, switch to "Manual" mode.

Remote command: 

TRIG:SEQ:LEV:POW:AUTO ON, see TRIGger[:​SEQuence]:​LEVel:​POWer:​AUTO
