[SENSe:]ESPectrum<sb>:SBCount <SubBlocks>
[SENSe:]ESPectrum<sb>:SCOunt <SubBlocks>

This command defines the number of sub blocks in the SEM measurement. (<sb> is irrelevant)

Parameters:
<SubBlocks>

Number of sub blocks in the SEM measurement.

Range: 1  to  3
*RST:1
Example: ESP:SCO 2
Manual operation: See "Sub Block Count"
