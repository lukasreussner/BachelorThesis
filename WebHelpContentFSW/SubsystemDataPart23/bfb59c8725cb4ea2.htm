SRS Subframe Configuration

Defines the subframe configuration of the SRS.

The subframe configuration of the SRS is specific to a cell. The UE sends a shortened PUCCH/PUSCH in these subframes, regardless of whether the UE is configured to send an SRS in the corresponding subframe or not.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SRS:​SUConfig
