NCP Modulation

Defines the modulation used by the Next Codeword Pointer (NCP).

The following modulation types are supported:
  • QPSK
  • 16-QAM
  • 64-QAM
Remote command: 

CONFigure:​DS:​CHANnel:​NCP:​MODulation

Query PLC information:

FETCh:​PLCMessages:​NCP:​MODulation?
