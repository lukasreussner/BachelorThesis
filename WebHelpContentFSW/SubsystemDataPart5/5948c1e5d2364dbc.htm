Welcome to the R&S FSW Avionics (VOR/ILS) measurements application

The R&S FSW-K15 is a firmware application that adds functionality to perform VOR/ILS measurements to the R&S FSW.

The R&S FSW-K15 features:

  • Demodulation of avionics (VOR/ILS) signals
  • Modulation accuracy evaluation
  • Maximum accuracy and temperature stability due to digital down-conversion
  • No evidence of typical errors of analog down-conversion and demodulation like AM ⇔ FM conversion, deviation error, frequency response or frequency drift at DC coupling

This user manual contains a description of the functionality that the application provides, including remote control operation.

All functions not discussed in this manual are the same as in the base unit and are described in the R&S FSW User Manual. The latest version is available for download at the product homepage.

For further information see the Rohde & Schwarz Application Note 1MA193: "Aeronautical radio navigation measurement solutions".

Installation
