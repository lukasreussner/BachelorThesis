Power Splitter Mode

Activates the use of the power splitter inserted between the [IF OUT 2 GHz/ IF OUT 5 GHz] connector of the R&S FSW and the [CH1] and [CH3] input connectors of the oscilloscope. Note that this mode requires an additional alignment with the power splitter.

For details see "Power Splitter Mode".

Remote command: 

SYSTem:​COMMunicate:​RDEVice:​OSCilloscope:​PSMode[:​STATe]
