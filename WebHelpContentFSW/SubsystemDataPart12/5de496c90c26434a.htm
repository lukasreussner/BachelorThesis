[SENSe:]DDEMod:KDATa:PRBS[:TYPE] <PRBSType>

Defines the type of the used PRBS model. The type of the model defines the degree, complexity and number of terms in the polynomial model. If the PRBS data is generated in accordance with the ITU-T standard, no further settings are required.

This function requires the R&S FSW-K70P option.

For details, see "Known Data from PRBS Generators".

Setting parameters: 
<PRBSType>
9 | 11 | 15 | 16 | 20 | 21 | 23
*RST:9
Example: DDEM:KDAT:PRBS:TYPE 11
Manual operation: See "PRBS Type"
