S-CPICH Code Nr

If a user-defined CPICH definition is to be used, enter the code of the CPICH based on the spreading factor 256. Possible values are 0 to 255.

Remote command: 

[SENSe:​]CDPower:​UCPich:​ANT<antenna>:​CODE
