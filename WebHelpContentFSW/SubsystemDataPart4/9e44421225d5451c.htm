READ:BURSt[:MACCuracy]:IQOFfset:SDEViation?

This command starts the measurement and reads out the standard deviation measurement of the IQ Offset taken over the selected number of bursts. When the measurement is started the analyzer is automatically set to single sweep mode. Further results of the measurement can then be queried without restart of the measurement via the FETCh:BURSt subsystem.

Return values: 
<Result>
numeric value

Standard deviation

Default unit: NONE
Example: READ:BURS:IQOF:SDEV?
Usage: Query only
