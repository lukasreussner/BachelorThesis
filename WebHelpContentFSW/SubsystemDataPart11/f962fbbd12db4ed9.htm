Phase parameters

Chirp phase parameters

All chirp phase deviation table values are calculated from the time domain result:

φdev(k)= φmeas(k) - φideal(k)

for chirp start ≤ k ≤ chirp start + chirp length

where:

φmeas(k): instantaneous phase of the measured signal

φideal(k): ideal phase trajectory obtained from weighted linear regression of φmeas(k) within the frequency meas range

Remote command: 

CALCulate<n>:​CHRDetection:​TABLe:​PHASe:​ALL[:​STATe]

 
└ Phase Deviation (Peak)
└ Phase Deviation (RMS)
└ Phase Deviation (Average)
