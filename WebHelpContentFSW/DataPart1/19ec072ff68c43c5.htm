General Network Settings

Access: SETUP > "Network + Remote" > "Network" tab

The R&S FSW can be operated in a local area network (LAN), for example to control the instrument from a remote PC or use a network printer.

Risk of network problems

All parameters can be edited here; however, beware that changing the computer name has major effects in a network.

For details, see "How to Set Up a Network and Remote Control".

Network settings in secure user mode

Be sure to store all network settings before "SecureUser Mode" is enabled; see "Protecting Data Using the Secure User Mode".

If the currently stored network settings are not suitable, you must correct them each time you switch on the R&S FSW in secure user mode, as the settings are not stored permanently in this case.
