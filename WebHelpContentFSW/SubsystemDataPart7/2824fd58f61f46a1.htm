CONFigure:DDPD:STARt

This command initiates a direct DPD sequence with the number of iterations you have defined.

You can define the number of iterations with CONFigure:DDPD:COUNt.

Prerequisites for this command
  • Turn on direct DPD (CONFigure:​DDPD[:​STATe]).
Example: 

//Initiate direct DPD sequence

CONF:DDPD:STAR
Usage: Event
Manual operation: See "Running a direct DPD sequence"
