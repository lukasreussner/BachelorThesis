SRS BW Conf. C_SRS

Defines the bandwidth configuration of the SRS.

The bandwidth configuration is a cell-specific parameter that, in combination with the SRS bandwidth and the channel bandwidth, defines the length of the sounding reference signal sequence. For more information on the calculation, refer to 3GPP TS 36.211 chapter 5.5.3 "Sounding Reference Signal".

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SRS:​CSRS
