Common Measurement Settings > Data Input and Output > Optional External Mixers > External Mixer Settings > Basic Settings > Bias Value > Write to CVL table
Write to CVL table

Stores the bias setting in the currently selected "Conversion Loss Table" for the range (see "Managing Conversion Loss Tables"). If no conversion loss table is selected yet, this function is not available ("CVL Table not selected").

Remote command: 

[SENSe:​]CORRection:​CVL:​BIAS
