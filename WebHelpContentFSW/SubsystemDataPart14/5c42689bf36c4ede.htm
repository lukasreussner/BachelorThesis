Signal Capture (Data Acquisition)

Access: "Overview" > "Signal Capture"

How much and how data is captured from the input signal are defined in the "Signal Capture" settings.

Not available for Power vs Time measurements.

MSRA operating mode

In MSRA operating mode, only the MSRA Master channel actually captures data from the input signal. The data acquisition settings for the TD-SCDMA BTS application in MSRA mode define the application data extract. See "Application Data (MSRA) ".

For details on the MSRA operating mode, see Applications and Operating Modes.
