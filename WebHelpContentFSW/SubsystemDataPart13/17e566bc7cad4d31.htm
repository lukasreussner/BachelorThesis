Signal Capturing

The following commands are required to configure how much and how data is captured from the input signal.

MSRA operating mode

In MSRA operating mode, only the MSRA Master channel actually captures data from the input signal. The data acquisition settings for the 3GPP FDD application in MSRA mode define the application data (see "Configuring the Slave Application Data Range (MSRA mode only)").

For details on the MSRA operating mode see Applications and Operating Modes.
