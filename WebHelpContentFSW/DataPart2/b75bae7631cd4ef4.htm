External Generator Control Option (R&S FSW-B10)

The external generator control option provides an additional GPIB and an AUX CONTROL connector.

The GPIB connector can be used to connect the external generator to the R&S FSW.

The female AUX CONTROL connector is required for TTL synchronization, if supported by the generator.

For details on connecting an external generator, see "External Generator Connections".
