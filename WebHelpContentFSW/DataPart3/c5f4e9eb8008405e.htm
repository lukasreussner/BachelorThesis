Common Measurement Settings > Bandwidth, Filter and Sweep Configuration > Bandwidth, Filter and Sweep Settings > Span/RBW
Span/RBW

Sets the coupling ratio if "RBW" is set to auto mode.

For more information see "Coupling Span and RBW".

"Auto[100]"

"Resolution Bandwidth" = "Span/100"

This coupling ratio is the default setting of the R&S FSW.

"Manual"

The coupling ratio is defined manually.

The span/resolution bandwidth ratio can be set in the range from 1 to 10000.

Remote command: 

[SENSe:​]BANDwidth[:​RESolution]:​RATio
