Standard Patterns - selecting an assigned pattern

The "Standard Patterns" are the patterns assigned to the currently selected standard. You can add existing patterns to the standard or remove patterns already assigned to the standard. The highlighted pattern is currently selected for pattern search.

Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​SELect
