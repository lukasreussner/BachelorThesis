CALCulate:UNCertainty:DATA:GAIN <Gain>

This command defines the gain of the DUT.

This command is available if you have turned automatic determination of the DUT characteristics off with CALCulate:​UNCertainty:​DATA:​RESults.

Parameters:
<Gain>

Gain of the DUT.

*RST:0 dB
Example: CALC:UNC:DATA:GAIN -5DB

Defines a DUT gain of -5 dB.

Manual operation: See "Use Measurement Values"
