Import Table

Imports a stored conversion loss table from any directory and copies it to the instrument's C:\R_S\INSTR\USER\cvl\ directory. It can then be assigned for use for a specific frequency range (see "Conversion Loss").

For details, see Conversion Loss Tables.

Note: When using the optional 2 GHz bandwidth extension (R&S FSW-B2000), special conversion loss tables are required.
The 5 MHz bandwidth extension option (R&S FSW-B5000) does not support external mixers.
Supported tables have the file extension .b2g, as opposed to .acl for common tables. While .acl files can be used, data acquisition with the B2000 option using such conversion loss tables will lead to substantial inaccuracy. Using no conversion loss tables at all during data acquisition with the B2000 option will cause even more inaccuracy.
Note that only common conversion loss tables (in .acl files) can be edited. Special B2000 tables (in b2g files) can only be imported and deleted.
For more details, see "External Mixers and B2000/B5000".
