SYSTem:COMMunicate:RDEVice:OSCilloscope:ALIGnment:DATE?

Returns the date of alignment of the [IF OUT 2 GHz/ IF OUT 5 GHz] to the oscilloscope for the optional 2 GHz / 5 GHz bandwidth extension (R&S FSW-B2000/B5000).

For details see Alignment.

Return values: 
<Date>

Returns the date of alignment.

Example: SYST:COMM:RDEV:OSC:DATE?
//Result: 2014-02-28
Usage: Query only
