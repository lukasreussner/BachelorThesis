TRACe:IQ:DATA:RRANge?

This command queries the I/Q data currently stored in the memory of the R&S FSW for the defined result range (see"Defining the Result Range").

Return values: 
<IQData>

Measured value pair (I,Q) for each sample that has been recorded.

The data format depends on FORMat[:​DATA].

Default unit: V
Example: TRAC:IQ:DATA:RRAN?
Usage: Query only
