Zero Span

Sets the span to 0 Hz (zero span). The x-axis becomes the time axis with the grid lines corresponding to 1/10 of the current sweep time ("SWT").

For details see "Basic Measurements".

Remote command: 

FREQ:SPAN 0Hz, see [SENSe:​]FREQuency:​SPAN
