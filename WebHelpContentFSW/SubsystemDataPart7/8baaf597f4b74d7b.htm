Selecting the result type displayed in the Parameter Sweep diagram

You can select one of several result types evaluated in the Parameter Sweep diagram. When you open more than one instance of the Parameter Sweep, you can select a different result for each of the instances.

For an extensive list of the supported result types see "Parameter Sweep: Table".

By default, the application shows the highest and lowest values that have been measured inside the diagram area.

You can turn that off with the "Display Min and Max" feature.

Remote command: 

CONFigure:​PSWeep:​Z<n>:​RESult
