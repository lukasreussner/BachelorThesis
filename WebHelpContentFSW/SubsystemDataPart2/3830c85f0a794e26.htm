CALCulate<n>:TRENd:PHASe:X <XAxis>

Configures the x-axis of the Parameter Trend result display.

The y-axis is configured using the CALCulate<n>:TRENd:<GroupName>:Y commands.

Suffix: 
<n>
1..n

Window

Setting parameters: 
<XAxis>
POINt | PPPHase | RERRor | PERRor | DEViation

Pulse parameter to be displayed on the x-axis. For a description of the available parameters see "Phase Parameters".

POINt

Pulse phase at measurement point

PPPHase

Pulse-Pulse Phase Difference

RERRor

Phase Error (RMS)

PERRor

Phase Error (Peak)

DEViation

Phase Deviation

*RST:POINt
Example: CALC2:TREN:PHAS:X PERR
Usage: Setting only
