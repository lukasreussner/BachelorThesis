Squelch Level

Defines the level threshold below which the audible AF is cut off if squelching is enabled. The video trigger level is set to the same value.

The squelch level is indicated by a red line in the diagram.

Remote command: 

[SENSe:​]DEMod:​SQUelch:​LEVel
