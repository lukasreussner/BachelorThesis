[SENSe:]MIXer<x>:BIAS:HIGH <BiasSetting>

This command defines the bias current for the high (second) range.

(See also Bias Current).

This command is only available if the external mixer is active (see [SENSe:​]MIXer<x>[:​STATe]).

Suffix: 
<x>

irrelevant

Parameters:
<BiasSetting>
*RST:0.0 A
Default unit: A
