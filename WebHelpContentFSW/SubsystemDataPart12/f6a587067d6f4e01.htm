CALCulate<n>:LIMit:MACCuracy:CFERror:CURRent:VALue <LimitValue>
CALCulate<n>:LIMit:MACCuracy:CFERror:MEAN:VALue <LimitValue>
CALCulate<n>:LIMit:MACCuracy:CFERror:PEAK:VALue <LimitValue>

This command defines the limit for the current, peak or mean center frequency error limit. Note that the limits for the current and the peak value are always kept identical.

Suffix: 
<n>

Window

Setting parameters: 
<LimitValue>
numeric value

the value x (x>0) defines the interval [-x; x]

Range: 0.0  to  1000000
*RST:1000.0 (mean: 750.0)
Default unit: Hz
