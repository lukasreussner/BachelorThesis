[SENSe:]POWer:SEM:CHBS:AMPower <Power>

This command defines the aggregated maximum power for home base stations or the TX power for medium range base stations.

In case of medium range base stations, the command is available after [SENSe:]POWer:SEM:CHBS:AMPower:AUTO has been turned off.

Parameters:
<Power>
Numeric value
Default unit: dBm
Example: POW:SEM:CHBS:AMP 0

Defines a power of 0 dBm.
