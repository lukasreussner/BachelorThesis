Meas Range

Turns the integration of the entire measurement range for residual noise calculations on and off.

The range defined here is applied to all traces.

"On"

The application calculates the residual noise over the entire measurement range.

"Off"

The application calculates the residual noise over a customized range.

The input fields next to the "On/Off" control become available to define a customized integration range. The application shows two red lines ("EL1" and "EL2") in the graphical result display to indicate the custom range.

Remote command: 

Turn customized range on and off:

CALCulate<n>:​EVALuation[:​STATe]

Define start point of custom range:

CALCulate<n>:​EVALuation:​STARt

Define end point of custom range:

CALCulate<n>:​EVALuation:​STOP
