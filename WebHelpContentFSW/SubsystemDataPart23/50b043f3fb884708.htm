[SENSe:][LTE:]UL:DEMod:CESTimation <Type>

This command selects the channel estimation type.

Parameters:
<Type>
PIL

Pilot only

PILP

Pilot and payload

*RST:PILP
Example: 

//Use the pilot signal for channel estimation

UL:DEM:CEST PIL
Manual operation: See "Channel Estimation Range"
