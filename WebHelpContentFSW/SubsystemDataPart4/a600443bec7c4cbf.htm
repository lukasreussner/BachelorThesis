This command starts the measurement and reads out the peak result of the Error Vector Magnitude taken over the selected number of bursts. When the measurement is started the analyzer is automatically set to single sweep mode. Further results of the measurement can then be queried without restart of the measurement via the FETCh:BURSt subsystem.

For details on the EVM results see Table "Modulation accuracy parameters"

Both the current result and the statistical evaluation of all results over a number of frames (specified by "Statistic Count") are provided.

For details on how the statistical evaluation is performed see Table "Calculated summary results".

Return values: 
<Result>
numeric value

EVM

Default unit: NONE
Example: READ:BURS:PEAK:AVER?
Usage: Query only

For a detailed example see "Programming Example: Determining the EVM".
