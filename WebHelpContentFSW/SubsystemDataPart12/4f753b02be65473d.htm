[SENSe:]FREQuency:CENTer:STEP:AUTO <LinkMode>

Defines the step width of the center frequency.

Setting parameters: 
<LinkMode>
ON | OFF
ON | 1

Links the step width to the current standard (currently 1 MHz for all standards)

OFF | 0

Sets the step width as defined using the FREQ:CENT:STEP command (see [SENSe:​]FREQuency:​CENTer:​STEP).

*RST:ON
Manual operation: See "Center Frequency Stepsize"
