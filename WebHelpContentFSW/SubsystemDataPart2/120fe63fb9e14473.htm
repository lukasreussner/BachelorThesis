Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Retrieving Results > Exporting I/Q Results to an iq-tar File > MMEMory:STORe<n>:IQ:RANGe
MMEMory:STORe<n>:IQ:RANGe <RangeType>

This command sets the range of the I/Q data to store.

The suffix <n> is irrelevant.

Suffix: 
<n>
1..n
Setting parameters: 
<RangeType>
CAPTure | RRANge
CAPTure

The entire capture buffer is exported.

RRANge

The result range only (that is, the currently selected pulse; see SENSe:​TRACe:​MEASurement:​DEFine:​PULSe:​SELected) is exported.

*RST:CAPTure
Example: MMEM:STOR:IQ:RANG RRAN
Manual operation: See "Export Range"
