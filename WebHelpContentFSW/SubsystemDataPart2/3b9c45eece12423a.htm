Segmented Capture: Only <XXX> segments available from RTO

This message appears during segmented capture (see "Segmented Capture") if the connected oscilloscope does not provide as many segments as were specified ("Events" setting). The number <XXX> indicates how many segments were actually provided.

The maximum number of segments the oscilloscope can provide depends on its memory depth, see "Segmented capture and optional 2 GHz bandwidth extension".
