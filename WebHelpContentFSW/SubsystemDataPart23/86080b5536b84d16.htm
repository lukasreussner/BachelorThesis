CONFigure[:LTE]:UL[:CC<cc>]:SRS:SUConfig <Configuration>

This command defines the SRS subframe configuration.

Prerequisites for this command
  • Turn on the sounding reference signal with CONFigure[:LTE]:UL[:CC<cc>]:SRS:STAT.
Suffix: 
<cc>

Component Carrier

Parameters:
<Configuration>
<numeric value> (integer only)
Range: 0  to  14
*RST:0
Example: 

//Select SRS subframe configuration 4

CONF:UL:SRS:SUC 4
Manual operation: See "SRS Subframe Configuration"
