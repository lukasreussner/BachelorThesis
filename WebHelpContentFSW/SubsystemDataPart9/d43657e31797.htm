DISPlay[:WINDow]:TRACe:X[:SCALe]:HDECade <HalfDecade>

This command selects the half decade to be displayed.

Before you can use the command you have to select the half decade scope for the x-axis with DISPlay[:​WINDow]:​TRACe:​X[:​SCALe]:​SCOPe.

Parameters:
<HalfDecade>

Start offset frequency of the half decade you want to display.

Note that the half decade you want to display has to be part of the current measurement range.

Range: 100 mHz...300 mHz  to  3 GHz...10GHz
*RST:Half decade display is off.
Example: DISP:TRAC:X:HDEC 1KHZ

Displays the half decade beginning with 1 kHz.

Manual operation: See "Half Decade"
