MSRA Mode > Remote Commands to Perform Measurements in MSRA Mode > Activating MSRA Measurements > SYSTem:PRESet:COMPatible
SYSTem:PRESet:COMPatible <OpMode>

This command defines the operating mode that is activated when you switch on the R&S FSW or press the [PRESET] key.

For details on operating modes see "Applications and Operating Modes".

Parameters:
<OpMode>
SANalyzer

(Default:)

Defines Signal and Spectrum Analyzer operating mode as the presetting.

MSRA

Defines Multi-Standard Radio Analysis (MSRA) as the preset default operating mode.

RTSM

Defines Multi-Standard Real-Time (MSRT) as the preset default operating mode.
