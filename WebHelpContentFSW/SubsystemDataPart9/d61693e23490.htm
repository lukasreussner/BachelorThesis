Phase Noise Limit Line

Selects the shape of the phase noise limit line.

For more information see "Using Limit Lines".

"None"

No limit line.

"Noise floor and x Ranges"

Limit line defined by the noise floor and x corner frequencies and slopes. The application supports up to 5 ranges.

Remote command: 

CALCulate:​PNLimit:​TYPE
