CONFigure[:LTE]:DL[:CC<cc>]:EPDCch:NPRB <NofPRBPairs>

This command selects the number of resource blocks that the EPDCCH-PRB set uses.

Suffix: 
<cc>

Component Carrier

Parameters:
<NofPRBPairs>
0 | 2 | 4 | 8

When you select "0", the EPDCCH i not active.

Example: CONF:DL:EPDC:NPRB?

would return, e.g.

2
Manual operation: See "EPDCCH PRB Pairs"
