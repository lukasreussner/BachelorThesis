Shifting the Display (Offset)

Defines an arithmetic level offset. This offset is added to the measured level irrespective of the selected unit. The scaling of the x-axis is changed accordingly. The setting range is ±200 dB in 0.1 dB steps.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​RLEVel:​OFFSet
