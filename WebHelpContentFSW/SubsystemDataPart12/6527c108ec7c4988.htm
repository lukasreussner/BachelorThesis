Frame Configuration

A frame is defined manually as a succession of subframes with specified characteristics. The frame structure can be defined directly in the R&S FSW VSA application.

By default, at least three subframes are required: the main signal frame, plus one frame before ("previous frame"), and one after ("next frame") this frame. The previous and next frames are required to demodulate the first and last symbols in the main frame correctly. For these frames, only the modulation and boosting can be configured.
