Symbols Per Frame (K)

Defines the number of symbols per frame to be expected. The available number of symbols per frame varies depending on the used bandwidth and "NFFT (FFT length)". Values between 6 and 18 are allowed for 4K mode, values between 6 and 36 for 2K mode.

Remote command: 

CONFigure:​US:​CHANnel:​SYMBols
