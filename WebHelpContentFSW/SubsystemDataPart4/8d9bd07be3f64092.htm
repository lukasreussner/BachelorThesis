This command starts the measurement and reads out the result of the Frequency Error.

This command is retained for compatibility with R&S FS-K5 only. Use the READ:BURSt[:MACCuracy]:FREQuency or FETCh:BURSt[:MACCuracy]:FREQuency commands in newer remote control programs.

Return values: 
<Result>
numeric value

Frequency error

Default unit: Hz
Example: READ:BURS:FERR:SDEV?
Usage: Query only
