Ncs Conf

Selects the Ncs configuration, i.e. determines the Ncs value set according to TS 36.211, table 5.7.2.-2 and 5.7.2-3.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​PRACh:​NCSC
