RF Attenuation

Defines or indicates the RF attenuation for the range.

Note: If you use the "Setting the Reference Level Automatically (Auto Level)" function, the RF attenuation is set automatically for all ranges.

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​INPut:​ATTenuation
