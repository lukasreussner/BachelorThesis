OUTPut:ADEMod[:ONLine]:SOURce <WindowName>

This command selects the result display whose results are output. Only active time domain results can be selected.

Parameters:
<WindowName>
<string>

String containing the name of the window.

By default, the name of a window is the same as its index. To determine the name and index of all active windows, use the LAYout:​CATalog[:​WINDow]? query.

FOCus

Dynamically switches to the currently selected window. If a window is selected that does not contain a time-domain result display, the selection is ignored and the previous setting is maintained.

Example: OUTP:ADEM:ONL:SOUR 'AnalogDemod'

OR:

DISP:WIND1:SEL
OUTP:ADEM:SOUR FOC
Manual operation: See "Output Selection"
