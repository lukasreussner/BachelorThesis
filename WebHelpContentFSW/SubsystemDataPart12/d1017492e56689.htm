[SENSe:]DDEMod:SRATe <SymbolRate>

This command defines the symbol rate.

The minimum symbol rate is 25 Hz. The maximum symbol rate depends on the defined "Sample Rate" (see "Sample Rate, Symbol Rate and I/Q Bandwidth").

Setting parameters: 
<SymbolRate>
numeric value
Range: 25  to  250e6
*RST:3.84e6
Default unit: Hz
