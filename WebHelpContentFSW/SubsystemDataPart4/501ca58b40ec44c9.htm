Automatic Trigger Offset

If activated, the trigger offset (for external and IF power triggers) are detected and automatically measured.

This function is not available in MSRA mode.

For details on the trigger offset refer to "Trigger Offset".

Remote command: 

CONF:AUTO:TRIG ONCE, see CONFigure[:​MS]:​AUTO:​TRIGger ONCE
