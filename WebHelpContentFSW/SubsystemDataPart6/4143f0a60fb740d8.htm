The Calibration (Reference) Measurement

The Multi-Carrier Group Delay measurement method requires a preliminary reference measurement of the input signal without the device under test before the actual group delay measurement. This reference measurement is also referred to as the calibration measurement. The results of this measurement are stored internally in the R&S FSW to be used as a reference for the group delay calculation in the subsequent measurement with the DUT.

The results of the calibration measurement can be stored to a file and loaded as reference data for future measurements with the same setup.

The date and time of the most recently performed or loaded calibration measurement is indicated in the channel bar.
