Selecting the Pulse

The pulse traces (frequency, magnitude and pulse vs. time) always display the trace for one specific pulse, namely the currently selected pulse. To select a pulse, use the following command:

List of Commands
SENSe:TRACe:MEASurement:DEFine:PULSe:SELected
