DISPlay[:WINDow<n>][:SUBWindow<w>]:TRACe<t>:Y[:SCALe]:AUTO <ONCE>

This command automatically scales the y-axis of a diagram based on the displayed results.

Suffix: 
<n>

Window

<w>

Subwindow

<t>

irrelevant

Setting parameters: 
<ONCE>
DEFault

Restores the default scale of the y-axis.

ONCE

Scales the y-axis for ideal viewing.

Example: 

//Automatically scale the y-axis in subwindow 2 of window 2

DISP:WIND2:SUBW2:TRAC:Y:AUTO ONCE
Usage: Setting only
Manual operation: See "Y-Axis Scale"
