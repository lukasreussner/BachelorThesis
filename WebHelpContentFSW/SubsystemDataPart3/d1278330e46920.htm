OUTPut:ADEMod[:ONLine]:AF[:CFRequency] <Frequency>

This command defines the cutoff frequency for the AC highpass filter (for AC coupling only, see [SENSe:​]ADEMod<n>:​AF:​COUPling).

Parameters:
<Frequency>
numeric value
Range: 10 Hz  to  DemodBW/10 (= 300 kHz for active demodulation output)
*RST:100 Hz
Example: OUTP:ADEM:ONL:AF:CFR 100Hz
Manual operation: See "AC Cutoff Frequency"
