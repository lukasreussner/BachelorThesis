Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Reference Signal Description > RIQ:FIQ:PATH
RIQ:FIQ:PATH <FileName>

Selects the I/Q data file which contains the reference waveform. The file must be in iq.tar format as specified in "I/Q Data File Format (iq-tar)".

Parameters:
<FileName>

String containing the path and name of the file.

Example: RIQ:FIQ:PATH 'C:\FSW\predefined\ReferencePulsesPredefined\RefIQ1.iq.tar'
Manual operation: See "Input File Selection"
