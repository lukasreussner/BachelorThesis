Analyzer IP Address

Defines the IP addresses of the slaves connected via LAN to the master.

Tip: the LED symbol in the LAN status column indicates the state of the Rx antenna:

Color

State

gray

antenna off or IP address not available/valid

red

antenna on and IP address valid, but not accessible

green

antenna on and IP address accessible
