[SENSe:]CDPower:SFACtor <SpreadingFactor>

This command defines the spreading factor. The spreading factor is only significant for Peak Code Domain Error evaluation.

Parameters:
<SpreadingFactor>
4 | 8 | 16 | 32 | 64 | 128 | 256 | 512
*RST:512
Example: 

SENS:CDP:SFACtor 16
