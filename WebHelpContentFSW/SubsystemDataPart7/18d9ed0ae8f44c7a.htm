CONFigure:DDPD:FINish

This command stops a DPD sequence before all iterations are done and keeps the predistorted I/Q data that have been calculated.

Prerequisites for this command
  • Turn on direct DPD (CONFigure:​DDPD[:​STATe]).
  • Initiate a DPD sequence (CONFigure:​DDPD:​STARt).
Example: 

//Stop a DPD sequence

CONF:DDPD:FIN
Usage: Event
