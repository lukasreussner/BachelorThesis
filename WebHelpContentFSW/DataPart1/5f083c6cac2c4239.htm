[SENSe:]FREQuency:STARt <Frequency>

This command defines a start frequency for measurements in the frequency domain.

Parameters:
<Frequency>
0 to (fmax - min span)
*RST:0
Example: FREQ:STAR 20MHz
Manual operation: See "Frequency Sweep"
See "Start / Stop"
