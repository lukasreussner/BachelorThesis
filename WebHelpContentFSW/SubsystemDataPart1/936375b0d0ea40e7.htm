I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Configuring I/Q Analyzer Measurements > Configuring Data Acquisition > [SENSe:]IQ:FFT:WINDow:OVERlap
[SENSe:]IQ:FFT:WINDow:OVERlap <Rate>

Defines the part of a single FFT window that is re-calculated by the next FFT calculation.

Parameters:
<Rate>
double value

Percentage rate

Range: 0  to  1
*RST:0.75
Example: IQ:FFT:WIND:OVER 0.5

Half of each window overlaps the previous window in FFT calculation.

Manual operation: See "Window Overlap"
