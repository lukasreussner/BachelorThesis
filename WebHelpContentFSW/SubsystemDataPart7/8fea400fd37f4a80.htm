Signal Length

Defines the number of samples that the reference signal consists of.

A number that is a power of 2 will speed up the internal signal processing. Thus, such a number should be specified if no other requirements limit the choice of the sample count.

For more information see "Pulse Duty Cycle".

Remote command: 

CONFigure:​REFSignal:​GOS:​SLENgth
