Configuring the Frequency

Access: "Overview" > "Input / Output" > "Frequency"

The "Frequency" tab of the "Input / Output" dialog box contains settings to configure frequency characteristics.

The frequency characteristics are similar to those available in the Spectrum application. For a comprehensive description of these settings, refer to the R&S FSW User Manual.

The remote commands required to configure the frequency are described in "Configuring the Frequency".
