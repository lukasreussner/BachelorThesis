Contents of the Status Registers

In the following sections, the contents of the status registers are described in more detail.

STATus:QUEStionable:DIQ register

The STATus:QUEStionable:DIQ register is used for digital I/Q data from the optional Digital Baseband Interface and is described in See Also.

STATus:QUEStionable:SYNC register

The STATus:QUEStionable:SYNC register is used by the R&S FSW applications and is described in the individual sections (manuals) for each application.
