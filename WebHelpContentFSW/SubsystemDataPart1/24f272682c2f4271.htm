INPut<ip>:IQ:OSC:TYPE <Type>

Defines the format of the input signal.

Suffix: 
<ip>
1 | 2

irrelevant

Parameters:
<Type>
IQ | I
IQ

Both components of the complex input signal (in-phase component, quadrature component) are filtered and resampled to the sample rate of the application.

The input signal is down-converted with the center frequency (Low IF I).

I

The input signal at the channel providing I data is resampled to the sample rate of the application.

The input signal is down-converted with the center frequency (Low IF I).

Example: INP:IQ:OSC:TYPE I
Manual operation: See "I/Q Mode"
