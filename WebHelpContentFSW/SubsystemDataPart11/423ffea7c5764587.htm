Applying a global frequency offset

Applies a global Frequency Offset value to all hop states in the table at once.

To edit the frequency offset for an individual hop state, select the value directly in the "Hop States" table and enter the new value.

Remote command: 

CALCulate<n>:​HOPDetection:​STATes:​TABLe:​OFFSet
