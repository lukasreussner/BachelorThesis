Reference: ENR Table File Format

ENR tables can be exported to a file in xml format for further evaluation in other applications. Transducer factors stored in the specified xml format can also be imported to the R&S FSW for other measurements (see "Import / Export"). This reference describes in detail the format of the export/import files for ENR tables.

<?xml version="1.0" encoding="UTF-8"?>
<TableAttributes>
    <Header comment=""/>
    <Data freq="" value=""/>
    ...
</TableAttributes>
Example: Import file for an ENR table
<?xml version="1.0" encoding="UTF-8"?>
<TableAttributes>
    <Header comment="ENR values for NC364B (example)"/>
    <Data freq="10000000" value="13.14"/>
    <Data freq="100000000" value="13.21"/>
    <Data freq="1000000000" value="13.22"/>
    <Data freq="2000000000" value="13.17"/>
    <Data freq="3000000000" value="13.26"/>
    <Data freq="4000000000" value="13.38"/>
    <Data freq="5000000000" value="13.53"/>
    <Data freq="6000000000" value="13.63"/>
    <Data freq="7000000000" value="13.81"/>
</TableAttributes>
