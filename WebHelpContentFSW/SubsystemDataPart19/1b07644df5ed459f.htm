WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Retrieving Results > Numeric Modulation Accuracy, Flatness and Tolerance Results > PPDU and Symbol Count Results > FETCh:SYMBol:COUNt?
FETCh:SYMBol:COUNt?

This command returns the number of symbols in each analyzed PPDU as a comma-separated list. The length of the list corresponds to the number of PPDUs, i.e. the result of FETCh:​BURSt:​COUNt:​ALL?.

Return values: 
<Result>
<list>
Usage: Query only
