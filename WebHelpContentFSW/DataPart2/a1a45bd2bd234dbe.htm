Digital Baseband Interface (R&S FSW-B17) and R&S EX-IQ-BOX

The R&S FSW Digital Baseband Interface option (R&S FSW-B17) provides an online digital I/Q data interface on the rear panel of the instrument for input and output. The digital input and output can be enabled in the base unit or in one of the applications (where available).

Optionally, an R&S EX-IQ-BOX can be connected to the Digital Baseband Interface to convert signal properties and the transmission protocol of the R&S FSW into user-defined or standardized signal formats and vice versa.

Since the Digital I/Q input and the Analog Baseband input use the same digital signal path, both cannot be used simultaneously. When one is activated, established connections for the other are disconnected. When the second input is deactivated, connections to the first are re-established. This can cause a short delay in data transfer after switching the input source.

For more information on the Digital Baseband Interface (R&S FSW-B17) and typical applications, see the R&S FSW I/Q Analyzer User Manual.
