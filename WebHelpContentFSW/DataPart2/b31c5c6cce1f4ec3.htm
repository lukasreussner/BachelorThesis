Remote Commands > Managing Settings and Results > Storing and Loading Instrument Settings > MMEMory:LOAD:AUTO
MMEMory:LOAD:AUTO 1, 'Factory' | <FileName>

This command restores an instrument configuration and defines that configuration as the default state.

The default state is restored after a preset (*RST) or after you turn on the R&S FSW.

Parameters:
1
'Factory' | <FileName>
'Factory'

Restores the factory settings as the default state.

'<file_name>

String containing the path and name of the configuration file.

Note that only instrument settings files can be selected for the startup recall function; channel files cause an error.

Example: MMEM:LOAD:AUTO 1,'C:\R_S\INSTR\USER\TEST'
Usage: Event
Manual operation: See "Startup Recall"
