[SENSe:]SWEep:EGATe:CONTinuous:PCOunt <Amount>

Defines the number of gate periods to be measured after a single trigger event.

Parameters:
<Amount>
integer
Range: 1  to  1023
Increment: 1
*RST:100
Example: SWE:EGAT:CONT:PCO 50
Manual operation: See "Gate Period Count"
