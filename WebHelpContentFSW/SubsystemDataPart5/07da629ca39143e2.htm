STATus:QUEStionable:SYNC<n> Register

This register contains application-specific information about synchronization errors or errors during burst detection for each window in each VOR/ILS channel. It can be queried with commands STATus:​QUEStionable:​SYNC:​CONDition? and STATus:​QUEStionable:​SYNC[:​EVENt]?.
