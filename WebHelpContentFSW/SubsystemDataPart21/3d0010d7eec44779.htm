[SENSe:]FRAMe:SELect:STATe <State>

If enabled, only the frame specified by the [SENSe:​]FRAMe:​SELect command is evaluated. Statistic evaluation for numeric results is not performed, as only one result is available for each frame parameter.

If disabled, all detected frames in the current capture buffer are evaluated for numeric results. For graphical results, the first frame to be detected in the capture buffer (frame 0) is automatically selected for evaluation.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: SENS:FRAM:SEL:STAT ON
SENS:FRAM:SEL 1
Manual operation: See "Analyzing a single frame (Specified Frame)"
