Triggering

The following remote commands are required to configure a triggered measurement in a remote environment. More details are described for manual operation in "Trigger Settings".

*OPC should be used after requesting data. This will hold off any subsequent changes to the selected trigger source, until after the sweep is completed and the data is returned.
