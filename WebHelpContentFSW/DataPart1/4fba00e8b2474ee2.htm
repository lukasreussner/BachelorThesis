Remote Commands > Configuring the R&S FSW > Remote Commands for Synchronizing Parameters > Predefined Parameter Coupling > INSTrument:COUPle:BANDwidth
INSTrument:COUPle:BANDwidth <State>
INSTrument:COUPle:BWIDth <State>

This command turns synchronization of the resolution bandwidth (and filter type) between measurement channels on and off.

Parameters:
<State>
ALL | NONE
ALL | 1

Turns on synchronization.

NONE | 0

Turns on synchronization.

*RST:NONE
Example: INST:COUP:BWID ALL

Synchronizes the resolution bandwidth.
