Configuring the RF Input

Access: "Overview" > "Input / Frontend" > "Input Source" > "RF Input"

The RF input captures the RF signal that you are measuring. It is always on.

The RF input source characteristics are similar to those available in the Spectrum application. For a comprehensive description of these settings, refer to the R&S FSW User Manual.

The remote commands required to configure the RF input are described in "Selecting and Configuring the Input Source".
