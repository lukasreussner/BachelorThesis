CONFigure[:MS]:MCARrier:BTSClass <BTSClass>

This command defines the base station class. The specified BTS Class effects the calculation of the limits according to the 3GPP standard for the modulation spectrum measurement, see 3GPP2 TS 45.005 (chapter 4.2.1. "Spectrum due to modulation and wide band noise" and chapter 4.3.2 "Base Transceiver Station", search for "Multicarrier BTS").

Note that this command is maintained for compatibility reasons only.

Parameters for setting and query: 
<BTSClass>
Range: 1  to  2
*RST:1
Default unit: NONE
Example: CONF:MCAR:BTSClass
