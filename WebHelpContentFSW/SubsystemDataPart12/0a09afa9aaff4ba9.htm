FORMat:DEXPort:HEADer <Header>

This command defines if a file header (including start frequency, sweep time, detector, etc.) is created or not. A small header with the instrument model, the version and the date is always transferred.

Setting parameters: 
<Header>
ON | OFF
*RST:OFF
