Radio Configuration

The radio configuration specifies various settings for transmission according to the CDMA2000 standard including:

  • Allowed data rates
  • Modulation types
  • Use of special channels
  • Transmit diversity

The standard describes nine RCs for BTS and six for MS signals, for different transmission scenarios.

In the BTS application, the radio configuration can be customized for two channel types: PDCH and CHAN (see "BTS Channel Types"). The applied RC is specified for each channel of these types in the channel tables. Predefined channel tables are provided for particular radio configurations (see "Reference: Predefined Channel Tables").
