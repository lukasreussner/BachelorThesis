CONFigure:EQUalizer:FILTer:LENGth <Samples>

This command defines the length of the filter that the equalizer training is based on.

Parameters:
<Samples>
<numeric value> (integer only)
Example: 

//Define equalizer filter length

CONF:EQU:FILT:LENG 25
Manual operation: See "Using the equalizer"
