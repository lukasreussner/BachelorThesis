Chirp Rate Auto Mode

If enabled, the chirp rate is estimated automatically for each individual pulse.

This setting is only available for linear FM pulse modulation.

Remote command: 

SENSe:​TRACe:​MEASurement:​DEFine:​FREQuency:​RATE:​AUTO
