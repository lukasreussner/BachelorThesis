Trigger Holdoff

Defines the minimum time (in seconds) that must pass between two trigger events. Trigger events that occur during the holdoff time are ignored.

For more information, see Trigger Holdoff.

This softkey is available for frequency and time domain measurements only.

Remote command: 

TRIGger[:​SEQuence]:​IFPower:​HOLDoff
