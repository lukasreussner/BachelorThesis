Object

The constellation diagram is restricted to the specified object.

If "All" is selected (default), the Constellation diagram is displayed for the following objects:

  • Pilots
  • Complementary Pilots (upstream only)
  • PLC preamble (downstream only)
  • PLC data (downstream only)
  • NCP all (downstream only)
  • Individual profile (A to P) (downstream) / current profile (upstream)
  • Scattered pilots (downstream only)
  • Continuous pilots (downstream only)
