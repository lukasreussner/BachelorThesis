CALCulate<n>:MARKer<m>:Z?

This command queries the value of a marker in 3-dimensional result displays ("MER vs Symbol X Carrier", "Power vs Symbol X Carrier").

To get a valid result, you have to perform a complete measurement with synchronization to the end of the measurement before reading out the result. This is only possible for single measurement mode.

See also INITiate<n>:​CONTinuous.

Suffix: 
<n>

Window

<m>

Marker

Return values: 
<Result>

Result at the marker position (symbol, carrier).

For "MER vs Symbol X Carrier": modulation error ratio (in dB or %)

For "Power vs Symbol X Carrier": power value (in dBm)

Example: CALC:MARK2:Z?

Outputs the measured value of marker 2.

Usage: Query only
Manual operation: See "Marker 1 / Marker 2 / Marker 3 / Marker 4"
