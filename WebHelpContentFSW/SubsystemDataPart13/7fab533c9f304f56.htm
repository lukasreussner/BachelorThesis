Selecting a Different Branch for a Window

By default, the same (common) branch is used by all windows, namely the one specified by the "Branch (UE measurements only)" setting.

In order to evaluate a different branch for an individual window, toggle the "Use Common Branch" setting to "No". Select the window from the list of active windows under "Specifics for", then select the "Branch".

Remote command: 

CALCulate<n>:​CDPower:​MAPPing
