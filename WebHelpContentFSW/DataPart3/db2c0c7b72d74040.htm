LAN Reset

Resets the LAN configuration to its default settings (LCI function).

According to the LXI standard, an LCI must set the following parameters to a default state.

Parameter

Value

TCP/IP Mode

DHCP + Auto IP Address

Dynamic DNS

Enabled

ICMP Ping

Enabled

Password for LAN configuration

LxiWebIfc

The LAN settings are configured in the "Network" tab of the "Network + Remote" dialog box or using the instrument's LXI Browser interface.

Remote command: 

SYSTem:​LXI:​LANReset
