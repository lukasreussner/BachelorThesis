PDCCH Format

Defines the format of the PDCCH (physical downlink control channel).

Note that PDCCH format "-1" is not defined in the standard. This format corresponds to the transmission of one PDCCH on all available resource element groups. As a special case for this PDCCH format, the center of the constellation diagram is treated as a valid constellation point.

Remote command: 

CONFigure[:​LTE]:​DL[:​CC<cc>]:​PDCCh:​FORMat
