I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Activating I/Q Analyzer Measurements > TRACe:IQ[:STATe]
TRACe:IQ[:STATe] <State>

This command changes the application of the current channel to I/Q Analyzer, activating the simple I/Q data acquisition mode (see "Different remote modes available").

Executing this command also has the following effects:

  • The sweep, amplitude, input and trigger settings from the previous application are retained
  • All measurements from the previous application (e.g. Spectrum) are turned off
  • All traces are set to "Blank" mode
  • The I/Q data analysis mode is turned off (TRAC:IQ:EVAL OFF, if previous application was also I/Q Analyzer)

Note: To turn trace display back on or to enable the evaluation functions of the I/Q Analyzer, execute the TRAC:IQ:EVAL ON command (see TRACe:​IQ:​EVAL).

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: TRAC:IQ ON

Switches on I/Q data acquisition
