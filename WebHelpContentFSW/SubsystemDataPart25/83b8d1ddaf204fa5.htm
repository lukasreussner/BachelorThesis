NB-IoT Downlink (R&S FSW-K106) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Parameter Estimation > Estimating Parameters > [SENSe:][LTE:]DL:DEMod:CESTimation
[SENSe:][LTE:]DL:DEMod:CESTimation <Type>

This command selects the channel estimation type for downlink signals.

Parameters:
<Type>
PIL

Optimal, pilot only

PILP

Optimal, pilot and payload

TGPP

3GPP EVM definition

*RST:TGPP
Example: 

//Select channel estimation type

DL:DEM:CEST TGPP
Manual operation: See "Channel Estimation"
