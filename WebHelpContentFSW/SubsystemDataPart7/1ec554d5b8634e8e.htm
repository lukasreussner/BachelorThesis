TRACe:IQ:EQUalized? <Input>

This command queries the equalized I/Q data.

You can query the raw, unequalized I/Q data with TRACe:​IQ:​DATA?.

Prerequisites for this command
  • Equalized data must be available.
Query parameters: 
<Input>
RF

You have to state this parameter, but it is always "RF".

Return values: 
<Result>

String containing the I/Q data.

Example: 

//Query equalized I/Q data

TRAC:IQ:EQU? RF

//Query raw I/Q data

TRAC:IQ:DATA?
Usage: Query only
Manual operation: See "Using the equalizer"
