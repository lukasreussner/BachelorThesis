DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe] <Range>

This command defines the display range of the y-axis (for all traces).

Suffix: 
<n>

Window

<t>

irrelevant

Parameters:
<Range>

If the y-axis shows the power, the unit is dB with a range from 10 dB to 200 dB.

If the y-axis shows the frequency, the unit is Hz with a variable range.

*RST:100 dB (frequency domain), 500 kHz (time domain)
Example: DISP:TRAC:Y 110dB
Example: 

See Programming Example: Performing a Basic Frequency Sweep.

Manual operation: See "Range"
