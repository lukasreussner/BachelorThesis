Hop/Chirp Results Table

Displays the automatically detected hop/chirp parameters in a table of results. This display requires additional option R&S FSW-K60C/-K60H.

Which parameters are displayed depends on the "Result Configuration" (see "Table Configuration"). The currently selected hop/chirp is highlighted blue. The remaining hops/chirps contained in the current capture buffer are highlighted green.

For details on the individual parameters see "Hop Parameters" or "Chirp Parameters".

Remote command: 

LAY:ADD:WIND '2',RIGH,RTAB see LAYout:​ADD[:​WINDow]?
