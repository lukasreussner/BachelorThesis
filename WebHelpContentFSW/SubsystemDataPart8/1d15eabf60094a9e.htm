Input / Output Match

Defines the match at the DUT input and output.

You can define the match either as the VSWR or as the return loss (RL). If you define the VSWR or the return loss, the application automatically calculates the other.

If these values are not defined in the DUT datasheet, determine these values, for example, with a network analyzer.

Remote command: 

CALCulate:​UNCertainty:​MATCh:​DUT:​IN:​[VSWR]

CALCulate:​UNCertainty:​MATCh:​DUT:​IN:​RL

CALCulate:​UNCertainty:​MATCh:​DUT:​OUT:​[VSWR]

CALCulate:​UNCertainty:​MATCh:​DUT:​OUT:​RL
