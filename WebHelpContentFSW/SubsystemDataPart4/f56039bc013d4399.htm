CONFigure[:MS]:SYNC:ONLY <State>

If activated, only results from frames (slots) where the "Slot to measure" was found are displayed and taken into account in the averaging of the results. The behavior of this function depends on the value of the"Synchronization" parameter (see CONFigure[:​MS]:​SYNC:​MODE).

Parameters for setting and query: 
<State>
ON | OFF | 1 | 0
*RST:1
Example: CONF:SYNC:MODE TSC

Search the capture buffer for the TSC of the "Slot to measure" as given in the frame configuration.

CONF:SYNC:ONLY ON

Only if the TSC is found, the results are displayed.

Manual operation: See "Measure only on Sync"
