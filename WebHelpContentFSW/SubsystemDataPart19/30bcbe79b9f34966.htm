WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Retrieving Results > Numeric Modulation Accuracy, Flatness and Tolerance Results > Error Parameter Results > FETCh:BURSt:MCPower:AVERage?
FETCh:BURSt:MCPower:AVERage?
FETCh:BURSt:MCPower:MAXimum?
FETCh:BURSt:MCPower:MINimum?

This command returns the MIMO cross power (average, maximum or minimum value) in dB for the IEEE 802.11n (MIMO) standard. For details see "Modulation Accuracy, Flatness and Tolerance Parameters".

Return values: 
<Result>
<Stream 1 result> ... <Stream n result>
Usage: Query only
