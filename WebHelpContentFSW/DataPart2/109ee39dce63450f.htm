Retrieving Trace Results

This chapter describes how to retrieve data from standard traces.

For spectrograms see also "Marker Search (Spectrograms)".

For details on the format of the retrieved trace data see also "Formats for Returned Values: ASCII Format and Binary Format".

Commands required to export traces (and other result data) are described in "Storing Measurement Results".
