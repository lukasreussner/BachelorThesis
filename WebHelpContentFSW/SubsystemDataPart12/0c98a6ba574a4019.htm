Import Symbols

Symbols from a displayed "Symbol Table" can be imported as a pattern. Select the number of symbols (starting from the first one) to be used as a pattern. By default, the entire "Result Length" is used.

This function is only available in single measurement mode and only if the "Symbol Table" is currently displayed.

Tip: If you do not know the pattern in advance, measure the signal and identify the pattern in the symbol table. Then align the result range to the pattern by defining an offset. The "Symbol Table" is aligned to the result range, and thus starts with the pattern. Now use the "Import Symbols" function to define the pattern for subsequent measurements.
