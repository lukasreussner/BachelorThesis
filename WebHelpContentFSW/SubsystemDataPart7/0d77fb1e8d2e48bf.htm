Defining Level Characteristics

Access: "Overview" > "Input / Output" > "Amplitude"

The "Amplitude" tab of the "Input / Output" dialog box contains settings to configure the signal level characteristics.

The level characteristics are the same as those available in the Spectrum application. For a comprehensive description of these settings, refer to the R&S FSW User Manual.

The remote commands required to configure the amplitude are described in "Defining Level Characteristics".

Functions available in the "Amplitude" dialog box described elsewhere:
