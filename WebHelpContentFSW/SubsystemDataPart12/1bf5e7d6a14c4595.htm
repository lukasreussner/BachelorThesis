[SENSe:]DDEMod:PATTern:MAPPing[:VALue] <Mapping>

This command selects the mapping for pattern demodulation.

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Setting parameters: 
<Mapping>
string

To obtain a list of available symbol mappings for the current modulation type use the [SENSe:​]DDEMod:​PATTern:​MAPPing:​CATalog?? query.

Example: DDEM:PATT:MAPP 'GSM'

Sets mapping zo GSM.

Manual operation: See "Modulation Mapping"
