[SENSe:]DEMod:FORMat:BANalyze:DBYTes:EQUal <State>

For IEEE 802.11b and g (DSSS) signals only:

If enabled, only PPDUs with a specific payload length are considered for measurement analysis.

If disabled, only PPDUs whose length is within a specified range are considered.

The payload length is specified by the [SENSe:​]DEMod:​FORMat:​BANalyze:​DBYTes:​MIN command.

A payload length range is defined as a minimum and maximum number of symbols the payload may contain (see [SENSe:​]DEMod:​FORMat:​BANalyze:​DBYTes:​MAX and [SENSe:​]DEMod:​FORMat:​BANalyze:​DBYTes:​MIN).

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Manual operation: See "Equal PPDU Length"
