CALCulate<n>:MARKer<m>:SGRam:SARea <SearchArea>
CALCulate<n>:MARKer<m>:SPECtrogram:SARea <SearchArea>

This command defines the marker search area for all spectrogram markers in the channel.

Suffix: 
<n>, <m>

irrelevant

Parameters:
<SearchArea>
VISible

Performs a search within the visible frames.

Note that the command does not work if the spectrogram is not visible for any reason (e.g. if the display update is off).

MEMory

Performs a search within all frames in the memory.

*RST:VISible
Manual operation: See "Marker Search Area"
