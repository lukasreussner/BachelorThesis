Spectrum Flatness

The spectrum flatness evaluation returns absolute power values per carrier (in dBm).

Two trace types are provided for this evaluation:

Query parameter and results for Spectrum Flatness

TRACE1

An average spectrum flatness value for each of the 53 (or 57/117 within the IEEE 802.11 n standard) carriers

TRACE2

All spectrum flatness values per channel

Supported data formats (see FORMat[:​DATA]): ASCii|REAL
