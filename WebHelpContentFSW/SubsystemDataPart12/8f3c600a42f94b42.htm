Save Standard

Saves the current measurement settings for a specific standard as a file with the defined name.

See "To store settings as a standard file" for details. For an overview of predefined standards and settings, see "Predefined Standards and Settings".

Remote command: 

[SENSe:​]DDEMod:​STANdard:​SAVE
