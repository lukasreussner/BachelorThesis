Abs Limit Start / Stop <n>

Sets an absolute limit value for the n-th power class at the start or stop frequency of the range [dBm].

This parameter is only available if the limit check is set accordingly (see "Limit Check <n>").

Remote command: 

[SENSe:​]ESPectrum<sb>:​RANGe<range>:​LIMit<PClass>:​ABSolute:​STARt

[SENSe:​]ESPectrum<sb>:​RANGe<range>:​LIMit<PClass>:​ABSolute:​STOP
