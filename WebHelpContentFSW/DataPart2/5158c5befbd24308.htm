Sweep/Average Count

Defines the number of sweeps to be performed in the single sweep mode. Values from 0 to 200000 are allowed. If the values 0 or 1 are set, one sweep is performed.

The sweep count is applied to all the traces in all diagrams.

If the trace modes "Average", "Max Hold" or "Min Hold" are set, this value also determines the number of averaging or maximum search procedures.

In continuous sweep mode, if "Sweep Count" = 0 (default), averaging is performed over 10 sweeps. For "Sweep Count" =1, no averaging, maxhold or minhold operations are performed.

For more information, see "How Much Data is Measured: Sweep Points and Sweep Count".

For spectrogram displays, the sweep count determines how many sweeps are combined in one frame in the spectrogram; that is: how many sweeps the R&S FSW performs to plot one trace in the spectrogram result display. For more details, see "Time Frames".

Remote command: 

[SENSe:​]SWEep:​COUNt

[SENSe:​]AVERage<n>:​COUNt
