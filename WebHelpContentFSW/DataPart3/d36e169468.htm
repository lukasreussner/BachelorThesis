CALibration:RESult?

This command returns the results collected during calibration.

Return values: 
<CalibrationData>

String containing the calibration data.

Example: CAL:RES?

would return, e.g.

Total Calibration Status: PASSED, Date (dd/mm/yyyy): 12/07/2004,
Time: 16:24:54,Runtime: 00.06
Usage: Query only
Manual operation: See "Alignment Results:"
