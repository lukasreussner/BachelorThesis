[SENSe:]CDPower:MAPPing <SignalBranch>

This command switches between I and Q branches of the signal for all evaluations (if not specified otherwise using CALCulate<n>:​CDPower:​MAPPing).

Parameters:
<SignalBranch>
I | Q
*RST:Q
Example: 

CDP:MAPP Q

Mode: UE application only
