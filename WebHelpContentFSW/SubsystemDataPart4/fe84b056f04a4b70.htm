Refresh (MSRA/MSRT only)

This function is only available if the Sequencer is deactivated and only for MSRA/MSRT slave applications.

The data in the capture buffer is re-evaluated by the currently active slave application only. The results for any other slave applications remain unchanged.

This is useful, for example, after evaluation changes have been made or if a new sweep was performed from another slave application; in this case, only that slave application is updated automatically after data acquisition.

Note: To update all active slave applications at once, use the "Refresh All" function in the "Sequencer" menu.
(See Refresh All ).

Remote command: 

INITiate<n>:​REFResh
