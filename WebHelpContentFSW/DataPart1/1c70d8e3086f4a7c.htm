DIAGnostic:SERVice:INPut:MC:CFRequency <Frequency>

This command defines the frequency of the calibration signal for R&S FSW models 43 MHz and higher.

This command only takes effect if a microwave calibration signal is selected for input (DIAGnostic:​SERVice:​INPut[:​SELect])

Parameters:
<Frequency>
*RST:8.004 GHz
Default unit: Hz
Manual operation: See "Calibration Frequency MW"
