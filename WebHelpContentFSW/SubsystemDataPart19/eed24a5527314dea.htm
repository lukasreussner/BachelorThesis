Equal PPDU Length

If enabled, only PPDUs with the specified "(Min./Max.) Payload Length" are considered for measurement analysis.

If disabled, a maximum and minimum "(Min./Max.) Payload Length" can be defined and all PPDUs whose length is within this range are considered.

Remote command: 

IEEE 802.11a, ac, ax, g (OFDM), j, n, p

[SENSe:​]DEMod:​FORMat:​BANalyze:​SYMBols:​EQUal

IEEE 802.11 b, g (DSSS):

[SENSe:​]DEMod:​FORMat:​BANalyze:​DURation:​EQUal

[SENSe:​]DEMod:​FORMat:​BANalyze:​DBYTes:​EQUal
