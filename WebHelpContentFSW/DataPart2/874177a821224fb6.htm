Power Mode

The measured power values can be displayed directly for each trace ("Clear/ Write"), or only the maximum values over a series of measurements can be displayed ("Max Hold"). In the latter case, the power values are calculated from the current trace and compared with the previous power value using a maximum algorithm. The higher value is retained. If "Max Hold" mode is activated, "Pwr Max" is indicated in the table header. Note that the trace mode remains unaffected by this setting.

Remote command:
