Mixer Type

The External Mixer option supports the following external mixer types:

(See also Two-port and Three-port Mixers)

"2 Port"

LO and IF data use the same port

"3 Port"

LO and IF data use separate ports

Remote command: 

[SENSe:​]MIXer<x>:​PORTs
