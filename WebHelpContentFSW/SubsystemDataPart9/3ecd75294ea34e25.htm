Phase Noise (R&S FSW-K40) > Remote Control Commands for Phase Noise Measurements > Configuring the Measurement Range > [SENSe:]LIST:RANGe<range>:SWEep:COUNt
[SENSe:]LIST:RANGe<range>:SWEep:COUNt <Measurements>

This command defines the number of measurements included in the averaging for a half decade.

Suffix: 
<range>
1...20

Selects the half decade.

For the suffix assignment see Table "<range> suffix assignment".

Parameters:
<Measurements>
Range: 1  to  10000
*RST:Depends on the half decade
Example: 

LIST:RANG9:SWE:COUN 15

Includes 15 measurements in the averaging of the ninth half decade.

Manual operation: See "Half Decades Configuration Table"
