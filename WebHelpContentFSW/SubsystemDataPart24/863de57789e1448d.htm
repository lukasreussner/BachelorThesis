TD-SCDMA BTS

The TD-SCDMA BTS slave application requires an instrument equipped with the TD-SCDMA BTS Measurements option, R&S FSW-K82. This slave application provides test measurements for TD-SCDMA BTS downlink signals (base station signals) according to the test specification.

RF measurements are not supported in MSRA mode.

For details refer to Welcome to the TD-SCDMA Applications.

Remote command: 

INST:SEL BTDS, see INSTrument[:​SELect]
