UNIT:CAXes <Caxes>

For result displays that evaluate a parameter per carrier (e.g. "MER vs Carrier"), this command defines whether the carrier number or the carrier frequency (in Hz) is displayed on the x-axis. Note, however, that this setting applies to ALL result displays based on carriers.

Parameters:
<Caxes>
CARR | HZ
CARR

The carrier number is displayed on the x-axis of all carrier-based result displays.

HZ

The carrier frequency (in Hz) is displayed on the x-axis of all carrier-based result displays.

*RST:HZ
Example: UNIT:CAX CARR
Manual operation: See "Carrier Axes Unit"
