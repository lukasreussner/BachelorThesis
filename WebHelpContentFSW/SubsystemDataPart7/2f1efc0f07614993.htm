CONFigure:GENerator:EXTernal:ROSCillator:LEDState?

This command queries the connection status of the generator to its frequency reference.

Return values: 
<State>
GREen

Connection to the reference was successful.

GREY

Unknown connection state.

RED

Connection to the reference was not successful.

Example: CONF:GEN:EXT:ROSC:LEDS?

would return, e.g.:

RED
Usage: Query only
Manual operation: See "Reference Frequency"
