Ref. Level

Defines or indicates the reference level for the range.

Note: If you use the "Setting the Reference Level Automatically (Auto Level)" function, the level is set automatically for all ranges.

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​RLEVel
