SYSTem:REVision:FACTory

Resets the response to the REV? query to the default value, e.g. after a user string was defined using the SYSTem:​REVision[:​STRing] command. (REV? query available for HP emulation only, see SYSTem:​LANGuage.)

Example: 

Define the system language:

SYST:LANG '8563E'

Set the response back to factory setting:

SYS:REV:FACT

Query the revision:

REV?

Response:

920528
Usage: Event
Manual operation: See "Resetting the Factory Revision"
