[SENSe:]LIST:SAVE <Filename>

Saves the current range setup to a user-defined comma-separated (.csv) file for later use. The values are stored in the following order for each range:

<No>,<Start>,<Stop>,<TNRStart>,<TNRStop>,<LimitOffset>,<PeakExcursion>,<SNR>,<AutoRBW>,<RBW>,<MaxFinalRBW>,<Detector>,<DetLength>,<Reserved>,<RefLevel>,<RFAttenuation>,<Preamp>

Setting parameters: 
<Filename>

String containing the path and name of the file.

Example: 
SENS:LIST:SAVE 'C:\R_S\userdata\RangeTable.csv'
//Result:
//RangeNo,StartFreq,StopFreq,DetThreshStart,DetThreshStop,LimitOffset,PeakExcursion,MinSNR,AutoRBW,RBW,MaxFinalRBW,Detector,FftBin,Reserved,RefLevel,RFAtt,Preamp
1,0,2000000000,-120,-120,10,3,10,On,,1000,Positive Peak,2,Reserved,-10,0,Off
2,2000000000,4000000000,-120,-120,10,3,10,On,,1000,Positive Peak,2,Reserved,-10,0,Off
3,4000000000,5500000000,-120,-110,10,3,10,On,,1000,Positive Peak,2,Reserved,-10,0,Off
4,6000000000,8000000000,-110,-110,10,3,10,On,,1000,Positive Peak,2,Reserved,-10,0,Off
Usage: Setting only
