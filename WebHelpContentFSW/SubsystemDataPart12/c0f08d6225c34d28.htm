[SENSe:]DDEMod:SEARch:SYNC:MODE <MeasOnlyOnPatt>

This command sets the vector analyzer so that the measurement is performed only if the measurement was synchronous to the selected sync pattern.

The command is available only if the pattern search is activated (see [SENSe:​]DDEMod:​SEARch:​SYNC:​STATe).

Setting parameters: 
<MeasOnlyOnPatt>
MEAS | SYNC
MEAS

The measurement is performed independently of successful synchronization

SYNC

The measured values are displayed and considered in the error evaluation only if the set sync pattern was found. Bursts with a wrong sync pattern (sync not found) are ignored. If an invalid or no sync pattern is found, the measurement waits and resumes running only when a valid sync pattern is found.

*RST:OFF
Manual operation: See "Meas only if Pattern Symbols Correct"
