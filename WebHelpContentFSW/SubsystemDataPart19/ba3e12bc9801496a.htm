Input Sample Rate

This is the sample rate the R&S FSW WLAN application expects the I/Q input data to have. If necessary, the R&S FSW has to resample the data.

During data processing in the R&S FSW, the sample rate usually changes (decreases). The RF input is captured by the R&S FSW using a high sample rate, and is resampled before it is processed by the R&S FSW WLAN application.

Remote command: 

TRACe:​IQ:​SRATe
