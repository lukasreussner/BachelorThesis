Handover Freq

If due to the LO frequency the conversion of the input signal is not possible using one harmonic, the band must be split. An adjacent, partially overlapping frequency range can be defined using different harmonics. In this case, the sweep begins using the harmonic defined for the first range. At the specified "handover frequency" in the overlapping range, it switches to the harmonic for the second range.

The handover frequency can be selected freely within the overlapping frequency range.

For details, see "Frequency Ranges".

Remote command: 

[SENSe:​]MIXer<x>:​FREQuency:​HANDover
