Number of RB

Sets the number of resource blocks the PUSCH allocation covers. The number of resource blocks defines the size or bandwidth of the PUSCH allocation.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SUBFrame<sf>:​ALLoc[:​CLUSter<cl>]:​RBCount
