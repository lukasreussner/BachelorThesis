Configuring RF Measurements

RF measurements are performed in the Spectrum application, with some predefined settings as described in "RF Measurements".

For details on configuring these RF measurements in a remote environment, see Configuring and Performing Measurements.

The cdma2000 RF measurements must be activated in a CDMA2000 application, see "Activating the Measurement Channel".

The individiual measurements are activated using the CONFigure:​CDPower[:​BTS]:​MEASurement command (see "Selecting a Measurement").
