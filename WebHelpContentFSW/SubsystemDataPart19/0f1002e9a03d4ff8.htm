Configuring the Trigger Output

The following commands are required to send the trigger signal to one of the variable [TRIGGER INPUT/OUTPUT] connectors on the R&S FSW.

The tasks for manual operation are described in How to Output a Trigger Signal.

List of Commands
OUTPut:TRIGger<port>:DIRection
OUTPut:TRIGger<port>:LEVel
OUTPut:TRIGger<port>:OTYPe
OUTPut:TRIGger<port>:PULSe:IMMediate
OUTPut:TRIGger<port>:PULSe:LENGth
