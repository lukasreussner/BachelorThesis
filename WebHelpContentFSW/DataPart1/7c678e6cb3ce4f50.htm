HCOPy:CMAP<item>:DEFault<colors>

This command defines the color scheme for print jobs.

Suffix: 
<item>

Selects the item for which the color scheme is to be defined.

For more information see "CMAP Suffix Assignment".

<colors>
1...4

1

Current colors with a white background and a black grid.

2

Optimized colors.

3

Customized colors.

4

Current screen colors (setting for hardcopies).

Example: HCOP:CMAP:DEF2

Selects the optimized color set for the color settings of a printout or a hardcopy.

Usage: Event
Manual operation: See "Print Colors"
