I/Q Analyzer > Configuration > Data Input and Output Settings > Input Source Settings > External Mixer Settings > Mixer Settings > Preset Band
Preset Band

Restores the presettings for the selected band.

Note: changes to the band and mixer settings are maintained even after using the [PRESET] function. This function allows you to restore the original band settings.

Remote command: 

[SENSe:​]MIXer<x>:​HARMonic:​BAND:​PRESet
