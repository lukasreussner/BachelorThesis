Configuring Time Alignment Error Measurements

Several settings supported by Time Alignment Error measurements are the same as those for I/Q measurements. For a comprehensive description of those, refer to the following chapters.

  • "Defining Signal Characteristics"
  • "Configuring the Synchronization Signal"
    (note that the Time Alignment Error measurement does not support all synchronization signal settings)
  • "Selecting the Input and Output Source"
  • "Defining the Frequency"
  • "Defining Level Characteristics"
  • "Configuring the Data Capture"
  • "Trigger"
  • "Configuring Demodulation Parameters"

For more information about configuring component carriers in the Time Alignment Error measurement see Carrier Aggregation.
