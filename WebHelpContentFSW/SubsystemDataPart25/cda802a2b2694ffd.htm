Analyzing Frequency Sweep Measurements

The NB-IoT application supports the same functionality to analyze the results of frequency sweep measurements (for example the ACLR) as the R&S FSW base unit does for those measurements.

Refer to the user manual of the R&S FSW for a detailed description on how to analyze these measurements.
