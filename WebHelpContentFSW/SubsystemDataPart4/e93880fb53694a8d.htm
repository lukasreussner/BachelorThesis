Adjusting the Center Frequency Automatically (Auto Freq)

This function adjusts the center frequency and "ARFCN" (I/Q mode only) automatically.

For multicarrier measurements, all carrier settings are automatically adjusted (see "Carrier Settings").

This command is not available when using the Digital Baseband Interface (R&S FSW-B17) or the Analog Baseband Interface (R&S FSW-B71).

Carriers are only detected in a range of approximately 25 MHz to 2 GHz. For further details see "Automatic Carrier Detection".

Remote command: 

[SENSe:​]ADJust:​FREQuency
