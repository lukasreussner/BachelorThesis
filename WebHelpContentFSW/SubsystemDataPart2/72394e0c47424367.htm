Pulse Phase

Displays the phase vs. time trace of the selected pulse. The length and alignment of the trace can be configured in the "Result Range" dialog box (see "Result Range").

Remote command: 

LAY:ADD:WIND '2',RIGH,PPH see LAYout:​ADD[:​WINDow]?

Results:

TRACe<n>[:​DATA]?
