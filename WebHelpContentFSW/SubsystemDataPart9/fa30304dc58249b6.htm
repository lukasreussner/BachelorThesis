Default Settings for Phase Noise Measurements

When you enter the phase noise application for the first time, a set of parameters is passed on from the currently active application:

  • nominal or center frequency
  • nominal or reference level
  • input coupling

After initial setup, the parameters for the measurement channel are stored upon exiting and restored upon re-entering the channel. Thus, you can switch between applications quickly and easily.

Apart from these settings, the following default settings are activated directly after a measurement channel has been set to the Phase Noise application, or after a channel preset:
