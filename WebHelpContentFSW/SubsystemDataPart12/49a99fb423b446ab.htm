VSA (R&S FSW-K70) > Configuration > Signal Description > Modulation > Transmit Filter Type > Load User Filter
Load User Filter

Opens a file-selection dialog box to select the user-defined transmit filter to be used.

Note: If a user-defined transmit filter is selected and the measurement filter is defined automatically (see "Using the Transmit Filter as a Measurement Filter (Auto)"), a Low-ISI measurement filter according to the selected user filter is calculated and used.

For details, see "Customized Filters".

For detailed instructions on working with user-defined filters, see "How to Select User-Defined Filters".

Remote command: 

[SENSe:​]DDEMod:​TFILter:​USER

[SENSe:​]DDEMod:​TFILter:​NAME
