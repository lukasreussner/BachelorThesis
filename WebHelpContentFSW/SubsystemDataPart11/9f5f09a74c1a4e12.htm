CALCulate<n>:CHRDetection:STATes[:DATA] {<ChirpRate>, <Tolerance>}...

This command sets and queries the chirp state detection table. It consists of a comma-separated list of value pairs, one for each possible chirp state.

Note that the state table can only be configured manually if CALCulate<n>:​CHRDetection:​STATes:​AUTO is OFF.

Suffix: 
<n>
irrelevant
Parameters:
<ChirpRate>
Default unit: HZ
<Tolerance>

Tolerance above or below the nominal chirp rate.

Default unit: HZ
Example: CALC:CHRD:STAT 1e6, 0.3, 1e5, 0.4
