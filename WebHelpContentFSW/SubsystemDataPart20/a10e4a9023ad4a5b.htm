Retrieving Trace Results

The following commands describe how to retrieve the trace data from the IEEE 802.11ad Modulation Accuracy measurement. Note that for these measurements, only 1 trace per window can be configured.

The traces for SEM measurements are identical to those in the Spectrum application.

Remote commands exclusive to retrieving trace results:
List of Commands
FORMat[:DATA]
TRACe<n>[:DATA]?
TRACe<n>[:DATA]:X?
