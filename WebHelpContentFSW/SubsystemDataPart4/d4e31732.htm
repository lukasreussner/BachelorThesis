Frame Configuration: Select Slot to Configure

This area shows a graphical representation of the configuration of each slot. Select a slot to display its "Slot" dialog box (see "Slot Settings").

For active slots the following information is shown:

  • The burst type, e.g. "Normal (NB)" for a normal burst.
  • The modulation, e.g. GMSK.
  • The training sequence TSC (and Set)

For details on how to interpret the graphic, see "Frame configuration and slot scope in the channel bar".
