Sub Block Count

Defines the number of sub blocks. By default, the familiar SEM measurement with just one single block of ranges is configured.

If more than one sub blocks are defined, additional tabs are inserted for each sub block in the individual tabs of the "Spectrum Emission Mask" configuration dialog box.

For details on Multi-SEM measurements, see "SEM with Multiple Sub Blocks ("Multi-SEM")".

Remote command: 

[SENSe:​]ESPectrum<sb>:​SCOunt
