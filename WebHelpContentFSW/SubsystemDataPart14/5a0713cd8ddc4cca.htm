CONFigure:CDPower:CTABle[:STATe] <State>

This command switches the use of a predefined channel table on or off. When switched on, the measured channel table is stored under the name "RECENT" and is selected for use. After the "RECENT" channel table is switched on, another channel table can be selected with the command CONFigure:​CDPower:​CTABle:​SELect.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: 

CONF:CDP:CTAB ON
