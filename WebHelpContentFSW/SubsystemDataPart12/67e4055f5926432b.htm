[SENSe:]DDEMod:EQUalizer[:STATe] <State>

This command activates or deactivates the equalizer.

For more information on the equalizer see "The Equalizer".

Setting parameters: 
<State>
ON | OFF
*RST:0
Example: DDEM:EQU OFF
Manual operation: See "State"
