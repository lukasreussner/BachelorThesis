I/Q Analyzer (MSRA mode only)

The I/Q Analyzer application (not Master) in MSRA mode can also perform measurements on the captured I/Q data in the time and frequency domain. This button is provided in order to return to the conventional I/Q analysis functions.

The I/Q Analyzer application is described in Configuration.

Remote command: 

CALCulate<n>:IQ:MODE
