Common Noise Source

Turns the use of a common ENR on and off.

Common ENRs have the same characteristics for the measurement and calibration. If you turn off common ENR, you can define an additional ENR to be used during calibration.

Define additional ENR for measurements on frequency converting DUTs, if one noise source does not cover the frequency range at the DUT input (RF) and for calibration (IF).

Remote command: 

[SENSe:​]CORRection:​ENR:​COMMon
