Spurious Measurements application (R&S FSW-K50) > Remote Commands to Perform Spurious Measurements > Configuring Spurious Measurements > Configuring the Result Displays > Configuring Tables and Diagrams > CALCulate:SSEarch:TABLe:COLumn
CALCulate:SSEarch:TABLe:COLumn <State>, <Headers>...

Select the numerical results to be displayed in the Spurious Detection Table.

For a description of the individual results see "Spurious Detection Table".

Parameters:
<Headers>
ALL | SID | STARt | STOP | RBW | FREQuency | POWer | DELTa | IDENt
ALL

All available results are displayed

STARt

Start frequency of range/span

STOP

Stop frequency of range/span

FREQuency

Spur frequency

POWer

Spur power

DELTa

Delta of spur to limit

RBW

Resolution bandwidth used for range

IDENt

Spur ID

Setting parameters: 
<State>
ON | OFF | 0 | 1
OFF | 0

Hides the result

ON | 1

Displays the result

*RST:1
Example: CALC:SSE:TABL:COL OFF,STARt
