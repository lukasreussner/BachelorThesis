[SENSe:]MIXer<x>:LOSS:HIGH <Average>

This command defines the average conversion loss to be used for the entire high (second) range.

Suffix: 
<x>

irrelevant

Parameters:
<Average>
numeric value
Range: 0  to  100
*RST:24.0 dB
Default unit: dB
Example: MIX:LOSS:HIGH 20dB
Example: 

See "Programming Example: Working with an External Mixer" and "Configuring a conversion loss table for a user-defined band".
