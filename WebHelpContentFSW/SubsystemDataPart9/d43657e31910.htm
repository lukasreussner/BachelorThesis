DISPlay[:WINDow]:TRACe:X[:SCALe]:STOP <StopFrequency>

This command selects the stop frequency of the display range.

Before you can use the command you have to select a manual display range for the x-axis with DISPlay[:​WINDow]:​TRACe:​X[:​SCALe]:​SCOPe.

Parameters:
<StopFrequency>

Stop offset frequency of a half decade.

Note that the stop offset you want to display has to be part of the current measurement range.

Range: 300 mHz  to  10 GHz
*RST:Manual display range is OFF
Example: DISP:TRAC:X:STOP 3KHZ

Defines 3 kHz as the end of the display range.

Manual operation: See "X-Axis Start / Stop"
