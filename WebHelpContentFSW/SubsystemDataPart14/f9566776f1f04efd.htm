Programming Examples (TD-SCDMA BTS)

The following programming examples are meant to demonstrate the operation of the R&S FSWTD-SCDMA application in a remote environment. They are performed with an R&S FSW equipped with option R&S FSW-K76. Only the commands required to control the R&S FSW-K76 application are provided, not the signal generator.

The measurements are performed using the following devices and accessories:

  • The R&S FSW with Application Firmware R&S FSW-K76: TD-SCDMA BTS
  • The Vector Signal Generator R&S SMU with option R&S SMU-B45: digital standard 3GPP (options R&S SMU-B20 and R&S SMU-B11 required)
  • 1 coaxial cable, 50Ω, approx. 1 m, N connector
  • 1 coaxial cable, 50Ω, approx. 1 m, BNC connector
Test setup

  1. Connect the RF output of the R&S SMU to the input of the R&S FSW.

  2. Connect the reference input (REF INPUT) on the rear panel of the R&S FSW to the reference input (REF) on the rear panel of the R&S SMU (coaxial cable with BNC connectors).

  3. Connect the external trigger input of the R&S FSW (TRIGGER INPUT) to the external trigger output of the R&S SMU (TRIGOUT1 of PAR DATA).
