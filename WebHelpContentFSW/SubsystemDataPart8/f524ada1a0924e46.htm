[SENSe:]CORRection:ENR:COMMon <State>

This command turns the use of a common ENR on or off.

For more information see "Common Noise Source".

Parameters:
<State>
ON | OFF | 1 | 0
Example: CORR:ENR:COMM ON

Turns the use of a common ENR on.

Manual operation: See "Common Noise Source"
