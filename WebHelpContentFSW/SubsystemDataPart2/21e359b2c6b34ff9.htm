Configuring a Parameter Spectrum

The parameter spectrum evaluations allow you to visualize the spectrum of results for a specific parameter for all measured pulses within the current capture buffer. For each parameter spectrum window you can configure which measured parameter is to be displayed.

For more information see "Parameter Spectrum Calculation".
