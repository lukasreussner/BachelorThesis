CALCulate<n>:STATistics:SCALe:Y:UPPer <Magnitude>

This command defines the upper vertical limit of the diagram.

Suffix: 
<n>

Window

Parameters:
<Magnitude>

The number is a statistical value and therefore dimensionless.

Range: 1E-5  to  1.0
*RST:1.0
Example: CALC:STAT:SCAL:Y:UPP 0.01
Example: 

See Programming Example: Measuring Statistics.

Manual operation: See "Defining Min and Max Values"
