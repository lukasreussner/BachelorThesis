CONFigure:REFSignal:GOS:CRESt <CrestFactor>

This command defines the crest factor of the internally generated reference signal.

Parameters:
<CrestFactor>
<numeric value>
Default unit: dB
Example: CONF:REFS:GOS:CRES 15

Defines a crest factor of 15 dB.

Manual operation: See "Crest Factor"
