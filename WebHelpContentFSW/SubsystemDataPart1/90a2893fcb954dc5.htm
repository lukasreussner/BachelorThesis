SOURce:EXTernal:FREQuency <Frequency>

This command defines a fixed source frequency for the external generator.

This command is only valid if External Generator Control (R&S FSW-B10) is installed.

Parameters:
<Frequency>

Source frequency of the external generator.

*RST:1100050000
Example: 

//Define frequency of the generator

SOUR:EXT:FREQ 10MHz
Manual operation: See "(Manual) Source Frequency"
