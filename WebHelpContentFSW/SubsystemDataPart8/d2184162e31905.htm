[SENSe:]CORRection:LOSS:INPut:MODE <Mode>

This command selects the input loss mode.

Parameters:
<Mode>
SPOT

Uses a constant input loss value for all measurement points (see [SENSe:​]CORRection:​LOSS:​INPut:​SPOT).

TABLe

Uses the contents of the input loss table.

*RST:SPOT
Example: 

CORR:LOSS:INP:MODE SPOT

Selects constant input loss.

Manual operation: See "Input Loss"
