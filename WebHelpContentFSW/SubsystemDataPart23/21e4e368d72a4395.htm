CONFigure[:LTE]:UL[:CC<cc>]:SRS:TRComb <TransComb>

This command defines the transmission comb (kTC).

Suffix: 
<cc>

Component Carrier

Parameters:
<TransComb>
<numeric value>
*RST:0
Example: CONF:UL:SRS:TRC 1

Sets transmission comb to 1.

Manual operation: See "Transm. Comb. k_TC"
