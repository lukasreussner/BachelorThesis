Profile Configuration (Downstream)

Access: "Overview" > "Signal Description" > "Profile Configuration"

or: MEAS CONFIG > "Signal Description" > "Profile Configuration"

Profiles define the modulation used by each subcarrier. For each set of modems with similar transmission conditions, a profile can then be assigned (see "Codeword / Frame Configuration").

For more information see also "Subcarriers and profiles".
