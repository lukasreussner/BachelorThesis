Remote Commands > Configuring the R&S FSW > Checking the System Configuration > SYSTem:FORMat:IDENt
SYSTem:FORMat:IDENt <IDNFormat>

This command selects the response format to the *IDN? query.

Parameters:
<IDNFormat>
LEGacy

Format is compatible to R&S FSP/FSU/FSQ/FSG family.

NEW | FSL

R&S FSW format

Format is also compatible to the R&S FSL and R&S FSV family

*RST:not reset!
Example: SYST:FORM:IDEN LEG

Adapts the return value of *IDN? to the R&S FSP/FSU/FSQ family.

Manual operation: See "*IDN Format"
