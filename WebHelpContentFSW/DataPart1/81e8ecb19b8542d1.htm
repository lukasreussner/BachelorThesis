[SENSe:]POWer:ACHannel:FILTer:ALPHa[:ALL] <Value>

This command defines the alpha value for the weighting filter for all channels.

Parameters:
<Value>
<numeric value>
*RST:0.22
Example: POW:ACH:FILT:ALPH:ALL 0.35
