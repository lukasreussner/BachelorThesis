CALCulate<n>:SGRam:TSTamp:DATA? <Frames>
CALCulate<n>:SPECtrogram:TSTamp:DATA? <Frames>

This command queries the starting time of the frames.

The return values consist of four values for each frame. If the Spectrogram is empty, the command returns '0,0,0,0'. The times are given as delta values, which simplifies evaluating relative results; however, you can also calculate the absolute date and time as displayed on the screen.

The frame results themselves are returned with TRAC:DATA? SGR

See TRACe<n>[:​DATA].

Suffix: 
<n>

irrelevant

Query parameters: 
<Frames>
CURRent

Returns the starting time of the current frame.

ALL

Returns the starting time for all frames. The results are sorted in descending order, beginning with the current frame.

Return values: 
<Seconds>

Number of seconds that have passed since 01.01.1970 till the frame start

<Nanoseconds>

Number of nanoseconds that have passed in addition to the <Seconds> since 01.01.1970 till the frame start.

<Reserved>

The third and fourth value are reserved for future uses.

Example: CALC:SGR:TST ON

Activates the time stamp.

CALC:SGR:TST:DATA? ALL

Returns the starting times of all frames sorted in a descending order.

Usage: Query only
Manual operation: See "Time Stamp"
