Symbol Constellation

For the Symbol Constellation result display, the command returns one value each for the real and imaginary parts of each symbol:

<Re0>, <Im0>, <Re1>, <Im1>, ...., <Ren>, <Imn>

The number of symbols depends on the spreading factor (see Table "Channel parameters and their dependencies").
