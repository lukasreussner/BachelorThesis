Automatic Grid Scaling

The y-axis is scaled automatically after each sweep according to the current measurement settings and results (continuously).

Tip: To update the scaling automatically once when this setting for continuous scaling is off, use the "Auto Scale Once" button or the softkey in the AUTO SET menu.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO
