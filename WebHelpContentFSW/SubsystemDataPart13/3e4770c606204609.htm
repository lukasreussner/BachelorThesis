Scrambling Code

The scrambling code identifying the base station transmitting the signal. This code can be defined in hexadecimal (default) or decimal format (see "Selecting the Scrambling Code Format").

The scrambling code for the reference carrier is taken from the Signal Description settings for CDA measurements (see "BTS Scrambling Code").

Remote command: 

[SENSe:​]TAERror:​CARRier<c>:​SCODe
