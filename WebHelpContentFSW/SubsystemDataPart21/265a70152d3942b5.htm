Signal Content Summary (downstream only)

This result display shows the summarized information for the NCPs and codewords in a specified number of frames or for a single frame. This is the currently "Selected Frame" as indicated in the "Magnitude Capture" display.

Note: This result display is not available for upstream measurements!

If more than one frame is evaluated (that is, "Analyzing a single frame (Specified Frame)" is not enabled), a statistical evaluation of the specified "Frame Statistic Count / Number of Frames to Analyze" or for all detected frames in the capture buffer is also performed. In this case, the minimum, maximum and mean values are displayed, as well as the defined limit, if available.

For details on the evaluation basis, see "Basis of (Statistical) Evaluation".

For details on individual entries, see "Signal Content Information".

Remote command: 

LAY:ADD? '1',RIGH,SCS, see LAYout:​ADD[:​WINDow]?

Results:

FETCh:​SCSummary:​ALL?

FETCh:​FRAMe:​COUNt?
