CALCulate<n>:DELTamarker<m>:LINK <State>

This command links delta marker <m> to marker 1.

If you change the horizontal position (x-value) of marker 1, delta marker <m> changes its horizontal position to the same value.

Tip: to link any marker to a different marker than marker 1, use the CALCulate<n>:​DELTamarker<m>:​LINK:​TO:​MARKer<m> or CALCulate<n>:​MARKer<m>:​LINK:​TO:​MARKer<m> commands.

Suffix: 
<n>

Window

<m>

Marker

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: CALC:DELT2:LINK ON
Manual operation: See "Linking to Another Marker"
