How to Work With Known Data Files

You can load xml files containing the possible sequences to the VSA application and use them to compare the measured data to. In particular, you can use known data for the following functions:

  • Fine synchronization during the demodulation process (see Figure "Demodulation stages of the vector signal analysis option" and "Fine Synchronization")
  • Calculation of the Bit Error Rate (BER), see "Bit Error Rate (BER)"
