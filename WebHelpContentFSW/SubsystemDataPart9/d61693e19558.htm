Phase Noise (R&S FSW-K40) > Configuration > Controlling the Measurement > On Verify Failed
On Verify Failed

Selects the way the application reacts if signal verification fails.

Takes effect on both frequency and level verification.

"Restart"

Restarts the measurement if verification has failed.

"Stop"

Stops the measurement if verification has failed.

"Run Auto All"

Starts an automatic frequency and level detection routine if verification has failed. After the new frequency and level have been set, the measurement restarts. For more information see "Automatic Measurement Configuration".

Remote command: 

[SENSe:​]SWEep:​SVFailed
