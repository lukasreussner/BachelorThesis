[SENSe:]DDEMod:PATTern:FRAMe:EDIT:PREVious:MODulation <Modulation>

Determines which modulation type is used to demodulate the frame previous to the first configured subframe.

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Setting parameters: 
<Modulation>
AUTO | DATA | PATTern
Data

The modulation type defined for data symbols is used (see [SENSe:​]DDEMod:​MAPPing[:​VALue])

Pattern

The modulation type defined for pattern symbols is used (see [SENSe:​]DDEMod:​PATTern:​MAPPing[:​VALue]).

Auto

The previous frame uses the same modulation as the last subframe of the frame configuration.

*RST:DATA
Example: SENS:DDEM:MAPP:VAL'16QAM'
SENS:DDEM:PATT:FRAM:EDIT:PREV:MOD DATA

The previous frame uses the same modulation as the data: 16QAM.

Manual operation: See "Modulation"
