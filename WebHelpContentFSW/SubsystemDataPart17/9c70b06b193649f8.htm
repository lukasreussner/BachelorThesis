Cut Off Frequency

Defines the frequency at which the passband of the RRC filter begins. Possible values are between 0.1 MHz and 2.4 MHz in 1 Hz steps. The default value is 1.25 MHz

This parameter is available for the RRC filter.

Remote command: 

CONFigure:​CDPower[:​BTS]:​MCARrier:​FILTer:​TYPE

CONFigure:​CDPower[:​BTS]:​MCARrier:​FILTer:​COFRequency
