CONFigure:CDPower[:BTS]:PVTime:FREStart <State>

If switched on, this command evaluates the limit line over all results at the end of a single sweep. The sweep is restarted if this result is FAILED.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: 

CONF:CDP:PVT:FRES ON

Restarts a single sweep if the result evaluation is failed.

Manual operation: See "Restart on Fail"
