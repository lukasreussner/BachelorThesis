Compensate for...[(FSK)]

If enabled, compensation for various effects is taken into consideration during demodulation. Thus, these distortions are not shown in the calculated error values.

Note: Note that compensation for all the listed distortions can result in lower EVM values.

  • Carrier Frequency Drift
  • FSK Deviation Error

For details on these effects see "Modulation Errors".

Remote command: 

[SENSe:​]DDEMod:​NORMalize:​CFDRift

[SENSe:​]DDEMod:​NORMalize:​FDERror
