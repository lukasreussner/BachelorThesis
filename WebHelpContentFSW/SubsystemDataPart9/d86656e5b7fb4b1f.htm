DISPlay[:WINDow<n>]:TRACe<t>:SMOothing:APERture <Percentage>

This command defines the degree (aperture) of the trace smoothing, if DISPlay[:​WINDow<n>]:​TRACe<t>:​SMOothing[:​STATe] TRUE.

Suffix: 
<n>

Window

<t>

Trace

Parameters:
<Percentage>
Range: 1  to  50
*RST:2
Default unit: PCT
Example: 

DISP3:TRAC2:SMO:APER 5

Defines an aperture of 5% for trace 2 in window 3

Manual operation: See "Smoothing"
