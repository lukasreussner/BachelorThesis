Automatic adjustment

When you select automatic adjustment of sample rate and measurement bandwidth, the application selects a bandwidth that is appropriate for the characteristics of the reference signal and adjusts the sample rate accordingly.

For more information about the reference signal, see "Designing a Reference Signal".

Remote command: 

Mode: TRACe:​IQ:​SRATe:​AUTO
