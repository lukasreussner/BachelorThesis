CONFigure[:LTE]:UL[:CC<cc>]:SUBFrame<sf>:ALLoc:PUCCh:NPAR <Parameter>

This command defines N_PUCCH on a subframe basis.

The command is available if CONFigure[:​LTE]:​UL[:​CC<cc>]:​PUCCh:​NPAR is turned on.

Suffix: 
<cc>

Component Carrier

<sf>

Subframe

Parameters:
<Parameter>
<numeric value>
Example: CONF:UL:SUBF:ALL:PUCC:NPAR 2

Sets N_PUCCH to 2.

Manual operation: See "Enhanced PUCCH Configuration"
