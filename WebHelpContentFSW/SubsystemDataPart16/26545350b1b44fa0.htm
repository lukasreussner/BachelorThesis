Subtype

Specifies the characteristics of the used transmission standard.

For details, see "Subtypes".

"0,1"

Single carrier

"2"

Increased number of active users

"3"

Modulation type 64QAM can be detected.

Remote command: 

CONFigure:​CDPower[:​BTS]:​SUBType
