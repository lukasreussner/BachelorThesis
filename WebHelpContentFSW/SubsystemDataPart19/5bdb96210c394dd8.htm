FETCh:SCDetailed:ALL?

Returns detailed signal information for each decoded RU and for each object. The result is a comma-separated list of values with 4 rows per RU, in the same order as the Signal Content Detailed result display (see "Signal Content Detailed (IEEE 802.11ax)").

The information for each decoded RU is returned in the following object order:
  • Long training field (HE-LTF)
  • Data + Pilot
  • Data only
  • Pilot only
Return values: 
<PPDU_index>
integer
<RU_index>
Range: 1 to 9
<RU_size>
26 | 52 | 106 | 242 | 484 | 996
<Object>
HE-LTF | Data + Pilot | Data | Pilot
<EVM>
Default unit: dB
<Power_per_SC>
Default unit: dBm
Example: 
FETC:SCD:ALL?
//Result:
//1,1,S996,HE-LTF,nan,-17.1766,
//1,1,S996,Data + Pilot,-83.2953,-17.1632,
//1,1,S996,Data,-83.2640,nan,
//1,1,S996,Pilot,-85.8353,nan

Usage: Query only
Manual operation: See "Signal Content Detailed (IEEE 802.11ax)"
