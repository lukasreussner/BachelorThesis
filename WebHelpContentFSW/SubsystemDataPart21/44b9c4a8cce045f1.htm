DOCSIS measurements (R&S FSW-K192/-K193) > Remote Commands for DOCSIS 3.1 Measurements > Configuring the DOCSIS 3.1 I/Q Measurement (Modulation Accuracy) > Signal Description > Continuous Pilots and Excluded Subcarrier Assignment > CONFigure:DS:CHANnel:CPES<i>:SUBCarrier:SET
CONFigure:DS:CHANnel:CPES<i>:SUBCarrier:SET <SCSet>[, <SCSet>(8191)]

Defines a comma-separated list of (discrete) subcarriers to be configured identically in the "Continuous Pilots and Excluded Subcarrier Assignment" table.

Suffix: 
<i>
1..n

index in the "Continuous Pilots and Excluded Subcarrier Assignment" table

Parameters:
<SCSet>
integer

Subcarrier number

Range: 1  to  4K mode: 4095; 8K mode: 8191
<SCSet>
Example: CONFigure:DS:CHANnel:CPES2:SUBCarrier:SET 301, 302
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Manual operation: See "Subcarrier Set"
