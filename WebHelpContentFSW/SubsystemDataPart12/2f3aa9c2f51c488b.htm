[SENSe:]DDEMod:SEARch:SYNC:FOUNd? <Name>

This command returns the name of the pattern that was found.

Return values: 
<Name>
string
Example: DDEM:SEAR:SYNC:FOUN?
Usage: Query only
Manual operation: See "Pattern Symbols Correct"
