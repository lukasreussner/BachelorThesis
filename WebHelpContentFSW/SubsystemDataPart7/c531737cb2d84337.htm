Spectrum FFT

The Spectrum FFT result display shows the frequency spectrum of the signal.

It is available for the data recorded on the RF input and both baseband inputs (I and Q channels).(Note that the Spectrum FFT of the I and Q channel are only available when parallel baseband capture has been turned on.)

The Spectrum FFT result shows the signal level in the spectrum around the center frequency. The unit is dBm.

In case of the RF spectrum, you can display the spectrum of the measured signal and the reference signal. In the best case, the measured signal has the same shape as the reference signal.

Remote command: 

Selection (RF): LAY:ADD RFS

Selection (I): LAY:OUT ISP

Selection (Q): LAY:ADD QSP

Result query: TRACe<t>[:​DATA]?
