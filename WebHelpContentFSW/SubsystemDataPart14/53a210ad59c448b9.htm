List Evaluation

The list evaluation provides the numerical results for the Power vs Time measurement.

The List Evaluation displays the following information:

Column

Description

Start / Stop

Start and stop time of the individual time intervals of the Emission Envelope Mask (in ns)

Avg

Average power measured in mask interval

Max

Maximum power measured in mask interval

Time @ MaxPower

The exact point in time when the maximum power occurred

For details, see "Power vs Time".

Remote command: 

LAY:ADD? '1',RIGH, LEV, see LAYout:​ADD[:​WINDow]?
