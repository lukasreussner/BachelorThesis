Phase Noise (R&S FSW-K40) > Analysis > Using Limit Lines > Selecting Standard Limit Lines > Select Limit Line > Compatible
Compatible

Shows if the limit line is compatible to the current measurement setup or not.

"Yes"

You can use the limit line because it is compatible to the current measurement setup.

"No"

You cannot use the limit line because it is compatible to the current measurement setup.
