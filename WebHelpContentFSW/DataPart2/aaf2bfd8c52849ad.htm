Deactivating Zoom (Selection Mode)

Deactivates any zoom mode.

Tapping the screen no longer invokes a zoom, but selects an object.

Remote command: 

Single zoom:

DISPlay[:​WINDow<n>]:​ZOOM[:​STATe]

Multiple zoom:

DISPlay[:​WINDow<n>]:​ZOOM:​MULTiple<zoom>[:​STATe] (for each multiple zoom window)
