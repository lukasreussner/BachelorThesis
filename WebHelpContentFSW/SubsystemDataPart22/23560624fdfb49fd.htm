LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > General Configuration > CONFigure[:LTE]:MEASurement
CONFigure[:LTE]:MEASurement <Measurement>

This command selects the measurement.

Parameters:
<Measurement>
ACLR

Selects the Adjacent Channel Leakage Ratio measurement.

CACLr

Selects the Cumulative ACLR measurement.

ESPectrum

Selects the Spectrum Emission Mask measurement.

EVM

Selects I/Q measurements.

MCAClr

Selects Multi-Carrier ACLR measurement.

MCESpectrum

Selects Multi-Carrier SEM measurement.

TAERor

Selects the Time Alignment Error measurement.

TPOO

Selects the Transmit On/Off Power measurement.

Example: CONF:MEAS ACLR

Selects the ACLR measurement.
