Time Sidelobe Range

Access: "Overview" > "Measurement" > "Time Sidelobe Range" tab

Or: [MEAS CONFIG] > "Pulse Meas" > "Time Sidelobe Range" tab

The time sidelobe range defines which part of the detected pulse is evaluated for sidelobe results, similarly to the result range for common pulse results. A keep-out time defines an excluded area around the center, assuming this is the mainlobe, in which no sidelobes are included.

The "Time Sidelobe Range" settings are only available if the additional option R&S FSW-K6S is installed.
