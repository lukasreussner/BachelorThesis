Subcarrier Set

Specifies the (discrete) subcarrier numbers to be configured in the same set.

When you select the input field, an edit dialog is displayed. Enter the individual subcarrier numbers in the dialog, selecting ENTER after each number.

To add further entry fields, select "Add".

Tip: to configure a series of subcarriers identically, use the "Subcarrier Range(Start/ Increment/ Stop)" settings.

Remote command: 

CONFigure:​DS:​CHANnel:​CPES<i>:​SUBCarrier:​SET
