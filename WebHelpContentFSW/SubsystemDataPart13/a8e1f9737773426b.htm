Channel Table

When the trace data for this evaluation is queried, 5 (7) values are transmitted for each channel (depending on the query parameter):

  • the class
  • the channel number
  • the absolute level
  • the relative level
  • the timing offset
  • the pilot length *)
  • the active flag *)

*) for CTAB query parameter only

For details on these parameters see TRAC:DATA? TRACE1 and TRAC:DATA? CTAB.
