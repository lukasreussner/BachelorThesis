TRIGger[:SEQuence]:LEVel:POWer:AUTO <State>

By default, the optimum trigger level for power triggers is automatically measured and determined at the start of each sweep (for Modulation Accuracy, Flatness, Tolerance... measurements).

This function is only considered for TRIG:SEQ:SOUR IFP and TRIG:SEQ:SOUR RFP, see TRIGger[:​SEQuence]:​SOURce

In order to define the trigger level manually, switch this function off and define the level using TRIGger[:​SEQuence]:​LEVel:​IFPower or TRIGger[:​SEQuence]:​LEVel:​RFPower.

Parameters for setting and query: 
<State>
OFF | 0

Switches the auto level detection function off

ON | 1

Switches the auto level detection function on

*RST:1
Manual operation: See "Trigger Level Mode"
