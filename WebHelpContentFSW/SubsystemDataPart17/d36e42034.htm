[SENSe:]CDPower:LCODe:Q <Mask>

Defines the long code mask of the Q branch of the mobile in hexadecimal form.

Parameters:
<Mask>
Range: #H0  to  #H4FFFFFFFFFF
*RST:#H0
Example: CDP:LCOD:Q '#HF'

'Define long code mask

Manual operation: See "Long Code Mask I/Long Code Mask Q"
