SYSTem:COMMunicate:RDEVice:OSCilloscope:LEDState <Color>

Returns the state of the LAN connection to the oscilloscope for the optional 2 GHz / 5 GHz bandwidth extension (R&S FSW-B2000/B5000).

For details see Alignment.

Parameters:
<Color>
GREY | RED | GREen
GREEN

Connection to the instrument has been established successfully.

GREY

Configuration state unknown, for example if you have not yet started transmission.

RED

Connection to the instrument could not be established.

Check the connection between the R&S FSW and the oscilloscope, and make sure the IP address of the oscilloscope has been defined (see SYSTem:​COMMunicate:​RDEVice:​OSCilloscope:​TCPip).

Example: SYST:COMM:RDEV:OSC:LEDS?
//Result: 'GREEN'
