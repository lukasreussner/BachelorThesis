Sequence Index (v)

Defines the sequence index (v).

The sequence index controls which of the 64 preambles available in a cell is used.

If you select the "Auto" menu item, the software automatically selects the required sequence index.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​PRACh:​SINDex
