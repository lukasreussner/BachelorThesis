[SENSe:]MIXer<x>:HARMonic:HIGH[:VALue] <HarmOrder>

This command specifies the harmonic order to be used for the high (second) range.

Suffix: 
<x>

irrelevant

Parameters:
<HarmOrder
numeric value
Range: 2 to 128 (USER band); for other bands: see band definition
Example: MIX:HARM:HIGH 2
Example: 

See "Programming Example: Working with an External Mixer" and "Configuring a conversion loss table for a user-defined band".

Manual operation: See "Harmonic Order"
