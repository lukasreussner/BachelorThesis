PMin/ PMax

Defines the power limits for each power class. The first range always starts at -200 dBm (-INF) and the last range always stops at 200 dBm (+INF). These fields cannot be modified. If more than one power class is defined, the value of "PMin" must be equal to the value of "PMax" of the previous power class and vice versa.

Note that the power level can be equal to the lower limit(s), but must be lower than the upper limit(s):

Pmin≦P<Pmax

Otherwise the ranges are corrected automatically.

Remote command: 

CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>:​MINimum

CALCulate<n>:​LIMit<k>:​ESPectrum<sb>:​PCLass<PClass>:​MAXimum
