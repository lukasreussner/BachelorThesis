Load User Modulation

This function is only available if the modulation type "User Modulation" is selected for the pattern.

Opens a file selection dialog box to select the file that contains the user-defined modulation (*.vam file).

For details on user-defined modulation files, see "User-defined Modulation".

Remote command: 

SENS:DDEM:PATT:FORM UQAM (see [SENSe:​]DDEMod:​PATTern:​FORMat)

[SENSe:​]DDEMod:​PATTern:​USER:​NAME
