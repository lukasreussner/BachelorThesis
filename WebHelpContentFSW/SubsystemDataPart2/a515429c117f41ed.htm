[SENSe:]BANDwidth:DEMod:TYPE <FilterType>
[SENSe:]BWIDth:DEMod:TYPE <FilterType>

This command defines the type of demodulation filter to be used. For information on supported filter bandwidths see the data sheet.

Parameters:
<FilterType>
FLAT | GAUSs
FLAT

Standard flat demodulation filter

GAUSs

Gaussian filter for optimized settling behavior

For Gaussian filters with a large 3dB bandwidth (> 40 MHz, only available with the bandwidth extension option) the actual filter shape deviates strongly from the ideal Gauss filter outside a range of approximately ±80 MHz. For this range the flat filter is more accurate.

For details see "Effects of Large Gauss Filters".

*RST:GAUS
