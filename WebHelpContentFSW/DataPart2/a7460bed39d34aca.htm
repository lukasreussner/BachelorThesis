[SENSe:]POWer:ACHannel:SBLock<sb>:TXCHannel:COUNt <Number>

This command defines the number of transmission channels the specific sub block contains.

This command is for MSR signals only (see CALCulate<n>:​MARKer<m>:​FUNCtion:​POWer:​PRESet).

For details on MSR signals see "Measurement on Multi-Standard Radio (MSR) Signals".

Suffix: 
<sb>
1 | 2 | 3 | 4 | 5

sub block number

Parameters:
<Number>
Range: 1  to  18
*RST:1
Example: 

See "Example: Configuring and Performing an MSR ACLR Measurement".

Manual operation: See "Number of Tx Channels (Tx Count)"
