RIQ:SELect <Source>

This command selects the reference IQ source for time sidelobe measurements.

Parameters:
<Source>
FIQ | PFM | BARKer | EBARker
FIQ

A custom waveform is loaded from an iq.tar file.

The file to be imported is defined by RIQ:​FIQ:​PATH.

PFM

A polynomial is used to define the signal's phase.

BARKer

A Barker waveform with a specified primary code is used.

EBARker

A Barker waveform with a specified primary and secondary code is used.

Manual operation: See "Reference Type"
