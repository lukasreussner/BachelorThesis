LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Signal Description > Control Channel > CONFigure[:LTE]:DL[:CC<cc>]:PDCCh:FORMat
CONFigure[:LTE]:DL[:CC<cc>]:PDCCh:FORMat <Format>

This command selects the PDCCH format.

Suffix: 
<cc>

Component Carrier

Parameters:
<Format>
-1 | 0 | 1 | 2 | 3
*RST:-1
Example: CONF:DL:PDCCH:FORM 0

Sets the PDDCH format to 0.

Manual operation: See "PDCCH Format"
