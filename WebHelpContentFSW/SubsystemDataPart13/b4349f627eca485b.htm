Code Domain Error Power

When the trace data for this evaluation is queried, 4 values are transmitted for each channel with code class 9:

code class

Highest code class of a downlink signal, always set to 9 (CC9)

code number

Code number of the evaluated CC9 channel [0...511]

CDEP

Code domain error power value of the CC9 channel in [dB]

channel flag

Indicates whether the CC9 channel belongs to an assigned code channel:

0b00-0d0: CC9 is inactive.

0b01-0d1: CC9 channel belongs to an active code channel.

0b11-0d3: CC9 channel belongs to an active code channel; sent pilot symbols are incorrect

The channels are sorted by code number.
