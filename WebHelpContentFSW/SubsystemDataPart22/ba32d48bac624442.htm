INPut:IMPedance <Impedance>

This command selects the nominal input impedance of the RF input. In some applications, only 50 Ω are supported.

Parameters:
<Impedance>
50 | 75
*RST:50 Ω
Example: INP:IMP 75
Manual operation: See "Impedance"
