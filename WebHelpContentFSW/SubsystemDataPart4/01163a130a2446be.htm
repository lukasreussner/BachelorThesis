Carrier Allocation

Defines whether a multicarrier measurement setup contains one subblock of regularly spaced carriers only (contiguous), or two subblocks of carriers with a gap in-between (non-contiguous).

For non-contiguous setups the position of the gap must be defined as the number of the active carrier after which the gap starts (see Gap start after carrier).

For details see "Contiguous vs Non-Contiguous Multicarrier Allocation".

Remote command: 

CONFigure[:​MS]:​MCARrier:​FALLocation
