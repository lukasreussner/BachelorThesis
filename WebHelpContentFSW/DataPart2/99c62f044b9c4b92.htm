[SENSe:]SWEep:DURation? <Time>

This command provides an estimation of the total time required to capture the data and process it. This time span may be considerably longer than the actual sweep time (see [SENSe:​]SWEep:​TIME).

Tip: To determine the necessary timeout for data capturing in a remote control program, double the estimated time and add 1 second.

Return values: 
<Time>
Example: SWE:TIME 1s
SWE:DUR?

Reply:

27.9734842578
Usage: Query only
Manual operation: See "Sweep Time"
See "Data capturing takes too long"
