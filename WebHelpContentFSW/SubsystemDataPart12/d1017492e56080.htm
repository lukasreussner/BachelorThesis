[SENSe:]DDEMod:MAPPing:CATalog?

This command queries the names of all mappings that are available for the current modulation type and order. A mapping describes the assignment of constellation points to symbols.

Return values: 
<Mappings>

A comma-separated list of strings, with one string for each mapping name.

Example: :SENSe:DDEMod:MAPPing:CATalog?

Result:

'CDMA2K_FWD','DVB_S2','GRAY','NATURAL','WCDMA'
Example: 

See "Measurement Example 1: User-defined Measurement of Continuous QPSK Signal".

Usage: Query only
Manual operation: See "Modulation Mapping"
