Segmented Capture: Timestamps inconsistent. Please reduce pre-trigger time.

This message appears during segmented capture (see "Segmented Capture") if a large pre-trigger time is used (negative trigger offset) such that multiple trigger events occur within the pre-trigger interval. In this case, an exact allocation of timestamps to segments is not possible. Reduce the pre-trigger time to avoid this situation.
