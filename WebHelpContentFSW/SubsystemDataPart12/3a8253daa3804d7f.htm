Eye Diagram Real (I)

The eye pattern of the inphase (I) channel; the x-axis value range is from -1 to +1 symbols (MSK: -2 to +2)

Available for source types:
  • Meas & Ref Signal
Result display "Eye Diagram Real (I)"

Display lines are available in eye diagrams which allow you to determine the size of the eye, see also "How to Measure the Size of an Eye".

Remote commands:

LAY:ADD? '1',BEL,MEAS

to define the required source type (see LAYout:​ADD[:​WINDow]?)

CALC:FORM IEYE

to define the result type (see CALCulate<n>:​FORMat)

TRAC:DATA? TRACE1

to query the trace results (see TRACe<n>[:DATA]? TRACE<n> and "Cartesian Diagrams")
