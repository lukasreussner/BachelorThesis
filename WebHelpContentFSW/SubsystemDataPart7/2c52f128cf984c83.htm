Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Configuring Amplifier Measurements > Controlling a Signal Generator > CONFigure:DUT:STIMe
CONFigure:DUT:STIMe <Time>

This command defines the settling time between generator setting changes and the start of the next measurement.

Parameters:
<Time>
<numeric value>
*RST:0
Default unit: s
Example: 

//Define settling delay

CONF:DUT:STIM 0.5
Manual operation: See "Settling Delay"
