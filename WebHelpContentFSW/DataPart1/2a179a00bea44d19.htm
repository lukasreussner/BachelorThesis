Example: Measuring Phase Noise

This example demonstrates how to measure phase noise using markers in a basic spectrum measurement in a remote environment. It assumes that the basic frequency sweep described in "Basic Frequency Sweep Measurement for Marker Function Examples" has been performed and thus does not begin by presetting the instrument.

//------------- Configuring the phase noise marker -------------------
DET SAMP
//Switches to Sample detector

CALC:MARK1 ON
//Activates marker1 and sets it to the maximum power level

CALC:DELT:FUNC:PNO ON
//Activates phase noise marker function

CALC:DELT1 ON
CALC:DELT1:X 100kHz

CALC:DELT2 ON
CALC:DELT2:X 500kHz

CALC:DELT3 ON
CALC:DELT3:X 1MHz

CALC:DELT4 ON
CALC:DELT4:X 1.5MHz

//Activates the phase noise measurement function for offsets 100kHz/500kHz/1MHz/1.5MHz.

BAND:VID?
//Queries the used VBW (= 0.1*RBW)


//-------- Querying the phase noise results ------------------------------

CALC:DELT1:FUNC:PNO:RES?
CALC:DELT2:FUNC:PNO:RES?
CALC:DELT3:FUNC:PNO:RES?
CALC:DELT4:FUNC:PNO:RES?
//Queries the difference in level between the peak and the noise power density
//measured at the deltamarkers, referred to the carrier power level (dBc)
