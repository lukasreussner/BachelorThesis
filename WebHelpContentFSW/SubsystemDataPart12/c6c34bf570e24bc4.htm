Range per Division

Defines the value range to be displayed per division on the x-axis. Since the display consists of 10 divisions by default, the displayed range is:

Range = 10 * <Range per Division>

Note: If fewer divisions are displayed (e.g. because the window is reduced in width), the range per division is increased to display the same result range in the smaller window. In this case, the per division value does not correspond to the actual display.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​X[:​SCALe]:​PDIVision
