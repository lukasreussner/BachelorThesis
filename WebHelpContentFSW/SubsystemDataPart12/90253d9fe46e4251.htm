Signal Capture

The signal capture commands define how much, how and when data is captured from the input signal.

MSRA/MSRT operating mode

In MSRA/MSRT operating mode, only the MSRA/MSRT Master channel actually captures data from the input signal. The data acquisition settings for the VSA slave application in MSRA/MSRT mode define the slave application data extract and analysis interval.

For details on the MSRA operating mode see Applications and Operating Modes. For details on the MSRT operating mode see Applications and Operating Modes.

The tasks for manual operation are described in "Signal Capture".
