CALCulate:LIMit<li>:TYPE <Result>

This command configures a limit line for a particular result type.

Suffix: 
<li>

Limit line

Parameters:
<Result>
GAIN

Assigns the limit line to gain reuslts.

NOISe

Assigns the limit line to noise figure results.

PCOLd

Assigns the limit line to power (cold) results.

PHOT

Assigns the limit line to power (hot) results.

TEMPerature

Assigns the limit line to noise temperature results.

YFACtor

Assigns the limit line to y-factor results.

Example: CALC:LIM2:TYPE GAIN

Assigns limit line 2 to the gain result display.
