Retrieving and Storing Trace Data

In order to retrieve the trace results in a remote environment, use the following command:

List of Commands
TRACe<n>[:DATA]?
TRACe<n>[:DATA]:X?
TRACe:IQ:DATA?
TRACe:IQ:DATA:MEMory?
TRACe:IQ:DATA:RRANge?
MMEMory:STORe<n>:TRACe
