Scaling the Y-Axis

Access: "Overview" > "Result Config" > "Scaling" > "Y Scaling"

By default, the application automatically scales the y-axis based on the current results. The scale changes when new measurement results are available. When you change the scale manually, the changes are shown in the diagram next to the settings.

Scope of the scaling

Scaling is applied only to the result display that you have selected from the "Specifics for:" dropdown menu at the bottom of the dialog box.

(In this case, the scale is applied to the Spectrum FFT result display.)
