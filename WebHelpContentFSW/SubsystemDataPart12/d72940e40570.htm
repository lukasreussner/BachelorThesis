Meas only if Pattern Symbols Correct

If enabled, measurement results are only displayed (and are only averaged) if a valid pattern has been found. When measuring signals that contain a pattern and are averaged over several measurements, it is recommended that you enable this option so that erroneous measurements do not affect the result of averaging.

Remote command: 

[SENSe:​]DDEMod:​SEARch:​SYNC:​MODE
