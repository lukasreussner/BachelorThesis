Evaluation

Defines whether the trace displays the evaluation of the measured signal or the reference signal (if "Meas & Ref Signal" is used as the evaluation data source, see "Signal Source").

For multi-source results, the evaluation for each trace can be defined as error or capture buffer/measurement (depending on the result type, see "Multi Source").

Remote command: 

CALCulate<n>:​TRACe<t>[:​VALue]
