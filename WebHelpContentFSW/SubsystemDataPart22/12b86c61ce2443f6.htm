PDSCH Power Ratio

Selects the PDSCH P_B parameter that defines the cell-specific ratio of rho_B to rho_A according to 3GPP TS 36.213, table 5.2-1.

The table below shows the resulting values as a function of the number of antennas.

If you select "p_B/p_A=1", the ratio is always 1, regardless of the number of antennas.

Remote command: 

CONFigure[:​LTE]:​DL[:​CC<cc>]:​PDSCh:​PB
