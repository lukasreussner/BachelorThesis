TRACe<t>[:DATA]? <Trace>

This command queries current trace data and measurement results from the selected window.

For details see "Measurement Results for TRACe<n>[:DATA]? TRACE<n>".

Suffix: 
<t>
1..n

irrelevant

Query parameters: 
<Trace>
TRACe1 | TRACe2 | TRACe3 | TRACe4 | TRACe5 | TRACe6

Selects the type of result to be returned.

TRACE1 | ... | TRACE6

Returns the trace data for the corresponding trace.

For most DOCSIS 3.1 I/Q measurements, only TRACE1 is available (unless specified otherwise in "Measurement Results for TRACe<n>[:DATA]? TRACE<n>").

Example: DISP:WIND2:SEL
TRAC? TRACE3

Queries the data of trace 3 in window 2.
