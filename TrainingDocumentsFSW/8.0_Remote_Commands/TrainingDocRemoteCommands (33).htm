INSTrument:MODE <OpMode>

The operating mode of the R&S FSW determines which applications are available and active. Whenever you change the operating mode, the currently active channels are closed. The default operating mode is Signal and Spectrum Analyzer mode, however, the presetting can be changed.

(See "Preset Mode").

For details on operating modes and applications see "Applications, Measurement Channels, and Operating Modes".

Parameters:
<OpMode>
SANalyzer

Signal and Spectrum Analyzer mode

MSRanalyzer

Multi-Standard Radio Analysis (MSRA) mode

RTMStandard

Multi-Standard Real-Time (MSRT) mode

Only available if one of the real-time options is installed.

(See See Also).

*RST:SAN
Example: INST:MODE MSR

Switches to Multi-Standard Radio Analysis (MSRA) mode.
