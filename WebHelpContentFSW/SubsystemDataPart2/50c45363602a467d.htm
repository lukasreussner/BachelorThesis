Segmented Capture

Access: "Overview" > "Trigger" > "Segmented Capture"

Or: [TRIG] > "Trigger Config" > "Segmented Capture"

Configures data capturing with a gating function, that is non-continuous data acquisition.

Segmented capture is only possible if an external, IF Power, or RF Power trigger is used (see "Trigger Source").

When using the new "Time Sidelobe Analysis" functions, set up the capture such that there are enough pre/post samples to account for the entire reference I/Q waveform length.

For details on segmented data capture and recommended settings see "Segmented Data Capturing".
