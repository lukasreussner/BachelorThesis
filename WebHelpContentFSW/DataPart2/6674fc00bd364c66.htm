Changing the Channel Name

The measurement channels are labeled with their default name. If that name already exists, a sequential number is added. You can change the name of the measurement channel by double-tapping the name in the channel bar and entering a new name.

For an overview of default names, see Table "Available channel types and default channel names in Signal and Spectrum Analyzer mode".

Remote command: 

INSTrument:​REName
