[SENSe:]ADEMod<n>:PRESet:RESTore

This command restores the default configurations of predefined Analog Demodulation standards.

Note that the command will overwrite customized standards that have the same name as predefined standards.

Suffix: 
<n>

irrelevant

Usage: Event
Manual operation: See "Restore Standard Files"
