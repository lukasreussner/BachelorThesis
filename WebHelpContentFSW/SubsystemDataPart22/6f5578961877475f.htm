LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Signal Description > CSI Reference Signal > CONFigure[:LTE]:DL[:CC<cc>]:CSIRs:CI
CONFigure[:LTE]:DL[:CC<cc>]:CSIRs:CI <Index>

This command selects the configuration index for the CSI reference signal.

Suffix: 
<cc>

Component Carrier

Parameters:
<Index>
MINimum | MAXimum | UP | DOWN | DEFault

Number of the configuration index.

Range: 0  to  31
Example: CONF:DL:CSIR:CI 12

Selects configuration index 12 for the CSI reference signal.

Manual operation: See "Configuration Index"
