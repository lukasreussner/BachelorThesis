Allocation ID vs Symbol x Carrier

The Allocation ID vs. Symbol X Carrier display shows the allocation ID of each carrier in each symbol of the received signal.

Each type of allocation is represented by a different color. Use a marker to get more information about the type of allocation.

Remote command: 

Selecting the result display: LAY:ADD ? '1',LEFT,AISC

Querying results: TRACe:DATA?
