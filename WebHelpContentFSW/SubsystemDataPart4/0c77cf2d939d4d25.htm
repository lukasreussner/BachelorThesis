CONFigure[:MS]:POWer:CLASs <Value>

This command the power class of the device under test.

Parameters for setting and query: 
<Value>
1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | E1 | E2 | E3 | M1 | M2 | M3 | P1 | NONE
1

MS and BTS power class 1

2

MS and BTS power class 2

3

MS and BTS power class 3

4

MS and BTS power class 4

5

MS and BTS power class 5

6

BTS power class 6

7

BTS power class 7

8

BTS power class 8

E1

MS power class E1

E2

MS power class E2

E3

MS power class E3

M1

BTS power class M1 (Micro)

M2

BTS power class M2 (Micro)

M3

BTS power class M3 (Micro)

P1

BTS power class P1 (Pico)

NONE

No power classes defined

*RST:2
Example: CONF:POW:CLAS 1
