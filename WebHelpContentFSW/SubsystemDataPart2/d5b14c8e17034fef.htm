Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Reference Signal Description > RIQ:FIQ:RANGe:LENGth
RIQ:FIQ:RANGe:LENGth <Time>

Defines the length of the reference pulse in the data file in seconds.

Parameters:
<Time>
Default unit: S
Example: RIQ:FIQ:RANG:AUTO OFF
RIQ:FIQ:RANG:OFFS 10 ms
RIQ:FIQ:RANG:LENG 1s

Defines the use of a reference range that is 1 second long and starts after 10 ms.

Manual operation: See "Length"
