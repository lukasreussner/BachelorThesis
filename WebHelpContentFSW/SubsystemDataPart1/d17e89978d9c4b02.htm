Marker Usage

Access: "Overview" > "Analysis"

The following marker settings and functions are available in the I/Q Analyzer application.

For "I/Q-Vector" displays markers are not available.

In the I/Q Analyzer application, the resolution with which the frequency can be measured with a marker is always the filter bandwidth, which is derived from the defined sample rate (see "Sample Rate and Maximum Usable I/Q Bandwidth for RF Input").

Marker settings are now window-specific.
