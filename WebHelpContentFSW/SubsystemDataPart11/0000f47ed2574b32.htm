Full Capture / Region Analysis / Hop / Chirp
For some result displays you can select the basis used for analysis:
  • The full capture buffer
  • The selected "Analysis Region"
  • An individual selected hop / chirp (only available if at least one of the additional options R&S FSW-K60C/-K60H are installed, see "Select Hop / Select Chirp")
    The currently selected hop / chirp is highlighted blue in the "Result Table" and "Statistic Table" displays.
    As soon as a new hop / chirp is selected, all hop/chirp-specific displays are automatically updated.
Remote command: 

DISPlay[:​WINDow<n>]:​EVALuate
