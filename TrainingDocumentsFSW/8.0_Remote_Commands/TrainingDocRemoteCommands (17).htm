CALCulate<n>:MARKer<m>:FUNCtion:POWer:PRESet <Standard>

This command loads a measurement configuration.

The measurement configuration for power measurements consists of weighting filter, channel bandwidth and spacing, resolution and video bandwidth, detector and sweep time.

If the "Multi-Standard Radio" standard is selected (see "Standard"), different commands are required to configure ACLR measurements (see "Configuring MSR ACLR Measurements").

Suffix: 
<n>

Window

<m>

Marker

Parameters:
<Standard>

For more information see "Reference: Predefined CP/ACLR Standards".

If you want to load a customized configuration, the parameter is a string containing the file name.
