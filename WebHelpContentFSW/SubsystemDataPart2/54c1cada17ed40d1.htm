I/Q Export

Opens a file selection dialog box to define an export file name to which the I/Q data is stored. This function is only available in single sweep mode.

For details, see I/Q Data Import and Export.

Note: Storing large amounts of I/Q data (several Gigabytes) can exceed the available (internal) storage space on the R&S FSW. In this case, it can be necessary to use an external storage medium.

Note: I/Q data cannot be imported in MSRT mode; however, you can use the I/Q Analyzer or (optional) Transient Analysis applications, for example, to import and analyze the exported I/Q data in Signal and Spectrum Analyzer mode later.

Note: Secure user mode.

In secure user mode, settings that are stored on the instrument are stored to volatile memory, which is restricted to 256 MB. Thus, a "memory limit reached" error can occur although the hard disk indicates that storage space is still available.

To store data permanently, select an external storage location such as a USB memory device.

For details, see Protecting Data Using the Secure User Mode.

Remote command: 

MMEMory:​STORe<n>:​IQ:​STATe

MMEMory:​STORe<n>:​IQ:​COMMent
