Composite Constellation

When the trace data for this evaluation is queried, the real and the imaginary branches of each chip are transferred:

<Re chip0>, <Im chip0>, <Re chip1>, <Im chip1>, ...., <Re chipn>, <Im chipn>

The number of value pairs corresponds to the number of chips from the 1024 chips in a half slot.
