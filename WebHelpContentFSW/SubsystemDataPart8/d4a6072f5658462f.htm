Output Match

Defines the output match of the noise source you are using.

You can define the output match either as the VSWR or as the return loss (RL).

Refer to the datasheet of the noise source for these values.

Remote command: 

CALCulate:​UNCertainty:​MATCh:​SOURce[:​VSWR]

CALCulate:​UNCertainty:​MATCh:​SOURce:​RL

CALCulate:​UNCertainty:​MATCh:​SOURce:​CALibration[:​VSWR]

CALCulate:​UNCertainty:​MATCh:​SOURce:​CALibration:​RL
