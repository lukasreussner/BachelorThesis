Auto RBW

Sets the RBW for the complete range such that the noise floor is displayed slightly below the user-defined threshold. Since the noise floor may vary across the frequencies, the RBW needs to be adapted to the signal. This is done by splitting the ranges into smaller segments with similar signal and noise characteristics.

"On"

The R&S FSW Spurious measurements application divides the ranges for which "Auto RBW" is active into segments according to the noise and power levels measured in the Spectral Overview sweep. The application then adapts the RBW setting within the segments so the noise floor lies below the threshold for the displayed signal. As a result, an additional "Segment" table is created with the used RBW for each segment (see "Show Segment Table").

"Off"

The RBW becomes available for manual editing again and the segment table is deleted. The default parameter values are restored.

Remote command: 

[SENSe:​]LIST:​RANGe<range>:​BANDwidth:​AUTO
