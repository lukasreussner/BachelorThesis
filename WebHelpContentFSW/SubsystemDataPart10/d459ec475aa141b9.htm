Carrier Reference Level

Access: "Overview" > "Carrier Reference Level"

The carrier, or the desired signal, is assumed to have the maximum power level in the input signal. Thus, determining the maximum peak allows for the R&S FSW Spurious measurements application to measure power values relative to the carrier.
