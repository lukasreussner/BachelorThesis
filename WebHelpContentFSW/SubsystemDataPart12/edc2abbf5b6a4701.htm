CALCulate<n>:LIMit:MACCuracy:FERRor:RPEak:VALue <LimitValue>

This command defines the value for the current, peak or mean frequency error (peak or RMS) limit. Note that the limits for the current and the peak value are always kept identical.

This command is available for FSK modulation only.

Setting parameters: 
<LimitValue>
numeric value

the value x (x>0) defines the interval [-x; x]

Range: 0.0  to  100
*RST:1.5 (mean: 1.0)
Default unit: Hz
