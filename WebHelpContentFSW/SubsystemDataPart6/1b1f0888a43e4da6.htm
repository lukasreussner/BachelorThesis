[SENSe:]ADEMod:SPECtrum:PHASe:TYPE <Trace1>, <Trace2>, <Trace3>

This command sets the modes of the first three traces of phase windows.

Note that this command is maintained for compatibility reasons only. Use DISPlay[:​WINDow<n>]:​TRACe<t>:​MODE for new remote control programs.

Parameters:
<Trace1>
WRITe | AVERage | MINHold | MAXHold | VIEW | OFF
<Trace2>
WRITe | AVERage | MINHold | MAXHold | VIEW | OFF
<Trace3>
WRITe | AVERage | MINHold | MAXHold | VIEW | OFF
