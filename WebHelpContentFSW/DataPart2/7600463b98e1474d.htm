Margins

Defines margins for the printout page on which no elements are printed. The margins are defined according to the selected unit.

Remote command: 

HCOPy:​PAGE:​MARGin<device>:​BOTTom

HCOPy:​PAGE:​MARGin<device>:​LEFT

HCOPy:​PAGE:​MARGin<device>:​RIGHt

HCOPy:​PAGE:​MARGin<device>:​TOP

HCOPy:​PAGE:​MARGin<device>:​UNIT
