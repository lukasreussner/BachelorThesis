Remote Commands > Commands for Remote Instrument Operation > SYSTem:CLOGging
SYSTem:CLOGging <State>

This command turns logging of remote commands on and off.

Parameters:
<State>
ON | 1

Writes all remote commands that have been sent to a file.

The destination is C:\Program Files (x86)\Rohde-Schwarz\FSW\<version>\ScpiLogging\ScpiLog.txt.

OFF | 0
*RST:0
Manual operation: See "I/O Logging"
