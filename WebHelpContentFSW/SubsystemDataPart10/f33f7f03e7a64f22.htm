Mark Residual Spurs

If enabled, residual spurs are indicated in the diagrams in light gray. If a residual spur coincides with a "true" spur, the spur is output using the residual color.

Note that some residuals detected in the spectral overview sweep might not be marked in the final spur results even if the "Mark Residual Spurs" setting is enabled, due to the different measurement parameters for the spectral overview sweep and the spot search.

Remote command: 

[SENSe:​]SSEarch:​RMARk
