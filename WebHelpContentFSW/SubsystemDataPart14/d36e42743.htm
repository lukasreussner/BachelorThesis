TD-SCDMA Measurements (R&S FSW-K76/-K77) > Remote Commands for TD-SCDMA Measurements > Configuring Code Domain Analysis > Channel Detection > Configuring Channel Tables > CONFigure:CDPower:CTABle:COMMent
CONFigure:CDPower:CTABle:COMMent <Comment>

This command defines a comment for the channel table selected with CONFigure:​CDPower:​CTABle:​NAME.

Parameters:
<Comment>

comment for the channel table

Example: CONF:CDP:CTAB:NAME 'CTAB_1'

Selects channel table 'CTAB_1'.

CONF:CDP:CTAB:COMM 'Comment for CTAB_1'

Writes a comment for 'CTAB_1'.

Manual operation: See "Comment"
