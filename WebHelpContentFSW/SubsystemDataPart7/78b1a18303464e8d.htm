FETCh:DDPD:OPERation:STATus?

This command queries the state of a direct DPD operation.

Return values: 
<State>
ON | OFF | 1 | 0
ON

Direct DPD operation was successful.

OFF

Direct DPD operation was not successful.

Example: 

//Query direct DPD state

FETC:DDPD:OPER:STAT?
Usage: Query only
