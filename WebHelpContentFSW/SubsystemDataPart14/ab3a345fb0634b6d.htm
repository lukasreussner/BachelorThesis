Evaluation Methods for Code Domain Analysis

Access: "Overview" > "Display Config"

The captured I/Q data can be evaluated using various different methods without having to start a new measurement. All evaluation methods available for the selected TD-SCDMA measurement are displayed in the evaluation bar in SmartGrid mode.

For details on working with the SmartGrid, see Laying out the Result Display with the SmartGrid.

The selected evaluation also affects the results of the trace data query (see "Measurement Results for TRACe<n>[:DATA]? TRACE<n>").
