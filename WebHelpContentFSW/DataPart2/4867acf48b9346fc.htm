X-Offset

Shifts a limit line that has been specified for relative frequencies or times (x-axis) horizontally.

This setting does not have any effect on limit lines that are defined by absolute values for the x-axis.

Remote command: 

CALCulate<n>:​LIMit<k>:​CONTrol:​OFFSet
