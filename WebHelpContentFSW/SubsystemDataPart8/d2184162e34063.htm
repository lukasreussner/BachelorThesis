CALCulate:UNCertainty:ENR:UNCertainty <Uncertainty>

This command defines the uncertainty of a noise source.

If the noise sources during calibration and measurement are different, the command defines the uncertainty of the measurement noise source.

Parameters:
<Uncertainty>

Uncertainty value of the noise source.

Refer to the data sheet of the noise source to determine its uncertainty.

*RST:0.1 dB
Example: CALC:UNC:ENR:UNC 0.05

Defines an uncertainty of 0.05 dB.

Manual operation: See "ENR Uncert(ainty)"
