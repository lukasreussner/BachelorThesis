A/N + SRS Simultaneous TX

Turns simultaneous transmission of the Sounding Reference Signal (SRS) and ACK/NACK messages (via PUCCH) on and off.

By turning the parameter on, you allow for simultaneous transmission of PUCCH and SRS in the same subframe.

If off, the SRS not transmitted in the subframe for which you have configured simultaneous transmission of PUCCH and SRS.

Note that simultaneous transmission of SRS and PUCCH is available only if the PUCCH format is either 1, 1a, 1b or 3. The other PUCCH formats contain CQI reports which are not transmitted with the SRS.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SRS:​ANST
