FETCh[:CC<cc>]:SUMMary:RSSI:MAXimum?
FETCh[:CC<cc>]:SUMMary:RSSI:MINimum?
FETCh[:CC<cc>]:SUMMary:RSSI[:AVERage]?

This command queries the RSSI.

Suffix: 
<cc>

Component Carrier

Return values: 
<RSSI>
<numeric value>

Minimum, maximum or average sampling error, depending on the last command syntax element.

Default unit: dBm
Example: 

//Query average RSSI

FETC:SUMM:RSSI?
Usage: Query only
Manual operation: See "Result Summary"
