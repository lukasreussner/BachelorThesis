EMI Final Measurement Configuration

The final EMI measurement can be performed with different settings than the initial peak search. These settings are described here.

The detector to be used for the final EMI test can be defined differently for each frequency, thus the detector is configured in the EMI marker settings, see "Final Test Detector".
