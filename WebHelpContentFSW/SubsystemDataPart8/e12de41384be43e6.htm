Automatic Control

Turns automatic control of the external generator on and off.

If on, the application controls the configuration of the external generator during measurements by transmitting the required remote commands. The connection is established automatically before a measurement if Init Before Meas is on. To establish the connection manually at any time, use the Init External Generator function.

Otherwise, you have to control the generator manually. The application assumes that the generator is configured correctly for the measurement. The remote connection to the generator is automatically terminated.

Turn on automatic control to perform Fixed IF measurements with the LO frequency tuned to the current measurement (RF) frequency. Manual control is recommended only for measurements whose generator settings remain the same during the measurement.

Remote command: 

SYSTem:​CONFigure:​GENerator:​CONTrol:​STATe
