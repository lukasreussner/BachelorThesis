[SENSe:]POWer:ACHannel:REFerence:TXCHannel:MANual <ChannelNumber>

This command defines a reference channel for relative ACLR measurements.

You need at least one channel for the command to work.

Note that this command is not available for MSR ACLR measurements (see CALCulate<n>:​MARKer<m>:​FUNCtion:​POWer:​PRESet).

Parameters:
<ChannelNumber>
Range: 1  to  18
*RST:1
Example: 

See "Example: Configuring and Performing an ACLR Measurement".

Manual operation: See "Reference Channel"
