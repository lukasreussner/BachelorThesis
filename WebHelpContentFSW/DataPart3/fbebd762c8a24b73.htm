Trace Math Function

Defines which trace is subtracted from trace 1. The result is displayed in trace 1.

The result refers to the zero point defined with the "Trace Math Position" setting. The following subtractions can be performed:

"T1-T2->T1"

Subtracts trace 2 from trace 1.

"T1-T3->T1"

Subtracts trace 3 from trace 1

"T1-T4->T1"

Subtracts trace 4 from trace 1

"T1-T5->T1"

Subtracts trace 5 from trace 1

"T1-T6->T1"

Subtracts trace 6 from trace 1

To switch off the trace math, use the "Trace Math Off" button.

Remote command: 

CALCulate<n>:​MATH[:​EXPRession][:​DEFine]

CALCulate<n>:​MATH:​STATe
