Trigger 2/3

Access: "Overview" > "Signal Capture" > "Trigger In/Out"

Or: TRIG > "Trigger Config" > "Trigger In/Out"

Defines the usage of the variable TRIGGER INPUT/OUTPUT connectors, where:

"Trigger 2": TRIGGER INPUT/OUTPUT connector on the front panel

"Trigger 3": TRIGGER 3 INPUT/ OUTPUT connector on the rear panel

(Trigger 1 is INPUT only.)

Note: Providing trigger signals as output is described in detail in How to Output a Trigger Signal.

"Input"

The signal at the connector is used as an external trigger source by the R&S FSW. Trigger input parameters are available in the "Trigger" dialog box.

"Output"

The R&S FSW sends a trigger signal to the output connector to be used by connected devices.

Further trigger parameters are available for the connector.

Remote command: 

OUTPut:​TRIGger<port>:​DIRection

 
└ Output Type
└ Level
└ Pulse Length
└ Send Trigger
