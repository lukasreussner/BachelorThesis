Result Range Configuration

Access: "Overview" > "Cut Result Ranges"

or: [MEAS CONFIG] > "Range Settings" > "Result Range" tab

The result range determines which part of the capture buffer, burst or pattern is displayed. For more information, see "Measurement Ranges".

A visualization of the result display with the current settings is displayed in the visualization area at the bottom of the dialog box.

The evaluation range settings are described in "Evaluation Range Configuration".

For multi-modulation analysis (requires option R&S FSW-K70M, see "Multi-Modulation Analysis (R&S FSW-K70M)"), the result length is set to the total number of symbols in the frame structure configuration by default (see "Frame Configuration"). The number can be decreased to exclude symbols at the beginning or end of the frame. The alignment and offset of the result range are automatically taken from the frame structure configuration and cannot be edited.
