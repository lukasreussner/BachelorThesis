CONFigure[:LTE]:UL[:CC<cc>]:PUCCh:FORMat <Format>

This command selects the PUCCH format.

Note that formats 2a and 2b are available for normal cyclic prefix length only.

Suffix: 
<cc>

Component Carrier

Parameters:
<Format>
F1 (F1)
F1A (F1a)
F1B (F1b)
F2 (F2)
F2A (F2a)
F2B (F2b)
F3 (F3)
SUBF

Allows you to define the PUCCH format for each subframe separately with .

*RST:F1
Example: CONF:UL:PUCC:FORM F1B

Sets the PUCCH format to F1B.

Manual operation: See "Format"
