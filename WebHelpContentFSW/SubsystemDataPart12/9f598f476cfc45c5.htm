Cartesian Diagrams

For cartesian diagrams (magnitude, phase, frequency, real/imag, eye diagrams), the command returns the y-values of the trace. The number of returned values is the product of the "Result Length" and the display points per symbol. The unit depends on the specified unit.

(See "Scaling and Units").

You can query the x-value that relates to the first value of the y-axis using DISPlay[:​WINDow<n>]:​TRACe<t>:​X[:​SCALe]:​STARt?.

The eye diagram result displays are the same as the real/imag result display; the results for eye diagrams are merely superimposed in the display.
