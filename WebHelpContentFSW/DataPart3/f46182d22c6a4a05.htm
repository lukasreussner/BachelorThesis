QuickSave 1 / ... / QuickSave 10

Selects one of the save sets to store the current settings in or to be recalled. At the time of storage, the "QuickSave 1 / ... / QuickSave 10" placeholder is replaced by a label indicating the storage date and time and the storage type.

During recall, save sets of type "Instrument" replace the settings of the entire instrument. All other save sets start a new channel with the stored settings.

Note: Saving instrument settings in secure user mode. Settings that are saved via Quick Save in secure user mode are only available during the current session. As soon as the power is switched off on the R&S FSW, the data is cleared (see "Protecting Data Using the Secure User Mode").
