Load User Modulation

This function is only available if the modulation type "User Modulation" is selected.

Opens a file selection dialog box to select the file that contains the user-defined modulation (*.vam file).

For details on user-defined modulation files, see "User-defined Modulation".

Remote command: 

SENS:DDEM:FORM UQAM (see [SENSe:​]DDEMod:​FORMat)

[SENSe:​]DDEMod:​USER:​NAME
