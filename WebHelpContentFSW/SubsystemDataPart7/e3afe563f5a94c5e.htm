Amplifier Measurements (R&S FSW-K18) > Remote Control Commands for Amplifier Measurements > Analyzing Results > Managing Measurement Data > MMEMory:LOAD:IQ:STATe
MMEMory:LOAD:IQ:STATe 1, <FileName>

This command restores the currently captured I/Q data to a file.

After restoring the I/Q data, the application also analyzes the data again.

Setting parameters: 
1
<FileName>

String containing the path and file name.

Example: MMEM:LOAD:IQ:STAT 1,'C:\IQData\Amplfier.iq.tar'

Restores the specified I/Q data.

Usage: Setting only
