Omitting the Digital Decimation Filter (No Filter)

This setting is only available when using the optional Digital Baseband Interface.

If enabled, no digital decimation filter is used during data acquisition. Thus, the "Analysis Bandwidth" is identical to the input sample rate configured for the Digital I/Q input source (see "Input Sample Rate").

Note, however, that in this case noise, artifacts, and the second IF side band may not be suppressed in the captured I/Q data.

Remote command: 

TRACe:​IQ:​DIQFilter
