CALCulate<n>:MSRA:WINDow<n>:MIVal?

This command queries the individual analysis intervals in the window specified by the WINDow suffix <n> (the CALC suffix is irrelevant). This command is only available for GSM measurement channels (R&S FSW-K10) in MSRA mode, and only for result displays that display traces with a history, i.e. maxhold, minhold or average traces.

The result is a comma-separated list of <IntStart>,<IntStop> values for each interval.

Return values: 
<IntStart>

Start value of the analysis interval in seconds (global time scale)

Default unit: s
<IntStop>

Stop value of the analysis interval in seconds (global time scale)

Example: CALC:MSRA:WIND2:MIV?

//Result:

+3.707922995E-003,+4.509000108E-003,+8.323308080E-003,+9.124384262E-003,+1.293869223E-002,+1.373976935E-002
Usage: Query only
