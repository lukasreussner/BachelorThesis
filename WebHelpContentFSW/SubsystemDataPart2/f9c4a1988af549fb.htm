SENSe:TRACe:MEASurement:DEFine:FREQuency:RATE <PulseChirpRate>

Defines a known frequency chirp rate (in Hz/μs) to be used to generate an ideal pulse waveform for computing frequency and phase error parameters. This value is assumed constant for all measured pulses.

Use the SENSe:​TRACe:​MEASurement:​DEFine:​FREQuency:​RATE:​AUTO to define the chirp rate automatically.

Parameters:
<PulseChirpRate>
*RST:0
Default unit: Hz/μs
Manual operation: See "Chirp Rate"
