General Instrument Setup > Transducers > Transducer Settings > Transducer Factors > Export
Export

Opens a file selection dialog box and stores the currently displayed transducer factor to the defined file in .CSV format.

For details on the file format see "Reference: Transducer Factor File Format".

The transducer factor can be imported again later by the R&S FSW for use in other measurements.

Remote command: 

MMEMory:​STORe<n>:​TFACtor
