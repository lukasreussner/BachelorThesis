DEVice:INFO:HWBand?

Queries the frequency bands used for measurement by the R&S FSW hardware. The start frequency of each band is provided.

The bands are instrument-specific and depend on the currently defined RBW, VBW and YIG preselector state. The precise frequency bands are required to define correction data for the correct bands, in particular for frequency-drifting DUTs.

This query is only available in zero span mode.

Example: 
//Set to zero span mode
FREQ:SPAN 0
//Set RBW
BAND:RES 1000000
//Set VBW
BAND:VID 10000
//Activate YIG filter
INP:FILT:YIG ON
//Query used hardware bands
DEV:INFO:HWB?
//Result:
//0,50000000,450000000,1000000000,3000000000,4000000000,5200000000,...

The used bands for this instrument and measurement setup are:

0 Hz to 49999999 Hz

50000000 Hz to 44999999

450000000 Hz to 999999999

1000000000 Hz to 2999999999

3000000000 Hz to 3999999999

4000000000 Hz to 5199999999

5200000000 Hz to ...

Usage: Query only
