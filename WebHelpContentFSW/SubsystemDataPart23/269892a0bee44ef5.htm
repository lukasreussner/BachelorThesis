[SENSe:][LTE:][CC<cc>:]SYMBol:SELect <Symbol>

This command filters the displayed results in the constellation diagram by a particular OFDM symbol.

Suffix: 
<cc>

Component Carrier

Parameters:
<Symbol>
ALL

Shows the results for all subcarriers.

<numeric_value>

Shows the results for a single OFDM symbol.

*RST:ALL
Example: 

//Display result for OFDM symbol 2

SYMB:SEL 2
Manual operation: See "Evaluation range for the constellation diagram"
