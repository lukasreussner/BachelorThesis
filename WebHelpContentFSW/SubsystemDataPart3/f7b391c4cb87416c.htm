CALCulate<n>:MARKer<m>:FUNCtion:PNOise[:STATe] <State>

This command turns the phase noise measurement at the marker position on and off.

Suffix: 
<n>

Window

<m>

Marker

Parameters:
<State>
ON | OFF | 1 | 0
*RST:0
Example: CALC:MARK2:FUNC:PNO ON

Switches on the phase-noise measurement for the marker 2.

Manual operation: See "Phase Noise Measurement State"
See "Switching All Phase Noise Measurements Off"
