MCS Index to use

Defines the PPDUs taking part in the analysis depending on their Modulation and Coding Scheme (MCS) index.

Note: The terms in brackets in the following description indicate how the setting is referred to in the Signal Field result display ("Format" column, see "Signal Field").

"Auto, same type as first PPDU:""(A1st)"

All PPDUs using the MCS index identical to the first recognized PPDU are analyzed.

"Auto, individually for each PPDU""(AI)"

All PPDUs are analyzed

"Meas only the specified MCS""(M ...)"

Only PPDUs with the MCS index specified for the "MCS Index" setting are analyzed

"Demod all with specified MCS""(D ...)"

The "MCS Index" setting is used for all PPDUs.

Remote command: 

[SENSe:​]DEMod:​FORMat:​MCSindex:​MODE
