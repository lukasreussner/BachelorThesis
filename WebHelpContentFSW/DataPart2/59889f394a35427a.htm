[SENSe:]POWer:ACHannel:SBCount <Number>

This command defines the number of sub blocks, i.e. groups of transmission channels in an MSR signal.

For more information see "Measurement on Multi-Standard Radio (MSR) Signals".

Parameters:
<Number>
Range: 1  to  5
*RST:1
Example: 

See "Example: Configuring and Performing an MSR ACLR Measurement".

Manual operation: See "Number of Sub Blocks"
