Pulse Measurements (R&S FSW-K6/6S) > Analysis > Trace Configuration > Trace Settings > Detector
Detector

Defines the trace detector to be used for trace analysis.

For details see Mapping Samples to sweep Points with the Trace Detector.

"Auto"

Selects the optimum detector for the selected trace and filter mode. This is the default setting.

"Type"

Defines the selected detector type.

Remote command: 

[SENSe:​][WINDow<n>:​]DETector<t>[:​FUNCtion]

[SENSe:​][WINDow<n>:​]DETector<t>[:​FUNCtion]:​AUTO
