1xEV-DO MS

The 1xEV-DO MS application requires an instrument equipped with the 1xEV-DO MS Measurements option, R&S FSW-K85. This application provides test measurements for 1xEV-DO MS uplink signals (mobile signals) according to the test specification.

For details refer to See Also.
