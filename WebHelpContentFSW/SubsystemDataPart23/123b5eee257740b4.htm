CONFigure[:LTE]:UL[:CC<cc>]:DRS:PLID <Identity>

This command defines the (cell) Identity of the Demodulation Reference Signal (DRS).

Suffix: 
<cc>

Component Carrier

Parameters:
<Identity>
FCID

From Cell ID: Uses the common Identity defined with CONFigure[:​LTE]:​UL[:​CC<cc>]:​PLC:​PLID.

<numeric value>

Custom Identity for the DRS (range: 1...2).

*RST:FCID
Example: CONF:UL:DRS:PLID?

would return, e.g.

FCID
Manual operation: See "Configuring the Physical Layer Cell Identity"
