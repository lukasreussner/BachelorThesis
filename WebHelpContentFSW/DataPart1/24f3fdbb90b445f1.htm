DISPlay[:WINDow<n>]:TRACe:X:SPACing <ScalingType>

This command selects the scaling of the x-axis.

Suffix: 
<n>

Window

Parameters:
<ScalingType>
LOGarithmic

Logarithmic scaling.

LINear

Linear scaling.

*RST:LINear
Example: DISP:TRAC:X:SPAC LOG
Manual operation: See "Frequency Axis Scaling"
