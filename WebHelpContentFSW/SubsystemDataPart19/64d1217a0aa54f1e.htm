CALCulate<n>:LIMit<li>:BURSt:TFALl[:AVERage]:RESult?
CALCulate<n>:LIMit<li>:BURSt:TFALl:MAXimum:RESult?

This command returns the result of the average or maximum fall time limit check. The limit value is defined by the standard or the user (see CALCulate<n>:​LIMit<li>:​BURSt:​TFALl:​MAXimum).

This command is only available for IEEE 802.11b.
