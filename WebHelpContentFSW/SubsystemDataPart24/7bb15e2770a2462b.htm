Refresh All

This function is only available if the Sequencer is deactivated, no sweep is currently running, and only in MSRA mode.

The data in the capture buffer is re-evaluated by all active slave applications, for example after a new sweep was performed while the Sequencer was off.

Note: To update only the displays in the currently active slave application, use the "Refresh" function in the "Sweep" menu for that slave application (see "Refresh (MSRA only)").

For details on the MSRA operating mode, see "Applications and Operating Modes". For details on the MSRT operating mode, see Applications and Operating Modes.

Remote command: 

INITiate<n>:​SEQuencer:​REFResh[:​ALL]
