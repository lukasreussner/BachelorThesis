Retrieving Trace Results

The following commands describe how to retrieve the trace data from the CDA. Note that for these measurements, only 1 trace per window can be configured.

List of Commands
FORMat[:DATA]
TRACe<n>[:DATA]?
TRACe<n>[:DATA]:X?
