Selecting a segment in a multi segment waveform file

If you are using a waveform file that contains several different waveforms, you have to select the segment that should be transferred to the signal generator.

Note that the segment that you have selected in the "Generator Setup" has to match the segment selected for the reference signal, regarding the signal characteristics.

Remote command: 

CONFigure:​GENerator:​SEGMent

CONFigure:​GENerator:​SEGMent:​LEDState?
