Synchronization

For BTS tests, the individual channels in the input signal need to be synchronized to detect timing offsets in the slot spacings. These commands are described here. they are only available in the 3GPP FDD BTS application

Useful commands for synchronization described elsewhere:
  • [SENSe:​]CDPower:​ANTenna
