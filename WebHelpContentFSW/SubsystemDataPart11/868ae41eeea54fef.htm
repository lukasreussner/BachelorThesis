Returns the statistical value for the FM settling length from the statistics table for the specified chirp(s).

Query parameters: 
<QueryRange>
CURRent | ALL
CURRent

Detected chirps in the current capture buffer

ALL

All chirps detected in the entire measurement

Return values: 
<Result>
Usage: Query only
