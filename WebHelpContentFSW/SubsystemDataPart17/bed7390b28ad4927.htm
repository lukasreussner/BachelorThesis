CONFigure:CDPower[:BTS]:BCLass|BANDclass <Bandclass>

This command selects the bandclass for the measurement. The bandclass defines the frequency band used for ACLR and SEM measurements. It also determines the corresponding limits and ACLR channel settings according to the 1xEV-DO standard.

Parameters:
<Bandclass>

For an overview of available bandclasses and the corresponding parameter values see "Reference: Supported Bandclasses".

*RST:0
Example: CONF:CDP:BCL 1

Selects band class 1, 1900 MHz

Manual operation: See "Bandclass"
