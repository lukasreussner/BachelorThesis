Measuring only if burst was found

If enabled, measurement results are only displayed (and are only averaged) if a valid burst has been found. When measuring bursted signals that are averaged over several measurements, it is recommended that you enable this option so that erroneous measurements do not affect the result of averaging.

Remote command: 

[SENSe:​]DDEMod:​SEARch:​BURSt:​MODE
