Time Reference (BTS mode)

Defines which slot is used as a time reference for synchronization.

"DwPTS"

Uses the Downlink Pilot Time Slot (DwPTS) as a time reference (see also "Frames, Subframes and Slots")

"Slot 0"

Uses slot 0 as a time reference.

Remote command: 

[SENSe:​]CDPower:​TREF
