Stop RUN on Limit Check Fail

If enabled, the measurement is stopped if the limit check fails at any time during the measurement. This is particularly useful if statistical evaluation is performed. If a value exceeds the limit, the R&S FSW WLAN application stops acquiring data, thus allowing you to determine the cause of the limit violation.

If the limit check fails, an error message in the status bar and a status bit in the SYNC register (bit 3) indicate the failure.

Remote command: 

[SENSe:​]SWEep:​LIMit:​ABORt:​STATe
