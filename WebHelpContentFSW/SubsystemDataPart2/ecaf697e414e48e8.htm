Display Configuration

Access: [MEAS]

Or:[MEAS CONFIG] > "Display Config"

The captured signal can be displayed using various evaluations. All evaluations available for the Pulse application are displayed in the evaluation bar in SmartGrid mode.

Up to six evaluation methods can be displayed simultaneously in separate windows. The Pulse evaluation methods are described in "Measurements and Result Displays".

For details on working with the SmartGrid see Laying out the Result Display with the SmartGrid.
