CONFigure:DS:CHANnel:PLC:INDex:AUTO <State>

If enabled, the start index of the physical link channel (PLC) is detected automatically.

If disabled, the numeric value defined by CONFigure:​DS:​CHANnel:​PLC:​INDex is used.

Parameters:
<State>
ON | OFF | 1 | 0
*RST:1
Example: CONF:DS:CHAN:PLC:IND:AUTO ON
Example: 

For a detailed example see "Measurement 1: Measuring Modulation Accuracy".

Manual operation: See "PLC Start Index L"
