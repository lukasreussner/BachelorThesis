DISPlay[:WINDow<n>]:TRACe<t>:X[:SCALe]:MAXimum <Max>
DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:MAXimum <Max>

Defines the maximum value to be displayed on the x-axis or y-axis of the specified evaluation diagram.

For automatic scaling with a fixed range (see DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​AUTO:​FIXed:​RANGe), the maximum defines the fixed upper limit.

This command is currently only supported for Amplitude Modulation measurements.

Suffix: 
<n>
1..n

Window

<t>
1..n

Trace

Parameters:
<Max>
Example: DISP:WIND2:TRAC:Y:SCAL:MAX 100
Manual operation: See "Minimum / Maximum"
