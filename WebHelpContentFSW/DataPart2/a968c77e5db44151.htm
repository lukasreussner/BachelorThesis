[SENSe:]CORRection:FRESponse:USER:VALid?

This command queries the validity of the user-defined correction settings.

Return values: 
<Validity>
0 | 1
1

The setting is valid.

0

The setting is not valid.

Example: SENS:CORR:FRES:USER:VAL?
Usage: Query only
Manual operation: See "State"
