[SENSe:]CDPower:ORDer <SortOrder>

This command sets the channel sorting for the Code Domain Power and Code Domain Error Power result displays.

Parameters:
<SortOrder>
HADamard | BITReverse
*RST:HADamard

For further details refer to "Code Display and Sort Order".

Example: CDP:ORD HAD

Sets Hadamard order.

TRAC? TRACE2

Reads out the results in Hadamard order.

CDP:ORD BITR

Sets BitReverse order.

TRAC? TRACE2

Reads out the results in BitReverse order.

Manual operation: See "Code Display Order"
