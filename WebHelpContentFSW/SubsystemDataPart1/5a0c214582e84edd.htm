Amplitude Settings for Analog Baseband Input

Access: "Overview" > "Amplitude"

The following settings and functions are available to define amplitude settings for input via the optional Analog Baseband Interface in the applications that support it.

The input and impedance settings provided here are identical to those in the "Input Source" > "Analog Baseband" tab, see "Analog Baseband Input Settings".

For more information, see "Processing Data from the Analog Baseband Interface".
