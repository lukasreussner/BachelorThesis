I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Importing and Exporting I/Q Data and Results > MMEMory:LOAD:IQ:STATe
MMEMory:LOAD:IQ:STATe 1,<FileName>

This command restores I/Q data from a file.

The file extension is *.iq.tar.

Parameters:
<FileName>

String containing the path and name of the source file.

Example: 

MMEM:LOAD:IQ:STAT 1, 'C:\R_S\Instr\user\data.iq.tar'

Loads IQ data from the specified file.

Usage: Setting only
Manual operation: See "I/Q Import"
