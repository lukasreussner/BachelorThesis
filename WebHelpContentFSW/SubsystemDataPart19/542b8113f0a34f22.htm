TRIGger[:SEQuence]:MODE <Source>

Defines the trigger source.

Note that this command is maintained for compatibility reasons only. Use the TRIGger[:​SEQuence]:​SOURce commands for new remote control programs.

This command configures how triggering is to be performed.
