Import ASCII File to Trace

Loads one trace or all traces from the selected file in the selected ASCII format (.DAT or .CSV) to the currently active result display.

For details see "Displaying a Reference Trace - Importing Trace Data".

Remote command: 

FORMat:​DIMPort:​TRACes
