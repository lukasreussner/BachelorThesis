[SENSe:]DDEMod:PATTern:USER:NAME <FileName>

Selects the file that contains a user-defined modulation.

For details on user-defined modulation files see "User-defined Modulation".

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Setting parameters: 
<FileName>
string

Path and file name of the *.vam file.

Example: SENS:DDEM:PATT:FORM UQAM
SENS:DDEM:PATT:USER:NAME 'D:\MyModulation.vam'
Manual operation: See "Load User Modulation"
