CONFigure:CDPower[:BTS]:CTABle:SELect <FileName>

This command selects a predefined channel table file for comparison during channel detection.

Before using this command, the channel table must be switched on first with the command CONFigure:​CDPower[:​BTS]:​CTABle[:​STATe].

Parameters:
<FileName>
*RST:RECENT
Example: CONF:CDP:CTAB ON

Switches the channel table on.

CONF:CDP:CTAB:SEL 'CTAB_1'

Selects the predefined channel table 'CTAB_1'.

Manual operation: See "Selecting a Table"
