Range Start / Range Stop

Sets the start and stop frequency of the selected range. Subsequent ranges must be defined in ascending order of frequencies; however, gaps between ranges are possible.

The initial range setup is defined automatically according to the currently available measurement span.

Remote command: 

[SENSe:​]LIST:​RANGe<range>[:​FREQuency]:​STARt

[SENSe:​]LIST:​RANGe<range>[:​FREQuency]:​STOP
