Pulse Measurements (R&S FSW-K6/6S) > Remote Commands for Pulse Measurements > Retrieving Results > Retrieving Information on Detected Pulses > [SENSe:]PULSe:NUMBer?
[SENSe:]PULSe:NUMBer? <QueryRange>

Queries the detected pulse numbers, i.e. the index within the capture buffer (as opposed to [SENSe:​]PULSe:​ID?).

Query parameters: 
<QueryRange>
CURRent | ALL
CURRent

Detected pulses in the current capture buffer

ALL

All detected pulses in the entire measurement.

Usage: Query only
