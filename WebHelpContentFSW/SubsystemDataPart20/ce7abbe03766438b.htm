Table Configuration

Access: "Overview" > "Result Config" > "Table Config"

Or: [MEAS CONFIG] > "Result Config" > "Table Config"

During each measurement, a large number of statistical and characteristic values are determined. The Result Summary provides an overview of the parameters selected here.

You can configure which results are displayed in Result Summary displays (see "Result Summary"). However, the results are always calculated, regardless of their visibility on the screen.

Note that the "Result Configuration" dialog box is window-specific; table configuration settings are only available if a table display is selected.

Select the parameters to be included in the table. For a description of the individual parameters see "Modulation Accuracy Parameters".
