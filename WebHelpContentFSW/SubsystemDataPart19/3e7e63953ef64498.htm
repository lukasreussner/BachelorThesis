WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Selecting a Measurement > Selecting the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > CONFigure:BURSt:PTRacking[:IMMediate]
CONFigure:BURSt:PTRacking[:IMMediate]

This remote control command configures the measurement type to be Phase Tracking vs Symbol.

Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the Magnitude Capture buffer in window 1 at the top of the screen and the selected result type in window 2 below that. Any other active windows are closed.

Use the LAYout commands to change the display (see "Working with Windows in the Display").

Manual operation: See "Phase Tracking"
