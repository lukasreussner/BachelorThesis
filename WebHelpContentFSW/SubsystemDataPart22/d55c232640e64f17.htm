LTE Downlink (R&S FSW-K10x) > Remote Control > Remote Commands to Read Numeric Results > Frame Results > FETCh[:CC<cc>]:SUMMary:EVM:DS1K:MAXimum?
FETCh[:CC<cc>]:SUMMary:EVM:DS1K:MAXimum?
FETCh[:CC<cc>]:SUMMary:EVM:DS1K:MINimum?
FETCh[:CC<cc>]:SUMMary:EVM:DS1K[:AVERage]?

This command queries the EVM of all resource elements of the PDSCH with a 1024QAM modulation.

Suffix: 
<cc>

Component Carrier

Return values: 
<EVM>
<numeric value>

EVM in % or dB, depending on the unit you have set.

Example: FETC:SUMM:EVM:DS1K?

Returns the PDSCH 1024QAM EVM.

Usage: Query only
Manual operation: See "Result Summary"
