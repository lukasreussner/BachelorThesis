Retrieving Trace Results

The following commands describe how to retrieve the trace data from the CDA and Time Alignment Error measurements. Note that for these measurements, only 1 trace per window can be configured.

  • FORMat[:​DATA]
  • TRACe<n>[:​DATA]?
  • TRACe<n>[:DATA]? TRACE1
  • TRACe<n>[:DATA]? ABITstream
  • TRACe<n>[:DATA]? ATRACE1
  • TRACe<n>[:DATA]? CTABle
  • TRACe<n>[:DATA]? CWCDp
  • TRACe<n>[:DATA]? FINal1
  • TRACe<n>[:DATA]? LIST
  • TRACe<n>[:DATA]? PWCDp
  • TRACe<n>[:DATA]? TPVSlot
