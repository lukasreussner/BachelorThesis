DISPlay[:WINDow<n>]:TRACe<t>:Y[:SCALe]:RLEVel <ReferenceLevel>

This command defines the reference level (for all traces in all windows).

Suffix: 
<n>, <t>

irrelevant

Parameters:
<ReferenceLevel>

The unit is variable.

Range: see datasheet
*RST:0 dBm
Example: DISP:TRAC:Y:RLEV -60dBm
Example: 

See Programming Example: Performing a Basic Frequency Sweep.

Manual operation: See "Reference Level"
