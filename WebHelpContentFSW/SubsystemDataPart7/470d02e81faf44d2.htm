Scaling the x-axis automatically

By default, the application scales the x-axis in all diagrams automatically (➙ "Auto" = ON).

Automatic scaling tries to obtain the ideal scale for the current measurement results. The application adjusts the scale each time the results change.

You can also force an automatic scaling of the x-axis at any time with the "Auto Scale Once" function. When you select this function, the application scales the x-axis even if the results have not been changed.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​X[:​SCALe]:​AUTO
