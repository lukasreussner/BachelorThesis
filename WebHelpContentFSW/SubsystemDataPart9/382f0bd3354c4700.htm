Configuring Graphical Result Displays

Access: "Overview" > "Graphical Results"

The "Graphical" tab of the "Results" dialog box and the "Trace" menu contain all functions necessary to set up and configure the graphical phase noise result displays.

Trace settings are described in "Configuring Traces".
