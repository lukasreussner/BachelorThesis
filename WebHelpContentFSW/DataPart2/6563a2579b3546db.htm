MMEMory:SELect[:ITEM]:NONE

This command does not include any of the following items when storing or loading a configuration file.

  • Hardware configuration: MMEMory:​SELect[:​ITEM]:​HWSettings
  • Limit lines: MMEMory:​SELect[:​ITEM]:​LINes:​ALL
  • Spectrogram data: MMEMory:​SELect[:​ITEM]:​SGRam
  • Trace data: MMEMory:​SELect[:​ITEM]:​TRACe[:​ACTive]
  • Transducers: MMEMory:​SELect[:​ITEM]:​TRANsducer:​ALL
Example: MMEM:SEL:NONE
Usage: Event
Manual operation: See "Items:"
