Selecting the Trace Information

Access: [TRACE] > "Trace Config" > "Traces"

Each result display contains one or several traces specific to the corresponding result type.

The number of traces available for each result display and the information these traces provide are described in "Performing Amplifier Measurements".

Remote command:

DISPlay[:​WINDow<n>]:​TRACe<t>:​MODE

Restoring default traces

You can press the "Preset All Traces" button anytime to restore the default trace configuration for each result display.
