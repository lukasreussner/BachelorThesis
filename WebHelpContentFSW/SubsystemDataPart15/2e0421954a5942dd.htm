Adjusting the Trigger Offset

The delay between the trigger event and the start of the PCG can be compensated for by adjusting the trigger offset.

  1. "TRIG > External Trigger 1"

  2. "TRIG > Trigger Offset" = 100 µs

    The following results are displayed: the first window shows the power of the code domain of the signal.

    In the second window, the Result Summary is displayed. The "Trigger to Frame" offset between the trigger event and and the start of the PCG has been eliminated.
