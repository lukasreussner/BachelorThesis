Store/Load Current Equalizer

Saves the current equalizer results to a file, or loads a user-defined equalizer.

The equalizer "Mode" must be set to USER in order to load a file.

For details see "The Equalizer".

Remote command: 

[SENSe:​]DDEMod:​EQUalizer:​SAVE

[SENSe:​]DDEMod:​EQUalizer:​LOAD
