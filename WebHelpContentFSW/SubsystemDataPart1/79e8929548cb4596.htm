SYSTem:COMMunicate:RDEVice:OSCilloscope:VDEVice?

Queries whether the connected instrument is supported by the 2 GHz / 5 GHz bandwidth extension option(B2000/B5000).

For details see "Prerequisites and Measurement Setup".

Return values: 
<State>
ON | 1

Instrument is supported

OFF | 0

Instrument is not supported

Example: SYST:COMM:RDEV:OSC:VDEV?
Usage: Query only
