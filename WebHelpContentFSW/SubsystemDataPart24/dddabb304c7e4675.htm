[SENSe:]IQ:BWIDth:RESolution <Bandwidth>

This command defines the resolution bandwidth manually if [SENSe:​]IQ:​BWIDth:​MODE is set to MAN.

Defines the resolution bandwidth. The available RBW values depend on the sample rate and record length.

Parameters:
<Bandwidth>
refer to data sheet
*RST:RBW: AUTO mode is used
Example: IQ:BAND:MODE MAN

Switches to manual RBW mode.

IQ:BAND:RES 120000

Sets the RBW to 120 kHz.

Manual operation: See "RBW"
