FS-Z11 Trigger

If activated, the measurement is triggered by a connected R&S FS-Z11 trigger unit, simultaneously for all connected analyzers. This is useful for MIMO measurements in simultaneous measurement mode (see "Simultaneous Signal Capture Setup").

The "Trigger Source" is automatically set to "External Trigger 1/2/3". The required connections between the analyzers, the trigger unit, and the DUT are indicated in the graphic.

For details see "Trigger Synchronization Using an R&S FS-Z11 Trigger Unit".

Remote command: 

TRIGger[:​SEQuence]:​SOURce
