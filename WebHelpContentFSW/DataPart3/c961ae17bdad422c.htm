General Instrument Setup > Frequency Response Correction (R&S FSW-K544) > User-defined Frequency Response Correction Settings > Add Freq Resp File
Add Freq Resp File

Loads an additional frequency response (.fres) file to the current configuration. The maximum number of files per configuration is 15.

The default directory for .fres files is C:\R_S\INSTR\USER\Fresponse.

Remote command: 

[SENSe:​]CORRection:​FRESponse:​USER:​FLISt<fli>:​CATalog?

[SENSe:​]CORRection:​FRESponse:​USER:​FLISt<fli>:​SELect
