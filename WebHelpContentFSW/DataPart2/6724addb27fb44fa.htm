Range <x> Start/Stop

Defines the start and stop points of the range within the tracing period. Make sure the value for the stopping time is smaller than the length of the period.

Note: You can define the time values with a greater numerical resolution than is displayed; the values are only rounded for display.

For details on using a gated trigger see "APD and CCDF Basics - Gated Triggering".

Remote command: 

[SENSe:​]SWEep:​EGATe:​TRACe<t>:​STARt<range>

[SENSe:​]SWEep:​EGATe:​TRACe<t>:​STOP<range>
