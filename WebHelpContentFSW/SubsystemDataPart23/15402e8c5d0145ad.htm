CONFigure[:LTE]:UL[:CC<cc>]:SRS:BHOP <Bandwidth>

This command defines the frequency hopping bandwidth bhop.

Suffix: 
<cc>

Component Carrier

Parameters:
<Bandwidth>
<numeric value>
*RST:0
Example: CONF:UL:SRS:BHOP 1

Sets the frequency hopping bandwidth to 1.

Manual operation: See "Hopping BW b_hop"
