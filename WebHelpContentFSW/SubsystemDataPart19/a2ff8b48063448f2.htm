Trigger Level (Repetition Interval)

Defines the trigger level for the specified trigger source. For time triggers, the repetition interval is defined.

For details on supported trigger levels, see the data sheet.

Remote command: 

TRIGger[:​SEQuence]:​LEVel:​IFPower

TRIGger[:​SEQuence]:​LEVel:​IQPower

TRIGger[:​SEQuence]:​LEVel[:​EXTernal<port>]

TRIGger[:​SEQuence]:​TIME:​RINTerval
