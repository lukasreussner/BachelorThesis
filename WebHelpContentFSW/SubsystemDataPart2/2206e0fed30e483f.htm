SENSe:TRACe:MEASurement:DEFine:TSRange:RANGe <Reference>

Defines which part of the detected pulse is evaluated for sidelobe results.

This command is only available if the additional option R&S FSW-K6S is installed.

Parameters:
<Reference>
RRANge | MANual
RRANge

The configured result range (see "Defining the Result Range") is also used to evaluate sidelobes.

MANual

You can define the length and alignment of the sidelobe range differently to the result range using the SENSe:​TRACe:​MEASurement:​DEFine:​TSRange:​ALIGnment and SENSe:​TRACe:​MEASurement:​DEFine:​TSRange:​LENGth commands.

*RST:RRANge
Manual operation: See "Range"
