Carrier Power Table

Displays the measured power levels and reference powers of all active carriers.

The following parameters are shown:

Carrier power measurement results

Parameter

Description

Carrier No.

Active carrier number (as defined in "Carrier Settings").

Additional labels:

  • "max": the carrier with the highest power level
    (If the reference power is determined by a reference measurement (see "Enabling a reference power measurement (Measure)"), and automatic carrier selection is active, see "Carrier Selection/Carrier".)
  • "ref": selected carrier for reference power
    (If the reference power is determined by a reference measurement (see "Enabling a reference power measurement (Measure)"), but the carrier is selected manually, see "Carrier Selection/Carrier".)
  • "man": manually defined reference powers (see "Defining Reference Powers Manually")

Carrier frequency

Frequency of the carrier at which power was measured

Power level

Measured power level in dBm

Reference power with RBW 300 kHz

Reference power for measurement with 300 kHz RBW (or manually defined reference value)

Reference power with RBW 100 kHz

Reference power for measurement with 100 kHz RBW (or manually defined reference value)

Reference power with RBW 30 kHz

Reference power for measurement with 30 kHz RBW (or manually defined reference value)

LAY:ADD? '1',RIGH,WSRP, see LAYout:​ADD[:​WINDow]?

Results:
