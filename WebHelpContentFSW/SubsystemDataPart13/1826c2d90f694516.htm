3G FDD Measurements (R&S FSW-K72/K73) > Remote Commands for 3GPP FDD Measurements > Activating 3GPP FDD Measurements > INSTrument:DELete
INSTrument:DELete <ChannelName>

This command deletes a channel.

If you delete the last channel, the default "Spectrum" channel is activated.

Parameters:
<ChannelName>

String containing the name of the channel you want to delete.

A channel must exist in order to be able delete it.

Example: INST:DEL 'IQAnalyzer4'

Deletes the channel with the name 'IQAnalyzer4'.

Usage: Event
