STATus:QUEStionable Register

The STATus:QUEStionable register contains information about indefinite states which can occur if the unit is operated without meeting the specifications.

Bit no

Meaning

0 to 7

Unavailable for noise figure measurements.

8

CALibration

This bit is set if the R&S FSW is not calibrated.

9

LIMit

This bit is set if a limit line is violated.

10

Unavailable for noise figure measurements.

11

CORRection

This bit is set if the noise figure calibration is questionable.

14

Unavailable for noise figure measurements.

15

This bit is always 0.
