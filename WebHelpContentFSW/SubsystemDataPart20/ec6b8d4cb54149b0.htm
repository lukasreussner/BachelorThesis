Bitstream

For a given OFDM symbol and a given subcarrier, the bitstream result is derived from the corresponding complex constellation point according to Std IEEE802.11-2012 "Figure 18-10—BPSK, QPSK, 16-QAM, and 64-QAM constellation bit encoding". The bit pattern (binary representation) is converted to its equivalent integer value as the final measurement result. The number of values returned for each analyzed OFDM symbol corresponds to the number of data subcarriers plus the number of pilot subcarriers (NSD+NSP) in remote mode.

As opposed to the graphical Bitstream results, the DC and NULL carriers are not available in remote mode.
