INSTrument:COUPle:USER<uc>:INFO?

This command queries additional information about the specified user-defined parameter coupling.

Suffix: 
<uc>

Index of a user-defined parameter coupling. Which indexes are available is defined by INSTrument:​COUPle:​USER<uc>:​NUMBers:​LIST?.

Return values: 
<Information>

String containing the message as displayed in the coupling manager.

If the coupling message contains no message, an empty string is returned.

Example: INST:COUP:USER2:INFO?

Queries possible information about the user coupling with index 2.

Result:

'Only one limit line allowed'
Usage: Query only
Manual operation: See "Info"
