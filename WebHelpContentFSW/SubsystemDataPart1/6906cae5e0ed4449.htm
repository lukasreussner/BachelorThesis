I/Q Analyzer > Remote Commands to Perform Measurements with I/Q Data > Configuring I/Q Analyzer Measurements > Configuring Data Acquisition > [SENSe:]IQ:FFT:WINDow:LENGth
[SENSe:]IQ:FFT:WINDow:LENGth <NoOfFFT>

Defines the number of samples to be included in a single FFT window when multiple FFT windows are used.

Parameters:
<NoOfFFT>
integer value
Range: 3  to  1001
*RST:1001
Example: IQ:FFT:WIND:LENG 500
Manual operation: See "Window Length"
