[SENSe:]DDEMod:KDATa[:NAME] <FileName>

This command selects the Known Data file. Note that known data must be activated ([SENSe:​]DDEMod:​KDATa:​STATe) before you can select a file.

Setting parameters: 
<FileName>
string
Example: 
DDEM:KDAT:STAT ON
DDEM:KDAT 'C:\TEMP\KNOWNDATA'
Manual operation: See "Load Data File"
