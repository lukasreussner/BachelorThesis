DISPlay[:WINDow<n>]:SGRam:COLor[:STYLe] <ColorScheme>
DISPlay[:WINDow<n>]:SPECtrogram:COLor[:STYLe] <ColorScheme>

This command selects the color scheme.

For details see Color Maps.

Parameters:
<ColorScheme>
HOT

Uses a color range from blue to red. Blue colors indicate low levels, red colors indicate high ones.

COLD

Uses a color range from red to blue. Red colors indicate low levels, blue colors indicate high ones.

RADar

Uses a color range from black over green to light turquoise with shades of green in between.

GRAYscale

Shows the results in shades of gray.

*RST:HOT
Example: DISP:WIND:SPEC:COL GRAY

Changes the color scheme of the spectrogram to black and white.

Manual operation: See "Hot / Cold / Radar / Grayscale"
