Export Data to ASCII File for Specific Window

Exports the data from the specified window in the R&S FSW Spurious measurements application for export to an ASCII file.

The results are output in the same order as they are displayed in the window: trace by trace, and table row by table row.

Remote command: 

MMEMory:​STORe<n>:​TABLe

MMEMory:​STORe<n>:​TRACe
