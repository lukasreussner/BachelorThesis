Adjusting Settings Automatically

The following remote commands are required to adjust settings automatically in a remote environment. These commands are only available for the MSRA Master channel. The tasks for manual operation are described in " Automatic Settings".

Settings related to data acquisition (measurement time, hysteresis) can only be adjusted in the MSRA Master, not in the slave applications.

[SENSe:]ADJust:CONFigure[:LEVel]:DURation
