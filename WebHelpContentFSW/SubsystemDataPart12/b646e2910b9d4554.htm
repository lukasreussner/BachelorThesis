[SENSe:]DDEMod:PATTern:FRAMe:LOAD <FileName>

Loads a user-defined frame structure configuration to be used by the measurement from an xml file. The default storage location for such files is C:\R_S\INSTR\USER\vsa\FrameRangeStructure.

This command is only available if the additional Multi-Modulation Analysis option (R&S FSW-K70M) is installed.

Setting parameters: 
<FileName>
string

Path and file name of the xml file.

Example: DDEM:PATT:FRAM:MODE USER
SENS:DDEM:PATT:FRAM:LOAD 'D:\MyFrameStructure.xml'
Manual operation: See "Load File"
