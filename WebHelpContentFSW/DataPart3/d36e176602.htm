Remote Commands > Emulating Other Instruments' Commands > Setting up Instrument Emulation > SYSTem:REVision[:STRing]
SYSTem:REVision[:STRing] <Name>

Sets the response to the REV? query to the defined string (HP emulation only, see SYSTem:​LANGuage).

Parameters:
<Name>
Example: 

Define the system language:

SYST:LANG '8563E'

Query the revision:

REV?

Response:

920528

Set the response to 'NewRevision':

SYST:REV:STR 'NewRevision'

Query the response:

SYST:REV:STR?

Response:

NewRevision
Manual operation: See "Revision String"
