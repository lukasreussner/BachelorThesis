Number of Samples

The number of samples is indicated for reference only. It is calculated from the "Capture Time" and the "Sample Rate" according to the following equation:

Number of samples = capture time * sample rate

The maximum number of samples for downstream is thus 96,256,000.

The maximum number of samples for upstream is thus 48,128,000.
