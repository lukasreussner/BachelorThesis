Spurious Measurements application (R&S FSW-K50) > Remote Commands to Perform Spurious Measurements > Retrieving Results > Exporting Table and Trace Results to an ASCII File > MMEMory:STORe:SPUR:MEAS
MMEMory:STORe:SPUR:MEAS <File>

This command stores the current measurement results (all enabled traces and tables of all windows) into the specified csv file.

The results are output in the same order as they are displayed on the screen: window by window, trace by trace, and table row by table row.

Setting parameters: 
<File>
Usage: Setting only
Manual operation: See "Export Data to ASCII File for All Windows"
