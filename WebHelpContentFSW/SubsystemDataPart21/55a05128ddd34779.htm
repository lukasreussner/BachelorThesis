CALCulate<n>:MARKer<m>:FUNCtion:POWer:RESult? <PowerResult>

This command queries the results of power measurements.

Suffix: 
<n>
1..n

irrelevant

<m>
1..n
Query parameters: 
<PowerResult>
OBANdwidth | OBWidth

Occupied bandwidth.

Returns the occupied bandwidth in Hz.
