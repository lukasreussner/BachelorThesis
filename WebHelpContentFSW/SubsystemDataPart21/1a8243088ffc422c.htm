CALCulate<n>:US:CHANnel:SBPower:BAND<n>:RESult?

This command queries the band power result of the specified band.

Suffix: 
<n>

irrelevant

<n>
1 to 20

Configured power band

Example: CALC:US:CHAN:SBP:BAND2:RES?
Usage: Query only
Manual operation: See "Synchronous Band Power (upstream only)"
