FETCh[:CC<cc>]:SUMMary:EVM:SDQP[:AVERage]?

This command queries the EVM of all DMRS PUSCH resource elements with QPSK modulation.

Suffix: 
<cc>

Component Carrier

Return values: 
<EVM>
<numeric value>

EVM in % or dB, depending on the unit you have set.

Example: 

//Query EVM

FETC:SUMM:EVM:SDQP?
Usage: Query only
Manual operation: See "Result Summary"
