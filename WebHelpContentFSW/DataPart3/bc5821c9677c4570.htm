Restore Standard Files

Restores the standards predefined by Rohde & Schwarz available at the time of delivery.

The XML files from the C:\Program Files (x86)\Rohde-Schwarz\FSW\<version>\sem_backup folder are copied to the C:\Program Files (x86)\Rohde-Schwarz\FSW\<version>\sem_std folder.

Note that this function will overwrite customized standards that have the same name as predefined standards.

Remote command: 

[SENSe:​]ESPectrum<sb>:​PRESet:​RESTore
