Limit Settings for Table Displays

Access: "Overview" > "Result Configuration" > "Table Config" > "Limits"

Measurement results can be checked against defined limits and the results of the limit check can then be indicated in the Result Table.

For details on limits see "Pulse Results".

The settings are window-specific and only available for result tables.

Optionally, limit lines can be displayed in the "Parameter Distribution" and "Parameter Trend" diagrams. You can drag these lines to a new position in the window. The new position is maintained, the limit check is repeated, and the results of the limit check in any active table displays are adapted.

For step-by-step instructions on how to configure limits, see "To configure a limit check for a pulse measurement".
