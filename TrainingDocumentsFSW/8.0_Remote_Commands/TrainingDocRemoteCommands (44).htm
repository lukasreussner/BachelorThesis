*RST

Reset

Sets the instrument to a defined default status. The default settings are indicated in the description of commands.

The command is equivalent to SYSTem:PRESet.

Note that the factory set default values can be modified to user-defined settings (see MMEMory:​LOAD:​STATe). For more details on default values see "Restoring the Default Instrument Configuration (Preset)".

Usage: Setting only
