Configuring Frequency Sweep Measurements on WLAN 802.11 Signals

The R&S FSW WLAN application uses the functionality of the R&S FSW base system (Spectrum application, see Configuring and Performing Measurements) to perform the WLAN 802.11 frequency sweep measurements. The R&S FSW WLAN application automatically sets the parameters to predefined settings as described in "Frequency Sweep Measurements".

The WLAN RF measurements must be activated for a measurement channel in the R&S FSW WLAN application, see "Activating WLAN 802.11 Measurements".

For details on configuring these RF measurements in a remote environment, see Configuring and Performing Measurements.
