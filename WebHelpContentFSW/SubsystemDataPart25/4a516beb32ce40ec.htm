NB-IoT Downlink (R&S FSW-K106) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Input / Frontend > Configuring the Input > INPut<ip>:IQ:BALanced[:STATe]
INPut<ip>:IQ:BALanced[:STATe] <State>

This command defines whether the input is provided as a differential signal via all 4 Analog Baseband connectors or as a plain I/Q signal via 2 single-ended lines.

Suffix: 
<ip>

irrelevant

Parameters:
<State>
ON | 1

Differential

OFF | 0

Single ended

*RST:1
Example: INP:IQ:BAL OFF
Example:
