Reference Level Offset

For simultaneous MIMO setups, you can set the reference level for all slave devices to the same setting as the master device, or define individual offsets.

Tip: the LED symbol in the "Ref Status" column indicates the connection state of the device providing the individual reference frequency for each channel:

Color

State

gray

external reference off or IP address not available/valid

red

external reference on and IP address valid, but not accessible

green

external reference on and IP address accessible

"From Master"

The slave analyzers' reference levels are set to match that of the master.

"Manual"

The slave analyzers' reference levels are specified individually and are not coupled to the reference level offset of the master analyzer.

Remote command: 

CONFigure:​WLAN:​ANTMatrix:​SOURce:​RLEVel:​OFFSet
