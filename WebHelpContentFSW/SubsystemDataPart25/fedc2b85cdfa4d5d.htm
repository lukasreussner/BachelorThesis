NB-IoT Downlink (R&S FSW-K106) > Remote Control > Remote Commands to Configure the Application > Configuring I/Q Measurements > Signal Capture > Trigger > TRIGger[:SEQuence]:PORT<in>
TRIGger[:SEQuence]:PORT<in> <Port>

This command selects the trigger port for measurements with devices that have several trigger ports.

Parameters:
<Port>
PORT1
PORT2
PORT3
Example: 

//Select trigger port 1

TRIG:PORT PORT1
