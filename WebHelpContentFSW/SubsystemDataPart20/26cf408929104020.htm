Channel Frequency Response

The Channel Frequency Response evaluation returns absolute power values per carrier.

Two trace types are provided for this evaluation:

Query parameter and results for Channel Frequency Response

TRACE1

All channel frequency response values per channel

TRACE2

An average channel frequency response value for each of the 53 (or 57/117 within the IEEE 802.11 n standard) carriers

Absolute power results are returned in dB.

Supported data formats (FORMat:DATA): ASCii|REAL
