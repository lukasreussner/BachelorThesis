Negate Feedback Path

The feedback bit consists of the xor connected bits from the specified feedback positions. If "Negate Feedback Path" is enabled, this bit is inverted before it is fed into the last shift register. By default, this setting is defined automatically according to the standard for the used PRBS type.

This function requires the R&S FSW-K70P option.

For details, see "Known Data from PRBS Generators".

Remote command: 

[SENSe:​]DDEMod:​KDATa:​PRBS:​FEEDback[:​STATe]

[SENSe:​]DDEMod:​KDATa:​PRBS:​FEEDback:​AUTO
