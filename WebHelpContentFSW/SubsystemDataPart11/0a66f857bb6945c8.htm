Frequency Deviation (Peak)

Maximum of Frequency Deviation vs Time trace

All hop frequency deviation table values are calculated from the time domain result:

fdev(k)= fmeas(k) - fideal(k)

for hop start ≤ k ≤ hop start + dwell time

where:

fmeas(k): instantaneous frequency of the measured signal

fideal(k): ideal frequency trajectory obtained from weighted linear regression of the instantaneous signal phase φmeas(k) within the frequency measurement range (see "Hop / Chirp Measurement Settings ")

The peak deviation is thus defined as:

fdevpeak = max(|fdev(k)|)

for k∈ {frequency meas range}
