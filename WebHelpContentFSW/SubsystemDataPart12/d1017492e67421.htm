[SENSe:]DDEMod:EQUalizer:SAVE <Name>

This command saves the current equalizer results to a file.

Setting parameters: 
<Name>
string

File name

Example: DDEM:EQU:SAVE 'D:\MyEqualizer''

Saves the current equalizer results to D:\MyEqualizer.vae.

Manual operation: See "Store/Load Current Equalizer"
