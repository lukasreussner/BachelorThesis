VSA (R&S FSW-K70) > Configuration > Measurement Filter Settings > Type > Load User Filter
Load User Filter

Opens a file-selection dialog box to select the user-defined measurement filter to be used.

This setting is only available if "User" is selected as the "Filter Type".

For detailed instructions on working with user-defined filters see "How to Select User-Defined Filters".

Remote command: 

[SENSe:​]DDEMod:​MFILter:​USER
