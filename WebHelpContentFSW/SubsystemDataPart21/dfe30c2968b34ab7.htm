CONFigure:US:CHANnel:PCONfig<pri>:MINislot:PPATtern <PilotPattern>

Defines the pilot pattern used by the specified minislots. Which patterns are available depends on the FFT mode.

For more information, see "Pilot patterns".

Suffix: 
<pri>
1..n

index in the "Profile Configuration (Upstream)" table

Parameters:
<PilotPattern>

The pattern values depend on the selected"NFFT (FFT length)".

Range: 2K mode: 1 to 7; 4K mode: 8 to 14
Manual operation: See "Minislot Pilot Pattern"
