If SER ≤

This setting is only available if "Known Data" is selected for "Fine Synchronization". You can define a maximum symbol error rate for the known data in reference to the evaluated data. Thus, if a wrong file was mistakenly loaded or the file proves to be unsuitable, it is not used for synchronization. Otherwise the results would be strongly distorted. If the SER of the known data exceeds this limit, the default synchronization using the detected data is performed.

Remote command: 

[SENSe:​]DDEMod:​FSYNc:​LEVel
