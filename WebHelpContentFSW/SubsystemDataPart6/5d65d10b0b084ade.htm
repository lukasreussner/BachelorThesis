Loading Calibration Data

Loads the calibration data (reference data) from the selected file (.csv format) to the instrument. The current reference data in the MCGD application is overwritten; the loaded data is used for the group delay calculation.

Remote command: 

MMEMory:​LOAD:​MCGD:​RCALibration
