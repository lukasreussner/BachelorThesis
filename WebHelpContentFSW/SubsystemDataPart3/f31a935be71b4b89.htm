Data Acquisition

Access: "Overview" > "Data Acquisition"

How data is to be acquired and then demodulated is configured in the "Data Acquisition" dialog box.

MSRA/MSRT operating mode

In MSRA/MSRT operating mode, only the MSRA/MSRT Master channel actually captures data from the input signal. The data acquisition settings for the Analog Demodulation application in MSRA/MSRT mode define the analysis interval.

For details on the MSRA operating mode see Applications and Operating Modes. For details on the MSRT operating mode see Applications and Operating Modes.
