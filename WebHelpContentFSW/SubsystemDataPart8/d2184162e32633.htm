[SENSe:]SWEep:COUNt <Averages>

This command defines the number of measurements that are used to average the results.

Parameters:
<Averages>

Number of measurements that are performed at a single frequency before average results are displayed.

If you set an average of 0 or 1, the application performs a single measurement at each frequency.

Range: 0  to  32767
*RST:1
Example: SWE:COUN 10

The application averages 10 measurements before it displays the results.

Manual operation: See "Average"
