I/Q Analyzer > Configuration > Data Input and Output Settings > Input Source Settings > Digital I/Q Input Settings > Full Scale Level
Full Scale Level

The "Full Scale Level" defines the level and unit that should correspond to an I/Q sample with the magnitude "1" .

If "Auto" is selected, the level is automatically set to the value provided by the connected device.

Remote command: 

INPut<ip>:​DIQ:​RANGe[:​UPPer]

INPut<ip>:​DIQ:​RANGe[:​UPPer]:​UNIT

INPut<ip>:​DIQ:​RANGe[:​UPPer]:​AUTO
