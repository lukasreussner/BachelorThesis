[SENSe:]ADEMod<n>:AF:STOP <Frequency>

This command sets the stop frequency for AF spectrum result display.

Suffix: 
<n>

irrelevant

Parameters:
<Frequency>
*RST:9 MHz
Example: 

ADEM:AF:STAR 0 kHz

Sets the AF start frequency to 0 kHz

ADEM:AF:STOP 500 kHz

Sets the AF stop frequency to 500 kHz

Manual operation: See "AF Stop"
