Y-Axis Reference Value

Defines a reference value for the y-axis in the current unit. The y-axis is adapted so that the reference value is displayed at the "Y-Axis Reference Position".

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​RVALue
