Windows Per Page

Defines how many windows are displayed on a single page of the printout. This setting is only available if "Print Multiple Windows" is active (see "Print Content Settings").

If more than one window is printed on one page, each window is printed in equal size.

Remote command: 

HCOPy:​PAGE:​WINDow<device>:​COUNt
