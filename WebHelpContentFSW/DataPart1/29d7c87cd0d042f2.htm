HCOPy:PAGE:COUNt:STATe <State>

This command includes or excludes the page number for printouts consisting of multiple pages (HCOP:CONT WIND).

Parameters:
<State>
1 | 0 | ON | OFF
1 | ON

The page number is printed.

0 | OFF

The page number is not printed.

*RST:1
Example: HCOP:PAGE:COUN:STAT ON
Manual operation: See "Print Page Count"
