Pulse Measurements (R&S FSW-K6/6S) > Analysis > Markers > Marker Positioning Functions > Search Next Peak
Search Next Peak

Sets the selected marker/delta marker to the next (lower) maximum of the assigned trace. If no marker is active, marker 1 is activated.

Remote command: 

CALCulate<n>:​MARKer<m>:​MAXimum:​NEXT

CALCulate<n>:​MARKer<m>:​MAXimum:​RIGHt

CALCulate<n>:​MARKer<m>:​MAXimum:​LEFT

CALCulate<n>:​DELTamarker<m>:​MAXimum:​NEXT

CALCulate<n>:​DELTamarker<m>:​MAXimum:​RIGHt

CALCulate<n>:​DELTamarker<m>:​MAXimum:​LEFT
