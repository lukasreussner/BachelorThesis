How to Import Traces

Trace data that was stored during a previous measurement can be imported to the Spectrum application, for example as a reference trace.

To import trace data

  1. Press the TRACE key.

  2. Select "Trace Config" > "Trace / Data Export"/"Import".

  3. Select "Import All Traces" to import traces for all the currently active traces, or select a specific trace to be imported in "Import to Trace".

  4. Select "Import ASCII File to Trace".

  5. Select the file format in which the data is stored.

  6. Select the file that contains the trace data.

  7. Select "Select" to close the dialog box and start the import.
