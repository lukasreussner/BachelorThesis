CONFigure:WLAN:SMAPping:TX<ch>:TIMeshift <TimeShift>

This remote control command specifies the timeshift for a specific antenna.

Suffix: 
<ch>
1..n
Parameters:
<TimeShift>

Time shift (in s) for specification of user defined CSD (cyclic delay diversity) for the Spatial Mapping.

Range: -32 ns to 32 ns
Default unit: S
Manual operation: See "User Defined Spatial Mapping"
