Configuring an Analysis Interval and Line (MSRA mode only)

In MSRA operating mode, only the MSRA Master actually captures data; the MSRA slave applications define an extract of the captured data for analysis, referred to as the analysis interval. The analysis line is a common time marker for all MSRA slave applications.

For the Pulse slave application, the commands to define the analysis interval are the same as those used to define the actual data acquisition (see "Data Acquisition". Be sure to select the correct measurement channel before executing these commands.

Useful commands related to MSRA mode described elsewhere:
  • INITiate<n>:​REFResh
  • INITiate<n>:​SEQuencer:​REFResh[:​ALL]
