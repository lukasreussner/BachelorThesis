Frame Number Offset

Defines a frame number offset for the analyzed frame.

The frame number offset assigns a number to the demodulated frame in order to identify it in a series of transmitted (and captured) frames.

Remote command: 

CONFigure[:​LTE]:​UL[:​CC<cc>]:​SFNO
