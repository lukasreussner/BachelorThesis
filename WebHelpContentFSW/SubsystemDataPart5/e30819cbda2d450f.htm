Select I/Q data file

Opens a file selection dialog box to select an input file that contains I/Q data.

The I/Q data must have a specific format (.iq.tar) as described in I/Q Data File Format (iq-tar).

The default storage location for I/Q data files is C:\R_S\INSTR\USER.

For details, see Basics on Input from I/Q Data Files.

Remote command: 

INPut<1|2>:​FILE:​PATH
