Frequency Response Group Delay

The Frequency Response Group Delay of the equalizer is the derivation of phase over frequency. It is a measure of phase distortion.

Available for source types:
  • Equalizer
Remote commands:

LAY:ADD? '1',BEL,EQU

to define the required source type (see LAYout:​ADD[:​WINDow]?)

CALC:FEED 'XFR:DDEM:RAT'

to define the frequency response result type (see CALCulate<n>:​FEED)

CALC:FORM GDEL

to define the group delay result type (see CALCulate<n>:​FORMat)

TRAC:DATA? TRACE1

to query the trace results (see TRACe<n>[:DATA]? TRACE<n> and "Equalizer")
