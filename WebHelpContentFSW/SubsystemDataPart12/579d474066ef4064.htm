INPut<ip>:EATT <Attenuation>

This command defines an electronic attenuation manually. Automatic mode must be switched off (INP:EATT:AUTO OFF, see INPut<ip>:​EATT:​AUTO).

If the current reference level is not compatible with an attenuation that has been set manually, the command also adjusts the reference level.

This command requires the electronic attenuation hardware option.

It is not available if the optional Digital Baseband Interface is active.

Suffix: 
<ip>
1 | 2

irrelevant

Parameters:
<Attenuation>
attenuation in dB
Range: see data sheet
Increment: 1 dB
*RST:0 dB (OFF)
Default unit: DB
Example: INP:EATT:AUTO OFF
INP:EATT 10 dB
