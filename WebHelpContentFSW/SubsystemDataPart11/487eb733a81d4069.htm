[SENSe:]SIGNal:MODel <Signal>

Defines which type of signal to expect (if known), thus determining the analysis method.

This command is only required if the additional options R&S FSW-K60C/-K60H are installed.

Parameters:
<Signal>
HOP | CHIRp
HOP

Signals "hop" between random carrier frequencies in short intervals

CHIRp

The carrier frequency is either increased or decreased linearly over time

NONE

No specific signal model is used; this is the default setting if no additional options are installed
