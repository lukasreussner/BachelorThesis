SYSTem:IFGain:MODE <Mode>

Configures the internal IF gain settings in HP emulation mode due to the application needs. This setting is only taken into account for resolution bandwidth < 300 kHz and is only available if a HP language is selected using SYSTem:​LANGuage.

Parameters:
<Mode>
NORM | PULS
NORM

Optimized for high dynamic range, overload limit is close to reference level.

PULS

Optimized for pulsed signals, overload limit up to 10 dB above reference level.

*RST:NORM
Example: SYST:IFG:MODE PULS
Manual operation: See "IF Gain"
