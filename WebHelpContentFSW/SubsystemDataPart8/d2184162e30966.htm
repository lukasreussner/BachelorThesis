[SENSe:]FREQuency:STEP <Stepsize>

This command defines the frequency stepsize in the frequency table.

The stepsize corresponds to the distance from one measurement point to another.

If you change the stepsize, the application creates a new frequency list.

Parameters:
<Stepsize>
Range: 0 Hz  to  span
*RST:100 MHz
Example: FREQ:STEP 100MHZ

Defines a stepsize of 100 MHz.

Manual operation: See "Step"
