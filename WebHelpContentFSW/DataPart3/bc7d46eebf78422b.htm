Mini Front Panel

If you require a front panel display but do not want to lose too much space for results in the display area, a mini front panel is available. The mini version displays only the main function keys in a separate window in the display area.

For more information, see "How to Work with the Soft Front Panels".

Note: You can also activate the mini front panel using the key combination ALT + M (be aware of the keyboard language defined in the operating system!). That is useful when you are working from a remote PC and the front panel function is not active.

Remote command: 

SYSTem:​DISPlay:​FPANel[:​STATe]
