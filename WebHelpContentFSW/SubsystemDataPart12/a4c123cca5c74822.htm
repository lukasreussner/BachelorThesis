Enabling Burst Searches

Enables or disables burst searches. If "Auto" is selected, burst search is enabled only if the signal structure defines a bursted signal (in the "Signal Structure" tab of the "Modulation & Signal Description" dialog box, see "Signal Type").

Remote command: 

[SENSe:​]DDEMod:​SEARch:​BURSt:​AUTO
