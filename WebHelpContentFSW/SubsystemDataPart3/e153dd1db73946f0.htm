CONFigure:ADEMod:RESults:PM:DETector<det>:REFerence:MEAStoref

Sets the reference value to be used for relative demodulation results to the currently measured value for all relative detectors.

If necessary, the detectors are activated.

A reference value 0 would provide infinite results and is thus automatically corrected to 0.1.

Suffix: 
<det>

irrelevant

Example: 

See CONFigure:​ADEMod:​RESults:​PM:​DETector<det>:​STATe

Manual operation: See "Meas -> Reference"
