Sweep Time

Defines the sweep time. Allowed values depend on the ratio of span to RBW and RBW to VBW. For details refer to the data sheet. Numeric input is always rounded to the nearest possible sweep time.

Remote command: 

[SENSe:​]SWEep:​TIME
