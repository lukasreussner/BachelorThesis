Temperature

Defines the absolute room temperature in degree Celsius or Fahrenheit.

The room temperature is required for the calculation of the real ENR of the noise source, because an ENR table is based on a temperature of 290K.

To change the unit of the temperature from Celsius to Fahrenheit, change the date format from "DE" to "US" in the "General" display settings.

  • Press the [SETUP] key.
  • Press the "Display" softkey.
  • Select the "General" tab.
  • Select "DE" for Celsius or "US" for Fahrenheit.

Note: If you define the temperature via remote control, the unit is degree Kelvin.
