FETCh:SFIeld:ALL?

Returns the results of the Signal Fields table, including column headers. The result is a comma-separated list of values for the selected PPDU. For details on the provided information see "Signal Fields (IEEE 802.11ax)"

This command is only available for the IEEE 802.11ax standard. For other standards, see TRACe[:​DATA].

Return values: 
<Result>
<Signal_field_type>[,<Occupied_bits>,<Field_name>, <Binary_value>,<Human_readable_value>],...

FETC:SFI:ALL?
