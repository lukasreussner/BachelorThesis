WLAN application (R&S FSW-K91) > Remote Commands for WLAN 802.11 Measurements > Configuring the WLAN IQ Measurement (Modulation Accuracy, Flatness and Tolerance) > Signal Capturing > General Capture Settings > TRACe:IQ:SRATe
TRACe:IQ:SRATe <SampleRate>

This command sets the final user sample rate for the acquired I/Q data. Thus, the user sample rate can be modified without affecting the actual data capturing settings on the R&S FSW.

Parameters:
<SampleRate>

The valid sample rates are described in "Sample Rate and Maximum Usable I/Q Bandwidth for RF Input".

*RST:32 MHz
