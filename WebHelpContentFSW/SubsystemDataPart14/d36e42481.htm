[SENSe:]CDPower:ICTReshold <ThresholdLevel>

This command defines the minimum power that a single channel must have compared to the total signal in order to be regarded as an active channel. Channels below the specified threshold are regarded as "inactive".

Parameters:
<ThresholdLevel>
Range: -100 dB  to  0 dB
*RST:-40 dB
Example: 

SENS:CDP:ICTR -100
