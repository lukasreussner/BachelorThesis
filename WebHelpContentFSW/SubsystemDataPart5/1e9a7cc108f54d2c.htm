CALCulate<n>:AVIonics:SDM?

This command queries the ILS Sum in Depth of Modulation (SDM) result.

Suffix: 
<n>
1..n

irrelevant

Return values: 
<Value>

Arithmetic sum of the modulation depth of the 90 Hz and the 150 Hz components without any influence of the phase between the components.

Default unit: %
Example: CALC:AVI:SDM?
Usage: Query only
Manual operation: See "SDM"
