Minimum / Maximum

Defines the minimum and maximum value to be displayed on the y-axis of the specified evaluation diagram.

For automatic scaling with a fixed range (see "Auto Fix Range"), the minimum defines the fixed lower limit, the maximum defines the fixed upper limit.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​MAXimum

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​MINimum
