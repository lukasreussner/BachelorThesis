CALibration:AIQ:HATiming[:STATe] <State>

Activates a mode with enhanced timing accuracy between analog baseband, RF and external trigger signals.

For more information see Analog Baseband Input.

Parameters:
<State>
ON | OFF | 1 | 0
ON | 1

The high accuracy timing function is switched on.

The cable for high accuracy timing must be connected to trigger ports 1 and 2.

OFF | 0

The high accuracy timing function is switched off.

*RST:0
Example: CAL:AIQ:HAT:STAT ON
