HCOPy:PAGE:WINDow<device>:CHANnel:STATe <ChannelName>, <State>

This command selects all windows of the specified channel to be included in the printout for HCOP:CONT WIND.

Suffix: 
<device>
1 | 2

Printing device.

Parameters:
<ChannelName>

String containing the name of the channel.

For a list of available channel types use INSTrument:​LIST?.

<State>
1 | 0 | ON | OFF
1 | ON

The channel windows are included in the printout.

0 | OFF

The channel windows are not included in the printout.

*RST:1
Example: HCOP:CONT WIND
HCOP:PAGE:WIND2:CHAN 'IQ Analyzer',0
HCOP:PAGE:WIND2:STAT 'IQ Analyzer','1',1

Prints only window 1 in the IQ Analyzer channel.

Manual operation: See "Print Multiple Windows"
