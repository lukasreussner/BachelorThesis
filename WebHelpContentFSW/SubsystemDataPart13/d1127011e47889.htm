CONFigure:WCDPower:MS:CTABle:DELete

This command deletes the selected channel table. The channel table to be deleted is selected with the command CONFigure:​WCDPower:​MS:​CTABle:​NAME.

Example: CONF:WCDP:MS:CTAB:NAME 'NEW_TAB'

Defines the channel table name to be deleted.

CONF:WCDP:MS:CTAB:DEL

Mode: UE application only
Manual operation: See "Deleting a Table"
