TRACe<n>[:DATA]? <Trace>

This command queries the y-values in the selected result display. It is only available for graphical displays.

For each trace point, the measured or calculated value is returned. For the Magnitude Capture display, the maximum y-value for each trace point is returned.

The unit depends on the display and on the unit you have currently set.

Suffix: 
<n>

Window

Query parameters: 
<Trace>
TRACE1 | TRACE2 | TRACE3 | TRACE4 | TRACE5 | TRACE6

The trace number whose values are to be returned.
