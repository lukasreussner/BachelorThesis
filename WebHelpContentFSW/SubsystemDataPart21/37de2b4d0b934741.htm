FETCh:SCDetailed:ALL?

This command returns all detailed signal content values as a comma-separated list.

Note this command is maintained for compatibility reasons only. For newer programs use the improved command FETCh:​SCDetailed:​ALL:​FORMatted?.

Example: FETC:SCD:ALL?
Usage: Query only
