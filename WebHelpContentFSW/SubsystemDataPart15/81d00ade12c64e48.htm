Long Code Mask

Defines the long code mask of the mobile in hexadecimal form. The value range is from 0 to 4FFFFFFFFFF.

For the default mask value of 0, the "Long Code Offset" is not considered.

For more information on long codes, see "Long code scrambling".

Remote command: 

[SENSe:​]CDPower:​LCODe:​MASK
