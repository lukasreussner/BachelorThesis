LISN Type

Selects the network type and activates output to the network via the user port of the R&S FSW. The network type determines the supported phases (see Table "Supported networks and phases").

"Off" disables LISN control and output.

For more information see Controlling V-Networks (LISN).

Remote command: 

INPut<ip>:​LISN[:​TYPE]
