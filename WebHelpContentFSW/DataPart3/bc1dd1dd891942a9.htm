Ref Level Position

Defines the reference level position, i.e. the position of the maximum AD converter value on the level axis in %, where 0  % corresponds to the lower and 100  % to the upper limit of the diagram.

For spectrograms, this value defines the position of the reference level value within the span covered by the color map. In this case, the value is given in %, where 0  % corresponds to the maximum (right end) and 100  % to the minimum (left end) of the color map.

Remote command: 

DISPlay[:​WINDow<n>]:​TRACe<t>:​Y[:​SCALe]:​RPOSition
