Number of Slots

Sets the number of slots you want to analyze.

The maximum number of slots is 36 for the BTS application, and 70 in the MS application. The default value is 3. To capture more slots, increase the "Number of Sets" to capture. In this case, the number of slots is <number of sets> x 32 (BTS application) or <number of sets> x 64 (MS application).

For more information on slots and sets, see "Slots and Sets".

Remote command: 

[SENSe:​]CDPower:​IQLength
